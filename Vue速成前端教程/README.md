## 你需要的最少编程知识

本系列教程假设你**不是零编程基础**。在开始前，你至少要知道：

- **变量**：`let name = "Alice"` 是什么
- **函数**：`function hello() {}` 和 `() => {}` 是什么意思
- **if/else**：条件判断的基本写法
- **数组**：`[1, 2, 3]` 是什么，怎么遍历
- **对象**：`{name: "Alice", age: 25}` 是什么，怎么取值

**如果上面这些你都不会？** 建议先学 Python 基础（`Python精通教程` 即将制作）。如果你会其中一部分，不会的在教程中遇到时现学现查也可以。

## 贯穿项目：简易博客前端

本系列所有代码都围绕一个项目展开——**一个简易博客系统前端**。你将一步步实现：

- 文章列表页（首页）
- 文章详情页
- 用户注册/登录
- 发布/编辑/删除文章
- 评论功能
- 响应式设计（手机/电脑都能看）
- 最终部署到 Vercel，通过 HTTPS 域名访问

**学完这个系列，你可以自信地在前端岗面试中说"我能独立搭建 Vue 项目"。**

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 语言 | JavaScript (ES6+) | 浏览器原生语言 |
| 框架 | Vue 3 | Composition API + `<script setup>` |
| 构建 | Vite | 秒级热更新 |
| 路由 | Vue Router 4 | 多页面导航 |
| 状态管理 | Pinia | 简单易用 |
| 样式 | scoped styles + Flexbox | 组件级隔离 |
| HTTP | fetch | 浏览器原生，不装额外库 |
| 部署 | Vercel | 免费、自动 HTTPS |
