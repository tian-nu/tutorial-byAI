# 第70章 · 响应处理

> "厨师做好了菜（handler处理完业务逻辑），装上盘子（格式化数据），递给服务员（返回HTTP响应），最后送到客人桌上（浏览器渲染）。Gin提供了多种'装盘'方式——你是想装成精致的法餐（JSON）、接地气的快餐盒（纯文本）、还是图文并茂的套餐（HTML模板）？"

---

## 70.1 JSON响应

JSON是现代API最通用的响应格式。Gin提供了非常简洁的JSON渲染。

### c.JSON：最基本

```go
r.GET("/user", func(c *gin.Context) {
    c.JSON(200, gin.H{
        "name": "张三",
        "age":  25,
    })
})
```

响应：

```json
{"name":"张三","age":25}
```

### c.JSON传结构体

```go
type User struct {
    Name  string `json:"name"`
    Age   int    `json:"age"`
    Email string `json:"email,omitempty"`
}

r.GET("/user", func(c *gin.Context) {
    user := User{Name: "张三", Age: 25}
    c.JSON(200, user)
})
```

emitempty效果：Email字段为零值（空字符串），不出现在JSON中。

### gin.H的本质

```go
type H map[string]interface{}
```

`gin.H` 就是 `map[string]interface{}` 的别名，方便快速构造JSON对象。

### PureJSON：禁用HTML转义

```go
c.JSON(200, gin.H{"html": "<b>Hello</b>"})
```

输出：`{"html":"\u003cb\u003eHello\u003c/b\u003e"}` — JSON默认对特殊字符做了Unicode转义。

```go
c.PureJSON(200, gin.H{"html": "<b>Hello</b>"})
```

输出：`{"html":"<b>Hello</b>"}` — 原样输出。

### AsciiJSON：强制ASCII

```go
c.AsciiJSON(200, gin.H{"name": "张三"})
```

输出：`{"name":"\u5f20\u4e09"}` — 中文字符被转成Unicode转义序列。某些旧系统只支持ASCII时用。

### SecureJSON：防JSON劫持

```go
c.SecureJSON(200, gin.H{"data": "sensitive"})
```

输出：`while(1);{"data":"sensitive"}`

在JSON前面加 `while(1);` 前缀，防止CSRF攻击通过 `<script>` 标签窃取JSON数据。现代浏览器已经修复了这个问题，但某些旧系统仍有需求。

### IndentedJSON：格式化输出

```go
c.IndentedJSON(200, gin.H{
    "name": "张三",
    "age":  25,
    "address": gin.H{
        "city":   "北京",
        "street": "长安街",
    },
})
```

响应（带缩进）：

```json
{
    "name": "张三",
    "age": 25,
    "address": {
        "city": "北京",
        "street": "长安街"
    }
}
```

调试时非常友好，生产环境用普通JSON（更紧凑，传输更快）。

### JSONP：跨域JSON

```go
c.JSONP(200, gin.H{"data": "hello"})
```

当URL中有 `callback` 参数时，返回 `callbackFunction({"data":"hello"})`。用于老旧浏览器的跨域方案，现代项目用CORS替代。

## 70.2 其他数据格式

### XML

```go
c.XML(200, gin.H{"message": "hello"})
```

响应：

```xml
<map>
    <message>hello</message>
</map>
```

传结构体同样支持 `xml` tag：

```go
type Person struct {
    Name string `xml:"name"`
    Age  int    `xml:"age"`
}
c.XML(200, Person{Name: "张三", Age: 25})
```

### YAML

```go
c.YAML(200, gin.H{"name": "张三", "age": 25})
```

响应：

```yaml
name: 张三
age: 25
```

### ProtoBuf

```go
c.ProtoBuf(200, &userProto)
```

用于gRPC通信的高性能二进制格式，需要预先定义 `.proto` 文件。

## 70.3 字符串与HTML

### 纯文本

```go
c.String(200, "Hello, %s!", "张三")
```

响应：`Hello, 张三!`

### HTML渲染

```go
r.GET("/welcome", func(c *gin.Context) {
    c.HTML(200, "welcome.html", gin.H{
        "name": "张三",
        "time": time.Now().Format("15:04:05"),
    })
})
```

模板文件 `templates/welcome.html`：

```html
<!DOCTYPE html>
<html>
<head><title>欢迎</title></head>
<body>
    <h1>你好，{{ .name }}！</h1>
    <p>当前时间：{{ .time }}</p>
</body>
</html>
```

先加载模板（见70.5节），然后 `c.HTML` 把数据注入模板。

## 70.4 静态文件

### 单个文件

```go
r.StaticFile("/favicon.ico", "./assets/favicon.ico")
```

访问 `http://localhost:8080/favicon.ico` 返回 `./assets/favicon.ico` 文件。

### 整个目录

```go
r.Static("/static", "./public")
```

把 `./public` 目录下的所有文件映射到 `/static` 路径下：

- `/static/css/main.css` → `./public/css/main.css`
- `/static/js/app.js` → `./public/js/app.js`
- `/static/images/logo.png` → `./public/images/logo.png`

### StaticFS：自定义文件系统

```go
r.StaticFS("/more_static", http.Dir("./my_file_system"))
```

当你需要把静态文件嵌入二进制（用 `embed` 包）或使用自定义文件系统时用。

## 70.5 模板渲染

Gin支持Go标准库的 `html/template` 模板引擎。

### 加载模板

```go
r.LoadHTMLGlob("templates/*")
```

加载 `templates/` 目录下所有文件。如果你的模板有子目录：

```go
r.LoadHTMLGlob("templates/**/*")
```

### 多模板目录

```go
r.LoadHTMLGlob("templates/*")
```

Gin会自动识别模板文件的 `{{ define "name" }}` 定义。

### 完整示例：博客主页

```go
func main() {
    r := gin.Default()

    r.LoadHTMLGlob("templates/*")

    r.GET("/", func(c *gin.Context) {
        articles := []gin.H{
            {"title": "Go入门教程", "author": "张三"},
            {"title": "Gin框架详解", "author": "李四"},
            {"title": "数据库设计", "author": "王五"},
        }

        c.HTML(200, "index.html", gin.H{
            "title":    "我的博客",
            "articles": articles,
        })
    })

    r.Run()
}
```

`templates/index.html`：

```html
<!DOCTYPE html>
<html>
<head><title>{{ .title }}</title></head>
<body>
    <h1>{{ .title }}</h1>
    <ul>
        {{ range .articles }}
        <li>{{ .title }} — 作者：{{ .author }}</li>
        {{ end }}
    </ul>
</body>
</html>
```

### 自定义分隔符

Go模板默认用 `{{ }}` 作为标记。如果你需要跟Vue.js的 `{{ }}` 共处，可以修改分隔符：

```go
r.Delims("{[{", "}]}")
r.LoadHTMLGlob("templates/*")
```

模板中使用 `{[{ .name }]}` 代替 `{{ .name }}`。

### SetFuncMap：自定义模板函数

```go
r.SetFuncMap(template.FuncMap{
    "formatTime": func(t time.Time) string {
        return t.Format("2006-01-02 15:04:05")
    },
    "add": func(a, b int) int {
        return a + b
    },
})
r.LoadHTMLGlob("templates/*")
```

模板中调用：`{{ formatTime .createdAt }}` 或 `{{ add 1 2 }}`。

## 70.6 统一响应格式设计

第65章提到过统一响应格式，这里做一个完整的Gin版实现。在项目中创建一个公共的响应工具：

```go
package response

import (
    "net/http"

    "github.com/gin-gonic/gin"
)

type Response struct {
    Code int         `json:"code"`
    Msg  string      `json:"msg"`
    Data interface{} `json:"data"`
}

type PageData struct {
    Items    interface{} `json:"items"`
    Total    int64       `json:"total"`
    Page     int         `json:"page"`
    PageSize int         `json:"page_size"`
}

func Success(c *gin.Context, data interface{}) {
    c.JSON(http.StatusOK, Response{
        Code: 0,
        Msg:  "success",
        Data: data,
    })
}

func SuccessWithMsg(c *gin.Context, msg string, data interface{}) {
    c.JSON(http.StatusOK, Response{
        Code: 0,
        Msg:  msg,
        Data: data,
    })
}

func Page(c *gin.Context, items interface{}, total int64, page, pageSize int) {
    c.JSON(http.StatusOK, Response{
        Code: 0,
        Msg:  "success",
        Data: PageData{
            Items:    items,
            Total:    total,
            Page:     page,
            PageSize: pageSize,
        },
    })
}

func Error(c *gin.Context, code int, msg string) {
    c.JSON(http.StatusOK, Response{
        Code: code,
        Msg:  msg,
        Data: nil,
    })
}

func ErrorWithStatus(c *gin.Context, httpStatus int, code int, msg string) {
    c.JSON(httpStatus, Response{
        Code: code,
        Msg:  msg,
        Data: nil,
    })
}
```

使用：

```go
import "myweb/pkg/response" // 项目目录结构详见第72章

r.GET("/users", func(c *gin.Context) {
    users := []User{{Name: "张三"}, {Name: "李四"}}
    response.Page(c, users, 100, 1, 20)
})

r.GET("/users/:id", func(c *gin.Context) {
    id := c.Param("id")
    user, err := findUser(id)
    if err != nil {
        response.Error(c, 10001, "用户不存在")
        return
    }
    response.Success(c, user)
})
```

## 70.7 文件下载

### c.File：浏览器内预览

```go
r.GET("/download", func(c *gin.Context) {
    c.File("./reports/sales.pdf")
})
```

浏览器收到PDF/图片时可能直接在浏览器内打开，不触发下载。

### c.FileAttachment：强制下载

```go
r.GET("/download", func(c *gin.Context) {
    c.FileAttachment("./reports/sales.pdf", "2024年销售报告.pdf")
})
```

浏览器会弹出下载对话框，文件名是第二个参数指定的 `2024年销售报告.pdf`。

### 从内存中下载

当你动态生成文件（不存硬盘，比如导出Excel）时，用 `c.Data`：

```go
r.GET("/export", func(c *gin.Context) {
    content := []byte("姓名,年龄\n张三,25\n李四,30\n")
    c.Data(200, "text/csv; charset=utf-8", content)
})
```

### 设置下载响应头

```go
r.GET("/invoice/:id", func(c *gin.Context) {
    c.Header("Content-Type", "application/pdf")
    c.Header("Content-Disposition", "attachment; filename=invoice_"+c.Param("id")+".pdf")
    c.File("./invoices/" + c.Param("id") + ".pdf")
})
```

## 🤔 想多一点：响应体的编码问题

Go的 `html/template` 会自动对输出做HTML转义，防止XSS攻击：

```go
c.HTML(200, "page.html", gin.H{
    "userContent": "<script>alert('xss')</script>",
})
```

模板中 `{{ .userContent }}` 输出：

```html
&lt;script&gt;alert(&#39;xss&#39;)&lt;/script&gt;
```

浏览器会把这串文字原样显示，不会执行JavaScript。这是安全机制。

但如果你确实需要输出原始HTML（比如富文本编辑器生成的内容），用 `template.HTML` 类型：

```go
c.HTML(200, "page.html", gin.H{
    "userContent": template.HTML("<b>加粗文字</b>"),
})
```

**安全警告**：只有在你100%信任数据来源时才能用 `template.HTML`。永远不要把用户输入的内容用 `template.HTML` 包裹——那等于把门打开让攻击者进来。

## 本章小结

| 知识点 | 要点 |
|--------|------|
| JSON | `c.JSON(code, data)` 最常用 |
| gin.H | `map[string]interface{}` 的别名 |
| PureJSON | 不转义HTML特殊字符 |
| IndentedJSON | 格式化输出，调试用 |
| SecureJSON | 加 `while(1);` 前缀防JSON劫持 |
| XML/YAML | `c.XML()` / `c.YAML()` |
| 字符串 | `c.String(code, format, args...)` |
| HTML | `c.HTML(code, templateName, data)` |
| 静态文件 | `r.Static("/prefix", "./dir")` / `r.StaticFile("/path", "./file")` |
| 模板 | `r.LoadHTMLGlob("templates/*")` + `r.SetFuncMap()` 自定义函数 |
| 模板安全 | 自动转义HTML，用 `template.HTML` 输出原始HTML（谨慎） |
| 统一响应 | `{code, msg, data}` 封装到独立包中 |
| 文件下载 | `c.File(path)` 预览，`c.FileAttachment(path, name)` 强制下载 |
| 内存下载 | `c.Data(code, contentType, []byte{})` |
