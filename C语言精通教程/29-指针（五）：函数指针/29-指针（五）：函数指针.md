## 我在做什么？

前面你学了指针可以指向变量，指向数组，甚至指向另一个指针。但指针还能指向一个更特殊的东西——**函数**。

函数存在内存里，有地址，所以可以有指向它的指针。函数指针是 C 语言实现 *回调函数* *此术语见附录F*、策略模式、插件系统的基石。读完这一章，你能声明函数指针，用 `qsort` 排序任意类型的数据，还能用函数指针数组替代繁琐的 `switch` 语句。

## 一、函数也有地址

### 1.1 函数存在内存中

代码编译后，每条指令都在内存中有一个地址。函数名就是函数的起始地址。

```c
#include <stdio.h>

void hello() {
    printf("Hello!\n");
}

int main() {
    printf("hello 函数的地址：%p\n", hello);  // 打印函数地址
    printf("main 函数的地址：%p\n", main);
    return 0;
}
```

输出示例：
```text
hello 函数的地址：0x401150
main 函数的地址：0x401170
```

### 1.2 比喻：电话本

```text
函数指针就像一个电话本条目：

普通变量：    电话本上写着"张三的电话号码：12345678"
函数指针：    电话本上写着"修水管师傅的电话：87654321"
              你不需要知道修水管的具体步骤，你只需要知道
              打这个电话就能叫来师傅。
```

当你调用函数指针时，CPU 跳转到那个地址执行代码，执行完再跳回来。

## 二、声明函数指针

### 2.1 语法

函数指针的声明语法是 C 语言中最让人头疼的之一，但拆开看其实有规律：

```c
返回类型 (*指针名)(参数类型列表);
```

```c
int (*func_ptr)(int, int);   // func_ptr 是一个指针，指向"接收两个 int、返回 int"的函数
```

**括号 `(*func_ptr)` 是必须的**，否则变成声明一个返回指针的函数：

```c
int (*func_ptr)(int, int);   // 函数指针
int *func_ptr(int, int);     // 函数声明：返回 int* 的函数（不是指针！）
```

### 2.2 完整示例

```c
#include <stdio.h>

// 两个普通函数
int add(int a, int b) {
    return a + b;
}

int multiply(int a, int b) {
    return a * b;
}

int main() {
    // 声明函数指针，指向接收两个 int、返回 int 的函数
    int (*operation)(int, int);

    // 让指针指向 add
    operation = add;          // 等价于 operation = &add;
    printf("add(3, 5) = %d\n", operation(3, 5));   // 输出 8

    // 让指针指向 multiply
    operation = multiply;
    printf("multiply(3, 5) = %d\n", operation(3, 5));  // 输出 15

    return 0;
}
```

```text
operation 指向 add 时：
┌──────────────┐      ┌─────────────────────┐
│ operation    │ ───→ │ add 函数的代码       │
│ 值 = 0x401150│      │ { return a + b; }   │
└──────────────┘      └─────────────────────┘

operation 指向 multiply 时：
┌──────────────┐      ┌─────────────────────┐
│ operation    │ ───→ │ multiply 函数的代码   │
│ 值 = 0x401180│      │ { return a * b; }   │
└──────────────┘      └─────────────────────┘
```

### 2.3 用 `typedef` 简化

函数指针的声明很丑，用 `typedef` 可以让它变得可读：

```c
// 不用 typedef：每次都要写完整声明
int (*op1)(int, int);
int (*op2)(int, int);
int (*op3)(int, int);

// 用 typedef：定义一个类型别名
typedef int (*BinaryOp)(int, int);

BinaryOp op1;   // 清晰！
BinaryOp op2;
BinaryOp op3;
```

## 三、回调函数

### 3.1 什么是回调函数

*回调函数* *此术语见附录F* 就是：你把一个函数 A 作为参数传给另一个函数 B，B 在适当的时候调用 A。

```c
#include <stdio.h>

// 回调函数类型
typedef void (*PrintFunc)(int);

// 打印函数
void print_star(int n) {
    for (int i = 0; i < n; i++) printf("*");
    printf("\n");
}

void print_dash(int n) {
    for (int i = 0; i < n; i++) printf("-");
    printf("\n");
}

// 接受回调函数的函数
void repeat_print(int times, int count, PrintFunc printer) {
    for (int i = 0; i < times; i++) {
        printer(count);   // 调用回调函数
    }
}

int main() {
    printf("打印 3 次，每次 5 个星号：\n");
    repeat_print(3, 5, print_star);

    printf("打印 2 次，每次 10 个短横线：\n");
    repeat_print(2, 10, print_dash);

    return 0;
}
```

输出：
```text
打印 3 次，每次 5 个星号：
*****
*****
*****
打印 2 次，每次 10 个短横线：
----------
----------
```

### 3.2 比喻：委托办事

```text
你（repeat_print）要办一件事，但不自己执行，而是：
  1. 找个会干这件事的人（print_star / print_dash）
  2. 告诉他"帮我干 X 次"
  3. 每次他干完，你接着让他干下一次

回调函数就是"被委托的那个人"。
```

### 3.3 回调函数的典型场景

- **排序**：`qsort` 的排序规则由你决定（后面讲）。
- **事件处理**：GUI 编程中，"按钮被点击时调用这个函数"。
- **遍历**："遍历数组，对每个元素执行这个函数"。
- **自定义过滤器**："筛选出满足条件的元素，条件由你定义"。

## 四、`qsort`：C 标准库的通用排序

### 4.1 `qsort` 是什么

`qsort` 是 C 标准库提供的通用排序函数，可以排序**任何类型**的数组。它需要你提供一个比较函数。

```c
void qsort(void *base,           // 数组首地址
           size_t nmemb,         // 元素个数
           size_t size,          // 每个元素的大小
           int (*compar)(const void*, const void*));  // 比较函数
```

### 4.2 比较函数的规则

比较函数接收两个 `const void*` 参数（指向要比较的两个元素），返回：

- 负数：第一个元素应该排在第二个前面
- 0：两个元素相等
- 正数：第一个元素应该排在第二个后面

```c
int compare(const void *a, const void *b) {
    // 把 void* 转成具体类型，然后比较
    int ia = *(int*)a;
    int ib = *(int*)b;
    return ia - ib;   // 升序排序
}
```

### 4.3 排序 int 数组

```c
#include <stdio.h>
#include <stdlib.h>

// 升序比较函数
int compare_asc(const void *a, const void *b) {
    return (*(int*)a - *(int*)b);
}

// 降序比较函数
int compare_desc(const void *a, const void *b) {
    return (*(int*)b - *(int*)a);
}

int main() {
    int arr[] = {5, 2, 8, 1, 9, 3};
    int size = sizeof(arr) / sizeof(arr[0]);

    printf("原始数组：");
    for (int i = 0; i < size; i++) printf("%d ", arr[i]);
    printf("\n");

    qsort(arr, size, sizeof(int), compare_asc);
    printf("升序排序：");
    for (int i = 0; i < size; i++) printf("%d ", arr[i]);
    printf("\n");

    qsort(arr, size, sizeof(int), compare_desc);
    printf("降序排序：");
    for (int i = 0; i < size; i++) printf("%d ", arr[i]);
    printf("\n");

    return 0;
}
```

### 4.4 排序字符串数组

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int compare_string(const void *a, const void *b) {
    // a 和 b 是指向 char* 的指针，即 char**
    return strcmp(*(char**)a, *(char**)b);
}

int main() {
    char *words[] = {"banana", "apple", "cherry", "date"};
    int size = sizeof(words) / sizeof(words[0]);

    qsort(words, size, sizeof(char*), compare_string);

    for (int i = 0; i < size; i++) {
        printf("%s\n", words[i]);
    }

    return 0;
}
```

输出：
```text
apple
banana
cherry
date
```

### 4.5 排序结构体

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct {
    char name[20];
    int score;
} Student;

int compare_by_score(const void *a, const void *b) {
    Student *sa = (Student*)a;
    Student *sb = (Student*)b;
    return sb->score - sa->score;  // 按分数降序
}

int main() {
    Student students[] = {
        {"Alice", 85},
        {"Bob",   92},
        {"Carol", 78},
        {"Dave",  95}
    };
    int size = sizeof(students) / sizeof(students[0]);

    qsort(students, size, sizeof(Student), compare_by_score);

    printf("按分数降序：\n");
    for (int i = 0; i < size; i++) {
        printf("  %s: %d\n", students[i].name, students[i].score);
    }

    return 0;
}
```

## 五、函数指针数组：替代 switch

### 5.1 问题：大量的 switch-case

```c
switch (operation) {
    case 1: result = add(a, b);    break;
    case 2: result = sub(a, b);    break;
    case 3: result = mul(a, b);    break;
    case 4: result = div(a, b);    break;
    // 如果再加 10 个操作，switch 会越来越长
}
```

### 5.2 解决方案：函数指针数组

```c
#include <stdio.h>

// 四则运算
int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }
int mul(int a, int b) { return a * b; }
int div_op(int a, int b) { return b != 0 ? a / b : 0; }

int main() {
    // 函数指针数组：索引对应操作编号
    int (*operations[])(int, int) = {add, sub, mul, div_op};
    char *names[] = {"加", "减", "乘", "除"};

    int a = 20, b = 4;

    for (int i = 0; i < 4; i++) {
        printf("%d %s %d = %d\n", a, names[i], b, operations[i](a, b));
    }

    return 0;
}
```

输出：
```text
20 加 4 = 24
20 减 4 = 16
20 乘 4 = 80
20 除 4 = 5
```

```text
函数指针数组的结构：

operations:
┌───────────┬───────────┬───────────┬───────────┐
│ operations[0] │ operations[1] │ operations[2] │ operations[3] │
│ 0x401150  │ 0x401180  │ 0x4011B0  │ 0x4011E0  │
└─────│─────┴─────│─────┴─────│─────┴─────│─────┘
      │           │           │           │
      ▼           ▼           ▼           ▼
   add()       sub()       mul()      div_op()
```

## 六、常见错误与排错

### 6.1 忘记括号

❌ 错误示例：
```c
int *func(int, int);        // 声明了一个返回 int* 的函数，不是函数指针
```

✅ 正确示例：
```c
int (*func)(int, int);      // 函数指针
```

### 6.2 函数签名不匹配

❌ 错误示例：
```c
int add(int a, int b) { return a + b; }
double (*fp)(int, int);   // 返回类型不匹配
fp = add;                 // 警告
```

✅ 正确示例：
```c
int (*fp)(int, int);      // 返回类型匹配
fp = add;
```

### 6.3 `qsort` 比较函数返回类型错误

❌ 错误示例：
```c
int compare(const void *a, const void *b) {
    return *(int*)a < *(int*)b;  // 返回 0 或 1，不是负数/0/正数
}
```

✅ 正确示例：
```c
int compare(const void *a, const void *b) {
    return (*(int*)a - *(int*)b);  // 返回负数、0 或正数
}
```

## 我做得对不对？

用以下程序验证：

```c
#include <stdio.h>
#include <stdlib.h>

// 函数指针类型
typedef int (*BinaryOp)(int, int);

int add(int a, int b) { return a + b; }
int multiply(int a, int b) { return a * b; }

// 接受回调函数
int compute(BinaryOp op, int x, int y) {
    return op(x, y);
}

// qsort 比较函数
int compare_int(const void *a, const void *b) {
    return (*(int*)a - *(int*)b);
}

int main() {
    // 测试1：基本函数指针
    BinaryOp op = add;
    printf("测试1：add(3,5) = %d (应为 8)\n", op(3, 5));
    op = multiply;
    printf("测试1：multiply(3,5) = %d (应为 15)\n", op(3, 5));

    // 测试2：回调函数
    printf("测试2：compute(add, 10, 7) = %d (应为 17)\n", compute(add, 10, 7));

    // 测试3：qsort
    int arr[] = {5, 2, 8, 1, 9};
    int size = sizeof(arr) / sizeof(arr[0]);
    qsort(arr, size, sizeof(int), compare_int);
    printf("测试3：排序后 arr[0] = %d (应为 1), arr[4] = %d (应为 9)\n",
           arr[0], arr[4]);

    // 测试4：函数指针数组
    int (*ops[])(int, int) = {add, multiply};
    printf("测试4：ops[0](3,4) = %d (应为 7)\n", ops[0](3, 4));
    printf("测试4：ops[1](3,4) = %d (应为 12)\n", ops[1](3, 4));

    return 0;
}
```

编译运行：
```bash
$ gcc -Wall -o test test.c
$ ./test
测试1：add(3,5) = 8 (应为 8)
测试1：multiply(3,5) = 15 (应为 15)
测试2：compute(add, 10, 7) = 17 (应为 17)
测试3：排序后 arr[0] = 1 (应为 1), arr[4] = 9 (应为 9)
测试4：ops[0](3,4) = 7 (应为 7)
测试4：ops[1](3,4) = 12 (应为 12)
```

## 不对怎么办？

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| 编译警告 `assignment from incompatible pointer type` | 函数签名不匹配 | 检查返回类型和参数类型是否完全一致 |
| `qsort` 排序结果不对 | 比较函数逻辑错误 | 确保返回负数/0/正数，而不是布尔值 |
| 声明函数指针报错 | 忘了括号 | 用 `(*fp)` 而不是 `*fp` |
| 函数指针类型声明太复杂 | 语法复杂 | 用 `typedef` 简化 |

## 本章小结

| 知识点 | 一句话总结 |
|--------|-----------|
| 函数指针 | 存储函数地址的指针，声明为 `返回类型 (*名)(参数列表)` |
| 函数地址 | 用函数名（不加括号）获取，等价于 `&函数名` |
| 回调函数 | 把函数作为参数传给另一个函数，由后者调用 |
| `qsort` | 标准库通用排序，需要提供比较函数 |
| 函数指针数组 | 用数组索引选择调用哪个函数，替代 `switch` |
| `typedef` 简化 | `typedef int (*BinOp)(int, int);` 让声明更清晰 |
