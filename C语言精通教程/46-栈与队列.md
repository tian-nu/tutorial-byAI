## 我在做什么？

栈和队列是两种**"受限"的线性结构**——它们不允许你在任意位置插入删除，只能在特定位置操作。但正是这种限制，让它们在很多场景下用起来特别顺手。

想象一个**托盘堆**：
- 你每次只能放托盘到**最顶上**，也只能从最顶上拿托盘。
- 后放上来的托盘，反而先被拿走。
- 这就是**栈（Stack）**，后进先出（Last In First Out，LIFO）。

想象一个**排队打饭**：
- 后来的人排在队伍**最后面**，前面的人打完走了才轮到你。
- 先到先得，后到后得。
- 这就是**队列（Queue）**，先进先出（First In First Out，FIFO）。

`*此术语见附录F*` 中：栈、LIFO、队列、FIFO、push、pop、enqueue、dequeue。

## 46.1 栈的定义

**栈**是只能在**一端**（栈顶）进行插入和删除操作的线性表。
- `push`：插入元素（压入栈顶）
- `pop`：删除元素（弹出栈顶）
- `peek`：查看栈顶元素，但不删除

```
push 10    push 20    push 30    pop        pop

  [  ]       [  ]       [30]       [  ]       [  ]
             [  ]       [20]       [20]       [  ]
  (空)     [10]       [10]       [10]      (空)
  栈顶↓     栈顶↓       栈顶↓       栈顶↓
  底←──────────────────────────────←顶

输出顺序：弹出 30 → 弹出 20 → 最后只剩 10
```

## 46.2 栈的数组实现

数组实现栈非常简单：
- `data`：存储元素的数组
- `capacity`：数组容量
- `top`：栈顶位置索引（从 0 开始，`top == -1` 表示空栈）

```c
/* stack_array.h */
#ifndef STACK_ARRAY_H
#define STACK_ARRAY_H

typedef struct {
    int *data;
    int  top;
    int  capacity;
} StackArray;

StackArray* stack_array_create(int init_capacity);
void        stack_array_destroy(StackArray *s);
int         stack_array_push(StackArray *s, int value);
int         stack_array_pop(StackArray *s, int *out);
int         stack_array_peek(const StackArray *s, int *out);
int         stack_array_is_empty(const StackArray *s);
int         stack_array_size(const StackArray *s);

#endif
```

```c
/* stack_array.c */
#include "stack_array.h"
#include <stdlib.h>

/* 创建栈，初始容量 init_capacity */
StackArray* stack_array_create(int init_capacity) {
    StackArray *s = (StackArray*)malloc(sizeof(StackArray));
    if (s == NULL) return NULL;
    s->data = (int*)malloc(sizeof(int) * init_capacity);
    if (s->data == NULL) {
        free(s);
        return NULL;
    }
    s->capacity = init_capacity;
    s->top = -1;  /* 空栈 */
    return s;
}

/* 销毁栈 */
void stack_array_destroy(StackArray *s) {
    if (s != NULL) {
        if (s->data != NULL) free(s->data);
        free(s);
    }
}

/* 压入栈顶 */
int stack_array_push(StackArray *s, int value) {
    if (s == NULL) return -1;
    /* 如果容量不够，需要扩容 */
    if (s->top + 1 >= s->capacity) {
        int new_capacity = s->capacity * 2;
        int *new_data = (int*)realloc(s->data, sizeof(int) * new_capacity);
        if (new_data == NULL) return -1;  /* 扩容失败 */
        s->data = new_data;
        s->capacity = new_capacity;
    }
    s->top++;
    s->data[s->top] = value;
    return 0;
}

/* 弹出栈顶 */
int stack_array_pop(StackArray *s, int *out) {
    if (s == NULL || stack_array_is_empty(s)) return -1;
    *out = s->data[s->top];
    s->top--;
    return 0;
}

/* 查看栈顶 */
int stack_array_peek(const StackArray *s, int *out) {
    if (s == NULL || stack_array_is_empty(s)) return -1;
    *out = s->data[s->top];
    return 0;
}

int stack_array_is_empty(const StackArray *s) {
    return s->top == -1;
}

int stack_array_size(const StackArray *s) {
    return s->top + 1;
}
```

优点：实现简单，内存连续，缓存友好。缺点：动态扩容需要 `realloc`。

## 46.3 栈的链表实现

用链表实现栈，头插头删（都在链表头部，O(1)）。

```
栈顶在链表头部：每次 push 头插，每次 pop 删除头节点。

  栈顶 ↓
  [30] -> [20] -> [10] -> NULL
  ^        ^        ^
  弹出先拿30，正好。
```

```c
/* stack_list.h */
#ifndef STACK_LIST_H
#define STACK_LIST_H

typedef struct StackListNode {
    int                    data;
    struct StackListNode  *next;
} StackListNode;

typedef struct {
    StackListNode *top;  /* 栈顶就是链表头 */
    int           size;
} StackList;

StackList* stack_list_create(void);
void       stack_list_destroy(StackList *s);
int        stack_list_push(StackList *s, int value);
int        stack_list_pop(StackList *s, int *out);
int        stack_list_peek(const StackList *s, int *out);
int        stack_list_is_empty(const StackList *s);
int        stack_list_size(const StackList *s);

#endif
```

```c
/* stack_list.c */
#include "stack_list.h"
#include <stdlib.h>

StackList* stack_list_create(void) {
    StackList *s = (StackList*)malloc(sizeof(StackList));
    if (s == NULL) return NULL;
    s->top = NULL;
    s->size = 0;
    return s;
}

void stack_list_destroy(StackList *s) {
    if (s == NULL) return;
    StackListNode *cur = s->top;
    while (cur != NULL) {
        StackListNode *next = cur->next;
        free(cur);
        cur = next;
    }
    free(s);
}

int stack_list_push(StackList *s, int value) {
    if (s == NULL) return -1;
    StackListNode *n = (StackListNode*)malloc(sizeof(StackListNode));
    if (n == NULL) return -1;
    n->data = value;
    n->next = s->top;
    s->top = n;
    s->size++;
    return 0;
}

int stack_list_pop(StackList *s, int *out) {
    if (s == NULL || stack_list_is_empty(s)) return -1;
    *out = s->top->data;
    StackListNode *to_del = s->top;
    s->top = s->top->next;
    free(to_del);
    s->size--;
    return 0;
}

int stack_list_peek(const StackList *s, int *out) {
    if (s == NULL || stack_list_is_empty(s)) return -1;
    *out = s->top->data;
    return 0;
}

int stack_list_is_empty(const StackList *s) {
    return s->size == 0;
}

int stack_list_size(const StackList *s) {
    return s->size;
}
```

优点：不需要预分配空间，动态增减。缺点：每个节点额外一个指针。

## 46.4 应用：括号匹配

栈最经典的应用之一：检查表达式中的括号是否匹配。

算法思路：
1. 遍历每个字符。
2. 如果是左括号 `(`，push 入栈。
3. 如果是右括号 `)`：
   - 如果栈空 → 右括号多了，不匹配。
   - 否则 pop 一个左括号 → 匹配一对。
4. 遍历结束后，如果栈空 → 全部匹配；否则左括号多了，不匹配。

```c
/* 括号匹配示例 */
int check_parentheses(const char *expr) {
    StackList *stack = stack_list_create();
    if (stack == NULL) return -1;  /* 内存错误 */

    for (int i = 0; expr[i] != '\0'; i++) {
        char c = expr[i];
        if (c == '(') {
            stack_list_push(stack, 1);
        } else if (c == ')') {
            if (stack_list_is_empty(stack)) {
                stack_list_destroy(stack);
                return 0;  /* 不匹配 */
            }
            int tmp;
            stack_list_pop(stack, &tmp);
        }
    }

    int result = stack_list_is_empty(stack) ? 1 : 0;
    stack_list_destroy(stack);
    return result;
}
```

测试：
- `"1+(2*3))"` → 不匹配（栈最后空？不，多一个右括号）
- `"(1+(2*3)"` → 不匹配（最后栈非空）
- `"(1+(2*3))"` → 匹配

## 46.5 队列的定义

**队列**是只能在**一端**插入，另一端删除的线性表。
- `enqueue`：插入到队尾
- `dequeue`：从队头删除
- `peek`：查看队头元素

```
enqueue 10    enqueue 20    enqueue 30    dequeue    dequeue

  头             头→[10]           头→[10]→[20]→[30]→尾   [20]→[30]→尾     [30]→尾
  (空)
输出顺序：弹出 10 → 弹出 20 → 最后只剩 30
先进去的先出来 → FIFO
```

## 46.6 队列的链表实现

用链表实现队列：头删尾插，都要记住头指针和尾指针。

```c
typedef struct QueueNode {
    int            data;
    struct QueueNode *next;
} QueueNode;

typedef struct {
    QueueNode *front;  /* 队头（出队从这里拿） */
    QueueNode *rear;   /* 队尾（入队插在这里） */
    int       size;
} Queue;
```

```c
/* queue_list.c */
#include "queue_list.h"
#include <stdlib.h>

Queue* queue_create(void) {
    Queue *q = (Queue*)malloc(sizeof(Queue));
    if (q == NULL) return NULL;
    q->front = q->rear = NULL;
    q->size = 0;
    return q;
}

void queue_destroy(Queue *q) {
    if (q == NULL) return;
    QueueNode *cur = q->front;
    while (cur != NULL) {
        QueueNode *next = cur->next;
        free(cur);
        cur = next;
    }
    free(q);
}

int queue_enqueue(Queue *q, int value) {
    if (q == NULL) return -1;
    QueueNode *n = (QueueNode*)malloc(sizeof(QueueNode));
    if (n == NULL) return -1;
    n->data = value;
    n->next = NULL;
    if (queue_is_empty(q)) {
        q->front = q->rear = n;
    } else {
        q->rear->next = n;
        q->rear = n;
    }
    q->size++;
    return 0;
}

int queue_dequeue(Queue *q, int *out) {
    if (q == NULL || queue_is_empty(q)) return -1;
    *out = q->front->data;
    QueueNode *to_del = q->front;
    q->front = q->front->next;
    free(to_del);
    q->size--;
    if (q->front == NULL) {
        q->rear = NULL;  /* 空了，rear也要设为 NULL */
    }
    return 0;
}

int queue_peek(const Queue *q, int *out) {
    if (q == NULL || queue_is_empty(q)) return -1;
    *out = q->front->data;
    return 0;
}

int queue_is_empty(const Queue *q) {
    return q->size == 0;
}

int queue_size(const Queue *q) {
    return q->size;
}
```
