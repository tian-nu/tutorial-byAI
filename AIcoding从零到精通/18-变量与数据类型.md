## 变量是什么：贴标签的盒子

想象你搬家，把东西装进纸箱子。为了防止搞混，你在每个箱子上贴标签：

```
📦 [冬衣]      📦 [厨具]      📦 [书]
```

**变量（Variable）**📖 就是这样一个贴了标签的盒子。标签是变量名，盒子里的东西是数据。

你可以：
- 往盒子里放东西（赋值）
- 打开盒子看里面是什么（读取）
- 把盒子里的东西换成别的（重新赋值）

看一段AI生成的代码，你就明白了：

> 以下代码由AI生成：

```python
# 把"Alice"放进一个贴了"name"标签的盒子
name = "Alice"

# 把25放进一个贴了"age"标签的盒子
age = 25

# 把True放进一个贴了"is_student"标签的盒子
is_student = True

# 打开盒子，看看里面是什么
print(name)        # 输出：Alice
print(age)         # 输出：25
print(is_student)  # 输出：True
```

每一行都在做同一件事：**`标签名 = 东西`**。等号左边是标签，右边是你放进去的东西。

> ⚠️ 编程里的等号 `=` 不是"等于"，而是"赋值"——把右边的东西放进左边的盒子。判断"等不等于"用的是 `==`（两个等号），这个第20章会讲。

## 常见数据类型：盒子可以装不同种类的东西

纸箱子可以装衣服、装书、装锅碗瓢盆。变量也一样，可以装不同**类型**的数据。

**数据类型（Data Type）**📖 = 变量盒子里装的东西属于什么种类。

Python里最常见的五种：

### ① 整数（int）—— 整数

```python
score = 95          # 考试分数
year = 2026         # 年份
count = 0           # 计数器，从0开始
```

就是数学里的整数，可以是正数、负数、零。

### ② 浮点数（float）—— 带小数点的数

```python
price = 19.99       # 价格
temperature = 36.5  # 体温
pi = 3.14159        # 圆周率
```

整数和浮点数统称"数字类型"，你可以对它们做加减乘除。

### ③ 字符串（String）📖 —— 文字，放在引号里

```python
name = "张三"           # 双引号
message = '你好，世界'   # 单引号也可以
empty = ""              # 空字符串，就像空盒子
```

**字符串就是文字**。可以是你的名字、一句话、一篇文章。特征：**必须用引号包起来**。没引号的话，Python会以为你在写变量名。

❌ 错误：`name = 张三` — Python不知道"张三"是什么
✅ 正确：`name = "张三"` — 引号告诉Python"这是一段文字"

### ④ 布尔值（Boolean）📖 —— 真或假，是或否

```python
is_logged_in = True     # 已登录
has_permission = False  # 没有权限
is_adult = True         # 是成年人
```

**布尔值只有两种可能：`True`（真/是）或 `False`（假/否）**。它就像一个开关——要么开，要么关。用来做判断最常用。

> 📖 **这个名字的来源**：布尔值是以数学家乔治·布尔（George Boole）命名的，他发明了用"真/假"做计算的逻辑系统。你不用管他是谁，知道 True/False 就够了。

### ⑤ 列表与字典 —— 多个数据的容器

这两种比较重要，第21章专门讲。这里先混个脸熟：

```python
# 列表：一排有编号的盒子
fruits = ["苹果", "香蕉", "橘子"]

# 字典：标签→内容的对照表
person = {"name": "张三", "age": 25, "city": "北京"}
```

## 五种类型速查表

| 类型 | Python名字 | 示例 | 一句话 | 特征 |
|------|-----------|------|--------|------|
| 整数 | `int` | `42` | 没有小数点的数字 | 可正可负可为零 |
| 浮点数 | `float` | `3.14` | 带小数点的数字 | 也叫"小数" |
| 字符串 | `str` | `"Hello"` | 文字 | **必须加引号** |
| 布尔值 | `bool` | `True` | 真/假 | 只有True和False |
| 列表 | `list` | `[1,2,3]` | 一排盒子 | 用方括号`[]` |
| 字典 | `dict` | `{"a":1}` | 标签对照表 | 用花括号`{}` |

## AI生成代码示例：个人信息卡片

> 以下代码由AI生成：

```python
# 个人信息 —— 用不同数据类型存储关于一个人的各种信息
name = "李明"                # 字符串：名字是文字
age = 28                     # 整数：年龄是整数
height = 1.75                # 浮点数：身高带小数
is_employed = True           # 布尔值：是否在职
hobbies = ["篮球", "读书", "编程"]  # 列表：多个爱好

# 打印出来看看
print("姓名：", name)
print("年龄：", age)
print("身高：", height, "米")
print("是否在职：", is_employed)
print("爱好：", hobbies)
```

**逐行讲解**：
- 第2行：把文字"李明"放进 `name` 盒子（字符串）
- 第3行：把数字28放进 `age` 盒子（整数）
- 第4行：把小数1.75放进 `height` 盒子（浮点数）
- 第5行：把 `True` 放进 `is_employed` 盒子（布尔值）
- 第6行：把三个爱好放进 `hobbies` 盒子（列表）
- 第8-12行：用 `print()` 把每个盒子里的内容展示出来

## 自测

1. `name = "Alice"` 中，`name` 是什么？`"Alice"` 是什么类型？
2. 下面这段代码中的变量各是什么类型？
   ```python
   score = 95.5
   passed = True
   name = "小明"
   ```
3. `=` 和 `==` 的区别是什么？（提示：一个叫赋值，一个叫判断相等）

> **答案**：
> 1. `name` 是变量名（标签），`"Alice"` 是字符串类型。
> 2. `score` 是浮点数（float），`passed` 是布尔值（bool），`name` 是字符串（str）。
> 3. `=` 是赋值（把右边放进左边盒子），`==` 是判断相等（问"两边一样吗？"）。
