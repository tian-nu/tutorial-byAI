## 数据结构是什么：管理数据的"容器"

你已经知道变量是一个贴标签的盒子。但如果你有100个学生的分数呢？定义100个变量？

```python
# ❌ 疯了吗
score1 = 85
score2 = 92
score3 = 78
# ... 写到 score100 = 88
```

显然不行。这时候你需要**数据结构（Data Structure）**📖 —— 把多个数据组织在一起的大容器。

> 📖 **数据结构** = 数据的管理方式。就像你衣服可以叠起来放衣柜（列表），也可以按季节分类挂好（字典）。同样的数据，不同的组织方式，查起来效率不一样。

本章讲两种最常用的：**列表**和**字典**。

---

## 列表（List）：一排有编号的盒子

**列表**就像一排编号的储物柜：

```
┌────┬────┬────┬────┬────┐
│ 0  │ 1  │ 2  │ 3  │ 4  │  ← 编号（索引）
├────┼────┼────┼────┼────┤
│苹果│香蕉│橘子│西瓜│葡萄│  ← 内容
└────┴────┴────┴────┴────┘
```

每个位置有一个编号，从0开始。第一个是0号，第二个是1号，第三个是2号……

> 🔥 **重点！编号从0开始！** 这是编程世界里最反直觉的设定。如果你觉得奇怪，用"电梯楼层"来理解：一楼是地面（0），二楼才是1。或者想"你出生时是0岁，满一周岁才是1岁"。总之，**列表的第一个元素编号是0，不是1**。

> 以下代码由AI生成：

```python
# 创建列表：用方括号 []，元素之间用逗号隔开
fruits = ["苹果", "香蕉", "橘子", "西瓜", "葡萄"]

# 按编号取东西（编号从0开始！）
print(fruits[0])    # 第1个：苹果
print(fruits[1])    # 第2个：香蕉
print(fruits[3])    # 第4个：西瓜

# 列表里有多少个元素？
print(len(fruits))  # 输出：5

# 往列表末尾加东西
fruits.append("草莓")  # 列表变成：["苹果", "香蕉", "橘子", "西瓜", "葡萄", "草莓"]

# 删除某个元素
fruits.remove("橘子")  # 列表变成：["苹果", "香蕉", "西瓜", "葡萄", "草莓"]
```

**逐行讲解**：
- 第2行：用 `[]` 创建列表，5个水果。
- 第5-7行：`fruits[0]` 取第1个（编号0），`fruits[1]` 取第2个（编号1）。
- 第10行：`len()` 是"length"的缩写，返回列表里有多少个元素。
- 第13行：`.append()` 是"追加"的意思，在列表末尾加一个。
- 第16行：`.remove()` 是"删除"的意思，找到指定元素并删掉。

## 字典（Dictionary）：标签 → 内容的对照表

**字典**就像一本真正的字典：

```
查"苹果"  →  一种水果，红色或绿色，甜的
查"Python" →  一种编程语言，简单易学
```

在代码里，字典是**键值对**（Key → Value）的集合：

```python
# 花括号 {}，冒号左边是标签（键），右边是内容（值）
person = {
    "name": "张三",       # 标签"name" → 内容"张三"
    "age": 28,            # 标签"age" → 内容28
    "city": "北京",       # 标签"city" → 内容"北京"
    "hobbies": ["篮球", "读书"]  # 值也可以是列表！
}

# 按标签查内容
print(person["name"])     # 输出：张三
print(person["age"])      # 输出：28

# 添加新标签
person["job"] = "工程师"  # 新增一个键值对

# 修改已有的标签
person["age"] = 29        # 把年龄改成29
```

**逐行讲解**：
- 第2-7行：用 `{}` 创建字典。`"name"` 是键（标签），`"张三"` 是值（内容）。冒号连接，逗号分隔。
- 第10-11行：`person["name"]` 就像查字典——给我标签"name"对应的内容。
- 第14行：直接给一个新标签赋值，就是添加。
- 第17行：给已有标签赋新值，就是修改。

---

## 什么时候用列表？什么时候用字典？

| 场景 | 用列表 | 用字典 |
|------|--------|--------|
| 存所有学生的名字 | ✅ 一排名字，按顺序取 | ❌ 没意义，不需要标签 |
| 存一个学生的各种信息 | ❌ 不知道哪个是名字哪个是年龄 | ✅ 标签明确：name, age, city |
| 排队处理 | ✅ 先进先出，按顺序 | ❌ 不需要标签 |
| 查特定信息 | ❌ 得记住编号 | ✅ 直接用标签查 |

**简单判断法**：
- 你需要"第几个" → 用列表
- 你需要"叫什么" → 用字典

## AI生成代码示例：通讯录

> 以下代码由AI生成：

```python
# 通讯录：用列表存多个人，每个人用字典存信息
contacts = [
    {"name": "张三", "phone": "13800138001", "city": "北京"},
    {"name": "李四", "phone": "13800138002", "city": "上海"},
    {"name": "王五", "phone": "13800138003", "city": "广州"},
]

# 遍历通讯录，打印每个人的信息
for person in contacts:                        # 从列表里每次取一个字典
    name = person["name"]                      # 从字典里取"name"标签的值
    phone = person["phone"]                    # 从字典里取"phone"标签的值
    city = person["city"]                      # 从字典里取"city"标签的值
    print(f"{name}，电话：{phone}，城市：{city}")
```

**输出**：
```
张三，电话：13800138001，城市：北京
李四，电话：13800138002，城市：上海
王五，电话：13800138003，城市：广州
```

**逐行讲解**：
- 第2-6行：`contacts` 是一个列表，里面每个元素都是一个字典。**列表套字典**——这是最常见的组合。
- 第9行：`for person in contacts` 每次从列表里取出一个字典（一个人）。
- 第10-12行：用 `person["name"]` 从字典里取特定标签的值。
- 第13行：格式化打印。
