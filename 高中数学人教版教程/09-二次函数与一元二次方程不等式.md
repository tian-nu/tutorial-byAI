## 09.3 三个"二次"的关系

> 二次函数、一元二次方程、一元二次不等式——三个名字里都有"二次"，它们到底是什么关系？

### 09.3.1 同一个故事的三张面孔

考虑这三个式子：

$$
\begin{aligned}
\text{二次函数：}&\quad y = x^2 - 3x - 10 \quad &\text{—— 输入 } x \text{，输出 } y \\[4pt]
\text{二次方程：}&\quad x^2 - 3x - 10 = 0 \quad &\text{—— 问：什么时候 } y = 0 \text{？} \\[4pt]
\text{二次不等式：}&\quad x^2 - 3x - 10 > 0 \quad &\text{—— 问：什么时候 } y > 0 \text{？}
\end{aligned}
$$

发现了吗？它们共享同一个式子 $x^2 - 3x - 10$，只是**问的问题不同**：

| 对象 | 问的问题 | 答案 |
|------|---------|------|
| 二次函数 | 对于每个 $x$，$y$ 等于多少？ | 一条抛物线 |
| 二次方程 | 哪些 $x$ 让 $y$ 恰好等于 0？ | 两个根（或一个根，或无根） |
| 二次不等式 | 哪些 $x$ 让 $y$ 大于 0（或小于 0）？ | 一段或多段区间 |

> 💡 **生活比喻**：想象你在一条起伏的山路上开车。
> - **二次函数**就是整条山路的**海拔曲线**（$y$ 是海拔，$x$ 是水平位置）。
> - **二次方程**问的是：**在哪里海拔恰好是 0 米？**（与海平面的交点）
> - **二次不等式**问的是：**哪里海拔高于 0 米？哪里低于 0 米？**

### 09.3.2 用抛物线统一三者的核心思想

画出 $y = x^2 - 3x - 10$ 的图像（抛物线）：

```
        y
        ↑
        |     ↗         ↗
        |    /           \
        |   /             \
    ────┼──/───────────────\──────→ x
        | /                 \
        |/                   \
    ────●─────────────────────●──── y = 0（x轴）
        |                       \
        |                        \
        |                         \
              ↑               ↑
            x = -2          x = 5
```

- 抛物线与 $x$ 轴的交点：$x = -2$ 和 $x = 5$。这两个点就是方程 $x^2 - 3x - 10 = 0$ 的根。
- 抛物线在 $x$ 轴**上方**的部分（$y > 0$）：$x < -2$ 或 $x > 5$。这就是不等式 $x^2 - 3x - 10 > 0$ 的解集。
- 抛物线在 $x$ 轴**下方**的部分（$y < 0$）：$-2 < x < 5$。这就是不等式 $x^2 - 3x - 10 < 0$ 的解集。

> 📊 **交互演示见 [visual/math1_quadratic_visual.html](visual/math1_quadratic_visual.html)**——拖拽参数 $a, b, c$，观察抛物线如何变化，以及不等式解集如何随之改变。**强烈建议现在就打开，边看边读下面的内容。**

> 🤔 **想多一点**：为什么数学家要把这三者放在一起讲？因为分开学是"三种题型"，合在一起学是"一种思想"——**数形结合**。以后你遇到任何二次不等式，脑海中浮现的不应是一堆符号，而应该是一条抛物线。这就是高中数学和初中数学最大的区别：初中你"算"出答案，高中你"看"出答案。

## 09.4 解一元二次不等式的标准步骤

### 09.4.1 四步法

对于不等式 $ax^2 + bx + c > 0$（或 $< 0, \geq 0, \leq 0$），按以下四步操作：

**第 1 步：化为标准形式**

确保不等式右边为 0，左边为 $ax^2 + bx + c$ 的形式，且 $a > 0$。

- 如果 $a < 0$，不等式两边同乘 $-1$，**注意不等号方向要反转**。

> 例：$-2x^2 + 3x - 1 > 0$ → 两边乘 $-1$ → $2x^2 - 3x + 1 < 0$（注意 $>$ 变成了 $<$）

**第 2 步：计算判别式 $\Delta = b^2 - 4ac$**

判别式告诉你抛物线与 $x$ 轴有几个交点：

| $\Delta$ 的值 | 与 $x$ 轴的交点 | 方程有几个实根 |
|:---:|:---:|:---:|
| $\Delta > 0$ | 两个不同交点 | 两个不等实根 $x_1, x_2$ |
| $\Delta = 0$ | 一个交点（相切） | 一个实根（重根）$x_0$ |
| $\Delta < 0$ | 没有交点 | 无实根 |

**第 3 步：若 $\Delta \geq 0$，求根**

用求根公式或十字相乘法求出一元二次方程的根：

$$x = \frac{-b \pm \sqrt{\Delta}}{2a}$$

设 $x_1 < x_2$（两根中较小的称为 $x_1$，较大的称为 $x_2$）。

**第 4 步：画抛物线草图，根据开口方向确定解集**

> ⚠️ **这是最关键的一步，也是最容易被跳过的一步。** 很多同学算完根就直接写答案，结果开口方向弄反、不等号弄反——**画了图就不会错**。

- 开口向上（$a > 0$）：抛物线 $\cup$ 形
- 开口向下（$a < 0$）：先化为 $a > 0$ 再解

> 📊 交互演示见 [visual/math1_quadratic_visual.html](visual/math1_quadratic_visual.html)

### 09.4.2 口诀："大于取两边，小于取中间"

这是解一元二次不等式最重要的口诀，但它有一个**前提条件**：**$a > 0$ 且 $\Delta > 0$**。

- **"大于取两边"**：不等式是 $ax^2 + bx + c > 0$（$a > 0$），解集是两根之外的区间：$x < x_1$ 或 $x > x_2$。
- **"小于取中间"**：不等式是 $ax^2 + bx + c < 0$（$a > 0$），解集是两根之间的区间：$x_1 < x < x_2$。

```
"大于取两边"：
        y↑
         |   ↗        ↗
         |  /          \
    ─────┼─/────────────\─────→ x
         |/              \
        ●                ●       ← 两根 x₁, x₂
    ←───┘                └───→   ← 解集：两边（x < x₁ 或 x > x₂）
         ↑ y > 0 的区域 ↑

"小于取中间"：
        y↑
         |   ↗        ↗
         |  /          \
    ─────┼─/────────────\─────→ x
         |/              \
        ●                ●       ← 两根 x₁, x₂
         └──────┬───────┘       ← 解集：中间（x₁ < x < x₂）
              y < 0 的区域
```

> 💡 **"两边"和"中间"指什么？**
> - "两边"：两根的**外侧**——$x$ 小于较小的根，或 $x$ 大于较大的根。写成区间是 $(-\infty, x_1) \cup (x_2, +\infty)$。
> - "中间"：两根的**内侧**——$x$ 在两根之间。写成区间是 $(x_1, x_2)$。
> - 为什么叫"两边"和"中间"？因为两根把数轴切成三段：左段（$x < x_1$）、中段（$x_1 < x < x_2$）、右段（$x > x_2$）。"大于"取左段和右段（两边），"小于"取中段（中间）。

## 09.5 分情况讨论

了解了口诀之后，我们来看**所有可能的情况**。下面的表格覆盖了 $a > 0$ 时 $\Delta$ 的三种取值，以及 $a < 0$ 时的处理方式。

### 09.5.1 情况一：$a > 0$，$\Delta > 0$（最典型的情况）

两根 $x_1 < x_2$，抛物线开口向上，与 $x$ 轴有两个交点。

| 不等式 | 解集 | 区间表示 |
|--------|------|---------|
| $ax^2 + bx + c > 0$ | $x < x_1$ 或 $x > x_2$ | $(-\infty, x_1) \cup (x_2, +\infty)$ |
| $ax^2 + bx + c \geq 0$ | $x \leq x_1$ 或 $x \geq x_2$ | $(-\infty, x_1] \cup [x_2, +\infty)$ |
| $ax^2 + bx + c < 0$ | $x_1 < x < x_2$ | $(x_1, x_2)$ |
| $ax^2 + bx + c \leq 0$ | $x_1 \leq x \leq x_2$ | $[x_1, x_2]$ |

> 这就是口诀"大于取两边，小于取中间"的完整版。注意：**带等号的不等式，区间端点也要带等号（用闭区间 `[ ]`）**。

### 09.5.2 情况二：$a > 0$，$\Delta = 0$（抛物线与 $x$ 轴相切）

只有一个根 $x_0$（重根），抛物线开口向上，顶点恰好在 $x$ 轴上。

```
        y↑
         |       ↗
         |      /
         |     /
    ─────┼────●────────→ x
         |   / x₀
         |  /
         | /
```

| 不等式 | 解集 | 区间表示 |
|--------|------|---------|
| $ax^2 + bx + c > 0$ | $x \neq x_0$ | $(-\infty, x_0) \cup (x_0, +\infty)$ |
| $ax^2 + bx + c \geq 0$ | 全体实数 | $\mathbb{R}$ |
| $ax^2 + bx + c < 0$ | 无解 | $\varnothing$ |
| $ax^2 + bx + c \leq 0$ | $x = x_0$ | $\{x_0\}$ |

> 🤔 **理解关键**：$\Delta = 0$ 时，抛物线除了在 $x_0$ 处等于 0，其余所有位置都大于 0（因为开口向上）。所以 $> 0$ 的解集是"除 $x_0$ 外的所有实数"，$< 0$ 无解（抛物线没有低于 $x$ 轴的部分）。

### 09.5.3 情况三：$a > 0$，$\Delta < 0$（抛物线与 $x$ 轴无交点）

方程无实根，抛物线开口向上，整个图像都在 $x$ 轴上方。

```
        y↑
         |    ↗     ↗
         |   /       \
         |  /         \
    ─────┼─/───────────\───→ x
         |/             \
         |               \
         | (永远在x轴上方)
```

| 不等式 | 解集 | 区间表示 |
|--------|------|---------|
| $ax^2 + bx + c > 0$ | 全体实数 | $\mathbb{R}$ |
| $ax^2 + bx + c \geq 0$ | 全体实数 | $\mathbb{R}$ |
| $ax^2 + bx + c < 0$ | 无解 | $\varnothing$ |
| $ax^2 + bx + c \leq 0$ | 无解 | $\varnothing$ |

> 💡 **记忆技巧**：$\Delta < 0$ 且 $a > 0$ → 抛物线永远在 $x$ 轴上方 → $y$ 永远大于 0 → $>0$ 恒成立，$<0$ 永远不成立。

### 09.5.4 情况四：$a < 0$（先化为 $a > 0$ 再解）

> ⚠️ **这是最容易出错的场景！** 当二次项系数为负时，**不要直接套口诀**，必须先化为 $a > 0$ 的形式。

**处理方法**：不等式两边同乘 $-1$，**不等号方向反转**，然后按 $a > 0$ 的情况处理。

**示例**：解 $-x^2 + 3x - 2 > 0$

- 第 1 步：两边乘 $-1$ → $x^2 - 3x + 2 < 0$（注意 $>$ 变成了 $<$）
- 第 2 步：$\Delta = (-3)^2 - 4 \cdot 1 \cdot 2 = 9 - 8 = 1 > 0$
- 第 3 步：求根 $x_1 = 1$，$x_2 = 2$
- 第 4 步：$a = 1 > 0$，$<$ 取中间 → $1 < x < 2$

- ❌ **错误做法**：不化成正的，直接套口诀。$-x^2 + 3x - 2 > 0$ 开口向下，"大于"的口诀就不再适用了。
- ✅ **正确做法**：始终先把 $a$ 化为正数，再套口诀。

## 09.6 含参一元二次不等式的讨论

当不等式中的系数包含参数（未知字母）时，不能直接套用固定公式，必须**分类讨论**。

### 09.6.1 参数在二次项系数：讨论 $a = 0$ / $a > 0$ / $a < 0$

当二次项系数含参数时，首先要判断它是不是真的"二次"不等式。

**策略**：

| $a$ 的取值 | 不等式类型 | 处理方法 |
|:---:|:---:|------|
| $a = 0$ | 退化为一次不等式 | 按一次不等式解（注意一次项系数是否为 0） |
| $a > 0$ | 标准二次不等式，开口向上 | 按 09.5 节步骤处理 |
| $a < 0$ | 二次不等式，开口向下 | 化为 $a > 0$ 后按 09.5 节处理 |

> ⚠️ **关键提醒**：看到含参二次不等式，第一反应不是求 $\Delta$，而是问"$a$ 是不是 0？"**忘了这一步，后面的讨论全部前功尽弃。**

### 09.6.2 参数在其他位置：讨论判别式、两根大小

当参数不出现在二次项系数上（或已确定 $a \neq 0$），需要讨论：

1. **判别式 $\Delta$ 的符号**：$\Delta > 0$、$\Delta = 0$、$\Delta < 0$ 三种情况，解集形式完全不同。
2. **两根的大小关系**：当 $\Delta > 0$ 时，两根 $x_1$ 和 $x_2$ 的大小可能随参数变化。如果题目没有明确 $x_1$ 和 $x_2$ 谁大谁小，需要根据参数范围判断。

> 💡 **讨论顺序口诀**：先看二次项（是不是二次？开口向哪？），再看判别式（有几个根？），最后比两根（谁大谁小？）。按这个顺序走，不会漏情况。

## 09.7 例题精讲

### 例题 1 —— 标准型一元二次不等式

**题目**：解不等式 $2x^2 - 5x + 2 < 0$

**解**：

**第 1 步**：已经是标准形式，$a = 2 > 0$，无需调整。

**第 2 步**：计算判别式。

$$\Delta = (-5)^2 - 4 \times 2 \times 2 = 25 - 16 = 9 > 0$$

$\Delta > 0$，有两个不等实根。

**第 3 步**：求根。

$$x = \frac{5 \pm \sqrt{9}}{2 \times 2} = \frac{5 \pm 3}{4}$$

$$x_1 = \frac{5 - 3}{4} = \frac{1}{2}, \quad x_2 = \frac{5 + 3}{4} = 2$$

**第 4 步**：画草图。

$a = 2 > 0$，开口向上。不等式是 $< 0$（小于），取"中间"。

```
        y↑
         |   ↗        ↗
         |  /          \
    ─────┼─/────────────\─────→ x
         |/              \
        ●                ●
       x=½              x=2
         └──────┬───────┘
           解集：½ < x < 2
```

**解集**：$\displaystyle \left(\frac{1}{2}, 2\right)$

---

### 例题 2 —— $\Delta = 0$ 的情况

**题目**：解不等式 $x^2 + 4x + 4 \geq 0$

**解**：

**第 1 步**：已是标准形式，$a = 1 > 0$。

**第 2 步**：计算判别式。

$$\Delta = 4^2 - 4 \times 1 \times 4 = 16 - 16 = 0$$

$\Delta = 0$，有一个重根。

**第 3 步**：求根。

$$x^2 + 4x + 4 = (x + 2)^2 = 0 \quad \Rightarrow \quad x = -2$$

**第 4 步**：画草图。

$a = 1 > 0$，开口向上。$\Delta = 0$，顶点在 $x$ 轴上。不等式是 $\geq 0$。

```
        y↑
         |       ↗
         |      /
         |     /
    ─────┼────●────────→ x
         |   / x=-2
         |  /
         | /
```

抛物线全部在 $x$ 轴及上方（$y \geq 0$ 对所有 $x$ 成立）。

**解集**：$\mathbb{R}$（全体实数）

> 🧠 **反思**：$(x + 2)^2 \geq 0$ 永远成立（任何实数的平方 $\geq 0$），所以解集是全体实数。这种题不需要算 $\Delta$，直接配方就能看出来。但按标准步骤走一遍也不会有错。

---

### 例题 3 —— 含参不等式的分类讨论

**题目**：解不等式 $ax^2 - (a+1)x + 1 > 0$（$a$ 为实数）

**解**：

**第 1 步**：判断二次项系数 $a$。

- 若 $a = 0$，不等式变为 $-x + 1 > 0$，即 $x < 1$。解集为 $(-\infty, 1)$。

- 若 $a \neq 0$，是二次不等式，需要继续讨论。

**第 2 步**（$a \neq 0$）：计算判别式。

$$\Delta = [-(a+1)]^2 - 4a \cdot 1 = (a+1)^2 - 4a = a^2 + 2a + 1 - 4a = a^2 - 2a + 1 = (a-1)^2$$

$\Delta = (a-1)^2 \geq 0$ 恒成立。

**第 3 步**：求根。

$$x = \frac{(a+1) \pm \sqrt{(a-1)^2}}{2a} = \frac{(a+1) \pm |a-1|}{2a}$$

**子情况 2.1**：$a = 1$ 时，$\Delta = 0$。

$$x = \frac{1+1}{2 \times 1} = 1$$

不等式变为 $x^2 - 2x + 1 > 0$，即 $(x-1)^2 > 0$。$a = 1 > 0$，开口向上，$\Delta = 0$。

解集：$x \neq 1$，即 $(-\infty, 1) \cup (1, +\infty)$。

**子情况 2.2**：$a > 0$ 且 $a \neq 1$。

此时 $\Delta > 0$，有两个不等实根。需要去掉绝对值：

- 若 $a > 1$，则 $|a-1| = a-1$：
  $$x_1 = \frac{a+1 - (a-1)}{2a} = \frac{2}{2a} = \frac{1}{a}, \quad x_2 = \frac{a+1 + (a-1)}{2a} = \frac{2a}{2a} = 1$$
  由于 $a > 1$，$\frac{1}{a} < 1$，所以 $x_1 = \frac{1}{a}$，$x_2 = 1$。

- 若 $0 < a < 1$，则 $|a-1| = 1-a$：
  $$x_1 = \frac{a+1 - (1-a)}{2a} = \frac{2a}{2a} = 1, \quad x_2 = \frac{a+1 + (1-a)}{2a} = \frac{2}{2a} = \frac{1}{a}$$
  由于 $0 < a < 1$，$\frac{1}{a} > 1$，所以 $x_1 = 1$，$x_2 = \frac{1}{a}$。

无论哪种情况，两根都是 $1$ 和 $\frac{1}{a}$，只是大小关系不同。$a > 0$ 开口向上，$> 0$ 取"两边"。

- 当 $a > 1$：$x_1 = \frac{1}{a} < x_2 = 1$，解集为 $(-\infty, \frac{1}{a}) \cup (1, +\infty)$。
- 当 $0 < a < 1$：$x_1 = 1 < x_2 = \frac{1}{a}$，解集为 $(-\infty, 1) \cup (\frac{1}{a}, +\infty)$。

**子情况 2.3**：$a < 0$。

$\Delta > 0$（因为 $a \neq 1$），两根仍为 $1$ 和 $\frac{1}{a}$。此时 $a < 0$，所以 $\frac{1}{a} < 0$。

由于 $a < 0$，开口向下。不等式两边同乘 $-1$（注意 $a$ 本身是负的，但规范做法是直接处理 $a < 0$ 的开口向下）：

不等式 $ax^2 - (a+1)x + 1 > 0$，$a < 0$ 开口向下，$> 0$ 取两根之间。

两根为 $\frac{1}{a}$（负）和 $1$（正），$\frac{1}{a} < 1$。

解集：$\frac{1}{a} < x < 1$，即 $(\frac{1}{a}, 1)$。

**综合**：

| $a$ 的范围 | 解集 |
|:---:|------|
| $a < 0$ | $(\frac{1}{a}, 1)$ |
| $a = 0$ | $(-\infty, 1)$ |
| $0 < a < 1$ | $(-\infty, 1) \cup (\frac{1}{a}, +\infty)$ |
| $a = 1$ | $(-\infty, 1) \cup (1, +\infty)$ |
| $a > 1$ | $(-\infty, \frac{1}{a}) \cup (1, +\infty)$ |

> 🧠 **反思**：这道题展示了含参讨论的完整思路——先看二次项系数（$a = 0$？），再看判别式（$\Delta = 0$？），最后比两根大小。每一步都画出草图辅助判断，就不会在开口方向和取"两边"还是"中间"上犯错。

## 09.8 常见错误辨析

| 错误 | 错在哪里？ | 正确做法 |
|------|-----------|---------|
| ❌ 不画图直接写解集，$a < 0$ 时套用"大于取两边" | 口诀前提是 $a > 0$，开口向下时口诀不适用 | 先化 $a > 0$ 再套口诀，或画图根据开口方向判断 |
| ❌ 解 $-x^2 + 4 > 0$ 得 $x < -2$ 或 $x > 2$ | 没有先化 $a > 0$，直接套口诀弄反了 | 化为 $x^2 - 4 < 0$，得 $-2 < x < 2$ |
| ❌ $\Delta = 0$ 时，$> 0$ 写成全体实数 | 忘记了 $x = x_0$ 时 $y = 0$，不满足 $>0$ | $>0$ 的解集是 $x \neq x_0$，$\geq 0$ 才是全体实数 |
| ❌ $\Delta < 0$ 时，$<0$ 写成全体实数 | 开口向上且 $\Delta < 0$ 时，抛物线永远在 $x$ 轴上方，$y$ 永远不小于 0 | $<0$ 无解，$>0$ 才是全体实数 |
| ❌ 含参不等式直接求 $\Delta$，忘了讨论 $a = 0$ | $a = 0$ 时不是二次不等式，求根公式不能用 | 第一件事就是判断 $a$ 是否为 0 |
| ❌ 解集写成 $\{x \mid x < 1 \text{ 或 } x > 2\}$ 时漏了"或" | 写成 $\{x \mid x < 1, x > 2\}$ 用逗号，意思变成"同时满足两个条件"——不存在这样的 $x$ | 两个区间之间必须用"或"（$\cup$）连接 |

## 09.9 练习题

### 基础题

**第 1 题**：解下列一元二次不等式。

(1) $x^2 - 5x + 6 > 0$
(2) $x^2 - 2x - 8 \leq 0$
(3) $3x^2 - 7x + 2 < 0$

**第 2 题**：解下列不等式。

(1) $-x^2 + 4x - 3 > 0$
(2) $x^2 + 6x + 9 \geq 0$
(3) $x^2 + x + 1 < 0$

### 提高题

**第 3 题**：解不等式 $x^2 - 2ax + a^2 - 1 \leq 0$（$a$ 为实数）。

**第 4 题**：已知不等式 $x^2 - 2x + k > 0$ 的解集为 $\mathbb{R}$，求 $k$ 的取值范围。

**第 5 题**：某商品每件成本为 $80$ 元，售价为 $x$ 元时，每天销量为 $(200 - x)$ 件。要使每天利润不低于 $2000$ 元，售价 $x$ 应定在什么范围？

---

### 练习题答案与解析

**第 1 题**：

(1) $x^2 - 5x + 6 > 0$

$\Delta = 25 - 24 = 1 > 0$，两根为 $x_1 = 2$，$x_2 = 3$。$a = 1 > 0$，$> 0$ 取两边。

解集：$(-\infty, 2) \cup (3, +\infty)$

(2) $x^2 - 2x - 8 \leq 0$

$\Delta = 4 + 32 = 36 > 0$，两根为 $x = \frac{2 \pm 6}{2}$，即 $x_1 = -2$，$x_2 = 4$。$a = 1 > 0$，$\leq 0$ 取中间（含端点）。

解集：$[-2, 4]$

(3) $3x^2 - 7x + 2 < 0$

$\Delta = 49 - 24 = 25 > 0$，两根为 $x = \frac{7 \pm 5}{6}$，即 $x_1 = \frac{1}{3}$，$x_2 = 2$。$a = 3 > 0$，$< 0$ 取中间。

解集：$\displaystyle \left(\frac{1}{3}, 2\right)$

---

**第 2 题**：

(1) $-x^2 + 4x - 3 > 0$

先化为 $a > 0$：两边乘 $-1$ → $x^2 - 4x + 3 < 0$。

$\Delta = 16 - 12 = 4 > 0$，两根为 $x_1 = 1$，$x_2 = 3$。$a = 1 > 0$，$< 0$ 取中间。

解集：$(1, 3)$

(2) $x^2 + 6x + 9 \geq 0$

$\Delta = 36 - 36 = 0$，重根 $x = -3$。$a = 1 > 0$，开口向上，$\geq 0$ 对全体实数成立。

解集：$\mathbb{R}$

(3) $x^2 + x + 1 < 0$

$\Delta = 1 - 4 = -3 < 0$。$a = 1 > 0$，开口向上，抛物线全部在 $x$ 轴上方，$y$ 永远不小于 0。

解集：$\varnothing$（无解）

---

**第 3 题**：$x^2 - 2ax + a^2 - 1 \leq 0$

先配方：$x^2 - 2ax + a^2 - 1 = (x - a)^2 - 1 \leq 0$

即 $(x - a)^2 \leq 1$，两边开方得 $-1 \leq x - a \leq 1$，即 $a - 1 \leq x \leq a + 1$。

（也可以用 $\Delta$ 法：$\Delta = 4a^2 - 4(a^2-1) = 4 > 0$，两根为 $x = a \pm 1$，$a > 0$ 取中间。）

解集：$[a-1, a+1]$

---

**第 4 题**：$x^2 - 2x + k > 0$ 的解集为 $\mathbb{R}$。

$a = 1 > 0$，开口向上。要使 $> 0$ 恒成立，需要 $\Delta < 0$（抛物线全部在 $x$ 轴上方）。

$\Delta = (-2)^2 - 4 \times 1 \times k = 4 - 4k < 0$ → $4k > 4$ → $k > 1$。

答案：$k > 1$

---

**第 5 题**（应用题）：

每天利润 = 每件利润 × 销量 = $(x - 80)(200 - x)$。

要求：$(x - 80)(200 - x) \geq 2000$

展开：$(x - 80)(200 - x) = 200x - x^2 - 16000 + 80x = -x^2 + 280x - 16000$

不等式：$-x^2 + 280x - 16000 \geq 2000$

移项：$-x^2 + 280x - 18000 \geq 0$

两边乘 $-1$：$x^2 - 280x + 18000 \leq 0$

$\Delta = 280^2 - 4 \times 18000 = 78400 - 72000 = 6400 > 0$，$\sqrt{\Delta} = 80$。

两根：$x = \frac{280 \pm 80}{2}$，即 $x_1 = 100$，$x_2 = 180$。

$a = 1 > 0$，$\leq 0$ 取中间：$100 \leq x \leq 180$。

同时，售价应大于成本（$x > 80$）且销量为正（$200 - x > 0$，即 $x < 200$）。$100 \leq x \leq 180$ 满足这两个条件。

答案：售价应定在 $[100, 180]$ 元之间。

|--------|---------|-----------|
| 三个二次关系 | 函数、方程、不等式是同一个式子的三种问法 | 用一张抛物线图统一理解三者 |
