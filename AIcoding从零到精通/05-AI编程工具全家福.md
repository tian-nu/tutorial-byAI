## 四大类AI编程工具

AI编程工具可以分为四类，就像交通工具有自行车、汽车、高铁、飞机——各有各的用途。

```
┌─────────────────────────────────────────────────────────┐
│  AI编程工具四大类                                        │
│                                                         │
│  1. 代码补全类    → 在你打字时自动补全                     │
│     GitHub Copilot、Tabnine、Codeium、Supermaven         │
│                                                         │
│  2. 对话类        → 在网页上和AI聊天，让它帮你写代码        │
│     ChatGPT网页版、Claude网页版、DeepSeek网页版            │
│                                                         │
│  3. IDE集成类     → AI直接嵌入编辑器，一站式AI编程          │
│     Cursor、Trae、Windsurf、Cline、GitHub Copilot Chat   │
│                                                         │
│  4. 终端类        → 在命令行里用AI                         │
│     Warp、Fig、Shell-GPT                                 │
└─────────────────────────────────────────────────────────┘
```

## 第一类：代码补全类 — "你打字，AI猜你下一步"

这类工具像你手机上的**输入法联想**，但比输入法强100倍。

### GitHub Copilot（📖 术语）

**它是什么**：GitHub（全球最大的代码托管平台）和OpenAI合作推出的AI编程助手。2021年发布，是AI编程工具的"鼻祖"。

**怎么用**：作为VS Code / JetBrains等编辑器的插件安装。你写代码，它实时弹出灰色建议，Tab键接受。

**优点**：最老牌、最成熟、社区最大、支持最多编辑器
**缺点**：只有补全功能（没有对话、没有Agent），月费$10
**适合谁**：已经在用VS Code/JetBrains、只需要代码补全的人

### Tabnine

**它是什么**：比Copilot更早的AI代码补全工具（2018年就有了）。特点是轻量、快速，有些版本支持本地运行（代码不上传到云端）。

**优点**：速度快、支持本地模型（隐私好）
**缺点**：补全质量不如Copilot和Cursor
**适合谁**：对隐私要求极高、不想代码上传云端的人

### Codeium

**它是什么**：后起之秀，主打"免费"的代码补全工具。功能和Copilot类似，但个人使用完全免费。

**优点**：免费、支持多编辑器、补全质量不错
**缺点**：功能不如Cursor/Trae全面（只有补全，没有Agent）
**适合谁**：不想花钱、只需要基础补全的人

---

## 第二类：对话类 — "在网页上问AI，复制代码到编辑器"

这类工具最原始但也最灵活——你打开网页，和AI聊天，AI帮你写代码，你复制粘贴到编辑器。

### ChatGPT网页版

**它是什么**：OpenAI的聊天机器人网页版。输入问题，AI回答。免费版用GPT-4o mini，付费版（$20/月）用GPT-4o。

**优点**：最知名的AI、功能全面、能上传文件、能联网搜索
**缺点**：需要手动复制粘贴代码、不能直接操作你的项目文件
**适合谁**：偶尔用AI写代码、还没决定用哪个IDE工具的人

### Claude网页版

**它是什么**：Anthropic的聊天机器人网页版。特长是长文本处理、代码风格规范、回答更谨慎。

**优点**：超长上下文（能一次处理整本书）、代码风格好、安全性高
**缺点**：同样需要手动复制粘贴、免费版有使用次数限制
**适合谁**：需要处理长代码文件、重视代码质量的人

### DeepSeek网页版

**它是什么**：中国深度求索公司的AI聊天机器人。特点是免费、中文友好、代码能力不错。

**优点**：完全免费、中文理解能力强、代码能力出色
**缺点**：功能不如ChatGPT/Claude丰富
**适合谁**：国内用户、不想付费、日常使用

## 第三类：IDE集成类 — "AI就在你的编辑器里"（⭐本书重点）

**IDE（📖 术语：Integrated Development Environment，集成开发环境）** = 把写代码、运行、调试、版本管理等功能整合在一起的软件。你可以把它理解成程序员的"工作台"。

IDE集成类AI工具，就是把AI深度嵌入到你的"工作台"里——你不需要在编辑器和网页之间来回切换，AI就在你手边。

### Cursor（⭐本书主力工具一）

**它是什么**：基于VS Code的AI IDE。保留了VS Code的所有功能和快捷键，但把AI深度嵌入到了编辑器的每一个角落。

**核心功能**：
- **Tab补全**：实时预测，比Copilot更快更准
- **Chat（Ctrl+L）**：和AI对话，问问题、改代码、解释代码
- **Composer（Ctrl+I）**：批量生成多个文件
- **Agent**：AI自动找文件、改代码、运行终端——"自动驾驶"模式

**优点**：功能最全面、AI集成最深、社区活跃、更新频繁
**缺点**：免费版有使用次数限制、Pro版$20/月
**适合谁**：**所有想学AI编程的人**（本书重点推荐）

### Trae（⭐本书主力工具二）

**它是什么**：字节跳动出品的AI IDE。和Cursor定位相似，但Builder模式是它的杀手锏。

**核心功能**：
- **Builder模式**：一句话说清楚你要做什么项目，Trae直接生成完整项目骨架
- **Chat模式**：和AI对话，改代码、问问题
- **项目感知**：默认就能看到你的所有文件，理解项目结构

**优点**：国内使用顺畅（不需要特殊网络环境）、Builder模式新建项目极快、免费
**缺点**：功能不如Cursor全面、社区和生态还在建设中
**适合谁**：**国内用户**、喜欢从零快速搭建项目的人（本书重点推荐）

### Windsurf

**它是什么**：Codeium公司推出的AI IDE，和Cursor直接竞争。特点是AI流式编辑体验（像"AI在帮你打字"）。

**优点**：免费版额度慷慨、流式编辑体验独特
**缺点**：功能不如Cursor成熟、Agent能力较弱
**适合谁**：想尝试不同AI IDE体验的人

### Cline

**它是什么**：VS Code插件形式的AI编程助手，特点是**完全开源**、支持自己配置API Key、可以接入任何模型。

**优点**：开源免费、支持任何模型（OpenAI/Anthropic/DeepSeek/本地模型）、高度可定制
**缺点**：需要自己配置API Key、对新手不够友好
**适合谁**：喜欢折腾、想用自己API Key、不想被特定工具绑定的人

## 第四类：终端类 — "在命令行里用AI"

如果你经常在终端里敲命令（`ls`、`cd`、`git`……），这类工具可以让AI帮你写命令、解释报错。

### Warp

**它是什么**：一个"AI增强版"的终端。你输入自然语言，AI帮你转成命令；报错了，AI帮你分析原因。

**优点**：现代UI、AI集成好、团队协作功能
**缺点**：目前只支持Mac和Linux（Windows版还在开发）
**适合谁**：Mac/Linux用户、经常用终端的人

### Fig

**它是什么**：终端自动补全工具（已被AWS收购）。在终端里打字时，自动弹出命令建议。

**优点**：轻量、补全好用
**缺点**：被AWS收购后发展方向不明
**适合谁**：想要终端自动补全的人

## 为什么本书只聚焦Cursor + Trae

看完以上全家福，你可能会想："工具这么多，我是不是该每个都试试？"

**答案是：不需要。** 原因很简单：

1. **选择困难症是效率杀手**：花10小时研究工具，不如花1小时装好Cursor然后开始写代码。工具是手段，写代码是目的。

2. **Cursor和Trae是当前最好的两个**：Cursor功能最全面，Trae对国内用户最友好。两者结合，覆盖了95%的AI编程场景。

3. **先精通一个，再尝试别的**：等你把Cursor和Trae用熟了，再回头看其他工具，你会发现它们的核心功能都差不多。你只是换了个"皮肤"，核心能力（Prompt Engineering）是一样的。

4. **其他工具在本书中不会消失**：当你需要用到某个特定功能时（比如需要超长上下文处理，Claude网页版更有优势），本书会提醒你。
