## 我在做什么？

之前我们学的都是线性结构——所有元素排成一条线，每个元素最多一个前驱一个后继。现在我们进入**非线性结构**的世界：**树**。

想象一个家谱：你有爸爸，爸爸有爷爷，爸爸还能有几个孩子（你和你的兄弟姐妹）。每个人都只有一个爸爸，但可以有多个孩子。这就是一棵树。

最常见的树是**二叉树**——每个节点最多只有两个孩子：左孩子和右孩子。

`*此术语见附录F*`：二叉树、根节点、叶子节点、深度。

这一章我们会：
1. 理解树的基本概念（根、叶子、深度、高度）
2. 学习**二叉搜索树（BST）**——左小右大，快速查找
3. 实现四种遍历：前序、中序、后序、层序
4. 完整实现BST的插入、查找、删除

```
一个简单的二叉树：
        根 →   50
               /  \
左孩子 ←    30   70 → 右孩子
             /  \    \
           20   40   80
           ↑
        叶子节点（没有孩子）
        深度：根在第0层（或第1层，看定义），叶子在第2层
```

## 47.1 树的基本概念

| 概念 | 含义 |
|------|------|
| **节点** | 存储数据的单元，包含指向孩子的指针 |
| **根节点** | 最顶层的节点，没有父节点 |
| **父节点** | 如果节点 A 有子节点 B，那么 A 是 B 的父节点 |
| **子节点** | B 是 A 的子节点 |
| **叶子节点** | 没有子节点的节点 |
| **兄弟节点** | 同一个父节点的多个节点互称兄弟 |
| **深度** | 从根到该节点的路径长度（边数） |
| **高度** | 从该节点到最远叶子的路径长度 |
| **二叉树** | 每个节点最多两个子节点（左、右） |

### 二叉树的几种特殊形态

```
1. 满二叉树：所有叶子都在同一层，每个非叶子都有两个孩子
         1
       /   \
      2     3
     / \   / \
    4  5  6  7

2. 完全二叉树：除了最后一层，其他层都填满，最后一层靠左排列
         1
       /   \
      2     3
     / \   /
    4  5  6

3. 斜树：所有节点都只有左孩子或都只有右孩子（退化成链表）
         1
        /
       2
      /
     3
    /
   4
```

## 47.2 二叉搜索树（BST）的定义

**二叉搜索树（Binary Search Tree, BST）** 是二叉树的一种，满足：
- 对于任意节点：
  - 左子树中**所有**节点的值都小于该节点的值
  - 右子树中**所有**节点的值都大于该节点的值

性质：**中序遍历 BST 得到的是升序序列**。这是判断是否BST的重要依据。

```
一个合法的BST：
        50
       /  \
      30  70
     /  \   \
    20  40   80

中序遍历：20 → 30 → 40 → 50 → 70 → 80 → 正好升序！
```

## 47.3 节点结构体定义

```c
/* BST节点 */
typedef struct TreeNode {
    int                key;    /* 关键字（用于排序和查找） */
    int                value;  /* 存储的数据 */
    struct TreeNode   *left;   /* 左孩子 */
    struct TreeNode   *right;  /* 右孩子 */

/* BST结构体（封装根指针） */
typedef struct BST {
```

创建新节点：

```c
TreeNode* bst_create_node(int key, int value) {

    node->key = key;
    node->value = value;
    node->left = NULL;
    node->right = NULL;
    return node;
}

/* 创建空BST */
BST* bst_create(void) {
    BST *tree = (BST*)malloc(sizeof(BST));
    if (tree == NULL) return NULL;
    tree->root = NULL;
    tree->size = 0;
    return tree;
}
```

## 47.4 查找——二分思想

查找关键字 `key` 在哪里：
1. 从根开始比较
2. 如果 `key == 当前节点.key` → 找到了
3. 如果 `key < 当前节点.key` → 去左子树找
4. 如果 `key > 当前节点.key` → 去右子树找
5. 如果走到 NULL 都没找到 → 不存在

```c
/* bst_search: 在树中查找关键字key，返回节点指针 */
TreeNode* bst_search(BST *tree, int key) {
    TreeNode *current = tree->root;
    while (current != NULL) {
        if (key == current->key) {
            return current;  /* 找到了 */
        } else if (key < current->key) {
            current = current->left;  /* 去左边 */
        } else {
            current = current->right; /* 去右边 */
        }
    }
    return NULL;  /* 没找到 */
}
```

递归版本更简洁：

```c
TreeNode* bst_search_recursive(TreeNode *node, int key) {
    if (node == NULL || node->key == key) {
        return node;
    }
    if (key < node->key) {
        return bst_search_recursive(node->left, key);
    } else {
        return bst_search_recursive(node->right, key);
    }
}
```

平均时间复杂度：O(log n)，最坏（斜树）O(n)。

## 47.5 插入——保持BST性质

插入新节点：
1. 如果树为空，新节点就是根
2. 否则，从根开始找，找到合适的叶子位置插入
3. `key < 当前节点` 往左走，`key > 当前节点` 往右走
4. 走到空位置，就把新节点挂在这里

注意：BST一般不允许有重复关键字（如果允许可以额外计数）。

```c
/* bst_insert: 插入(key, value)，已存在则更新value */
int bst_insert(BST *tree, int key, int value) {
    if (tree == NULL) return -1;

    /* 空树，直接作为根 */
    if (tree->root == NULL) {
        tree->root = bst_create_node(key, value);
        if (tree->root == NULL) return -1;
        tree->size = 1;
        return 0;
    }

    /* 找插入位置 */
    TreeNode *parent = NULL;
    TreeNode *current = tree->root;
    while (current != NULL) {
        parent = current;
        if (key == current->key) {
            /* 已存在，更新值 */
            current->value = value;
            return 0;
        } else if (key < current->key) {
            current = current->left;
        } else {
            current = current->right;
        }
    }

    /* current 是 NULL，parent 是父节点，插入 */
    TreeNode *new_node = bst_create_node(key, value);
    if (new_node == NULL) return -1;
    if (key < parent->key) {
        parent->left = new_node;
    } else {
        parent->right = new_node;
    }
    tree->size++;
    return 0;
}
```

## 47.6 四种遍历方式

遍历就是按某种顺序访问所有节点一次。常见四种方式：

### 1. 前序遍历（根 → 左 → 右）

先访问根，再遍历左子树，再遍历右子树。

```c
void bst_preorder(TreeNode *node) {
    if (node == NULL) return;
    printf("%d ", node->key);   /* 先访问根 */
    bst_preorder(node->left);    /* 再左子树 */
    bst_preorder(node->right);   /* 再右子树 */
}
```

### 2. 中序遍历（左 → 根 → 右）

先遍历左子树，再访问根，再遍历右子树。**对BST来说，中序遍历就是升序输出**！

```c
void bst_inorder(TreeNode *node) {
    if (node == NULL) return;
    bst_inorder(node->left);     /* 先左子树 */
    printf("%d ", node->key);   /* 再访问根 */
    bst_inorder(node->right);    /* 再右子树 */
}
```

### 3. 后序遍历（左 → 右 → 根）

先遍历左子树，再遍历右子树，最后访问根。删除节点时常用（先删孩子再删自己）。

```c
void bst_postorder(TreeNode *node) {
    if (node == NULL) return;
    bst_postorder(node->left);   /* 先左 */
    bst_postorder(node->right);  /* 再右 */
    printf("%d ", node->key);   /* 最后根 */
}
```

### 4. 层序遍历（广度优先）

按层从上到下，每层从左到右。需要队列辅助。

```c
/* 层序遍历需要队列，我们用链表实现简单队列 */
typedef struct QueueNode {
    TreeNode *tree_node;
    struct QueueNode *next;
} QueueNode;

typedef struct Queue {
    QueueNode *front;
    QueueNode *rear;
} Queue;

void enqueue(Queue *q, TreeNode *node) {
    QueueNode *qn = (QueueNode*)malloc(sizeof(QueueNode));
    qn->tree_node = node;
    qn->next = NULL;
    if (q->rear == NULL) {
        q->front = q->rear = qn;
    } else {
        q->rear->next = qn;
        q->rear = qn;
    }
}

TreeNode* dequeue(Queue *q) {
    if (q->front == NULL) return NULL;
    QueueNode *qn = q->front;
    TreeNode *node = qn->tree_node;
    q->front = qn->next;
    if (q->front == NULL) {
        q->rear = NULL;
    }
    free(qn);
    return node;
}

int queue_is_empty(Queue *q) {
    return q->front == NULL;
}

void bst_level_order(BST *tree) {
    if (tree->root == NULL) return;
    Queue q = {NULL, NULL};
    enqueue(&q, tree->root);
    while (!queue_is_empty(&q)) {
        TreeNode *node = dequeue(&q);
        printf("%d ", node->key);
        if (node->left != NULL) enqueue(&q, node->left);
        if (node->right != NULL) enqueue(&q, node->right);
    }
}
```

## 47.7 删除——最复杂的操作

删除分三种情况：

**情况1：删除叶子节点**（左右都为空）
- 直接删掉，把父节点对应的指针设为 NULL。

**情况2：删除只有一个孩子的节点**
- 让父节点直接指向孩子，跳过当前节点，然后删掉当前节点。

**情况3：删除有两个孩子的节点**
- 找**前驱**（左子树中最大的节点）或者**后继**（右子树中最小的节点）
- 把前驱/后继的数据复制到当前节点
- 然后删除前驱/后继（它最多只有一个孩子，一定会落入情况1或2）

这里我们用**后继**法：

```c
/* 找最小节点（一直往左走） */
TreeNode* bst_find_min(TreeNode *node) {
    while (node->left != NULL) {
        node = node->left;
    }
    return node;
}

/* 找最大节点（一直往右走） */
TreeNode* bst_find_max(TreeNode *node) {
    while (node->right != NULL) {
        node = node->right;
    }
    return node;
}

/* bst_delete_node: 删除以node为根的子树中key所在节点
 * 返回删除后的新根（因为可能删除根，所以要返回）
 */
TreeNode* bst_delete_node(BST *tree, TreeNode *node, int key) {
    if (node == NULL) return NULL;

    if (key < node->key) {
        /* 在左子树删 */
        node->left = bst_delete_node(tree, node->left, key);
        return node;
    } else if (key > node->key) {
        /* 在右子树删 */
        node->right = bst_delete_node(tree, node->right, key);
        return node;
    } else {
        /* 找到了，就是这个节点，分三种情况 */
        tree->size--;

        /* 情况1：没有孩子 或 只有一个孩子 */
        if (node->left == NULL) {
            TreeNode *right = node->right;
            free(node);
            return right;  /* 返回给父节点，接上 */
        }
        if (node->right == NULL) {
            TreeNode *left = node->left;
            free(node);
            return left;   /* 返回给父节点，接上 */
        }

        /* 情况3：两个孩子都有，找后继（右子树最小） */
        TreeNode *successor = bst_find_min(node->right);
        /* 复制数据 */
        node->key = successor->key;
        node->value = successor->value;
        /* 删除后继（在右子树中删） */
        node->right = bst_delete_node(tree, node->right, successor->key);
        return node;
    }
}

/* 对外接口 */
int bst_delete(BST *tree, int key) {
    if (tree == NULL || tree->root == NULL) return -1;
    TreeNode *new_root = bst_delete_node(tree, tree->root, key);
    tree->root = new_root;
    return 0;
}
```

## 47.8 销毁整个树

后序遍历，先删左右孩子再删自己：

```c
void bst_destroy_node(TreeNode *node) {
    if (node == NULL) return;
    bst_destroy_node(node->left);
    bst_destroy_node(node->right);
    free(node);
}

    if (tree == NULL) return;
    bst_destroy_node(tree->root);
    free(tree);
```

## 47.9 完整代码清单

```c
/*
 * bst.c —— 二叉搜索树完整实现
 * 编译: gcc -std=c99 -Wall -o bst bst.c
 */

#include <stdio.h>
#include <stdlib.h>

/* ==================== BST节点和树结构 ==================== */
typedef struct TreeNode {
    int                key;
    int                value;
    struct TreeNode   *left;
    struct TreeNode   *right;
} TreeNode;

typedef struct BST {
    TreeNode *root;
    int      size;
} BST;

/* ==================== 创建 ==================== */
TreeNode* bst_create_node(int key, int value) {
    TreeNode *node = (TreeNode*)malloc(sizeof(TreeNode));
    if (node == NULL) return NULL;
    node->key = key;
    node->value = value;
    node->left = NULL;
    node->right = NULL;
    return node;
}

BST* bst_create(void) {
    BST *tree = (BST*)malloc(sizeof(BST));
    if (tree == NULL) return NULL;
    tree->root = NULL;
    tree->size = 0;
    return tree;
}

/* ==================== 查找 ==================== */
TreeNode* bst_search(BST *tree, int key) {
    TreeNode *current = tree->root;
    while (current != NULL) {
        if (key == current->key) return current;
        else if (key < current->key) current = current->left;
        else current = current->right;
    }
    return NULL;
}

/* ==================== 插入 ==================== */
int bst_insert(BST *tree, int key, int value) {
    if (tree == NULL) return -1;
    if (tree->root == NULL) {
        tree->root = bst_create_node(key, value);
        if (tree->root == NULL) return -1;
        tree->size = 1;
        return 0;
    }

    TreeNode *parent = NULL;
    TreeNode *current = tree->root;
    while (current != NULL) {
        parent = current;
        if (key == current->key) {
            current->value = value;
            return 0;
        } else if (key < current->key) {
            current = current->left;
        } else {
            current = current->right;
        }
    }

    TreeNode *new_node = bst_create_node(key, value);
    if (new_node == NULL) return -1;
    if (key < parent->key) {
        parent->left = new_node;
    } else {
        parent->right = new_node;
    }
    tree->size++;
    return 0;
}

/* ==================== 工具：找最小最大 ==================== */
TreeNode* bst_find_min(TreeNode *node) {
    while (node->left != NULL) node = node->left;
    return node;
}

TreeNode* bst_find_max(TreeNode *node) {
    while (node->right != NULL) node = node->right;
    return node;
}

/* ==================== 删除 ==================== */
TreeNode* bst_delete_node(BST *tree, TreeNode *node, int key) {
    if (node == NULL) return NULL;

    if (key < node->key) {
        node->left = bst_delete_node(tree, node->left, key);
        return node;
    } else if (key > node->key) {
        node->right = bst_delete_node(tree, node->right, key);
        return node;
    } else {
        tree->size--;
        /* 情况1: 0或1个孩子 */
        if (node->left == NULL) {
            TreeNode *right = node->right;
            free(node);
            return right;
        }
        if (node->right == NULL) {
            TreeNode *left = node->left;
            free(node);
            return left;
        }
        /* 情况3: 两个孩子，找后继 */
        TreeNode *succ = bst_find_min(node->right);
        node->key = succ->key;
        node->value = succ->value;
        node->right = bst_delete_node(tree, node->right, succ->key);
        return node;
    }
}

int bst_delete(BST *tree, int key) {
    if (tree == NULL || tree->root == NULL) return -1;
    tree->root = bst_delete_node(tree, tree->root, key);
    return 0;
}

/* ==================== 遍历 ==================== */
void bst_preorder(TreeNode *node) {
    if (node == NULL) return;
    printf("%d ", node->key);
    bst_preorder(node->left);
    bst_preorder(node->right);
}

void bst_inorder(TreeNode *node) {
    if (node == NULL) return;
    bst_inorder(node->left);
    printf("%d ", node->key);
    bst_inorder(node->right);
}

void bst_postorder(TreeNode *node) {
    if (node == NULL) return;
    bst_postorder(node->left);
    bst_postorder(node->right);
    printf("%d ", node->key);
}

/* 层序遍历需要队列 */
typedef struct QueueNode {
    TreeNode *tree_node;
    struct QueueNode *next;
} QueueNode;

typedef struct Queue {
    QueueNode *front;
    QueueNode *rear;
} Queue;

void enqueue(Queue *q, TreeNode *node) {
    QueueNode *qn = (QueueNode*)malloc(sizeof(QueueNode));
    qn->tree_node = node;
    qn->next = NULL;
    if (q->rear == NULL) {
        q->front = q->rear = qn;
    } else {
        q->rear->next = qn;
        q->rear = qn;
    }
}

TreeNode* dequeue(Queue *q) {
    if (q->front == NULL) return NULL;
    QueueNode *qn = q->front;
    TreeNode *node = qn->tree_node;
    q->front = qn->next;
    if (q->front == NULL) q->rear = NULL;
    free(qn);
    return node;
}

int queue_is_empty(Queue *q) {
    return q->front == NULL;
}

void bst_level_order(BST *tree) {
    if (tree->root == NULL) return;
    Queue q = {NULL, NULL};
    enqueue(&q, tree->root);
    while (!queue_is_empty(&q)) {
        TreeNode *node = dequeue(&q);
        printf("%d ", node->key);
        if (node->left != NULL) enqueue(&q, node->left);
        if (node->right != NULL) enqueue(&q, node->right);
    }
}

/* ==================== 销毁 ==================== */
void bst_destroy_node(TreeNode *node) {
    if (node == NULL) return;
    bst_destroy_node(node->left);
    bst_destroy_node(node->right);
    free(node);
}

void bst_destroy(BST *tree) {
    if (tree == NULL) return;
    bst_destroy_node(tree->root);
    free(tree);
}

/* ==================== 演示 ==================== */
int main(void) {
    BST *tree = bst_create();

    printf("===== 插入测试 =====\n");
    int keys[] = {50, 30, 70, 20, 40, 60, 80};
    int n = sizeof(keys) / sizeof(keys[0]);
    for (int i = 0; i < n; i++) {
        bst_insert(tree, keys[i], keys[i] * 10);
    }
    printf("大小: %d\n", tree->size);

    printf("\n===== 遍历 =====\n");
    printf("前序: "); bst_preorder(tree->root); printf("\n");
    printf("中序: "); bst_inorder(tree->root); printf("\n");
    printf("后序: "); bst_postorder(tree->root); printf("\n");
    printf("层序: "); bst_level_order(tree); printf("\n");

    printf("\n===== 查找 =====\n");
    TreeNode *found = bst_search(tree, 40);
    printf("查找40: %s, value=%d\n", found ? "找到" : "未找到", found ? found->value : -1);
    found = bst_search(tree, 99);
    printf("查找99: %s\n", found ? "找到" : "未找到");

    printf("\n===== 删除测试 =====\n");
    printf("删除叶子节点 20:\n");
    bst_delete(tree, 20);
    printf("中序: "); bst_inorder(tree->root); printf("\n");

    printf("删除有一个孩子 30:\n");
    bst_delete(tree, 30);
    printf("中序: "); bst_inorder(tree->root); printf("\n");

    printf("删除有两个孩子 50（根）:\n");
    bst_delete(tree, 50);
    printf("中序: "); bst_inorder(tree->root); printf("\n");
    printf("大小: %d\n", tree->size);

    bst_destroy(tree);
    return 0;
}
```

## 我做得对不对？

运行完整代码，观察输出：
1. **插入后中序**应该是 `20 30 40 50 60 70 80`（升序）。
2. **删除叶子20**后中序应该是 `30 40 50 60 70 80`。
3. **删除30**后中序应该是 `40 50 60 70 80`。
4. **删除根50**后中序应该是 `40 60 70 80`。
5. 最终大小应该是 4。

## 不对怎么办？

### 常见Bug 1：插入时忘记增加 `tree->size`

```c
/* ❌ 插入成功但 size 没变，导致计数错误 */
if (key < parent->key) {
    parent->left = new_node;
} else {
    parent->right = new_node;
}
/* 忘记写 tree->size++; */
```

✅ 插入成功后一定要 `tree->size++`。

### 常见Bug 2：删除两个孩子时，错误删除了当前节点而不是后继

```c
/* ❌ 复制后直接删除当前节点，指针关系错了 */
node->key = succ->key;
node->value = succ->value;
free(succ);  /* 不行！succ 还挂在树上，不能直接 free，要递归删除 */
```

✅ 应该：`node->right = bst_delete_node(tree, node->right, succ->key);`

### 常见Bug 3：递归删除忘记返回新节点

```c
/* ❌ 删除后父节点指针不更新 */
if (key < node->key) {
    bst_delete_node(node->left, key);
    /* 忘记写 node->left = ... */
}
```

✅ `bst_delete_node` 返回删除后的新子树根，必须赋值回 `node->left` 或 `node->right`。

### 常见Bug 4：销毁树时先删自己再删孩子（顺序错了）

```c
/* ❌ 先删自己，然后递归删孩子，孩子指针已经失效 */
void destroy_wrong(TreeNode *node) {
    free(node);
    destroy_wrong(node->left);
    destroy_wrong(node->right);
}
```

✅ 后序：先递归删左右，再删自己。

## 本章小结

| 要点 | 说明 |
|------|------|
| 二叉搜索树性质 | 左子树所有关键字 < 当前节点 < 右子树所有关键字 |
| 查找 | 从根开始二分比较，平均 O(log n) |
| 插入 | 找叶子位置挂上去，O(log n) |
| 删除分三种情况 | 叶子直接删；单孩子让父接孩子；双孩子用后继替换后删后继 |
| 四种遍历 | 前序(根左右)、中序(左根右)、后序(左右根)、层序(广度优先) |
| BST性质 | 中序遍历一定得到升序序列 |
