## 我在做什么？

第43章我们学会了建链表。但建好的链表是"死"的——你只能往里塞东西，不能拿掉、不能修改、不能找某个特定值。就像你买了一排储物柜，但只能往里放东西，想取出来却只能隔空看。

本章给链表加上四个核心能力：**增（任意位置插入）、删（移除节点并释放内存）、改（修改节点数据）、查（按值或按位置查找）**。再加上一个压轴操作——**链表反转**。

学完本章，你就能对链表做任何想做的事了。

## 44.1 查找节点——按值查找和按位置查找

### 按值查找：找到第一个匹配的节点

```
链表: [10] -> [25] -> [30] -> [25] -> [50] -> NULL
查找 25: 从头开始遍历，第2个节点匹配，返回它的指针
```

```c
/*
 * find_by_value: 查找第一个值为 target 的节点
 * @head: 链表头指针
 * @target: 要查找的值
 * @return: 找到返回节点指针，未找到返回 NULL
 */
Node* find_by_value(Node *head, int target) {
    Node *current = head;
    while (current != NULL) {
        if (current->data == target) {
            return current;  /* 找到了，返回节点指针 */
        }
        current = current->next;
    }
    return NULL;  /* 遍历完都没找到 */
}
```

### 按位置查找：返回第 n 个节点

```c
/*
 * get_node_at: 返回第 index 个节点（从0开始计数）
 * @head: 链表头指针
 * @index: 索引位置
 * @return: 找到返回节点指针，越界返回 NULL
 */
Node* get_node_at(Node *head, int index) {
    if (index < 0) return NULL;

    Node *current = head;
    int   i = 0;
    while (current != NULL && i < index) {
        current = current->next;
        i++;
    }
    return current;  /* 可能是 NULL（越界） */
}
```

## 44.2 插入——任意位置

第43章讲了头插和尾插。现在统一为**在指定位置插入**。

```
原始链表:  head -> [A] -> [B] -> [D] -> NULL
想在索引 2 处（B 和 D 之间）插入 [C]：

步骤1: 找到索引 1 的节点（即 [B]）
         prev 指向 [B]

步骤2: 让新节点 [C] 的 next 指向 [B] 的 next（即 [D]）
         [C].next = [D]

步骤3: 让 [B] 的 next 指向 [C]
         [B].next = [C]

结果:     head -> [A] -> [B] -> [C] -> [D] -> NULL
```

```c
/*
 * insert_at: 在指定索引位置插入节点
 * @head: 指向头指针的指针
 * @index: 插入位置（0 表示头部，大于等于长度表示尾部）
 * @value: 要插入的数据
 * @return: 成功返回 0，失败返回 -1
 */
int insert_at(Node **head, int index, int value) {
    if (index < 0) return -1;

    /* 插入头部（包括空链表的情况） */
    if (index == 0) {
        return insert_head(head, value);
    }

    /* 找到 index-1 位置的节点 */
    Node *prev = get_node_at(*head, index - 1);
    if (prev == NULL) {
        /* index-1 位置不存在，说明索引越界，插入尾部 */
        return insert_tail(head, value);
    }

    /* 在 prev 之后插入 */
    Node *new_node = create_node(value);
    if (new_node == NULL) return -1;

    new_node->next = prev->next;
    prev->next = new_node;
    return 0;
}
```

**插入操作的三种情况图解**：

```
情况1: 插入头部 (index == 0)
        同 insert_head，新节点成为头节点

情况2: 插入中间 (0 < index < length)
        prev 指向 index-1 的节点
        new_node->next = prev->next
        prev->next = new_node

情况3: 插入尾部 (index >= length)
        同 insert_tail，遍历到末尾再挂载
        （上面的代码中，prev 为 NULL 时走这个分支）
```

## 44.3 删除——最危险也最重要

删除操作有三个难点：
1. 找到要删除的节点**前一个节点**（因为要修改它的 next 指针）
2. 在释放前**保存好**断开的链表
3. 处理**空链表**和**删除头节点**的特殊情况

```
删除节点 [B] 的图解：

删除前:  head -> [A] -> [B] -> [C] -> NULL

步骤1: 找到 [B] 的前一个节点 [A]（记为 prev）
步骤2: 记录要删除的节点 to_delete = prev->next（即 [B]）
步骤3: 跳过 [B]：prev->next = [B].next（即 [C]）
步骤4: 释放 [B]：free(to_delete)

删除后:  head -> [A] -> [C] -> NULL
```

```c
/*
 * delete_at: 删除指定索引位置的节点
 * @head: 指向头指针的指针
 * @index: 要删除的节点索引（从0开始）
 * @return: 成功返回 0，失败返回 -1
 */
int delete_at(Node **head, int index) {
    if (*head == NULL || index < 0) {
        return -1;  /* 空链表或非法索引 */
    }

    Node *to_delete = NULL;

    /* 情况1：删除头节点 */
    if (index == 0) {
        to_delete = *head;
        *head = (*head)->next;  /* 头指针指向第二个节点 */
        free(to_delete);
        return 0;
    }

    /* 情况2：删除中间或尾部节点 */
    /* 找到被删节点的前一个节点 */
    Node *prev = get_node_at(*head, index - 1);
    if (prev == NULL || prev->next == NULL) {
        return -1;  /* 索引越界 */
    }

    to_delete = prev->next;          /* 记录要删除的节点 */
    prev->next = to_delete->next;    /* 跳过被删节点 */
    free(to_delete);                 /* 释放内存 */
    return 0;
}
```

### 按值删除：删除第一个匹配值的节点

```c
/*
 * delete_by_value: 删除第一个值为 target 的节点
 * @head: 指向头指针的指针
 * @target: 要删除的值
 * @return: 成功返回 0，未找到返回 -1
 */
int delete_by_value(Node **head, int target) {
    if (*head == NULL) return -1;

    /* 情况1：头节点就是要删除的 */
    if ((*head)->data == target) {
        Node *to_delete = *head;
        *head = (*head)->next;
        free(to_delete);
        return 0;
    }

    /* 情况2：在中间或尾部查找 */
    Node *current = *head;
    while (current->next != NULL && current->next->data != target) {
        current = current->next;
    }

    if (current->next == NULL) {
        return -1;  /* 没找到 target */
    }

    /* current->next 就是要删除的节点 */
    Node *to_delete = current->next;
    current->next = to_delete->next;
    free(to_delete);
    return 0;
}
```

## 44.4 修改节点数据

修改是最简单的操作——找到节点，直接改 `data` 字段即可。

```c
/*
 * update_at: 修改指定索引位置节点的数据
 * @head: 链表头指针
 * @index: 节点索引
 * @new_value: 新值
 * @return: 成功返回 0，失败返回 -1
 */
int update_at(Node *head, int index, int new_value) {
    Node *target = get_node_at(head, index);
    if (target == NULL) {
        return -1;
    }
    target->data = new_value;
    return 0;
}
```

## 44.5 链表反转——三个指针的舞蹈

链表反转是面试高频题，也是理解指针操作的试金石。核心思想：**逐个翻转每个节点的 next 方向**。

```
反转前:  head -> [A] -> [B] -> [C] -> NULL
                   方向：A→B→C

反转后:  NULL <- [A] <- [B] <- [C] <- head
                   方向：C→B→A
```

**迭代法使用三个指针**：

```
      prev        curr        next
      NULL        [A]         [B]      初始状态
        │          │           │
        └──────────┴───────────┘

      第1步: curr->next = prev    (A的next指向NULL)
      第2步: prev = curr          (prev前进到A)
      第3步: curr = next          (curr前进到B)
      第4步: next = next->next    (next前进到C)

      重复直到 curr == NULL，此时 prev 就是新头节点
```

```c
/*
 * reverse_list: 就地反转链表（迭代法）
 * @head: 指向头指针的指针
 *
 * 时间复杂度 O(n)，空间复杂度 O(1)
 */
void reverse_list(Node **head) {
    Node *prev = NULL;
    Node *curr = *head;
    Node *next = NULL;

    while (curr != NULL) {
        next = curr->next;    /* 保存下一个节点，防止断链 */
        curr->next = prev;    /* 反转！当前节点指向前一个 */
        prev = curr;          /* prev 前进 */
        curr = next;          /* curr 前进 */
    }
    /* 循环结束时，prev 指向原链表的最后一个节点，即新头节点 */
    *head = prev;
}
```

**反转过程慢动作演示**：

```
初始:  prev=NULL   curr=A.next=B   next=?
         NULL        A -> B -> C -> NULL

第1轮:  next = B
         A.next = NULL   →   NULL <- A    B -> C -> NULL
         prev = A, curr = B

第2轮:  next = C
         B.next = A      →   NULL <- A <- B    C -> NULL
         prev = B, curr = C

第3轮:  next = NULL
         C.next = B      →   NULL <- A <- B <- C
         prev = C, curr = NULL

结束:  *head = prev = C
结果:  C -> B -> A -> NULL
```

## 44.6 完整代码清单

```c
/*
 * linked_list_operations.c —— 单向链表增删改查 + 反转
 * 编译: gcc -std=c99 -Wall -o list_ops linked_list_operations.c
 */

#include <stdio.h>
#include <stdlib.h>

/* ==================== 节点与基础操作（同第43章） ==================== */
typedef struct Node {
    int          data;
    struct Node *next;
} Node;

Node* create_node(int value) {
    Node *n = (Node*)malloc(sizeof(Node));
    if (n == NULL) return NULL;
    n->data = value;
    n->next = NULL;
    return n;
}

int insert_head(Node **head, int value) {
    Node *n = create_node(value);
    if (n == NULL) return -1;
    n->next = *head;
    *head = n;
    return 0;
}

int insert_tail(Node **head, int value) {
    Node *n = create_node(value);
    if (n == NULL) return -1;
    if (*head == NULL) { *head = n; return 0; }
    Node *cur = *head;
    while (cur->next != NULL) cur = cur->next;
    cur->next = n;
    return 0;
}

void print_list(Node *head) {
    if (head == NULL) { printf("(空链表)\n"); return; }
    Node *cur = head;
    while (cur != NULL) {
        printf("%d", cur->data);
        if (cur->next) printf(" -> ");
        cur = cur->next;
    }
    printf(" -> NULL\n");
}

void free_list(Node **head) {
    Node *cur = *head;
    while (cur != NULL) {
        Node *next = cur->next;
        free(cur);
        cur = next;
    }
    *head = NULL;
}

/* ==================== 查找 ==================== */
Node* find_by_value(Node *head, int target) {
    Node *cur = head;
    while (cur != NULL) {
        if (cur->data == target) return cur;
        cur = cur->next;
    }
    return NULL;
}

Node* get_node_at(Node *head, int index) {
    if (index < 0) return NULL;
    Node *cur = head;
    int i = 0;
    while (cur != NULL && i < index) {
        cur = cur->next;
        i++;
    }
    return cur;
}

/* ==================== 插入（任意位置） ==================== */
int insert_at(Node **head, int index, int value) {
    if (index < 0) return -1;
    if (index == 0) return insert_head(head, value);

    Node *prev = get_node_at(*head, index - 1);
    if (prev == NULL) return insert_tail(head, value);

    Node *n = create_node(value);
    if (n == NULL) return -1;
    n->next = prev->next;
    prev->next = n;
    return 0;
}

/* ==================== 删除 ==================== */
int delete_at(Node **head, int index) {
    if (*head == NULL || index < 0) return -1;

    if (index == 0) {
        Node *to_del = *head;
        *head = (*head)->next;
        free(to_del);
        return 0;
    }

    Node *prev = get_node_at(*head, index - 1);
    if (prev == NULL || prev->next == NULL) return -1;

    Node *to_del = prev->next;
    prev->next = to_del->next;
    free(to_del);
    return 0;
}

int delete_by_value(Node **head, int target) {
    if (*head == NULL) return -1;
    if ((*head)->data == target) {
        Node *to_del = *head;
        *head = (*head)->next;
        free(to_del);
        return 0;
    }
    Node *cur = *head;
    while (cur->next != NULL && cur->next->data != target)
        cur = cur->next;
    if (cur->next == NULL) return -1;
    Node *to_del = cur->next;
    cur->next = to_del->next;
    free(to_del);
    return 0;
}

/* ==================== 修改 ==================== */
int update_at(Node *head, int index, int new_value) {
    Node *target = get_node_at(head, index);
    if (target == NULL) return -1;
    target->data = new_value;
    return 0;
}

/* ==================== 反转 ==================== */
void reverse_list(Node **head) {
    Node *prev = NULL;
    Node *curr = *head;
    Node *next = NULL;
    while (curr != NULL) {
        next = curr->next;
        curr->next = prev;
        prev = curr;
        curr = next;
    }
    *head = prev;
}

/* ==================== 演示 ==================== */
int main(void) {
    Node *list = NULL;

    printf("===== 构建链表 =====\n");
    insert_tail(&list, 10);
    insert_tail(&list, 20);
    insert_tail(&list, 30);
    insert_tail(&list, 40);
    print_list(list);  /* 10 -> 20 -> 30 -> 40 -> NULL */

    printf("\n===== 查找 =====\n");
    Node *found = find_by_value(list, 30);
    printf("查找 30: %s\n", found ? "找到了" : "没找到");
    found = find_by_value(list, 99);
    printf("查找 99: %s\n", found ? "找到了" : "没找到");
    Node *n = get_node_at(list, 2);
    printf("索引 2 的值: %d\n", n ? n->data : -1);

    printf("\n===== 插入（任意位置） =====\n");
    insert_at(&list, 0, 5);     /* 头部插入 5 */
    insert_at(&list, 3, 25);    /* 索引 3 处插入 25 */
    insert_at(&list, 99, 50);   /* 超出范围，尾部插入 50 */
    print_list(list);  /* 5 -> 10 -> 20 -> 25 -> 30 -> 40 -> 50 -> NULL */

    printf("\n===== 修改 =====\n");
    update_at(list, 2, 200);    /* 索引 2 的值改为 200 */
    print_list(list);  /* 5 -> 10 -> 200 -> 25 -> 30 -> 40 -> 50 -> NULL */

    printf("\n===== 删除 =====\n");
    delete_at(&list, 0);        /* 删除头节点 */
    print_list(list);  /* 10 -> 200 -> 25 -> 30 -> 40 -> 50 -> NULL */
    delete_at(&list, 2);        /* 删除索引 2（值 25） */
    print_list(list);  /* 10 -> 200 -> 30 -> 40 -> 50 -> NULL */
    delete_by_value(&list, 50); /* 按值删除 50 */
    print_list(list);  /* 10 -> 200 -> 30 -> 40 -> NULL */

    printf("\n===== 反转 =====\n");
    reverse_list(&list);
    print_list(list);  /* 40 -> 30 -> 200 -> 10 -> NULL */

    free_list(&list);
    return 0;
}
```

## 我做得对不对？

1. **插入**：在头部、中间、尾部各插入一个值，验证顺序正确。
2. **删除**：删除头节点、中间节点、尾节点各一次，验证链表不"断"。
3. **反转**：反转后打印，与反转前比较，顺序应完全颠倒。
4. **空链表**：所有操作在空链表上调一次，确保不崩溃。

---

## 不对怎么办？

### 常见Bug 1：删除时忘记保存 next 导致断链

```c
/* ❌ 直接 free 当前节点，链表断了 */
while (current != NULL) {
    free(current);          /* 释放了 current */
    current = current->next;  /* 访问已释放的内存！ */
}
```

✅ 先 `Node *next = current->next;` 再 `free(current);` 再 `current = next;`

### 常见Bug 2：反转时丢失头节点

```c
/* ❌ 反转后忘记更新 *head */
void reverse_list_wrong(Node **head) {
    /* ... 反转逻辑 ... */
    /* 忘记写 *head = prev; */
}
/* 调用后 head 还是指向原来的第一个节点（现在变成最后一个） */
```

✅ 反转循环结束后，必须写 `*head = prev;`

### 常见Bug 3：`delete_at` 中 `prev->next` 为 NULL 时继续删除

```c
/* ❌ 没有检查 prev->next 是否为 NULL */
Node *prev = get_node_at(*head, index - 1);
Node *to_del = prev->next;  /* 如果 prev->next 是 NULL */
prev->next = to_del->next;  /* to_del 是 NULL，to_del->next 崩溃 */
```

✅ 在获取 `to_del` 之前，检查 `prev == NULL || prev->next == NULL`

### 常见Bug 4：`insert_at` 中 `prev` 为 NULL 时崩溃

```c
/* ❌ 没有处理 prev 为 NULL 的情况 */
Node *prev = get_node_at(*head, index - 1);
Node *n = create_node(value);
n->next = prev->next;  /* prev 是 NULL，崩溃！ */
```

✅ 如果 `prev == NULL`，说明索引越界，改为尾插。

## 本章小结

| 操作 | 核心思路 | 时间复杂度 |
|------|----------|-----------|
| 按值查找 | 遍历比较 `data` | O(n) |
| 按位置查找 | 遍历计数 | O(n) |
| 任意位置插入 | 找到 `prev`，修改 `prev->next` | O(n) |
| 删除节点 | 找到 `prev`，跳过被删节点，`free` | O(n) |
| 修改节点 | 找到节点，改 `data` | O(n) |
| 反转链表 | 三个指针逐个翻转 `next` 方向 | O(n) |

**题2（编程）**：实现 `Node* find_middle(Node *head)`，返回链表的中间节点。如果节点数为偶数，返回中间偏右的那个。要求只遍历一次（提示：快慢指针）。

**题3（编程）**：实现 `int has_cycle(Node *head)`，判断链表是否有环（即某个节点的 next 指向了前面的节点，形成循环）。返回 1 表示有环，0 表示无环。要求 O(n) 时间、O(1) 额外空间。（提示：快慢指针，Floyd 判圈算法）

**题4（编程）**：实现 `Node* merge_sorted(Node *a, Node *b)`，将两个升序链表合并为一个新的升序链表，不创建新节点，只修改指针。返回合并后的头节点。
