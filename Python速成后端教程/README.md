## 你需要的最少 Python 知识

本系列教程假设你**不是零编程基础**。在开始前，你至少要知道：

- **变量**：`name = "Alice"` 是在干什么
- **函数**：`def hello():` 是什么意思，怎么调用
- **if/else**：条件判断的基本写法
- **列表**：`[1, 2, 3]` 是什么，`list.append()` 怎么用
- **字典**：`{"key": "value"}` 是什么，怎么取值
- **class 基础**：`class Dog:` 长什么样，`self` 是干嘛的
- **import**：`import json` 是引入别人写好的代码

**如果上面这些你都不会？** 建议先学 Python 基础（我们后续会制作 Python 教程，届时可以从这里跳转过去）。如果你会其中一部分，不会的在教程中遇到时现学现查也可以。

## 贯穿项目：简易博客系统

本系列所有代码都围绕一个项目展开——**一个简易博客系统**。你将一步步实现：

- 用户可以注册、登录
- 登录后可以发表文章、修改文章、删除文章
- 所有人可以查看文章列表和单篇文章
- 用户可以给文章写评论
- 文章支持封面图上传
- 支持分页和关键词搜索
- 最终部署到云服务器，通过 HTTPS 域名访问

**学完这个系列，你拿到任何一个"做一个 XXX 系统"的需求，都不会再害怕。**

## 代码文件

所有代码会随着教程逐步构建，最终项目结构：

```
blog_backend/
├── main.py           # 主入口，所有接口
├── schemas.py        # 请求/响应数据模型
├── models.py         # 数据库表模型
├── database.py       # 数据库连接配置
├── auth.py           # 认证相关（密码哈希、JWT）
├── middleware.py      # 中间件
├── test_main.py      # 测试用例
├── conftest.py       # 测试配置
├── Dockerfile        # Docker 镜像
├── docker-compose.yml
└── requirements.txt  # 依赖清单
```
