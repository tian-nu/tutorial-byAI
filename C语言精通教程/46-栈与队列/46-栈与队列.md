## 我在做什么？

栈和队列是两种**"受限"的线性结构**——它们不允许你在任意位置插入删除，只能在特定位置操作。但正是这种限制，让它们在很多场景下用起来特别顺手。

想象一个**托盘堆**：
- 你每次只能放托盘到**最顶上**，也只能从最顶上拿托盘。
- 后放上来的托盘，反而先被拿走。
- 这就是**栈（Stack）**，后进先出（Last In First Out，LIFO）。

想象一个**排队打饭**：
- 后来的人排在队伍**最后面**，前面的人打完走了才轮到你。
- 先到先得，后到后得。
- 这就是**队列（Queue）**，先进先出（First In First Out，FIFO）。

`*此术语见附录F*`：栈、LIFO、push、pop、队列、FIFO、enqueue、dequeue。

## 46.1 栈的定义

**栈**是只能在**一端**（栈顶）进行插入和删除操作的线性表。
- `push`：插入元素（压入栈顶）
- `pop`：删除元素（弹出栈顶）
- `peek`：查看栈顶元素，但不删除

```
push 10    push 20    push 30    pop        pop

  [  ]       [  ]       [30]       [  ]       [  ]
             [  ]       [20]       [20]       [  ]
  (空)     [10]       [10]       [10]      (空)
  栈顶↓     栈顶↓       栈顶↓       栈顶↓
  底←──────────────────────────────←顶

输出顺序：弹出 30 → 弹出 20 → 最后只剩 10
```

## 46.2 栈的数组实现

数组实现栈非常简单：
- `data`：存储元素的数组
- `capacity`：数组容量
- `top`：栈顶位置索引（从 0 开始，`top == -1` 表示空栈）

```c
/* stack_array.h */
#ifndef STACK_ARRAY_H
#define STACK_ARRAY_H

typedef struct {
    int *data;
    int  top;
    int  capacity;
} StackArray;

StackArray* stack_array_create(int init_capacity);
void        stack_array_destroy(StackArray *s);
int         stack_array_push(StackArray *s, int value);
int         stack_array_pop(StackArray *s, int *out);
int         stack_array_peek(const StackArray *s, int *out);
int         stack_array_is_empty(const StackArray *s);
int         stack_array_size(const StackArray *s);

#endif
```

```c
/* stack_array.c */
#include "stack_array.h"
#include <stdlib.h>

/* 创建栈，初始容量 init_capacity */
StackArray* stack_array_create(int init_capacity) {
    StackArray *s = (StackArray*)malloc(sizeof(StackArray));
    if (s == NULL) return NULL;
    s->data = (int*)malloc(sizeof(int) * init_capacity);
    if (s->data == NULL) {
        free(s);
        return NULL;
    }
    s->capacity = init_capacity;
    s->top = -1;  /* 空栈 */
    return s;
}

/* 销毁栈 */
void stack_array_destroy(StackArray *s) {
    if (s != NULL) {
        if (s->data != NULL) free(s->data);
        free(s);
    }
}

/* 压入栈顶 */
int stack_array_push(StackArray *s, int value) {
    if (s == NULL) return -1;
    /* 如果容量不够，需要扩容 */
    if (s->top + 1 >= s->capacity) {
        int new_capacity = s->capacity * 2;
        int *new_data = (int*)realloc(s->data, sizeof(int) * new_capacity);
        if (new_data == NULL) return -1;  /* 扩容失败 */
        s->data = new_data;
        s->capacity = new_capacity;
    }
    s->top++;
    s->data[s->top] = value;
    return 0;
}

/* 弹出栈顶 */
int stack_array_pop(StackArray *s, int *out) {
    if (s == NULL || stack_array_is_empty(s)) return -1;
    *out = s->data[s->top];
    s->top--;
    return 0;
}

/* 查看栈顶 */
int stack_array_peek(const StackArray *s, int *out) {
    if (s == NULL || stack_array_is_empty(s)) return -1;
    *out = s->data[s->top];
    return 0;
}

int stack_array_is_empty(const StackArray *s) {
    return s->top == -1;
}

int stack_array_size(const StackArray *s) {
    return s->top + 1;
}
```

优点：实现简单，内存连续，缓存友好。缺点：动态扩容需要 `realloc`。

## 46.3 栈的链表实现

用链表实现栈，头插头删（都在链表头部，O(1)）。

```
栈顶在链表头部：每次 push 头插，每次 pop 删除头节点。

  栈顶 ↓
  [30] -> [20] -> [10] -> NULL
  ^        ^        ^
  弹出先拿30，正好。
```

```c
/* stack_list.h */
#ifndef STACK_LIST_H
#define STACK_LIST_H

typedef struct StackListNode {
    int                    data;
    struct StackListNode  *next;
} StackListNode;

typedef struct {
    StackListNode *top;  /* 栈顶就是链表头 */
    int           size;
} StackList;

StackList* stack_list_create(void);
void       stack_list_destroy(StackList *s);
int        stack_list_push(StackList *s, int value);
int        stack_list_pop(StackList *s, int *out);
int        stack_list_peek(const StackList *s, int *out);
int        stack_list_is_empty(const StackList *s);
int        stack_list_size(const StackList *s);

#endif
```

```c
/* stack_list.c */
#include "stack_list.h"
#include <stdlib.h>

StackList* stack_list_create(void) {
    StackList *s = (StackList*)malloc(sizeof(StackList));
    if (s == NULL) return NULL;
    s->top = NULL;
    s->size = 0;
    return s;
}

void stack_list_destroy(StackList *s) {
    if (s == NULL) return;
    StackListNode *cur = s->top;
    while (cur != NULL) {
        StackListNode *next = cur->next;
        free(cur);
        cur = next;
    }
    free(s);
}

int stack_list_push(StackList *s, int value) {
    if (s == NULL) return -1;
    StackListNode *n = (StackListNode*)malloc(sizeof(StackListNode));
    if (n == NULL) return -1;
    n->data = value;
    n->next = s->top;
    s->top = n;
    s->size++;
    return 0;
}

int stack_list_pop(StackList *s, int *out) {
    if (s == NULL || stack_list_is_empty(s)) return -1;
    *out = s->top->data;
    StackListNode *to_del = s->top;
    s->top = s->top->next;
    free(to_del);
    s->size--;
    return 0;
}

int stack_list_peek(const StackList *s, int *out) {
    if (s == NULL || stack_list_is_empty(s)) return -1;
    *out = s->top->data;
    return 0;
}

int stack_list_is_empty(const StackList *s) {
    return s->size == 0;
}

int stack_list_size(const StackList *s) {
    return s->size;
}
```

优点：不需要预分配空间，动态增减。缺点：每个节点额外一个指针。

## 46.4 应用：括号匹配

栈最经典的应用之一：检查表达式中的括号是否匹配。

算法思路：
1. 遍历每个字符。
2. 如果是左括号 `(`，push 入栈。
3. 如果是右括号 `)`：
   - 如果栈空 → 右括号多了，不匹配。
   - 否则 pop 一个左括号 → 匹配一对。
4. 遍历结束后，如果栈空 → 全部匹配；否则左括号多了，不匹配。

```c
/* 括号匹配示例 */
int check_parentheses(const char *expr) {
    StackList *stack = stack_list_create();
    if (stack == NULL) return -1;  /* 内存错误 */

    for (int i = 0; expr[i] != '\0'; i++) {
        char c = expr[i];
        if (c == '(') {
            stack_list_push(stack, 1);
        } else if (c == ')') {
            if (stack_list_is_empty(stack)) {
                stack_list_destroy(stack);
                return 0;  /* 不匹配 */
            }
            int tmp;
            stack_list_pop(stack, &tmp);
        }
    }

    int result = stack_list_is_empty(stack) ? 1 : 0;
    stack_list_destroy(stack);
    return result;
}
```

测试：
- `"1+(2*3))"` → 不匹配（多一个右括号）
- `"(1+(2*3)"` → 不匹配（最后栈非空）
- `"(1+(2*3))"` → 匹配

## 46.5 队列的定义

**队列**是只能在**一端**插入，另一端删除的线性表。
- `enqueue`：插入到队尾
- `dequeue`：从队头删除
- `peek`：查看队头元素

```
enqueue 10    enqueue 20    enqueue 30    dequeue    dequeue

  头             头→[10]           头→[10]→[20]→[30]→尾   [20]→[30]→尾     [30]→尾
  (空)
输出顺序：弹出 10 → 弹出 20 → 最后只剩 30
先进去的先出来 → FIFO
```

## 46.6 队列的链表实现

用链表实现队列：头删尾插，都要记住头指针和尾指针。

```c
typedef struct QueueNode {
    int            data;
    struct QueueNode *next;
} QueueNode;

typedef struct {
    QueueNode *front;  /* 队头（出队从这里拿） */
    QueueNode *rear;   /* 队尾（入队插在这里） */
    int       size;
} Queue;
```

```c
/* queue_list.c */
#include "queue_list.h"
#include <stdlib.h>

Queue* queue_create(void) {
    Queue *q = (Queue*)malloc(sizeof(Queue));
    if (q == NULL) return NULL;
    q->front = q->rear = NULL;
    q->size = 0;
    return q;
}

void queue_destroy(Queue *q) {
    if (q == NULL) return;
    QueueNode *cur = q->front;
    while (cur != NULL) {
        QueueNode *next = cur->next;
        free(cur);
        cur = next;
    }
    free(q);
}

int queue_enqueue(Queue *q, int value) {
    if (q == NULL) return -1;
    QueueNode *n = (QueueNode*)malloc(sizeof(QueueNode));
    if (n == NULL) return -1;
    n->data = value;
    n->next = NULL;
    if (queue_is_empty(q)) {
        q->front = q->rear = n;
    } else {
        q->rear->next = n;
        q->rear = n;
    }
    q->size++;
    return 0;
}

int queue_dequeue(Queue *q, int *out) {
    if (q == NULL || queue_is_empty(q)) return -1;
    *out = q->front->data;
    QueueNode *to_del = q->front;
    q->front = q->front->next;
    free(to_del);
    q->size--;
    if (q->front == NULL) {
        q->rear = NULL;  /* 空了，rear也要设为 NULL */
    }
    return 0;
}

int queue_peek(const Queue *q, int *out) {
    if (q == NULL || queue_is_empty(q)) return -1;
    *out = q->front->data;
    return 0;
}

int queue_is_empty(const Queue *q) {
    return q->size == 0;
}

int queue_size(const Queue *q) {
    return q->size;
}
```

## 46.7 循环队列——避免假溢出

数组实现队列有一个问题：如果你把数组当作固定缓冲区，出队元素后，前面的空间就空出来了，但 `rear` 可能已经到达数组末尾，导致"假溢出"——明明前面有空位，却没法插入新元素。

```
初始:      [  ] [  ] [  ] [  ]
          front=0, rear=0

enqueue 10, 20, 30:
            [10] [20] [30] [  ]
          front=0,       rear=3

dequeue 两次:
            [  ] [  ] [30] [  ]
                   front=2, rear=3

enqueue 40:
            [  ] [  ] [30] [40]
                   front=2, rear=4  ← rear 已经越界！但前面有2个空位！
```

这就是"假溢出"——实际数组未满，但 rear 到了末尾。解决方法是**循环队列**：让 rear 绕回到数组开头。

```
循环队列：把数组看成一个环。

rear = (rear + 1) % capacity;  ← 每次前进都取模，绕回去
front = (front + 1) % capacity; ← 同理

判断队空：front == rear
判断队满：(rear + 1) % capacity == front  ← 留一个空位来区分空和满
```

```c
/* circular_queue.c —— 循环队列完整实现 */
#include <stdio.h>
#include <stdlib.h>

typedef struct {
    int *data;
    int  front;
    int  rear;
    int  capacity;
} CircularQueue;

CircularQueue* cqueue_create(int capacity) {
    CircularQueue *q = (CircularQueue*)malloc(sizeof(CircularQueue));
    if (q == NULL) return NULL;
    q->data = (int*)malloc(sizeof(int) * capacity);
    if (q->data == NULL) {
        free(q);
        return NULL;
    }
    q->capacity = capacity;
    q->front = 0;
    q->rear = 0;
    return q;
}

void cqueue_destroy(CircularQueue *q) {
    if (q != NULL) {
        if (q->data != NULL) free(q->data);
        free(q);
    }
}

int cqueue_is_empty(const CircularQueue *q) {
    return q->front == q->rear;
}

int cqueue_is_full(const CircularQueue *q) {
    return (q->rear + 1) % q->capacity == q->front;
}

int cqueue_enqueue(CircularQueue *q, int value) {
    if (q == NULL || cqueue_is_full(q)) return -1;
    q->data[q->rear] = value;
    q->rear = (q->rear + 1) % q->capacity;
    return 0;
}

int cqueue_dequeue(CircularQueue *q, int *out) {
    if (q == NULL || cqueue_is_empty(q)) return -1;
    *out = q->data[q->front];
    q->front = (q->front + 1) % q->capacity;
    return 0;
}

int cqueue_size(const CircularQueue *q) {
    return (q->rear - q->front + q->capacity) % q->capacity;
}

/* 演示 */
int main(void) {
    CircularQueue *q = cqueue_create(5);  /* 实际能存4个元素 */

    cqueue_enqueue(q, 10);
    cqueue_enqueue(q, 20);
    cqueue_enqueue(q, 30);
    cqueue_enqueue(q, 40);
    printf("满? %s\n", cqueue_is_full(q) ? "是" : "否");

    int val;
    cqueue_dequeue(q, &val);
    printf("出队: %d\n", val);
    cqueue_dequeue(q, &val);
    printf("出队: %d\n", val);

    cqueue_enqueue(q, 50);
    cqueue_enqueue(q, 60);
    printf("当前队列大小: %d\n", cqueue_size(q));

    while (!cqueue_is_empty(q)) {
        cqueue_dequeue(q, &val);
        printf("%d ", val);
    }
    printf("\n");

    cqueue_destroy(q);
    return 0;
}
```

## 46.8 完整代码清单（栈 + 队列统一演示）

```c
/*
 * stack_queue_demo.c —— 栈和队列完整演示
 * 编译: gcc -std=c99 -Wall -o stack_queue stack_queue_demo.c
 */

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

/* ==================== 栈（链表实现） ==================== */
typedef struct StackNode {
    int               data;
    struct StackNode *next;
} StackNode;

typedef struct {
    StackNode *top;
    int       size;
} Stack;

Stack* stack_create(void) {
    Stack *s = (Stack*)malloc(sizeof(Stack));
    s->top = NULL;
    s->size = 0;
    return s;
}

void stack_destroy(Stack *s) {
    StackNode *cur = s->top;
    while (cur != NULL) {
        StackNode *next = cur->next;
        free(cur);
        cur = next;
    }
    free(s);
}

int stack_push(Stack *s, int value) {
    StackNode *n = (StackNode*)malloc(sizeof(StackNode));
    if (n == NULL) return -1;
    n->data = value;
    n->next = s->top;
    s->top = n;
    s->size++;
    return 0;
}

int stack_pop(Stack *s, int *out) {
    if (s->size == 0) return -1;
    *out = s->top->data;
    StackNode *to_del = s->top;
    s->top = s->top->next;
    free(to_del);
    s->size--;
    return 0;
}

/* ==================== 队列（链表实现） ==================== */
typedef struct QueueNode {
    int               data;
    struct QueueNode *next;
} QueueNode;

typedef struct {
    QueueNode *front;
    QueueNode *rear;
    int       size;
} Queue;

Queue* queue_create(void) {
    Queue *q = (Queue*)malloc(sizeof(Queue));
    q->front = q->rear = NULL;
    q->size = 0;
    return q;
}

void queue_destroy(Queue *q) {
    QueueNode *cur = q->front;
    while (cur != NULL) {
        QueueNode *next = cur->next;
        free(cur);
        cur = next;
    }
    free(q);
}

int queue_enqueue(Queue *q, int value) {
    QueueNode *n = (QueueNode*)malloc(sizeof(QueueNode));
    if (n == NULL) return -1;
    n->data = value;
    n->next = NULL;
    if (q->size == 0) {
        q->front = q->rear = n;
    } else {
        q->rear->next = n;
        q->rear = n;
    }
    q->size++;
    return 0;
}

int queue_dequeue(Queue *q, int *out) {
    if (q->size == 0) return -1;
    *out = q->front->data;
    QueueNode *to_del = q->front;
    q->front = q->front->next;
    free(to_del);
    q->size--;
    if (q->size == 0) q->rear = NULL;
    return 0;
}

/* ==================== 括号匹配 ==================== */
int check_brackets(const char *expr) {
    Stack *s = stack_create();
    for (int i = 0; expr[i] != '\0'; i++) {
        char c = expr[i];
        if (c == '(' || c == '[' || c == '{') {
            stack_push(s, (int)c);
        } else if (c == ')' || c == ']' || c == '}') {
            if (s->size == 0) { stack_destroy(s); return 0; }
            int top_char;
            stack_pop(s, &top_char);
            if ((c == ')' && top_char != '(') ||
                (c == ']' && top_char != '[') ||
                (c == '}' && top_char != '{')) {
                stack_destroy(s);
                return 0;
            }
        }
    }
    int result = (s->size == 0) ? 1 : 0;
    stack_destroy(s);
    return result;
}

/* ==================== 演示 ==================== */
int main(void) {
    printf("===== 栈演示 =====\n");
    Stack *s = stack_create();
    stack_push(s, 10);
    stack_push(s, 20);
    stack_push(s, 30);
    int val;
    while (s->size > 0) {
        stack_pop(s, &val);
        printf("弹出: %d\n", val);
    }
    stack_destroy(s);

    printf("\n===== 队列演示 =====\n");
    Queue *q = queue_create();
    queue_enqueue(q, 10);
    queue_enqueue(q, 20);
    queue_enqueue(q, 30);
    while (q->size > 0) {
        queue_dequeue(q, &val);
        printf("出队: %d\n", val);
    }
    queue_destroy(q);

    printf("\n===== 括号匹配演示 =====\n");
    printf("'()[]{}'  -> %s\n", check_brackets("()[]{}") ? "匹配" : "不匹配");
    printf("'([)]'    -> %s\n", check_brackets("([)]") ? "匹配" : "不匹配");
    printf("'((()))'  -> %s\n", check_brackets("((()))") ? "匹配" : "不匹配");
    printf("'(()'     -> %s\n", check_brackets("(()") ? "匹配" : "不匹配");

    return 0;
}
```

## 我做得对不对？

1. **栈 LIFO 验证**：push 10, 20, 30 后依次 pop，顺序应为 30, 20, 10。
2. **队列 FIFO 验证**：enqueue 10, 20, 30 后依次 dequeue，顺序应为 10, 20, 30。
3. **括号匹配**：测试 `"()[]{}"`（匹配）、`"([)]"`（不匹配，交叉）、`"((()))"`（匹配）、`"(()"`（不匹配）。
4. **循环队列**：验证满队状态不能插入，出队后能继续插入，且元素不丢失。

---

## 不对怎么办？

### 常见Bug 1：栈的 `pop` 后忘记 `free`

```c
/* ❌ 只改了 top 指针，没有释放节点内存 */
int stack_pop_wrong(Stack *s, int *out) {
    if (s->size == 0) return -1;
    *out = s->top->data;
    s->top = s->top->next;  /* 前一个节点泄漏了！ */
    s->size--;
    return 0;
}
```

✅ 在修改 `s->top` 之前，先用临时指针保存要释放的节点。

### 常见Bug 2：队列 `dequeue` 后忘记更新 `rear`

```c
/* ❌ 队列变空后 rear 还指向已释放的节点 */
int queue_dequeue_wrong(Queue *q, int *out) {
    /* ... 删除逻辑 ... */
    q->size--;
    /* 忘记写：if (q->size == 0) q->rear = NULL; */
}
```

✅ 当队列变空时，必须同时将 `rear` 设为 `NULL`。

### 常见Bug 3：循环队列判断空和满混淆

```c
/* ❌ 用 front == rear 同时判断空和满，无法区分 */
int is_empty(CircularQueue *q) { return q->front == q->rear; }
int is_full(CircularQueue *q)  { return q->front == q->rear; }  /* 和上面一样！ */
```

✅ 方案：牺牲一个存储空间，`(rear + 1) % capacity == front` 表示满。

### 常见Bug 4：循环队列 `size` 计算错误

```c
/* ❌ 直接用 rear - front，没考虑环绕 */
int size_wrong(CircularQueue *q) { return q->rear - q->front; }
```

✅ 正确公式：`(rear - front + capacity) % capacity`

## 本章小结

| 要点 | 说明 |
|------|------|
| 栈（Stack） | 后进先出（LIFO），push/pop/peek 都在栈顶操作 |
| 栈的数组实现 | 用 `top` 索引 + 动态扩容，实现简单 |
| 栈的链表实现 | 头插头删，无需扩容，但每个节点多一个指针 |
| 队列（Queue） | 先进先出（FIFO），enqueue 在队尾，dequeue 在队头 |
| 队列的链表实现 | 维护 front 和 rear 两个指针，头删尾插 |
| 循环队列 | 解决数组假溢出，利用取模运算让 rear 和 front 绕回 |
| 括号匹配 | 栈的经典应用，左括号入栈，右括号出栈匹配 |

**题1（编程）**：用栈实现一个十进制转二进制的函数 `void dec_to_bin(int n)`。思路：不断将 n 除以 2 取余数，余数依次入栈，最后依次出栈输出。
