## 函数是什么：把一段逻辑打包，起个名字

你有没有过这种经历：跟朋友说"老规矩"，对方就知道你要干什么？

比如你每周五都去吃火锅，后来你只需要说"老规矩"，朋友就知道是"周五晚上6点，海底捞，鸳鸯锅"。

**函数（Function）**📖 就是你给一段代码起的"老规矩"名字。把一段逻辑打包，贴个标签，以后喊这个标签，那段逻辑就自动跑一遍。

不用函数的时候，你可能要这样：

```python
# 每次计算长方形面积都要写一遍
length = 5
width = 3
area = length * width
print(area)

# 换一个长方形，又要写一遍
length = 10
width = 4
area = length * width
print(area)
```

用了函数之后，一句话搞定：

```python
def rectangle_area(length, width):
    return length * width

print(rectangle_area(5, 3))   # 输出：15
print(rectangle_area(10, 4))  # 输出：40
```

**一次定义，到处使用。** 这就是函数。

## 函数的三个零件：输入 → 加工 → 输出

每个函数都像一台榨汁机：

```
水果（输入）→ 🧃 榨汁机（加工）→ 果汁（输出）
```

对应到代码里：

| 生活中的概念 | 编程里的名字 | 说明 |
|-------------|-------------|------|
| 水果 | **参数（Parameter）**📖 | 你给函数的东西，放在括号里 |
| 榨汁机 | **函数体** | 函数内部的处理逻辑 |
| 果汁 | **返回值（Return Value）**📖 | 函数处理完后还给你东西 |

> 以下代码由AI生成：

```python
# 定义（def = define，定义的意思）
def add(a, b):          # a和b是参数——你要把两个数给我
    result = a + b       # 加工：把两个数加起来
    return result        # 返回值：把结果还给你

# 调用（用的时候）
answer = add(3, 5)       # 3和5是具体输入
print(answer)            # 输出：8
```

**逐行讲解**：
- 第2行：`def` 是"我要定义一个函数"的意思。`add` 是函数名。`(a, b)` 是参数——使用这个函数时必须给它两个数。
- 第3行：函数体——把 a 和 b 加起来，放进 `result`。
- 第4行：`return` 是"把结果还给你"的意思。调用 `add(3, 5)` 的人会收到 `8`。
- 第7行：`add(3, 5)` 是"喊这个函数跑一遍，输入是3和5"。跑完会返回8，存在 `answer` 里。
- 第8行：打印出 `8`。

## AI生成代码示例：长方形面积计算器

> 以下代码由AI生成：

```python
def rectangle_area(length, width):
    """计算长方形的面积"""    # 这是函数的说明，叫"文档字符串"
    area = length * width     # 面积 = 长 × 宽
    return area               # 把面积还给你

# 使用这个函数
area1 = rectangle_area(5, 3)       # 长5宽3，面积=15
area2 = rectangle_area(10.5, 4.2)  # 长10.5宽4.2，面积=44.1

print(f"第一个长方形面积：{area1}")   # 输出：第一个长方形面积：15
print(f"第二个长方形面积：{area2}")   # 输出：第二个长方形面积：44.1
```

**逐行讲解**：
- 第1行：定义一个叫 `rectangle_area` 的函数，需要两个参数：长和宽。
- 第2行：`"""..."""` 是函数的说明书，告诉别人这个函数干嘛的。
- 第3行：计算面积。
- 第4行：把结果返回。
- 第7-8行：两次调用，第一次算5×3，第二次算10.5×4.2。
- 第10-11行：`f"...{变量}..."` 是一种特殊字符串，可以把变量嵌到文字里。

## 什么时候需要函数？

一条简单的判断规则：

> 🔥 **同一段代码出现了两次以上 → 抽成函数。**

不用函数（❌ 不好的写法）：

```python
# 给三个人的分数做评价，重复写三遍逻辑
if score1 >= 60:
    print("及格")
else:
    print("不及格")

if score2 >= 60:
    print("及格")
else:
    print("不及格")

if score3 >= 60:
    print("及格")
else:
    print("不及格")
```

用函数（✅ 好的写法）：

```python
def evaluate(score):
    if score >= 60:
        return "及格"
    else:
        return "不及格"

print(evaluate(score1))
print(evaluate(score2))
print(evaluate(score3))
```

**好处**：逻辑只写一次，改起来也只需要改一处。万一及格线从60改到70，函数里改一个数字就行，不用改三遍。

## 函数可以不返回东西，也可以没有参数

并非所有函数都像榨汁机那样"输入→加工→输出"。有些函数只是"做一件事"：

```python
def say_hello():              # 没有参数
    print("你好，欢迎来到AI编程世界！")  # 只打印，不返回

say_hello()  # 调用：输出 → 你好，欢迎来到AI编程世界！
```

这种函数就像你喊"服务员，倒杯水"——不需要给服务员什么，也不需要服务员还你什么，他就去做这件事。
