# 06-安装Python与终端入门

> **本篇属于**：环境准备篇 | **预计阅读**：25分钟 | **难度**：★★☆☆☆
>
> **本章是第一个动手章节。** 如果你之前从没装过编程环境、从没打开过"那个黑窗口"——别怕，每一步都会说清楚。

你可能会问："为什么搞AI编程要先装Python？"

答案很简单：**AI最擅长写Python代码。** 你给AI用中文描述需求，AI用Python把它实现出来——你只需要装好Python，让AI写的代码能跑起来就行。

换句话说：**Python是AI和你之间的"共同语言"**。你不会Python没关系，AI会。但你得把Python装到电脑上，否则AI写了代码你也跑不了。

---

## 6.1 第一个比喻：Python是你的"游戏引擎"

如果你玩过游戏，你一定知道：游戏光盘你买回来，但光有光盘不行，你得先把游戏装到电脑上。Python也一样——它不是那种"双击就能用"的应用（比如微信、浏览器），而是一个底层的"引擎"，你装好它之后，电脑才能"理解"和"运行"Python代码。

装好Python就等同于给电脑装上了一台"代码翻译机"——以后AI写的Python代码，这台翻译机就能帮你执行。

---

## 6.2 Windows 安装 Python

### Step 1：下载安装包

1. 打开浏览器，访问 Python 官网：**https://www.python.org/downloads/**
2. 页面会自动推荐最新版本（比如 Python 3.12.x），点击那个黄色的大按钮 **"Download Python 3.x.x"**
3. 等待下载完成（大约25MB，很快）

### Step 2：安装（⚠️ 这是最容易出错的地方！）

1. 双击刚才下载的 `.exe` 文件，启动安装程序
2. 此时你会看到一个安装界面，**下方有一个复选框**，写着：
   ```
   ☐ Add Python 3.x to PATH
   ```
3. **⚠️ 请务必勾选这个复选框！务必！务必！务必！**

> 这一步如果不勾选，后面你在终端输入 `python` 的时候，电脑会一脸懵："python是啥？我找不到。"——这是本章最常见的坑，99%的新手都栽在这里。

4. 勾选后，点击 **"Install Now"**（现在安装）
5. 等待进度条走完（大约1分钟）
6. 出现 "Setup was successful" 就搞定了

### Step 3：关闭"路径长度限制"（推荐）

安装完成后，安装程序可能会弹出一个窗口，写着 "Disable path length limit"。点击它，然后关闭。这一步不是必须的，但做了之后能避免一些奇怪的路径报错。

## 6.3 Mac 安装 Python

### Step 1：下载安装包

1. 打开浏览器，访问：**https://www.python.org/downloads/**
2. 点击黄色按钮下载 `.pkg` 文件

### Step 2：安装

1. 双击下载的 `.pkg` 文件
2. 一路点击 "Continue"（继续）→ "Agree"（同意）→ "Install"（安装）
3. 输入你的 Mac 登录密码（这是给安装程序授权，正常的）
4. 安装完成，点击 "Close"

### Step 3：Mac 的 PATH 问题

Mac 安装 Python 通常会自动配置好 PATH，但偶尔会出问题。别担心，我们在 6.5 节会验证，如果出错了 6.8 节有解决方案。

## 6.4 什么是终端？——你人生中第一个"黑窗口"

现在 Python 已经装好了。但你还没法和它"说话"。你需要一个"对话窗口"——这就是 **📖 终端（Terminal）**。

> **📖 术语：终端（Terminal）**——就是那个黑底白字的窗口。它不是黑客工具，不是病毒，不是"电脑坏了"。它只是你和电脑**用文字交流**的地方。平时你用鼠标点击按钮是"图形交流"，终端里你打字发命令是"文字交流"。

**一张图胜过千言万语的比喻**：

| 你和电脑的交流方式 | 类比 |
|-------------------|------|
| 用鼠标点来点去（桌面、文件夹、浏览器） | 在餐厅指着菜单说"我要这个" |
| 在终端里打字发命令 | 在餐厅厨房直接对厨师说"番茄切块，鸡蛋打散，少油" |

终端更直接、更精准。给AI编程用的很多操作，在终端里一行命令就搞定了，用鼠标可能要点击十几次。

## 6.5 打开终端

### Windows

**方法1（最推荐）**：
1. 按下键盘上的 **`Win + R`**（Win键就是那个Windows图标的键，通常在Ctrl和Alt之间）
2. 弹出的"运行"窗口里输入 **`cmd`**
3. 按回车

**方法2**：
1. 点击左下角的"开始"菜单
2. 直接在搜索框里输入 **`cmd`** 或 **`命令提示符`**
3. 点击出现的"命令提示符"

打开后你会看到一个黑色（或白色）窗口，里面有一行字，后面有个闪烁的光标——这就是终端了。

### Mac

**方法1（最推荐）**：
1. 按下 **`Command + 空格`**，这会打开 Spotlight（聚光灯搜索）
2. 输入 **`Terminal`**（或中文输入"终端"）
3. 按回车

**方法2**：
1. 打开 Finder（访达）
2. 左侧菜单 → "应用程序" → "实用工具" → "终端"

打开后你会看到一个白色（或黑色）窗口，里面有一行字，最后是一个 `$` 符号——这就是 Mac 的终端。

## 6.6 验证安装：Python到底装成功了没有？

在你刚打开的终端里，输入下面这行命令，然后按回车：

```bash
python --version
```

**如果你看到类似这样的输出**：

```
Python 3.12.3
```

✅ **恭喜！Python 安装成功！**版本号大于等于 3.9 就没问题。

**如果你用 Mac，上面命令不行，试试**：

```bash
python3 --version
```

Mac 系统自带 Python 2.x，所以有时候 `python` 指向的是旧版本。如果用 `python3` 能看到 3.x 版本，说明也装好了——只不过以后你要记住用 `python3` 而不是 `python`。

---

## 6.7 你的第一个 Python 命令：Hello World

现在我们来运行人生的第一行 Python 代码。在终端输入：

```bash
python -c "print('Hello World')"
```

（Mac 用户如果上一步用的是 `python3`，这里也换成 `python3`）

按回车后，你应该看到：

```
Hello World
```

✅ 这行 `Hello World` 就是 Python 帮你"打印"出来的。你用一行命令，让 Python 完成了一次执行——这就是编程最基础的形态：**你发命令，电脑执行，给你结果**。

以后，AI 写的就是类似这样的代码（当然会更复杂），而你只需要把它交给 Python 运行。

## 6.8 ⚠️ 常见安装失败（这是本章最重要的部分！）

如果你在上面的验证步骤中没看到版本号，而是看到了乱七八糟的报错——别慌。下面按"症状"分类，找到你的问题。

### 🔴 问题1：`'python' 不是内部或外部命令，也不是可运行的程序`

**这是什么意思？**

翻译成大白话：**你让电脑打开"python"，但电脑找不到 Python 装在哪儿**。

**原因**：你在 6.2 节安装时**没有勾选 "Add Python to PATH"**。

> 📖 **术语：PATH（环境变量）**——想象你有一本"地址黄页"。当你在终端输入 `python`，电脑就去这本黄页里查"Python 住在哪"。PATH 就是这本黄页。如果安装时没勾选"Add Python to PATH"，就等于你的黄页里没有 Python 的地址，电脑当然找不到。

**解决方案（两选一）**：

**方案A：重新安装（最简单）**
1. 去"控制面板" → "程序和功能"（或"应用和功能"），找到 Python，卸载
2. 回到 6.2 节重新安装，**这次一定勾选 "Add Python to PATH"！**

**方案B：手动添加 PATH（不用重装，但多几步）**
1. 在 Windows 搜索框输入"环境变量"，点击"编辑系统环境变量"
2. 在弹出的窗口点击"环境变量"按钮
3. 在"系统变量"区域找到 `Path`，双击
4. 点击"新建"，添加以下两条路径（把 `YourUsername` 换成你的用户名）：
   ```
   C:\Users\YourUsername\AppData\Local\Programs\Python\Python312\
   C:\Users\YourUsername\AppData\Local\Programs\Python\Python312\Scripts\
   ```
5. 逐个点击"确定"关闭所有窗口
6. **关掉当前终端，重新打开一个新的终端**（这步很重要！终端不会自动刷新 PATH）
7. 重新执行 `python --version` 验证

### 🔴 问题2：Mac 提示 `python: command not found`

**原因**：Mac 安装 Python 后，可能需要用 `python3` 而不是 `python`。

**解决方案**：
```bash
# 试试这个
python3 --version
```

如果 `python3 --version` 能看到版本号，那你装成功了。只是以后每次输入命令时用 `python3` 替代 `python`。本书后续章节如果写 `python`，你自己换成 `python3` 就行。

### 🔴 问题3：版本号是 2.x（比如 `Python 2.7.18`）

**这是什么意思？**

Python 有两个大版本：Python 2 和 Python 3。它们是**不兼容的**——就像普通话和粤语，虽然都是中文，但互相听不懂。Python 2 已经"退休"了（2020年就停止维护了），**你必须用 Python 3**。

**解决方案**：

- **Mac**：用 `python3 --version` 试试，Mac 可能两个版本都有
- **Windows**：说明你可能不小心装了 Python 2。去控制面板卸载它，然后重新按 6.2 节安装 Python 3

### 🔴 问题4：Mac 使用 `python3` 还是报权限错误

**症状**：出现 `Permission denied` 之类的字样。

**解决方案**：给终端"全磁盘访问权限"：
1. 打开"系统设置" → "隐私与安全性" → "完全磁盘访问权限"
2. 把"终端"加到允许列表里
3. 重新打开终端再试

### 🔴 问题5：安装过程中 Windows 弹出"用户账户控制"（蓝框）

这是 Windows 的安全机制，在问"你确定要让这个程序修改电脑吗？"——**点击"是"**。这是正常的，安装 Python 需要管理员权限。

## 6.9 终端速成：你只需要会5个命令

终端能干的事非常多，但你现在**只需要会5个**就够了。其他的以后慢慢遇到。

| 命令 | 作用 | 大白话解释 |
|------|------|-----------|
| `cd 文件夹名` | 进入某个文件夹 | "带我去那个房间" |
| `cd ..` | 返回上一级文件夹 | "退出这个房间，回走廊" |
| `dir`（Windows）/ `ls`（Mac） | 列出当前文件夹里的所有文件 | "这个房间里都有啥？" |
| `python 文件名.py` | 运行一个 Python 文件 | "帮我把这个文件执行了" |
| `Ctrl + C` | 强制停止正在运行的程序 | "住手！别跑了！" |

> **别记笔记**，你现在不需要背。什么时候用到就翻回这一页。用三次你就记住了。
