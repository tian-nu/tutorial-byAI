# 附录A：Cursor快捷键速查

> 本附录可直接打印或收藏，随时查阅。

---

## 核心快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+K` | In-line Chat | 在代码中直接和AI对话，生成或修改代码 |
| `Ctrl+L` | 打开Chat侧边栏 | 和AI对话，支持@引用文件/文件夹/网络 |
| `Ctrl+I` | 打开Composer | 批量创作模式，一次生成多个文件 |
| `Ctrl+Shift+I` | 打开Composer (Agent模式) | AI自主找文件、改代码、运行终端 |
| `Tab` | 接受AI补全 | 接受灰色AI补全代码 |
| `Esc` | 拒绝AI补全 | 取消灰色AI补全 |
| `Ctrl+→` | 部分接受补全 | 只接受AI补全的下一个词 |
| `Ctrl+Enter` | 搜索代码库 | 用AI语义搜索整个项目 |

---

## 上下文引用

| 操作 | 说明 |
|------|------|
| `@file` | 引用特定文件，AI会读取该文件 |
| `@folder` | 引用整个文件夹 |
| `@web` | 让AI搜索网络获取最新信息 |
| `@git` | 引用Git提交历史 |
| `@code` | 引用代码片段 |
| `@docs` | 引用文档 |

---

## 模式切换

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+P` | 命令面板 |
| `Ctrl+B` | 切换侧边栏 |
| `Ctrl+J` | 切换终端面板 |
| `Ctrl+` (反引号) | 打开终端 |

---

## 编辑操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存文件 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 替换 |
| `Ctrl+D` | 选中下一个相同词 |
| `Alt+↑/↓` | 移动当前行 |
| `Ctrl+/` | 注释/取消注释 |

---

## 模型切换

在Chat或Composer窗口底部可以切换AI模型：
- **GPT-4o**：日常主力，速度快，适合大多数场景
- **Claude 3.5 Sonnet**：长文本理解好，适合分析大型文件
- **DeepSeek V3**：便宜，适合大量便宜调用
- **本地模型**：隐私保护，不联网

---

> **提示**：这些快捷键是默认设置。如果你导入了VS Code配置，部分快捷键可能和VS Code一致。
