## 学习路径建议

**完全零基础**：Git → Python后端 或 Go后端 → 数据库 → Docker

**有编程基础**：直接选你想学的教程

**想做全栈**：Go后端 + Vue前端 → 数据库 → Docker → 部署

**想用AI提效**：AI编程从零到精通 → 任意后端教程
