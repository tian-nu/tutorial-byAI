# 第0章 · 电脑基础准备——给第一次接触编程的你

> "我不认识终端，也不会用命令行。我就用过浏览器看网页，用 Word 写过文档，用微信聊过天。我能学会 Git 和 GitHub 吗？"
>
> **能。** 这一章的存在，就是为了让你从零开始——从"不知道怎么打开终端"到"能自己输入命令看到结果"。我们不会跳过任何一步。如果你已经知道怎么打开终端和浏览器，可以直接翻到第1章。

## 0.1 什么是终端（命令行）——为什么程序员要去一个黑乎乎的窗口里打字？

### 0.1.1 你平时怎么用电脑？

你打开微信，用的是它的界面——有头像、有聊天框、有发送按钮。你点一下"发送"，消息就发出去了。这叫**图形界面**（GUI，Graphical User Interface）。此术语需进附录。

图形界面很好，但你有没有发现一个问题：假如你想让电脑帮你做一件事——比如"把文件夹里所有的 `.txt` 文件都重命名为 `.md` 文件"——在图形界面上你要一个一个点右键 → 重命名 → 输入新名字 → 回车。如果有 100 个文件，你要重复 100 次。

### 0.1.2 终端是另一种方式

终端（也叫命令行、命令行界面）是另一种跟电脑说话的方式。在图形界面里你**点按钮**，在终端里你**打字**。

打个比方：

- **图形界面** = 你去餐厅，对着菜单用手指指："我要这个、这个、还有这个。"
- **终端** = 你给服务员写一张纸条："一份宫保鸡丁，一碗米饭。"

这两种方式结果一样——你都吃到了饭。但写纸条更精确、更快，而且可以一次写好几道菜。更重要的是：如果你想请服务员做一件菜单上没有的菜，你只能写纸条说清楚要求；指着菜单是没有用的。

### 0.1.3 终端长什么样？

当你打开终端，你会看到一个这样的窗口：

- 背景通常是**黑色**或**深蓝色**的
- 上面有一些白色的文字
- 有一个**闪烁的小横线或小方块**——这叫光标，它在等你打字

看上去像是黑客电影里的画面，但别怕。你只需要在光标后面打字，然后按回车键，电脑就会执行你打的命令。

### 0.1.4 不同系统上的终端叫什么

| 你的系统 | 终端叫什么 | 图标长什么样 |
|---------|-----------|------------|
| Windows 10 / 11 | **PowerShell**（推荐）或**命令提示符**或 **Git Bash**（装 Git 后才有） | 蓝色背景上有白色的 `>_` 符号 |
| Mac（苹果电脑） | **终端**（Terminal） | 黑色背景上有白色的 `>_` 符号 |

> 🤔 **PowerShell 和命令提示符有什么区别？**
>
> PowerShell 是命令提示符的"升级版"——功能更多，命令更强大。你把它理解成"高级版的黑窗口"就行。本教程所有 Windows 截图和说明都以 PowerShell 为准。如果你打开的是命令提示符（标题栏写着"命令提示符"），关了它，换成 PowerShell。

## 0.2 如何打开终端——精确到每一步鼠标点击

> ⚠️ 这一节是本教程最重要的基础操作。**后面每一章都会从"打开终端"开始**，所以请务必跟着做一遍，确保你能成功打开。

### 0.2.1 Windows 10 或 Windows 11 用户

**方法一（推荐，最快）：**

1. 找到你键盘**左下角**的那个键。它上面画着**四个小方块**，像一个飘动的窗户。这个键叫 **Windows 键**。有些键盘上这个键的图标是圆圈中间一个竖线，本质上一样。
2. 用你的左手小拇指或无名指**按一下**这个 Windows 键，不要按住不放，按一下就可以松开。
3. 你的屏幕左下角会弹出一个菜单（Windows 10）或者屏幕中央偏下会弹出一个搜索框（Windows 11），旁边还有一个可以打字的地方。
4. 直接在键盘上打字，输入这几个字母（不区分大小写）：

```
PowerShell
```

5. 打完之后，你会看到菜单或搜索框里出现了一个选项。它的名字是 **"Windows PowerShell"**，左边有一个**蓝色的图标**，图标上有白色的 `>_` 符号。
6. 用鼠标左键点击它。或者用键盘的上下箭头键选中它，然后按回车键。

> 📺 **你现在应该看到什么？**
>
> 一个窗口弹出来了。窗口的顶部标题栏写着 "Windows PowerShell"（或 "PowerShell 7"）。窗口里面大部分是黑色或深蓝色的背景，最上面几行有一些白色文字，最后一行有一个**闪烁的白色小方块或小横线**。恭喜你——这就是终端！

**方法二（备用）：**

如果你用方法一找不到 PowerShell：

1. 用鼠标左键点击屏幕左下角的**开始按钮**——它是一个四个方块拼在一起的图标，在屏幕最左下角。
2. 在弹出的菜单里，找到一个叫 **"Windows PowerShell"** 的文件夹。你可能需要往下滚动才能看到它（把鼠标放在菜单区域，用鼠标中间的滚轮往下滚，或者用鼠标左键拖动菜单右侧的滚动条）。
3. 点击这个文件夹，它会展开。
4. 点击里面的 **"Windows PowerShell"**——蓝色图标那个。

### 0.2.2 Mac（苹果电脑）用户

**方法一（推荐）：**

1. 看你的屏幕底部，有一条横着的图标栏——这叫 **Dock 栏**。上面有很多彩色图标。
2. 在 Dock 栏上找到**启动台（Launchpad）**的图标。它看起来像是一个灰色圆形，里面有九个小方块，或者是一个火箭的图案。
3. 用鼠标左键点击它。屏幕会切换到一个全是图标的页面。
4. 在这个页面里，找到**一个叫"其他"（Other）的文件夹**。图标通常是一个文件夹里面有一些工具的形状。
5. 点击这个"其他"文件夹。
6. 在打开的新页面里，找到**"终端"（Terminal）**的图标——黑色背景，白色的 `>_` 符号。
7. 点击它，终端就打开了。

**方法二（更快）：**

1. 同时按下键盘上的两个键：**Command 键**（键盘最下面一排，上面有 ⌘ 图案的那个键）和**空格键**，然后松开它们。
2. 屏幕中央会出现一个搜索框。
3. 在搜索框里输入（中文也可以）：

```
终端
```

4. 搜索框下面会出现一个叫"终端"的选项。
5. 看到后，按键盘上的**回车键**（Return 键），或者用鼠标双击那个选项。

> 📺 **你现在应该看到什么？**
>
> 一个窗口弹出来了。窗口的标题栏写着"终端"或"Terminal"。背景是白色的（或黑色的，取决于你的系统设置）。窗口里面有几行文字，最后一行有一个**灰色的小方块在闪烁**。这就是终端！

### 0.2.3 Linux 用户

你大概率已经知道怎么打开终端了。按 `Ctrl + Alt + T`，或者在应用菜单里搜索"Terminal"。

## 0.3 如何在终端里做最基本的事

终端打开了，但你可能在想："然后呢？我能在里面干什么？"

我们来学 5 个最基础的操作。每一个都会在你后面的学习中反复用到。

### 0.3.1 光标在等你

打开终端后，你看最后一行。有一个**小横线或小方块在一闪一闪**——这叫**光标（Cursor）**。它告诉你："我在这里，你打字会出现在我前面。"

在光标的左边，通常还有一串文字，比如：

```
PS C:\Users\你的名字>
```

或者在 Mac 上：

```
你的名字@MacBook-Pro ~ %
```

这串文字叫**提示符（Prompt）**——它在"提示"你：当前在什么位置、以什么身份在操作。你不用管它的具体含义，只要知道**光标在提示符后面**就行。

### 0.3.2 输入命令并按回车

在终端里，你做的事情就是：**打字 → 按回车 → 看结果**。

来试一下：

1. 确认光标在闪烁（如果不在，用鼠标点一下终端窗口内部）。
2. 在键盘上打这几个字母：

```
echo Hello
```

（e-c-h-o，空格，H-e-l-l-o）

3. 打完以后，用右手小拇指按一下键盘上的 **Enter 键**（也叫回车键、Return 键）——它在键盘右边，字母区的右侧，是一个比较大的横着的键，上面可能写着 "Enter" 或者画着一个向左拐的箭头 `↵`。

> 📺 **你现在应该看到什么？**
>
> 你打完 `echo Hello` 并按了回车之后，终端里出现了两行内容：
>
> ```
> PS C:\Users\你的名字> echo Hello
> Hello
> PS C:\Users\你的名字>
> ```
>
> 第一行是你输入的命令（电脑原样显示给你看）。第二行是电脑的回答——`Hello`。第三行又出现了一个新的提示符，光标在闪烁，表示"我准备好了，请说下一句"。

**你刚刚做了什么？**

- `echo` = 一个命令，意思是"把我后面写的东西重复一遍"
- `Hello` = 你让电脑重复的内容
- 按回车 = "好了，我说完了，请执行"

这就是终端的核心工作方式：**你打一条命令，按回车，电脑执行，给你看结果，然后等你打下一条命令。**

### 0.3.3 如何把文字粘贴到终端

后面教程里会给你一些命令让你执行。你不需要一个字一个字手打——可以复制粘贴。

**Windows PowerShell 用户：**

1. 先在你看到教程的地方（比如浏览器里的教程页面），用鼠标**选中一段命令文字**：把鼠标指针移到文字开头，按住鼠标左键不放，拖动鼠标到文字末尾，松开。被选中的文字会变成蓝色背景。
2. 按键盘上的 **Ctrl + C**（按住 Ctrl 不放，再按一下 C，然后两个键同时松开）。文字就被"复制"了。
3. 回到终端窗口，用鼠标**右键**点击终端窗口内部的黑色区域。文字就粘贴进去了。

> ⚠️ **注意：** Windows PowerShell 里不能用 `Ctrl + V` 粘贴！你只能右键粘贴。如果你按了 `Ctrl + V`，可能会输入一个奇怪的字符 `^V`。如果不小心按了，按一次退格键（Backspace，回车键上方那个向左的箭头键）删掉。

**Mac 用户：**

1. 在教程页面选中命令文字。
2. 按键盘上的 **Command + C**（⌘ + C）复制。
3. 回到终端窗口，按键盘上的 **Command + V**（⌘ + V）粘贴。

### 0.3.4 如何复制终端里的文字

有时候你需要把终端里显示的内容复制出来（比如把公钥复制到浏览器里）。

**Windows PowerShell 用户：**

1. 在终端窗口里，用鼠标**选中你想复制的文字**：按住鼠标左键从开头拖到结尾。被选中的文字会变成白色背景（或蓝色背景）。
2. 选好后，**按一下鼠标右键**，或者**按键盘上的 Ctrl + C**。文字就被复制了。

**Mac 用户：**

1. 在终端窗口里，用鼠标选中你想复制的文字。
2. 按键盘上的 **Command + C**（⌘ + C）复制。

### 0.3.5 如果打错了想重来

你正在终端里打字，打到一半发现前面有个字母打错了。你有两个选择：

**选择一：删掉重打。**

按键盘上的 **Backspace 键**（也叫退格键。它在键盘最上面一排的右侧，Enter 键的上方，上面画着一个向左的箭头 `←`）。每按一次删掉一个字符。按住不放可以连续删除。

**选择二：整行取消，从头再来。**

按键盘上的 **Ctrl + C**（Windows）或 **Control + C**（Mac）。

- 按住 Ctrl 键不放
- 再按一下 C 键
- 两个键同时松开

这会让当前这一行被取消，你会看到光标跳到新的一行，提示符重新出现。你现在可以重新打字了。

> ⚠️ **Ctrl + C 很重要！** 在终端里，`Ctrl + C` 的意思是"停止、取消"。如果某个命令一直在跑停不下来，按 `Ctrl + C` 可以强行终止它。后面教程里遇到任何"停不下来"的情况，请记住这个操作。

## 0.4 什么是浏览器——以及地址栏在哪

后面的教程你需要访问 GitHub 网站。这就需要用到**浏览器**。

### 0.4.1 浏览器是什么

浏览器是一个能让你**看网页**的程序。你电脑上大概率已经装了下面其中之一：

| 浏览器名字 | 图标长什么样 |
|-----------|------------|
| Google Chrome | 红黄绿蓝四色组成的一个圆形 |
| Microsoft Edge | 蓝色和绿色组成的一个漩涡形状 |
| Mozilla Firefox | 橙色狐狸绕着一个蓝色地球 |
| Safari（Mac 自带） | 蓝色指南针 |

### 0.4.2 如何打开浏览器

在桌面上找到上面任意一个浏览器图标，用鼠标**左键双击**它。或者点击屏幕底部任务栏（Windows）或 Dock 栏（Mac）上的浏览器图标。

### 0.4.3 地址栏在哪——这是最重要的东西

浏览器打开后，看**最上面**。有一条**白色的长条框**，里面可能会显示一些文字。这就是**地址栏**。

地址栏的作用是：你在这里输入一个网站的地址，按回车，浏览器就会打开那个网站。

比如说，如果地址栏里现在写着 `www.baidu.com`，表示你当前在百度。如果你想访问 GitHub，你需要：

1. 用鼠标左键**点击地址栏里的文字**，让文字全部变蓝（被选中）。
2. 直接输入新的网址。
3. 按键盘上的**回车键**。

我们来练习一下：

1. 用鼠标点击地址栏，清空里面的内容。
2. 输入：

```
github.com
```

3. 按回车键。

> 📺 **你现在应该看到什么？**
>
> 浏览器页面的内容变了。你会看到一个网站，页面可能显示一些代码动画、介绍文字，或者一个猫脸章鱼身的吉祥物图案（GitHub 的吉祥物叫 Octocat）。页面顶部有导航栏，右上角有几个按钮。如果你看到了这个，说明你成功访问了 GitHub。**GitHub 是世界上最大的代码存放网站**，全世界的程序员都在上面存放和分享代码。此术语需进附录。

在后面的教程里，每当我说"打开浏览器访问某个网址"，意思就是：打开浏览器 → 点击地址栏 → 输入网址 → 按回车。

## 0.5 什么是文件、文件夹、路径——用文件柜来理解

### 0.5.1 什么是文件

你在电脑上做的每一件事，最终都保存在**文件**里：

- 你用 Word 写了一篇文档，它保存为一个 `.docx` 文件。
- 你用手机拍了一张照片，它是一个 `.jpg` 文件。
- 你下载了一首歌，它是一个 `.mp3` 文件。
- 程序员写的代码，保存在 `.go`、`.py`、`.js` 等文件里。

**一句话：文件 = 存了内容的"一张纸"。**

### 0.5.2 什么是文件夹

你的电脑里有成千上万个文件。如果全部堆在一起，根本找不到东西。所以你需要**文件夹**来整理它们。

**一句话：文件夹 = 一个"抽屉"，里面可以放文件，也可以放更小的抽屉。**

打个比方：

- 你的电脑 = 一个文件柜
- C 盘、D 盘 = 文件柜里的大隔层
- 文件夹 = 隔层里的抽屉
- 子文件夹 = 抽屉里面的小格子
- 文件 = 格子里的纸

### 0.5.3 什么是路径

"路径"回答了这个问题：**从文件柜顶层，经过哪些抽屉和格子，最后找到那张纸？**

比如，如果你桌面上有一个文件叫 `我的笔记.txt`，它的路径在 Windows 上是这样的：

```
C:\Users\你的名字\Desktop\我的笔记.txt
```

从左到右读这句话：

- `C:` = C 盘，你的主硬盘
- `Users` = "用户"文件夹，所有用户的资料都在这里
- `你的名字` = 你登录电脑时用的用户名
- `Desktop` = 桌面（英文 Desktop 就是桌面的意思）
- `我的笔记.txt` = 最终的那个文件

中间的 `\` 叫**路径分隔符**，用来分隔每一层。在 Mac 上，分隔符是 `/`（正斜杠）。

### 0.5.4 你已经有一个重要的文件夹：桌面

桌面（Desktop）也是一个文件夹。你把文件拖到桌面上，这个文件实际上就放在了 `C:\Users\你的名字\Desktop\` 这个文件夹里。

在后面的教程里，我们会在桌面或其他位置创建项目文件夹。当你看到 `C:\Users\你的名字\` 这样的路径时，把"你的名字"换成你登录电脑时用的实际用户名。

## 0.6 如何创建文件——文本文件怎么建

### 0.6.1 在桌面创建一个文本文件

**Windows 用户：**

1. 先把所有窗口最小化，看到桌面背景。
2. 在桌面**空白处**（没有图标的地方），用鼠标**右键**点击一下。
3. 会弹出一个菜单。把鼠标移到菜单里的**"新建"**那一行——它右边有一个小箭头。
4. 鼠标移到"新建"上之后，旁边会再弹出一个子菜单。
5. 在子菜单里找到**"文本文档"**——它的图标是一张纸上面有几行字。用鼠标**左键**点击它。
6. 桌面上出现了一个新文件，名字是蓝底白字的"新建文本文档.txt"（等着你改名）。
7. 直接在键盘上输入你想用的名字，比如 `test`。
8. 输入完后，用鼠标点击桌面空白处，或者按回车键。文件名变成了 `test.txt`。

**Mac 用户：**

1. 打开"文本编辑"（TextEdit）应用——在 Dock 栏或启动台里找到它。
2. 菜单栏（屏幕最顶部）点击"文件"（File）→"新建"（New）。
3. 写一些内容。
4. 菜单栏点击"文件"→"存储"（Save）。
5. 在弹出的窗口顶部"位置"（Where）下拉菜单里选"桌面"（Desktop）。
6. "存储为"（Save As）里输入文件名，比如 `test.txt`。
7. 点击"存储"按钮。

### 0.6.2 如何用记事本打开和编辑文件

**Windows 用户：**

1. 在桌面上找到你刚创建的 `test.txt` 文件。
2. 用鼠标**右键**点击这个文件。
3. 在弹出的菜单里，找到**"打开方式"**，鼠标移到上面。
4. 在子菜单里找到**"记事本"**——图标是一本蓝色的小本子。用鼠标左键点击它。
5. 记事本打开了，里面是你的文件内容。你可以编辑（打字、删除），完成后按 **Ctrl + S**（按住 Ctrl 不放，再按 S，然后同时松开）保存。

**Mac 用户：**

1. 在桌面上找到你的 `.txt` 文件。
2. 用鼠标**双击**它。它会在"文本编辑"应用中打开。
3. 编辑完成后按 **Command + S** 保存。

### 0.6.3 修改文件的后缀名

文件的名字分两部分：前缀和后缀。比如 `test.txt`：

- `test` = 文件名（前缀）
- `.txt` = 后缀名，告诉电脑"这是文本文件"

不同的后缀表示不同类型的文件。`.go` 是 Go 代码文件，`.py` 是 Python 代码文件，`.md` 是 Markdown 文档文件。

**Windows 用户修改后缀名：**

1. 右键点击文件。
2. 在弹出的菜单里选择**"重命名"**。
3. 把 `.txt` 改成你需要的后缀（比如 `.md`）。
4. 电脑可能会弹出一个警告框："如果改变文件扩展名，可能会导致文件不可用。确实要更改吗？"
5. 点击**"是"**。

> ⚠️ 如果你在文件名里看不到 `.txt` 这个后缀名，说明你的电脑"隐藏了已知文件类型的扩展名"。你需要先让后缀名显示出来：
>
> 1. 打开任意一个文件夹（双击桌面上的"此电脑"图标）。
> 2. 看文件夹窗口的顶部，有一排标签。点击**"查看"**标签。
> 3. 在"查看"标签的右侧区域，找到**"文件扩展名"**这个复选框。
> 4. 用鼠标左键点击它，让它前面的方框里出现一个**对勾** ✓。
> 5. 现在所有文件的后缀名都显示出来了。你可以回到桌面继续操作。

**Mac 用户修改后缀名：**

1. 右键点击文件（或按住 Control 键再点击文件）。
2. 选择"重命名"。
3. 直接改后缀名。
4. 在弹出的警告框里点击"保留 .xxx"（确认修改）。
