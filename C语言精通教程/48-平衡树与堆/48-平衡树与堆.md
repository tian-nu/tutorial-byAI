## 我在做什么？

上一章我们学会了BST——它能快速查找，但有一个致命的弱点：**如果插入的数据是有序的（比如 1, 2, 3, 4, 5），BST就会退化成一条斜线，查找从 O(log n) 变成 O(n)**。

想想看，你建了一个"快速查找"的结构，结果因为插入顺序不对，变成了一个链表——这等于白费了功夫。

本章解决两个问题：
1. **平衡树**：如何让BST保持"两边差不多高"，避免退化。我们不会实现完整的AVL代码，但会讲清楚旋转原理。
2. **堆**：一种完全不同的树结构，不是用来查找，而是用来快速获取最大/最小值。它是堆排序和优先队列的基础。

`*此术语见附录F*`：平衡树、旋转、堆、大顶堆、小顶堆、完全二叉树。

## 48.1 BST的退化问题

```
插入顺序: 1, 2, 3, 4, 5

BST变成:
    1
     \
      2
       \
        3
         \
          4
           \
            5

查找 5 需要走 5 步 → O(n)！
这不是树，这是链表 —— 叫做"斜树"。
```

为什么会出现这个问题？因为BST的插入算法**不关心树的形状**，只管往下挂。为了保持平衡，我们需要在插入和删除后**调整树的结构**。

## 48.2 AVL树——自平衡的BST

**AVL树**（以发明者Adelson-Velsky和Landis命名）是第一种自平衡二叉搜索树。它的核心规则：

**每个节点的左右子树高度差不超过1**。

这个高度差称为**平衡因子**（Balance Factor）：

```
平衡因子 = 左子树高度 - 右子树高度
平衡因子 只能是 -1, 0, 1
```

如果某节点的平衡因子超出了这个范围（变成 -2 或 2），就需要用**旋转**来修复。

```
平衡因子为1的节点（左高右低）:
        A
       / \
      B   C
     / \
    D   E
   左子树高度 = 2, 右子树高度 = 1, 平衡因子 = 1 (OK)

平衡因子为2的节点（需要旋转修复）:
        A
       / \
      B   C
     / \
    D   E
   /
  F
   左子树高度 = 3, 右子树高度 = 1, 平衡因子 = 2 (需要旋转!)
```

## 48.3 四种旋转情况

AVL树通过四种旋转操作来恢复平衡：

### 情况1：LL（左左）—— 右旋（Right Rotation）

```
插入导致左子树的左子树过高：

旋转前:               旋转后:
      A                    B
     / \                  / \
    B   C    ======>     D   A
   / \                      / \
  D   E                    E   C

右旋操作：
1. B 的右孩子 E 变成 A 的左孩子
2. A 变成 B 的右孩子
3. B 成为新的根
```

```c
/* 右旋：LL情况 */
TreeNode* rotate_right(TreeNode *y) {
    TreeNode *x = y->left;
    TreeNode *T2 = x->right;

    /* 执行旋转 */
    x->right = y;
    y->left = T2;

    return x;  /* 新的根 */
}
```

### 情况2：RR（右右）—— 左旋（Left Rotation）

```
插入导致右子树的右子树过高：

旋转前:               旋转后:
      A                    C
     / \                  / \
    B   C    ======>     A   E
       / \              / \
      D   E            B   D

左旋操作：
1. C 的左孩子 D 变成 A 的右孩子
2. A 变成 C 的左孩子
3. C 成为新的根
```

```c
/* 左旋：RR情况 */
TreeNode* rotate_left(TreeNode *x) {
    TreeNode *y = x->right;
    TreeNode *T2 = y->left;

    /* 执行旋转 */
    y->left = x;
    x->right = T2;

    return y;  /* 新的根 */
}
```

### 情况3：LR（左右）—— 先左旋再右旋

```
左子树的右子树过高（需要两次旋转）：

旋转前:               先左旋(B):           再右旋(A):
      A                    A                    C
     / \                  / \                  / \
    B   C    ======>     C   C2   ======>     B   A
   / \                  / \                  / \ / \
  D   C1               B  C1r              D  C1l C1r C2
     / \              / \
   C1l C1r           D  C1l

操作：先对 B 左旋，再对 A 右旋
```

```c
/* LR：先左旋再右旋 */
TreeNode* rotate_lr(TreeNode *node) {
    node->left = rotate_left(node->left);
    return rotate_right(node);
}
```

### 情况4：RL（右左）—— 先右旋再左旋

```
右子树的左子树过高（需要两次旋转）：

操作：先对右孩子右旋，再对当前节点左旋
```

```c
/* RL：先右旋再左旋 */
TreeNode* rotate_rl(TreeNode *node) {
    node->right = rotate_right(node->right);
    return rotate_left(node);
}
```

**四种旋转快速记忆**：
- LL：右旋一次
- RR：左旋一次
- LR：左旋左孩子，再右旋自己
- RL：右旋右孩子，再左旋自己
- 规律：名字是反的！LR = 先左后右，RL = 先右后左

## 48.4 AVL插入完整示例

以下是一个带平衡因子和旋转的AVL插入函数（简化版，仅展示核心逻辑）：

```c
/* 获取高度 */
int height(TreeNode *node) {
    if (node == NULL) return 0;
    return node->height;
}

/* 获取平衡因子 */
int get_balance(TreeNode *node) {
    if (node == NULL) return 0;
    return height(node->left) - height(node->right);
}

/* 更新高度 */
void update_height(TreeNode *node) {
    int lh = height(node->left);
    int rh = height(node->right);
    node->height = (lh > rh ? lh : rh) + 1;
}

/* AVL插入（递归版本，带自动平衡） */
TreeNode* avl_insert(TreeNode *node, int key) {
    /* 1. 普通BST插入 */
    if (node == NULL) {
        TreeNode *n = (TreeNode*)malloc(sizeof(TreeNode));
        n->key = key;
        n->left = NULL;
        n->right = NULL;
        n->height = 1;
        return n;
    }

    if (key < node->key) {
        node->left = avl_insert(node->left, key);
    } else if (key > node->key) {
        node->right = avl_insert(node->right, key);
    } else {
        return node;  /* 不允许重复 */
    }

    /* 2. 更新高度 */
    update_height(node);

    /* 3. 检查平衡因子并旋转 */
    int balance = get_balance(node);

    /* LL情况 */
    if (balance > 1 && key < node->left->key) {
        return rotate_right(node);
    }
    /* RR情况 */
    if (balance < -1 && key > node->right->key) {
        return rotate_left(node);
    }
    /* LR情况 */
    if (balance > 1 && key > node->left->key) {
        node->left = rotate_left(node->left);
        return rotate_right(node);
    }
    /* RL情况 */
    if (balance < -1 && key < node->right->key) {
        node->right = rotate_right(node->right);
        return rotate_left(node);
    }

    return node;
}
```

有了AVL树，无论插入顺序如何，查找始终是 O(log n)。

## 48.5 堆——另一种"树"

堆是一种完全不同的树结构，它不用于查找，而用于快速获取**最大**或**最小**值。

**大顶堆（Max Heap）**：每个节点的值 ≥ 所有子节点的值。根节点是最大值。
**小顶堆（Min Heap）**：每个节点的值 ≤ 所有子节点的值。根节点是最小值。

```
大顶堆:
        50
       /  \
      30  40
     / \  / \
    10 20 15 35

性质：每个父节点 >= 子节点
根 50 是最大值
```

堆通常用**数组**实现，因为它是**完全二叉树**——除最后一层外，每层都填满，最后一层靠左排列。

```
数组索引规则（根从索引0开始）：
- 下标 i 的父节点: (i-1)/2
- 下标 i 的左孩子: 2*i + 1
- 下标 i 的右孩子: 2*i + 2

数组表示:
索引:  0   1   2   3   4   5   6
值:   50  30  40  10  20  15  35

对应树:
        50(0)
       /      \
    30(1)    40(2)
   /   \     /   \
 10(3) 20(4) 15(5) 35(6)
```

## 48.6 堆的数组实现（大顶堆）

```c
/* heap.h */
#ifndef HEAP_H
#define HEAP_H

typedef struct {
    int *data;
    int  size;
    int  capacity;
} MaxHeap;

MaxHeap* heap_create(int capacity);
void     heap_destroy(MaxHeap *h);
int      heap_insert(MaxHeap *h, int value);
int      heap_extract_max(MaxHeap *h, int *out);
int      heap_peek_max(const MaxHeap *h, int *out);
int      heap_is_empty(const MaxHeap *h);
int      heap_size(const MaxHeap *h);

#endif
```

```c
/* heap.c */
#include "heap.h"
#include <stdlib.h>

MaxHeap* heap_create(int capacity) {
    MaxHeap *h = (MaxHeap*)malloc(sizeof(MaxHeap));
    if (h == NULL) return NULL;
    h->data = (int*)malloc(sizeof(int) * capacity);
    if (h->data == NULL) {
        free(h);
        return NULL;
    }
    h->capacity = capacity;
    h->size = 0;
    return h;
}

void heap_destroy(MaxHeap *h) {
    if (h != NULL) {
        free(h->data);
        free(h);
    }
}

/* 上浮：从索引 i 开始，如果比父节点大就交换，一直上浮到合适位置 */
static void heap_sift_up(MaxHeap *h, int i) {
    int parent = (i - 1) / 2;
    while (i > 0 && h->data[i] > h->data[parent]) {
        /* 交换 */
        int tmp = h->data[i];
        h->data[i] = h->data[parent];
        h->data[parent] = tmp;
        /* 继续上浮 */
        i = parent;
        parent = (i - 1) / 2;
    }
}

/* 插入：放到数组末尾，然后上浮 */
int heap_insert(MaxHeap *h, int value) {
    if (h == NULL || h->size >= h->capacity) return -1;
    h->data[h->size] = value;
    heap_sift_up(h, h->size);
    h->size++;
    return 0;
}

/* 下沉：从索引 i 开始，找到最大的孩子，如果比孩子小就交换，一直下沉 */
static void heap_sift_down(MaxHeap *h, int i) {
    while (1) {
        int largest = i;
        int left = 2 * i + 1;
        int right = 2 * i + 2;

        if (left < h->size && h->data[left] > h->data[largest]) {
            largest = left;
        }
        if (right < h->size && h->data[right] > h->data[largest]) {
            largest = right;
        }

        if (largest == i) break;  /* 已经满足堆性质 */

        /* 交换 */
        int tmp = h->data[i];
        h->data[i] = h->data[largest];
        h->data[largest] = tmp;

        i = largest;  /* 继续下沉 */
    }
}

/* 取出最大值：取根，把最后一个元素移到根，然后下沉 */
int heap_extract_max(MaxHeap *h, int *out) {
    if (h == NULL || h->size == 0) return -1;
    *out = h->data[0];
    h->data[0] = h->data[h->size - 1];
    h->size--;
    heap_sift_down(h, 0);
    return 0;
}

int heap_peek_max(const MaxHeap *h, int *out) {
    if (h == NULL || h->size == 0) return -1;
    *out = h->data[0];
    return 0;
}

int heap_is_empty(const MaxHeap *h) {
    return h->size == 0;
}

int heap_size(const MaxHeap *h) {
    return h->size;
}
```

**上浮（sift_up）图解**：

```
插入 60 到大顶堆中:

             50
           /    \
          30    40
         /  \   / \
        10  20 15 35

1. 放到末尾: 数组变为 [50, 30, 40, 10, 20, 15, 35, 60]
   60 在索引7，父节点索引 (7-1)/2=3，即 10

2. 60 > 10，交换: 60 到索引3，10 到索引7
   父节点索引 (3-1)/2=1，即 30

3. 60 > 30，交换: 60 到索引1，30 到索引3
   父节点索引 (1-1)/2=0，即 50

4. 60 > 50，交换: 60 到索引0，50 到索引1
   当前在索引0，是根，停止

最终堆:
             60
           /    \
          50    40
         /  \   / \
        30  20 15 35
       /
      10
```

**下沉（sift_down）图解**：

```
取出最大值 60 后，把最后一个元素10移到根:

             10
           /    \
          50    40
         /  \   / \
        30  20 15 35

1. 比较10和孩子：50 > 10，交换10和50
   50 到根，10 到索引1

2. 比较10和现在的孩子：30 > 10，交换10和30
   30 到索引1，10 到索引3

3. 索引3 没有孩子（或孩子超出范围），停止

最终堆:
             50
           /    \
          30    40
         /  \   / \
        10  20 15 35
```

## 48.7 堆排序

堆排序利用堆的性质来排序，分两步：
1. **建堆**：把所有元素插入堆（或原地建堆）
2. **依次取出最大值**：每次取根（最大值），放到数组末尾

```c
/* 堆排序：将数组原地排序为升序 */
void heap_sort(int arr[], int n) {
    /* 1. 建堆：从最后一个非叶子节点开始，依次下沉 */
    /* 最后一个非叶子节点的索引是 (n/2 - 1) */
    for (int i = n / 2 - 1; i >= 0; i--) {
        /* 下沉操作（需要临时把 arr 当作堆来操作） */
        int root = i;
        while (1) {
            int largest = root;
            int left = 2 * root + 1;
            int right = 2 * root + 2;

            if (left < n && arr[left] > arr[largest])
                largest = left;
            if (right < n && arr[right] > arr[largest])
                largest = right;

            if (largest == root) break;

            int tmp = arr[root];
            arr[root] = arr[largest];
            arr[largest] = tmp;
            root = largest;
        }
    }

    /* 2. 依次取出最大值，放到数组末尾 */
    for (int i = n - 1; i > 0; i--) {
        /* 把根（最大值）交换到末尾 */
        int tmp = arr[0];
        arr[0] = arr[i];
        arr[i] = tmp;

        /* 对缩小的堆重新下沉 */
        int root = 0;
        while (1) {
            int largest = root;
            int left = 2 * root + 1;
            int right = 2 * root + 2;

            if (left < i && arr[left] > arr[largest])
                largest = left;
            if (right < i && arr[right] > arr[largest])
                largest = right;

            if (largest == root) break;

            tmp = arr[root];
            arr[root] = arr[largest];
            arr[largest] = tmp;
            root = largest;
        }
    }
}
```

时间复杂度：O(n log n)，空间复杂度 O(1)。

## 48.8 优先队列——堆的典型应用

**优先队列**（Priority Queue）是一种特殊的队列，每次出队的不一定是先入队的，而是**优先级最高**的。用大顶堆实现，插入和删除都是 O(log n)。

```c
/* 优先队列就是前面的大顶堆 */
typedef MaxHeap PriorityQueue;

/* 入队：插入新任务 */
int pq_enqueue(PriorityQueue *pq, int priority) {
    return heap_insert(pq, priority);
}

/* 出队：取出最高优先级的任务 */
int pq_dequeue(PriorityQueue *pq, int *out) {
    return heap_extract_max(pq, out);
}
```

典型应用场景：
- 操作系统任务调度（高优先级任务先执行）
- 事件驱动的模拟系统
- Dijkstra 最短路径算法
- Huffman 编码

 * 编译: gcc -std=c99 -Wall -o heap heap_demo.c

#include <stdio.h>
#include <stdlib.h>

typedef struct {
    int *data;
    int  size;

MaxHeap* heap_create(int capacity) {

    h->size = 0;
    return h;
}

    if (h != NULL) { free(h->data); free(h); }
}

        int tmp = h->data[i];
        h->data[i] = h->data[parent];
        h->data[parent] = tmp;
        i = parent;
        parent = (i - 1) / 2;
    }
}

int heap_insert(MaxHeap *h, int value) {
    if (h->size >= h->capacity) return -1;
    h->data[h->size] = value;
    sift_up(h, h->size);
    h->size++;
    return 0;
}

static void sift_down(MaxHeap *h, int i) {
    while (1) {
        int largest = i;
        int left = 2 * i + 1;
        int right = 2 * i + 2;
        if (left < h->size && h->data[left] > h->data[largest])
            largest = left;
        if (right < h->size && h->data[right] > h->data[largest])
            largest = right;
        if (largest == i) break;
        int tmp = h->data[i];
        h->data[i] = h->data[largest];
        h->data[largest] = tmp;
        i = largest;
    }
}

int heap_extract_max(MaxHeap *h, int *out) {
    if (h->size == 0) return -1;
    *out = h->data[0];
    h->data[0] = h->data[h->size - 1];
    h->size--;
    sift_down(h, 0);
    return 0;
}

void heap_print(MaxHeap *h) {
    printf("堆内容: ");
    for (int i = 0; i < h->size; i++) {
        printf("%d ", h->data[i]);
    }
    printf("\n");
}

/* ==================== 堆排序 ==================== */
void heap_sort(int arr[], int n) {
    /* 建堆 */
    for (int i = n / 2 - 1; i >= 0; i--) {
        int root = i;
        while (1) {
            int largest = root;
            int left = 2 * root + 1;
            int right = 2 * root + 2;
            if (left < n && arr[left] > arr[largest]) largest = left;
            if (right < n && arr[right] > arr[largest]) largest = right;
            if (largest == root) break;
            int tmp = arr[root]; arr[root] = arr[largest]; arr[largest] = tmp;
            root = largest;
        }
    }
    /* 排序 */
    for (int i = n - 1; i > 0; i--) {
        int tmp = arr[0]; arr[0] = arr[i]; arr[i] = tmp;
        int root = 0;
        while (1) {
            int largest = root;
            int left = 2 * root + 1;
            int right = 2 * root + 2;
            if (left < i && arr[left] > arr[largest]) largest = left;
            if (right < i && arr[right] > arr[largest]) largest = right;
            if (largest == root) break;
            tmp = arr[root]; arr[root] = arr[largest]; arr[largest] = tmp;
            root = largest;
        }
    }
}

/* ==================== 演示 ==================== */
int main(void) {
    printf("===== 大顶堆演示 =====\n");
    MaxHeap *h = heap_create(20);
    int values[] = {3, 10, 5, 20, 15, 8, 30};
    int n = sizeof(values) / sizeof(values[0]);

    for (int i = 0; i < n; i++) {
        heap_insert(h, values[i]);
        printf("插入 %d: ", values[i]);
        heap_print(h);
    }

    printf("\n依次取出最大值: ");
    int val;
    while (h->size > 0) {
        heap_extract_max(h, &val);
        printf("%d ", val);
    }
    printf("\n");

    heap_destroy(h);

    printf("\n===== 堆排序演示 =====\n");
    int arr[] = {12, 11, 13, 5, 6, 7};
    int arr_n = sizeof(arr) / sizeof(arr[0]);
    printf("排序前: ");
    for (int i = 0; i < arr_n; i++) printf("%d ", arr[i]);
    printf("\n");

    heap_sort(arr, arr_n);

    printf("排序后: ");
    for (int i = 0; i < arr_n; i++) printf("%d ", arr[i]);
    printf("\n");

    return 0;
}

## 我做得对不对？

1. **堆插入**：每次插入后，根节点应该保持最大值。
2. **堆取出**：依次取出应该得到降序序列（最大值 → 最小值）。
3. **堆排序**：排序后数组应该升序。
4. **堆性质**：`heap_print` 打印的数组符合堆的数组表示，父节点索引 `(i-1)/2` 的值 >= 子节点。

---

## 不对怎么办？

### 常见Bug 1：数组索引算错

```c
/* ❌ 父节点索引算错 */
int parent = i / 2;   /* 当 i=0 时 parent=0，死循环！ */
int left = 2 * i;      /* 应该是 2*i+1 */
int right = 2 * i + 1; /* 应该是 2*i+2 */
```

✅ 正确：`parent = (i-1)/2`，`left = 2*i+1`，`right = 2*i+2`

### 常见Bug 2：上浮循环条件错误

```c
/* ❌ 没有检查 i>0，i=0 时 parent=(0-1)/2 在整数除法下也是0 */
while (h->data[i] > h->data[parent]) {
    /* 如果 i=0 且 parent=0，会死循环 */
}
```

✅ 正确：`while (i > 0 && h->data[i] > h->data[parent])`

### 常见Bug 3：堆排序第二轮没有更新堆大小

```c
/* ❌ 第二轮下沉时，范围还是 n，没有缩小到 i */
for (int i = n - 1; i > 0; i--) {
    /* 交换后... */
    /* 下沉时 left < n（错误！应该 left < i） */
}
```

✅ 第二轮下沉时，左孩子应该 `< i`（因为已排序部分在数组末尾）

### 常见Bug 4：下沉时忘记 `break` 导致死循环

```c
/* ❌ largest==i 时没有 break */
while (1) {
    /* ... 比较逻辑 ... */
    /* 如果 largest == i，说明不需要交换，应该 break */
}
```

✅ 当 `largest == i` 时，必须 `break` 退出循环。

## 本章小结

| 要点 | 说明 |
|------|------|
| BST退化问题 | 插入有序数据退化成链表，查找 O(n) |
| AVL平衡因子 | 左右子树高度差 ≤ 1，否则旋转 |
| 四种旋转 | LL(右旋)、RR(左旋)、LR(先左后右)、RL(先右后左) |
| 堆的定义 | 大顶堆(父>=子)、小顶堆(父<=子) |
| 堆的数组实现 | 完全二叉树，索引公式：父(i-1)/2, 左2i+1, 右2i+2 |
| 上浮(sift_up) | 新元素从末尾向上冒泡，O(log n) |
| 下沉(sift_down) | 根元素向下调整，O(log n) |
| 堆排序 | 建堆 O(n) + n次取最大 O(n log n)，原地排序 |
| 优先队列 | 用堆实现，每次取最高优先级元素 |
