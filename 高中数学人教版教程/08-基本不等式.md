## 08.1 基本不等式的推导 —— 从完全平方公式出发

### 08.1.1 一个平方数的启示

请回忆初中学过的完全平方公式：

$$(a - b)^2 = a^2 - 2ab + b^2$$

现在，把 $a$ 和 $b$ 换成 $\sqrt{a}$ 和 $\sqrt{b}$（假设 $a \geq 0$，$b \geq 0$）：

$$(\sqrt{a} - \sqrt{b})^2 = a - 2\sqrt{ab} + b$$

> 💡 **为什么要换？** 因为 $(\sqrt{a})^2 = a$，$(\sqrt{b})^2 = b$，这样展开后就没有根号了，式子变得干净。

---

### 08.1.2 关键一步：平方数永远非负

任何一个实数的平方都 $\geq 0$，这是数学中最基本的事实之一。所以：

$$(\sqrt{a} - \sqrt{b})^2 \geq 0$$

展开左边：

$$a - 2\sqrt{ab} + b \geq 0$$

把 $-2\sqrt{ab}$ 移到右边（或者说两边同时加上 $2\sqrt{ab}$）：

$$a + b \geq 2\sqrt{ab}$$

两边同时除以 2：

$$\frac{a + b}{2} \geq \sqrt{ab}$$

> 🎉 **这就是"基本不等式"！** 完整表述为：

$$\boxed{\frac{a + b}{2} \geq \sqrt{ab} \quad (a > 0,\; b > 0)}$$

当且仅当 $a = b$ 时，等号成立。

---

### 08.1.3 每一步都不要跳——完整推导链

下面把推导过程完整写出来，每一步标出依据：

$$\begin{aligned}
(\sqrt{a} - \sqrt{b})^2 &\geq 0                                           &&\text{—— 任何实数的平方 ≥ 0} \\[4pt]
a - 2\sqrt{ab} + b &\geq 0                                                &&\text{—— 展开平方} \\[4pt]
a + b &\geq 2\sqrt{ab}                                                    &&\text{—— 两边同时加 } 2\sqrt{ab} \\[4pt]
\frac{a + b}{2} &\geq \sqrt{ab}                                            &&\text{—— 两边同时除以 2（2 > 0，不等号方向不变）}
\end{aligned}$$

**等号成立条件**：$(\sqrt{a} - \sqrt{b})^2 = 0 \;\Leftrightarrow\; \sqrt{a} = \sqrt{b} \;\Leftrightarrow\; a = b$。

> 🤔 **想多一点**：为什么推导的起点是 $(\sqrt{a} - \sqrt{b})^2 \geq 0$？因为平方非负是数学中"最可靠的真理"之一——它不需要任何前提条件，永远成立。从这样一个"绝对真理"出发推导出的结论，也一定是可靠的。这是数学证明中常用的策略：**找一个绝对成立的起点，一步步推出目标**。

---

### 08.1.4 两个关键名称

| 名称 | 表达式 | 含义 |
|------|--------|------|
| **算术平均数** | $\frac{a + b}{2}$ | 两个数加起来除以 2——日常说的"平均数" |
| **几何平均数** | $\sqrt{ab}$ | 两个数乘积的平方根——"几何"一词来源于面积类比 |

> 💡 **为什么叫几何平均数？** 考虑一个长为 $a$、宽为 $b$ 的矩形，面积是 $ab$。$\sqrt{ab}$ 就是与这个矩形面积相等的正方形的边长。所以几何平均数本质上是一个"等面积正方化"的操作——把矩形变成等面积的正方形，新正方形的边长就是几何平均数。这个解释来自几何，所以叫"几何平均数"。

**基本不等式用文字表述就是**：

> 两个正数的算术平均数 $\geq$ 它们的几何平均数。

## 08.2 "一正二定三相等" —— 用基本不等式求最值的口诀

> 基本不等式不只是一个漂亮的数学结论，它更是一个**求最值的工具**。但使用这个工具有三个前提条件，缺一不可。这句口诀帮你记住它们。

### 08.2.1 一正：$a$ 和 $b$ 必须是正数

**这是使用基本不等式的绝对前提。**

为什么？因为 $\sqrt{ab}$ 要求 $a, b \geq 0$（否则根号下会出现负数），而等号成立条件 $a = b$ 加上 $a, b > 0$ 才能保证整个推导有意义。

- ❌ **错误**：$a = -1, b = -4$，写出 $\frac{(-1) + (-4)}{2} \geq \sqrt{(-1)(-4)}$，即 $-2.5 \geq 2$ —— **荒谬！** 两边都是负数时，左边小于右边，不等式不成立。
- ✅ **正确**：使用基本不等式前，必须先确认 $a > 0$ 且 $b > 0$。

> ⚠️ **常见陷阱**：题目给 $a, b$ 没有明确说正数，但隐含了正数条件（比如是边长、面积、价格等）。你需要自己判断并**先写出"$a > 0, b > 0$"**，再使用基本不等式。

---

### 08.2.2 二定：和或积必须为定值

基本不等式联系了"和"（$a + b$）和"积"（$ab$）。如果其中一个为定值，另一个就有最值。具体来说：

| 已知条件 | 结论 | 等号成立时 |
|---------|------|-----------|
| $a + b = S$（定值） | $ab \leq \frac{S^2}{4}$ —— 积有最大值 | $a = b = \frac{S}{2}$ |
| $ab = P$（定值） | $a + b \geq 2\sqrt{P}$ —— 和有最小值 | $a = b = \sqrt{P}$ |

**推导**：

**情况一：和定积最大。**
由 $\frac{a + b}{2} \geq \sqrt{ab}$，两边平方得 $\frac{(a + b)^2}{4} \geq ab$。
若 $a + b = S$（定值），则 $ab \leq \frac{S^2}{4}$，当 $a = b$ 时取等号。

**情况二：积定和最小。**
由 $\frac{a + b}{2} \geq \sqrt{ab}$，两边乘以 2 得 $a + b \geq 2\sqrt{ab}$。
若 $ab = P$（定值），则 $a + b \geq 2\sqrt{P}$，当 $a = b$ 时取等号。

> 💡 **记忆口诀**：
> - 和一定 → 积有最大值（$a = b$ 时达到）
> - 积一定 → 和有最小值（$a = b$ 时达到）
> - 共同点：都是 $a = b$ 时取到最值——"两个数相等时最极端"

---

### 08.2.3 三相等：验证等号能否取到

**这是最容易被忽略的一步，但也是考试扣分最多的点。**

写基本不等式求最值时，必须验证等号成立条件是否满足题目条件。如果不满足，就不能取到那个最值。

**验证方法**：
1. 写出等号成立条件：$a = b$。
2. 代入题目条件，看 $a = b$ 是否与题目条件兼容。
3. 如果兼容，给出 $a = b$ 的具体值；如果不兼容，说明不能取到等号，最值实际是另一个值。

> ⚠️ **经典反例**：已知 $a > 0, b > 0$，$a + b > 2$，求 $ab$ 的最大值。
> - 如果直接写 $ab \leq \left(\frac{a + b}{2}\right)^2$，但 $a + b$ 不是定值，就不能用"和定积最大"。
> - 即使 $a + b = 2$ 是定值，等号成立时 $a = b = 1$，也需要验证 $a = b = 1$ 确实满足 $a + b = 2$。

---

### 08.2.4 "一正二定三相等"完整示例

**题目**：已知 $x > 0$，求 $y = x + \frac{1}{x}$ 的最小值。

**解**：

**第一步：验证"一正"。**
$x > 0$，$\frac{1}{x} > 0$，满足正数条件。✅

**第二步：验证"二定"。**
$x \cdot \frac{1}{x} = 1$（定值）。积为定值，和有最小值。✅

**第三步：使用基本不等式。**
$$y = x + \frac{1}{x} \geq 2\sqrt{x \cdot \frac{1}{x}} = 2\sqrt{1} = 2$$

**第四步：验证"三相等"。**
等号成立当且仅当 $x = \frac{1}{x}$，即 $x^2 = 1$，解得 $x = 1$（因为 $x > 0$，舍去 $x = -1$）。
$x = 1$ 满足 $x > 0$ 的条件。✅

**结论**：$y_{\min} = 2$，当 $x = 1$ 时取得。

> 🧠 **反思**：这道题是基本不等式最经典的入门题。很多同学做到第三步就停了，但第四步"三相等"才是完整答案不可或缺的部分。**没有写等号成立条件的答案，在考试中会扣分。**

---

## 08.3 基本不等式的应用 —— 典型题型

### 08.3.1 题型一：已知和求积的最大值

**例题**：已知 $a > 0, b > 0$，$a + b = 6$，求 $ab$ 的最大值。

**解**：

**一正**：$a > 0, b > 0$。✅

**二定**：$a + b = 6$ 为定值，积有最大值。✅

由基本不等式：$\frac{a + b}{2} \geq \sqrt{ab}$。

代入 $a + b = 6$：$\frac{6}{2} \geq \sqrt{ab}$，即 $3 \geq \sqrt{ab}$。

两边平方（$3 > 0$，$\sqrt{ab} \geq 0$，平方不改变不等号方向）：$9 \geq ab$，即 $ab \leq 9$。

**三相等**：等号成立当且仅当 $a = b$。由 $a + b = 6$ 且 $a = b$，得 $a = b = 3$。✅

**结论**：$ab$ 的最大值为 $9$，当 $a = b = 3$ 时取得。

> 💡 **生活比喻**：用固定长度的篱笆围一个矩形菜地，什么时候面积最大？答案是正方形时。$a + b$ 固定（周长的二分之一固定），$ab$（面积）在 $a = b$ 时最大。这就是"和定积最大"的几何含义。

---

### 08.3.2 题型二：已知积求和的最小值

**例题**：已知 $a > 0, b > 0$，$ab = 16$，求 $a + b$ 的最小值。

**解**：

**一正**：$a > 0, b > 0$。✅

**二定**：$ab = 16$ 为定值，和有最小值。✅

由基本不等式：$a + b \geq 2\sqrt{ab} = 2\sqrt{16} = 2 \times 4 = 8$。

**三相等**：等号成立当且仅当 $a = b$。由 $ab = 16$ 且 $a = b$，得 $a^2 = 16$，$a = 4$（因为 $a > 0$，舍去 $-4$）。所以 $a = b = 4$。✅

**结论**：$a + b$ 的最小值为 $8$，当 $a = b = 4$ 时取得。

---

### 08.3.3 题型三：配凑法 —— 创造"积定"或"和定"的条件

> 很多题目不会直接给"和"或"积"为定值，需要你**自己配凑**出来。

**例题**：已知 $x > 0$，求 $y = 2x + \frac{8}{x}$ 的最小值。

**解**：

**一正**：$x > 0$，所以 $2x > 0$，$\frac{8}{x} > 0$。✅

**二定**：$2x \cdot \frac{8}{x} = 16$（定值）。积为定值，和有最小值。✅

由基本不等式：
$$y = 2x + \frac{8}{x} \geq 2\sqrt{2x \cdot \frac{8}{x}} = 2\sqrt{16} = 2 \times 4 = 8$$

**三相等**：等号成立当且仅当 $2x = \frac{8}{x}$，即 $2x^2 = 8$，$x^2 = 4$，$x = 2$（因为 $x > 0$）。✅

**结论**：$y_{\min} = 8$，当 $x = 2$ 时取得。

---

### ⚠️ 08.3.4 常见错误

**错误 1：忽略"一正"条件**

- ❌ 已知 $x < 0$，求 $y = x + \frac{1}{x}$ 的最小值。
  - 如果直接套 $y \geq 2$，就犯了"忽略正数条件"的错误。实际上 $x < 0$ 时，$x + \frac{1}{x} \leq -2$（通过令 $t = -x > 0$ 转化后再用基本不等式）。

**错误 2：忘记验证"三相等"**

- ❌ 已知 $a > 0, b > 0$，$a + b \geq 3$，求 $ab$ 的最小值。
  - 如果写 $ab \leq \left(\frac{a + b}{2}\right)^2$，但 $a + b$ 不是定值（只是 $\geq 3$），根本不能用"和定积最大"。而且这里求的是最小值，套路不对。这种题需要用其他方法。

**错误 3：错误地创造"定值"**

- ❌ 已知 $x > 0$，求 $y = x + \frac{1}{x + 1}$ 的最小值。
  - 如果直接写 $x \cdot \frac{1}{x + 1} \neq$ 常数，不能直接用基本不等式。需要先变形：$y = (x + 1) + \frac{1}{x + 1} - 1$，令 $t = x + 1 > 1$，则 $y = t + \frac{1}{t} - 1 \geq 2 - 1 = 1$。

## 08.4 重要不等式链

> 基本不等式只是"均值不等式家族"中的一个成员。完整的家族有四个成员，它们之间有一个漂亮的不等式链。

### 08.4.1 四个均值

对于两个正数 $a, b$，定义：

| 名称 | 表达式 | 通俗理解 |
|------|--------|---------|
| **调和平均数** $H$ | $\frac{2}{\frac{1}{a} + \frac{1}{b}}$ | "倒数的平均的倒数" |
| **几何平均数** $G$ | $\sqrt{ab}$ | 与矩形等积的正方形的边长 |
| **算术平均数** $A$ | $\frac{a + b}{2}$ | 日常说的平均数 |
| **平方平均数** $Q$ | $\sqrt{\frac{a^2 + b^2}{2}}$ | 先平方、再平均、再开方 |

### 08.4.2 不等式链

对于任意正数 $a, b$，有：

$$\boxed{\frac{2}{\frac{1}{a} + \frac{1}{b}} \;\leq\; \sqrt{ab} \;\leq\; \frac{a + b}{2} \;\leq\; \sqrt{\frac{a^2 + b^2}{2}}}$$

即：

$$H \leq G \leq A \leq Q$$

> 📝 **记忆口诀**：调几算平 —— 调和 $\leq$ 几何 $\leq$ 算术 $\leq$ 平方

等号成立条件：所有等号都在 $a = b$ 时同时成立。

> 💡 **为什么叫"调和平均"？** 调和这个词来自音乐——如果两根弦的长度比为 $a : b$，它们的调和平均数对应的弦长所产生的音高，恰好是这两根弦音高的"调和"（和谐组合）。在物理学中，并联电阻的等效电阻、平均速度等问题中都会出现调和平均。

---

### 08.4.3 不等式链的证明（只证 $G \leq A$ 和 $A \leq Q$，其余从略）

**$G \leq A$（几何 ≤ 算术）**：就是基本不等式 $\frac{a + b}{2} \geq \sqrt{ab}$，已在 08.1 中完整证明。

**$A \leq Q$（算术 ≤ 平方）**：

要证 $\frac{a + b}{2} \leq \sqrt{\frac{a^2 + b^2}{2}}$。

两边平方（两边均为正数）：$\frac{(a + b)^2}{4} \leq \frac{a^2 + b^2}{2}$。

两边乘以 4：$(a + b)^2 \leq 2(a^2 + b^2)$。

展开：$a^2 + 2ab + b^2 \leq 2a^2 + 2b^2$。

移项：$0 \leq a^2 - 2ab + b^2$。

即 $0 \leq (a - b)^2$。这显然成立（平方非负）。

等号成立当且仅当 $a = b$。∎

> 🤔 **想多一点**：$A \leq Q$ 的证明最后也归结到 $(a - b)^2 \geq 0$——和基本不等式是同一个根源。实际上，整个均值不等式链的根基都是"平方非负"这个简单事实。数学的美妙就在于此：一个简单的起点，推导出丰富的结论。

---

## 08.5 例题精讲

### 例题 1 —— 基本不等式直接应用

**题目**：已知 $a > 0, b > 0$，$a + b = 10$，求 $\frac{1}{a} + \frac{1}{b}$ 的最小值。

**解**：

**第一步**：尝试直接使用基本不等式。

$\frac{1}{a} + \frac{1}{b} = \frac{a + b}{ab} = \frac{10}{ab}$。

要求 $\frac{10}{ab}$ 的最小值，等价于求 $ab$ 的最大值。

**第二步**：求 $ab$ 的最大值。

由 $a > 0, b > 0$，$a + b = 10$（定值），根据"和定积最大"：
$$ab \leq \left(\frac{a + b}{2}\right)^2 = \left(\frac{10}{2}\right)^2 = 25$$

等号成立当且仅当 $a = b = 5$。

**第三步**：代入。

$\frac{1}{a} + \frac{1}{b} = \frac{10}{ab} \geq \frac{10}{25} = \frac{2}{5}$。

**三相等**：$a = b = 5$ 时，$\frac{1}{a} + \frac{1}{b} = \frac{1}{5} + \frac{1}{5} = \frac{2}{5}$。✅

**结论**：$\frac{1}{a} + \frac{1}{b}$ 的最小值为 $\frac{2}{5}$，当 $a = b = 5$ 时取得。

---

### 例题 2 —— 配凑法

**题目**：已知 $x > 1$，求 $y = x + \frac{1}{x - 1}$ 的最小值。

**解**：

**第一步**：$x > 1$ 意味着 $x - 1 > 0$。但 $x$ 和 $\frac{1}{x - 1}$ 的乘积不是常数，不能直接套用基本不等式。

**配凑**：把 $x$ 拆成 $(x - 1) + 1$：
$$y = (x - 1) + 1 + \frac{1}{x - 1} = (x - 1) + \frac{1}{x - 1} + 1$$

**第二步**：令 $t = x - 1 > 0$，则 $y = t + \frac{1}{t} + 1$。

**一正**：$t > 0$，$\frac{1}{t} > 0$。✅

**二定**：$t \cdot \frac{1}{t} = 1$（定值）。✅

由基本不等式：$t + \frac{1}{t} \geq 2\sqrt{t \cdot \frac{1}{t}} = 2$。

所以 $y = t + \frac{1}{t} + 1 \geq 2 + 1 = 3$。

**三相等**：$t = \frac{1}{t}$，即 $t^2 = 1$，$t = 1$（$t > 0$）。$t = 1$ 即 $x - 1 = 1$，$x = 2$。✅

**结论**：$y_{\min} = 3$，当 $x = 2$ 时取得。

> 🧠 **反思**：配凑的核心思想是**把不是"积定"的式子变成"积定"**。$x$ 和 $\frac{1}{x - 1}$ 积不是常数，但 $(x - 1)$ 和 $\frac{1}{x - 1}$ 的积是常数 1。所以把 $x$ 拆成 $(x - 1) + 1$，多出来的 $+1$ 放在外面，不参与基本不等式。

---

### 例题 3 —— 不等式链的应用

**题目**：已知 $a > 0, b > 0$，且 $a + b = 1$，求证：

(1) $ab \leq \frac{1}{4}$
(2) $a^2 + b^2 \geq \frac{1}{2}$
(3) $\frac{1}{a} + \frac{1}{b} \geq 4$

**证明**：

**(1)** 由基本不等式：$\frac{a + b}{2} \geq \sqrt{ab}$。

代入 $a + b = 1$：$\frac{1}{2} \geq \sqrt{ab}$。

两边平方：$\frac{1}{4} \geq ab$，即 $ab \leq \frac{1}{4}$。

等号成立当且仅当 $a = b = \frac{1}{2}$。∎

**(2)** 由不等式链 $A \leq Q$：$\frac{a + b}{2} \leq \sqrt{\frac{a^2 + b^2}{2}}$。

两边平方：$\frac{(a + b)^2}{4} \leq \frac{a^2 + b^2}{2}$。

代入 $a + b = 1$：$\frac{1}{4} \leq \frac{a^2 + b^2}{2}$。

两边乘以 2：$\frac{1}{2} \leq a^2 + b^2$，即 $a^2 + b^2 \geq \frac{1}{2}$。

等号成立当且仅当 $a = b = \frac{1}{2}$。∎

**(3)** $\frac{1}{a} + \frac{1}{b} = \frac{a + b}{ab} = \frac{1}{ab}$。

由 (1) 知 $ab \leq \frac{1}{4}$，两边取倒数（$ab > 0$，不等号反转）：$\frac{1}{ab} \geq 4$。

即 $\frac{1}{a} + \frac{1}{b} \geq 4$。

等号成立当且仅当 $a = b = \frac{1}{2}$。∎

> 🧠 **反思**：这道题展示了基本不等式和不等式链在一个具体问题上的联合应用。注意 (3) 用了 (1) 的结论——"一步推导、多次使用"是数学证明中的常见策略。

---

### 例题 4 —— 综合应用：变形与配凑

**题目**：已知 $a > 0, b > 0$，$ab = 1$，求 $\frac{1}{a} + \frac{1}{b} + \frac{1}{a + b}$ 的最小值。

**解**：

**第一步**：因为 $ab = 1$，所以 $\frac{1}{a} = b$，$\frac{1}{b} = a$。

$$\frac{1}{a} + \frac{1}{b} + \frac{1}{a + b} = b + a + \frac{1}{a + b} = (a + b) + \frac{1}{a + b}$$

**第二步**：令 $t = a + b$。由 $a > 0, b > 0$ 得 $t > 0$。原式变为 $t + \frac{1}{t}$。

**一正**：$t > 0$。✅

**二定**：$t \cdot \frac{1}{t} = 1$（定值）。✅

由基本不等式：$t + \frac{1}{t} \geq 2\sqrt{t \cdot \frac{1}{t}} = 2$。

**三相等**：$t = \frac{1}{t}$，即 $t^2 = 1$，$t = 1$（$t > 0$）。

由 $a + b = 1$ 且 $ab = 1$，解方程组：
代入 $b = 1 - a$ 到 $ab = 1$：$a(1 - a) = 1$，即 $a^2 - a + 1 = 0$。

判别式 $\Delta = 1 - 4 = -3 < 0$，无实数解。

> ⚠️ **等号取不到！**

这意味着 $a + b = 1$ 且 $ab = 1$ 不能同时成立。实际上，由 $ab = 1$ 和基本不等式得 $a + b \geq 2\sqrt{ab} = 2$，所以 $t = a + b \geq 2$。

**第四步**：因为 $t \geq 2$，而 $y = t + \frac{1}{t}$ 在 $t \geq 2$ 时，由基本不等式：

$t + \frac{1}{t} \geq 2$，但等号在 $t = 1$ 时取到，而 $t \geq 2$，所以这里不能直接用基本不等式求最小值。

我们需要单独分析 $y = t + \frac{1}{t}$ 在 $t \geq 2$ 时的最小值。

令 $f(t) = t + \frac{1}{t}$，$t \geq 2$。

对 $t_2 > t_1 \geq 2$：
$$f(t_2) - f(t_1) = (t_2 - t_1) + \left(\frac{1}{t_2} - \frac{1}{t_1}\right) = (t_2 - t_1) - \frac{t_2 - t_1}{t_1 t_2} = (t_2 - t_1)\left(1 - \frac{1}{t_1 t_2}\right)$$

当 $t_1, t_2 \geq 2$ 时，$t_1 t_2 \geq 4$，$1 - \frac{1}{t_1 t_2} \geq 1 - \frac{1}{4} > 0$，所以 $f(t_2) > f(t_1)$，$f(t)$ 在 $[2, +\infty)$ 上单调递增。

因此 $f(t)_{\min} = f(2) = 2 + \frac{1}{2} = \frac{5}{2}$。

当 $t = 2$ 时，即 $a + b = 2$ 且 $ab = 1$，解得 $a = b = 1$。

**结论**：最小值为 $\frac{5}{2}$，当 $a = b = 1$ 时取得。

> 🧠 **反思**：这道题是一个"陷阱题"——表面上能用基本不等式，但等号成立条件与题目条件冲突。**这正是"三相等"的价值所在**：验证等号能否取到，如果取不到，需要用其他方法求解。这道题也展示了"等号取不到时，用单调性判断"的进阶思路。

## 08.6 常见错误辨析

| 错误 | 错在哪里？ | 正确做法 |
|------|-----------|---------|
| ❌ $a, b < 0$ 时直接用 $\frac{a + b}{2} \geq \sqrt{ab}$ | 基本不等式要求 $a, b > 0$ | 先判断 $a, b$ 是否为正，若为负则令 $t = -a > 0$ 转化 |
| ❌ 求最值后不写等号成立条件 | 缺少"三相等"步骤，答案不完整 | 必须写出等号成立时 $a, b$ 的具体值 |
| ❌ $a + b$ 不是定值却用"和定积最大" | "二定"条件不满足 | 需要先配凑出定值，或者换方法 |
| ❌ $ab$ 不是定值却用"积定和最小" | "二定"条件不满足 | 同上 |
| ❌ 等号成立条件与题目条件冲突，但不管 | 等号取不到，最值实际不是那个值 | 需要验证等号能否取到，取不到则用其他方法 |
| ❌ 直接写 $a + \frac{1}{a + 1} \geq 2$ | $a$ 与 $\frac{1}{a + 1}$ 的积不是常数 | 先变形：$a + \frac{1}{a + 1} = (a + 1) + \frac{1}{a + 1} - 1$ |

## 08.7 练习题

### 基础题

**第 1 题**：已知 $x > 0$，求下列函数的最小值，并写出等号成立条件。

(1) $y = x + \frac{4}{x}$
(2) $y = 3x + \frac{12}{x}$

**第 2 题**：已知 $a > 0, b > 0$，$a + b = 8$，求 $ab$ 的最大值及此时的 $a, b$ 值。

**第 3 题**：已知 $a > 0, b > 0$，$ab = 9$，求 $a + b$ 的最小值及此时的 $a, b$ 值。

### 提高题

**第 4 题**：已知 $x > 2$，求 $y = x + \frac{1}{x - 2}$ 的最小值。

**第 5 题**：已知 $a > 0, b > 0$，$a + b = 1$，求 $\frac{1}{a} + \frac{4}{b}$ 的最小值。（提示：$\frac{1}{a} + \frac{4}{b} = \frac{b + 4a}{ab} = \frac{4a + b}{ab}$，但这样不好。更好的方法是：$\frac{1}{a} + \frac{4}{b} = \left(\frac{1}{a} + \frac{4}{b}\right) \cdot 1 = \left(\frac{1}{a} + \frac{4}{b}\right)(a + b)$，展开后使用基本不等式。）

**第 6 题**：已知 $a > 0, b > 0$，$a + b = 1$，求证：$\left(a + \frac{1}{a}\right)^2 + \left(b + \frac{1}{b}\right)^2 \geq \frac{25}{2}$。

## 08.8 练习题答案

### 第 1 题

**(1)** $y = x + \frac{4}{x}$

**一正**：$x > 0$，$\frac{4}{x} > 0$。✅

**二定**：$x \cdot \frac{4}{x} = 4$（定值）。✅

**基本不等式**：$y \geq 2\sqrt{x \cdot \frac{4}{x}} = 2\sqrt{4} = 4$。

**三相等**：$x = \frac{4}{x}$，即 $x^2 = 4$，$x = 2$（$x > 0$）。✅

**答**：$y_{\min} = 4$，当 $x = 2$ 时取得。

**(2)** $y = 3x + \frac{12}{x}$

**一正**：$x > 0$，$3x > 0$，$\frac{12}{x} > 0$。✅

**二定**：$3x \cdot \frac{12}{x} = 36$（定值）。✅

**基本不等式**：$y \geq 2\sqrt{3x \cdot \frac{12}{x}} = 2\sqrt{36} = 12$。

**三相等**：$3x = \frac{12}{x}$，即 $3x^2 = 12$，$x^2 = 4$，$x = 2$（$x > 0$）。✅

**答**：$y_{\min} = 12$，当 $x = 2$ 时取得。

---

### 第 2 题

**一正**：$a > 0, b > 0$。✅

**二定**：$a + b = 8$（定值），积有最大值。✅

由基本不等式：$\frac{a + b}{2} \geq \sqrt{ab}$。

代入：$\frac{8}{2} \geq \sqrt{ab}$，即 $4 \geq \sqrt{ab}$。

两边平方：$16 \geq ab$，即 $ab \leq 16$。

**三相等**：$a = b$，且 $a + b = 8$，得 $a = b = 4$。✅

**答**：$ab$ 的最大值为 $16$，此时 $a = b = 4$。

---

### 第 3 题

**一正**：$a > 0, b > 0$。✅

**二定**：$ab = 9$（定值），和有最小值。✅

由基本不等式：$a + b \geq 2\sqrt{ab} = 2\sqrt{9} = 6$。

**三相等**：$a = b$，且 $ab = 9$，得 $a^2 = 9$，$a = 3$（$a > 0$），所以 $a = b = 3$。✅

**答**：$a + b$ 的最小值为 $6$，此时 $a = b = 3$。

---

### 第 4 题

$y = x + \frac{1}{x - 2}$

**配凑**：$x = (x - 2) + 2$，所以：
$$y = (x - 2) + 2 + \frac{1}{x - 2} = (x - 2) + \frac{1}{x - 2} + 2$$

令 $t = x - 2 > 0$（因为 $x > 2$），则 $y = t + \frac{1}{t} + 2$。

**一正**：$t > 0$，$\frac{1}{t} > 0$。✅

**二定**：$t \cdot \frac{1}{t} = 1$（定值）。✅

由基本不等式：$t + \frac{1}{t} \geq 2\sqrt{t \cdot \frac{1}{t}} = 2$。

所以 $y \geq 2 + 2 = 4$。

**三相等**：$t = \frac{1}{t}$，即 $t^2 = 1$，$t = 1$（$t > 0$）。$t = 1$ 即 $x - 2 = 1$，$x = 3$。✅

**答**：$y_{\min} = 4$，当 $x = 3$ 时取得。

---

### 第 5 题

$\frac{1}{a} + \frac{4}{b} = \left(\frac{1}{a} + \frac{4}{b}\right) \cdot 1 = \left(\frac{1}{a} + \frac{4}{b}\right)(a + b)$

展开：
$$\begin{aligned}
\left(\frac{1}{a} + \frac{4}{b}\right)(a + b) &= \frac{1}{a} \cdot a + \frac{1}{a} \cdot b + \frac{4}{b} \cdot a + \frac{4}{b} \cdot b \\
&= 1 + \frac{b}{a} + \frac{4a}{b} + 4 \\
&= 5 + \frac{b}{a} + \frac{4a}{b}
\end{aligned}$$

**一正**：$a > 0, b > 0$，所以 $\frac{b}{a} > 0$，$\frac{4a}{b} > 0$。✅

**二定**：$\frac{b}{a} \cdot \frac{4a}{b} = 4$（定值）。✅

由基本不等式：$\frac{b}{a} + \frac{4a}{b} \geq 2\sqrt{\frac{b}{a} \cdot \frac{4a}{b}} = 2\sqrt{4} = 4$。

所以原式 $= 5 + \frac{b}{a} + \frac{4a}{b} \geq 5 + 4 = 9$。

**三相等**：$\frac{b}{a} = \frac{4a}{b}$，即 $b^2 = 4a^2$，$b = 2a$（$a, b > 0$）。结合 $a + b = 1$，得 $a + 2a = 1$，$a = \frac{1}{3}$，$b = \frac{2}{3}$。✅

**答**：最小值为 $9$，当 $a = \frac{1}{3}, b = \frac{2}{3}$ 时取得。

> 💡 **技巧**：乘 $1 = (a + b)$ 是处理"分母含一次项、分子为常数"的常用配凑技巧。展开后会出现 $\frac{b}{a}$ 和 $\frac{a}{b}$ 这种互为倒数的项，乘积为常数，正好用基本不等式。

---

### 第 6 题

**证明**：

展开：$\left(a + \frac{1}{a}\right)^2 + \left(b + \frac{1}{b}\right)^2 = a^2 + 2 + \frac{1}{a^2} + b^2 + 2 + \frac{1}{b^2} = (a^2 + b^2) + \left(\frac{1}{a^2} + \frac{1}{b^2}\right) + 4$

由例题 3(2)，$a^2 + b^2 \geq \frac{1}{2}$。

对于 $\frac{1}{a^2} + \frac{1}{b^2}$，由基本不等式：
$$\frac{1}{a^2} + \frac{1}{b^2} \geq 2\sqrt{\frac{1}{a^2} \cdot \frac{1}{b^2}} = \frac{2}{ab}$$

由例题 3(1)，$ab \leq \frac{1}{4}$，所以 $\frac{1}{ab} \geq 4$，$\frac{2}{ab} \geq 8$。

因此：
$$\left(a + \frac{1}{a}\right)^2 + \left(b + \frac{1}{b}\right)^2 \geq \frac{1}{2} + 8 + 4 = \frac{25}{2}$$

等号成立当且仅当 $a = b = \frac{1}{2}$。∎

## 本章小结

| 知识点 | 核心内容 | 一句话提醒 |
|--------|---------|-----------|
| 基本不等式 | $\frac{a + b}{2} \geq \sqrt{ab}$（$a, b > 0$） | 推导起点：$(\sqrt{a} - \sqrt{b})^2 \geq 0$ |
| 算术平均数 | $\frac{a + b}{2}$ | 日常说的平均数 |
| 几何平均数 | $\sqrt{ab}$ | 等积正方形的边长 |
| 一正 | $a, b$ 必须为正数 | 先判断，再用！ |
| 二定 | 和或积必须为定值 | 和定积最大，积定和最小 |
| 三相等 | 验证等号能否取到 | ⚠️ 不写就扣分，取不到要换方法 |
| 不等式链 | $H \leq G \leq A \leq Q$ | 调几算平 |
| 配凑法 | 把不是定值的式子变形为定值 | 拆项、乘 1、换元 |
