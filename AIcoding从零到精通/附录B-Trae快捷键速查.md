# 附录B：Trae快捷键速查

> 本附录可直接打印或收藏，随时查阅。

---

## 核心快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+L` | 打开Chat | 和AI对话编程 |
| `Ctrl+I` | 打开Builder | 用自然语言创建完整项目 |
| `Tab` | 接受AI补全 | 接受AI灰色补全建议 |
| `Esc` | 拒绝AI补全 | 取消AI补全 |

---

## Builder模式操作

| 操作 | 说明 |
|------|------|
| 输入项目描述 | 用自然语言描述你想要的项目 |
| 查看规划 | Builder会展示项目结构和文件列表 |
| 确认/修改 | 确认生成或提出修改意见 |
| 预览 | 生成后可以预览效果 |

---

## Chat模式操作

| 操作 | 说明 |
|------|------|
| 选中代码 + 输入指令 | 针对选中代码进行操作 |
| `@file` | 引用文件 |
| `@folder` | 引用文件夹 |
| Apply | 将AI修改应用到文件 |

---

## 编辑操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存文件 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 替换 |
| `Ctrl+/` | 注释/取消注释 |

---

## 模式切换

| 操作 | 功能 |
|------|------|
| 点击Builder标签 | 切换到Builder模式 |
| 点击Chat标签 | 切换到Chat模式 |
| 点击文件标签 | 切换到代码编辑 |

---

> **提示**：Trae目前是新型IDE，快捷键可能随版本更新变化。如发现不生效，请查看Trae菜单栏的快捷键提示。
