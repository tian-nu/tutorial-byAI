## 13.3 奇偶性 —— 函数的"对称"

### 13.3.1 什么是奇偶性？从对称说起

想象你对着镜子看自己——镜子里外的你是**对称**的。函数的图像也有类似的"对称"现象，而奇偶性就是把这种对称用**代数式子**精确描述出来。

**偶函数**：图像关于 $y$ 轴对称。

把图像沿 $y$ 轴折叠，左右两边完全重合。这意味着：对于定义域内的任意 $x$，$x$ 和 $-x$ 对应的函数值相等。

$$f(-x) = f(x)$$

**奇函数**：图像关于原点对称。

把图像绕原点旋转 $180^\circ$，图像与自身重合。这意味着：对于定义域内的任意 $x$，$-x$ 对应的函数值是 $x$ 对应函数值的相反数。

$$f(-x) = -f(x)$$

> 💡 **记忆技巧**：
> - **偶**函数 → 关于 $y$ **轴**对称 → 折"叠"能重合 → $f(-x) = f(x)$（两个 $f$ 值相同）
> - **奇**函数 → 关于原点对称 → 旋"转"能重合 → $f(-x) = -f(x)$（$f$ 值变号）

### 13.3.2 绝对前提：定义域必须关于原点对称

> ⚠️ **这是判断奇偶性最容易被忽略的步骤，也是考试中最常见的扣分点。**

判断奇偶性之前，**第一步**永远是检查定义域是否关于原点对称。如果定义域不关于原点对称，那么函数既不是奇函数也不是偶函数——直接下结论，不需要再算。

**什么叫做"定义域关于原点对称"？**

意思是：如果 $x$ 在定义域内，那么 $-x$ 也必须在定义域内。

**✅ 关于原点对称的定义域**：
- $(-3, 3)$：$-3$ 和 $3$ 都不在域内，但对称。
- $[-2, 2]$：$-2$ 和 $2$ 都在域内，对称。
- $\mathbb{R}$（全体实数）：对称。
- $(-\infty, 0) \cup (0, +\infty)$：正负都取，对称。

**❌ 不关于原点对称的定义域**：
- $(0, +\infty)$：只有正数，没有负数，不对称。
- $[-2, 3]$：$-2$ 对应的 $+2$ 在域内，但 $3$ 对应的 $-3$ 不在域内，不对称。
- $(-1, 2]$：$-1$ 不在域内，但 $1$ 在域内——不对称。

> 💡 **快速判断法**：把定义域画在数轴上，看它是否关于 $0$ 这个点对称。如果从 $0$ 向左和向右"走"的距离一样，就是对称的。

```
✅ 关于原点对称：          ❌ 不关于原点对称：
    ───●══════════●───→         ───────●══════●───→
      -3    0     3               0    1     3
    （左右各3个单位）            （0到1是1个单位，
                                 0到3是3个单位，不等）
```

### 13.3.3 偶函数详解

**定义**：对于函数 $f(x)$ 定义域内的任意 $x$，都有 $f(-x) = f(x)$。

**几何意义**：图像关于 $y$ 轴对称。把图像沿 $y$ 轴对折，左右两边完全重合。

**典型例子**：

| 函数 | $f(-x)$ | $f(x)$ | 是否相等？ |
|------|---------|--------|:---:|
| $f(x) = x^2$ | $(-x)^2 = x^2$ | $x^2$ | ✅ 偶函数 |
| $f(x) = \|x\|$ | $\|-x\| = \|x\|$ | $\|x\|$ | ✅ 偶函数 |
| $f(x) = 3$（常数函数） | $3$ | $3$ | ✅ 偶函数 |
| $f(x) = \dfrac{1}{x^2}$ | $\dfrac{1}{(-x)^2} = \dfrac{1}{x^2}$ | $\dfrac{1}{x^2}$ | ✅ 偶函数 |

> 🤔 **想多一点**：为什么 $f(x) = 3$ 是偶函数？因为它的图像是一条水平线 $y = 3$，这条线关于 $y$ 轴对称（左边和右边完全一样）。这也说明：**偶函数不一定是抛物线**，任何满足 $f(-x) = f(x)$ 的函数都是偶函数。

### 13.3.4 奇函数详解

**定义**：对于函数 $f(x)$ 定义域内的任意 $x$，都有 $f(-x) = -f(x)$。

**几何意义**：图像关于原点对称。把图像绕原点旋转 $180^\circ$，与自身重合。

**典型例子**：

| 函数 | $f(-x)$ | $-f(x)$ | 是否相等？ |
|------|---------|---------|:---:|
| $f(x) = x$ | $-x$ | $-x$ | ✅ 奇函数 |
| $f(x) = x^3$ | $(-x)^3 = -x^3$ | $-x^3$ | ✅ 奇函数 |
| $f(x) = \dfrac{1}{x}$ | $\dfrac{1}{-x} = -\dfrac{1}{x}$ | $-\dfrac{1}{x}$ | ✅ 奇函数 |
| $f(x) = 0$（零函数） | $0$ | $0$ | ✅ 奇函数（也是偶函数） |

> 🤔 **想多一点**：$f(x) = 0$（零函数）是唯一一个**既是奇函数又是偶函数**的函数。因为它同时满足 $f(-x) = f(x) = 0$ 和 $f(-x) = -f(x) = 0$。在考试中，最常考的"既是奇函数又是偶函数"的答案就是 $f(x) = 0$（且定义域关于原点对称）。

### 13.3.5 既不是奇函数也不是偶函数（非奇非偶）

大多数函数既不是奇函数也不是偶函数。判断标准很简单：只要 $f(-x)$ 既不等于 $f(x)$ 也不等于 $-f(x)$，就是非奇非偶。

**典型例子**：
- $f(x) = x^2 + x$：$f(-x) = x^2 - x$，既不等于 $x^2 + x$ 也不等于 $-(x^2 + x)$。
- $f(x) = x + 1$：$f(-x) = -x + 1$，既不等于 $x + 1$ 也不等于 $-(x + 1)$。

## 13.4 判断奇偶性的标准步骤

按以下四步操作，不会漏掉任何一种情况。

### 第 1 步：求定义域，检查是否关于原点对称

写出函数的定义域。如果定义域不关于原点对称，直接写"非奇非偶"，停止。

### 第 2 步：计算 $f(-x)$ 并化简

将 $-x$ 代入函数表达式，化简到最简形式。

### 第 3 步：比较 $f(-x)$ 与 $f(x)$、$-f(x)$ 的关系

有三种可能的结果：

| 比较结果 | 结论 |
|---------|------|
| $f(-x) = f(x)$ 对定义域内所有 $x$ 成立 | **偶函数** |
| $f(-x) = -f(x)$ 对定义域内所有 $x$ 成立 | **奇函数** |
| $f(-x) = f(x)$ 且 $f(-x) = -f(x)$ 同时成立 | **既是奇函数又是偶函数**（只有 $f(x) = 0$） |
| 以上都不成立 | **非奇非偶** |

### 第 4 步：写下结论

根据比较结果，写出"$f(x)$ 是奇函数"或"$f(x)$ 是偶函数"或"$f(x)$ 既不是奇函数也不是偶函数"。

## 13.5 奇偶性的运算性质

当两个函数各自有奇偶性时，它们的和、差、积、商也有确定的奇偶性。掌握这些运算性质，可以快速判断复杂函数的奇偶性，而不需要每次都从定义出发重新计算。

### 13.5.1 加减法性质

| 运算 | 结果 |
|------|------|
| 奇函数 + 奇函数 | **奇函数** |
| 偶函数 + 偶函数 | **偶函数** |
| 奇函数 + 偶函数 | **非奇非偶**（一般情况） |

> ⚠️ **注意**："奇 + 偶 = 非奇非偶"是"一般情况"，但有两个例外：(1) 如果两者都是零函数，结果是零函数（既是奇又是偶）；(2) 如果两者恰好抵消，可能出现特殊情况，但高中数学中极少出现。

### 13.5.2 乘法性质

| 运算 | 结果 | 口诀 |
|------|------|------|
| 奇函数 × 奇函数 | **偶函数** | 奇×奇 = 偶 |
| 偶函数 × 偶函数 | **偶函数** | 偶×偶 = 偶 |
| 奇函数 × 偶函数 | **奇函数** | 奇×偶 = 奇 |

> 💡 **记忆口诀**：把"奇"看作"负号"，把"偶"看作"正号"。
> - 奇×奇 = 负×负 = 正 = 偶 ✅
> - 偶×偶 = 正×正 = 正 = 偶 ✅
> - 奇×偶 = 负×正 = 负 = 奇 ✅

### 13.5.3 为什么奇 × 奇 = 偶？

用定义来证明，这是理解乘法性质的关键：

设 $f(x)$ 和 $g(x)$ 都是奇函数，即 $f(-x) = -f(x)$，$g(-x) = -g(x)$。

令 $h(x) = f(x) \cdot g(x)$，计算 $h(-x)$：

$$h(-x) = f(-x) \cdot g(-x) = [-f(x)] \cdot [-g(x)] = f(x) \cdot g(x) = h(x)$$

因为 $h(-x) = h(x)$，所以 $h(x)$ 是偶函数。**两个负号相乘变成正号，所以奇×奇 = 偶。**

> 🤔 **想多一点**：用同样的方法可以自己推导"偶×偶 = 偶"和"奇×偶 = 奇"。设 $f(x)$ 和 $g(x)$ 都是偶函数，则 $h(-x) = f(-x) \cdot g(-x) = f(x) \cdot g(x) = h(x)$，所以偶×偶 = 偶。设 $f(x)$ 是奇函数，$g(x)$ 是偶函数，则 $h(-x) = f(-x) \cdot g(-x) = [-f(x)] \cdot g(x) = -h(x)$，所以奇×偶 = 奇。动手推导一遍，比背口诀记得牢。

### 13.5.4 除法性质

除法的奇偶性规律与乘法完全一致（因为除法本质上是乘以倒数）：

| 运算 | 结果 |
|------|------|
| 奇函数 ÷ 奇函数 | **偶函数** |
| 偶函数 ÷ 偶函数 | **偶函数** |
| 奇函数 ÷ 偶函数 | **奇函数** |

> ⚠️ **注意**：做除法时，分母不能为 0，需要考虑定义域的限制。

### 13.5.5 复合函数的奇偶性

| 复合方式 | 结果 |
|---------|------|
| 奇函数 复合 奇函数 | **奇函数** |
| 奇函数 复合 偶函数 | **偶函数** |
| 偶函数 复合 奇函数 | **偶函数** |
| 偶函数 复合 偶函数 | **偶函数** |

> 💡 **记忆口诀**：复合函数中，只要外层的函数是偶函数，结果就是偶函数；外层是奇函数时，结果取决于内层。

## 13.6 例题精讲

### 例题 1 —— 标准判断：多项式函数

**题目**：判断函数 $f(x) = x^3 - x$ 的奇偶性。

**解**：

**第 1 步**：求定义域。$f(x) = x^3 - x$ 是一个多项式函数，定义域为 $\mathbb{R}$（全体实数）。$\mathbb{R}$ 关于原点对称。✅

**第 2 步**：计算 $f(-x)$。

$$f(-x) = (-x)^3 - (-x) = -x^3 + x$$

**第 3 步**：比较 $f(-x)$ 与 $f(x)$、$-f(x)$。

- $f(x) = x^3 - x$
- $-f(x) = -(x^3 - x) = -x^3 + x$

发现 $f(-x) = -x^3 + x = -f(x)$。✅

**第 4 步**：结论。$f(x) = x^3 - x$ 是**奇函数**。

> ✅ **验证**：取 $x = 2$，$f(2) = 8 - 2 = 6$，$f(-2) = -8 + 2 = -6$，确实有 $f(-2) = -f(2)$。取 $x = 1$，$f(1) = 0$，$f(-1) = 0$，$f(-1) = -f(1) = 0$ 也成立。

---

### 例题 2 —— 分式函数的奇偶性

**题目**：判断函数 $f(x) = \dfrac{1}{x^2}$ 的奇偶性。

**解**：

**第 1 步**：求定义域。分母 $x^2 \neq 0$，即 $x \neq 0$。定义域为 $(-\infty, 0) \cup (0, +\infty)$。这个区间关于原点对称。✅

**第 2 步**：计算 $f(-x)$。

$$f(-x) = \frac{1}{(-x)^2} = \frac{1}{x^2}$$

**第 3 步**：比较。

$f(-x) = \dfrac{1}{x^2} = f(x)$。✅

**第 4 步**：结论。$f(x) = \dfrac{1}{x^2}$ 是**偶函数**。

> 💡 **规律**：$x$ 的偶次幂（$x^2, x^4, x^6, \ldots$）都是偶函数。因为 $(-x)^{2k} = x^{2k}$。

---

### 例题 3 —— 定义域不对称的陷阱

**题目**：判断函数 $f(x) = \dfrac{x}{x-1}$ 的奇偶性。

**解**：

**第 1 步**：求定义域。分母 $x - 1 \neq 0$，即 $x \neq 1$。定义域为 $(-\infty, 1) \cup (1, +\infty)$。

检查定义域是否关于原点对称：$x = -1$ 在定义域内，但 $x = 1$ 不在定义域内。定义域**不关于原点对称**。

**结论**：$f(x) = \dfrac{x}{x-1}$ **既不是奇函数也不是偶函数**（非奇非偶）。

> ⚠️ **关键提醒**：这道题到了第 1 步就结束了，不需要再算 $f(-x)$。很多同学会直接去算 $f(-x) = \dfrac{-x}{-x-1}$，然后去比较——但这是没有意义的，因为定义域本身就不满足判断奇偶性的前提条件。**定义域不关于原点对称 → 直接非奇非偶，一步到位。**

---

### 例题 4 —— 利用运算性质快速判断

**题目**：判断函数 $f(x) = x^2 \cdot \sin x$ 的奇偶性。（已知 $\sin x$ 是奇函数）

**解**：

**分析**：$f(x) = x^2 \cdot \sin x$ 是两个函数的乘积。

- $x^2$ 是偶函数（$(-x)^2 = x^2$）。
- $\sin x$ 是奇函数（已知条件）。

根据运算性质：偶 × 奇 = 奇。

**结论**：$f(x) = x^2 \cdot \sin x$ 是**奇函数**。

> 💡 **对比**：如果不用运算性质，需要计算 $f(-x) = (-x)^2 \cdot \sin(-x) = x^2 \cdot (-\sin x) = -x^2 \sin x = -f(x)$，得出同样结论。但用运算性质更快——只需要知道 $x^2$ 是偶函数、$\sin x$ 是奇函数，然后套"偶×奇 = 奇"即可。

## 13.7 常见错误辨析

| 错误 | 错在哪里？ | 正确做法 |
|------|-----------|---------|
| ❌ 判断奇偶性前不检查定义域是否关于原点对称 | 奇偶性定义的前提是 $x$ 和 $-x$ 同时在定义域内。定义域不对称，$f(-x)$ 根本没意义 | 第 1 步永远先求定义域，检查是否关于原点对称 |
| ❌ 只验证一个 $x$ 值就下结论 | 奇偶性要求"对定义域内的**任意** $x$"成立，一个例子不足以证明 | 必须用代数推导 $f(-x)$ 的表达式，然后与 $f(x)$ 和 $-f(x)$ 比较 |
| ❌ 把 $f(-x) = f(x)$ 和 $f(-x) = -f(x)$ 搞反 | 偶函数 $f(-x) = f(x)$（值不变），奇函数 $f(-x) = -f(x)$（值变号） | 记忆：**偶**函数的图像折"**叠**"能重合，值不变；**奇**函数的图像旋"**转**"能重合，值变号 |
| ❌ 认为 $f(x) = 0$ 只是奇函数 | $f(0) = 0$ 是奇函数的必要条件，但 $f(x) \equiv 0$ 既是奇函数又是偶函数 | 如果 $f(-x) = f(x) = 0$ 且 $f(-x) = -f(x) = 0$ 同时成立，则既是奇函数又是偶函数 |
| ❌ 认为"奇函数 + 偶函数 = 非奇非偶"总是成立 | 一般情况下成立，但零函数是例外 | 对于 $f(x) = 0$ 和 $g(x) = 0$，它们本身既是奇又是偶，和也是零函数 |
| ❌ 认为 $f(x) = x^3$ 是奇函数，所以 $f(x) = x^3 + 1$ 也是奇函数 | $x^3$ 是奇函数，但 $+1$ 破坏了奇偶性 | 奇函数 + 常数（非零）= 非奇非偶（除非常数恰好使得整个函数变为零函数，但那不可能） |

## 13.8 练习题

### 基础题

**第 1 题**：判断下列函数的奇偶性。

(1) $f(x) = x^4 + 2x^2$
(2) $f(x) = x^3 - 2x$
(3) $f(x) = x^2 + x$

**第 2 题**：判断下列函数的奇偶性，注意先检查定义域。

(1) $f(x) = \dfrac{1}{x}$（$x \neq 0$）
(2) $f(x) = \sqrt{x}$
(3) $f(x) = \dfrac{x}{x^2 + 1}$

**第 3 题**：已知 $f(x)$ 是定义在 $\mathbb{R}$ 上的奇函数，且当 $x > 0$ 时 $f(x) = x^2 + 1$，求 $f(0)$、$f(-2)$ 和 $f(-3)$。

### 提高题

**第 4 题**：已知 $f(x)$ 是定义在 $\mathbb{R}$ 上的奇函数，$g(x)$ 是定义在 $\mathbb{R}$ 上的偶函数，且 $f(x) + g(x) = x^2 + x + 1$。求 $f(x)$ 和 $g(x)$ 的解析式。

**第 5 题**：判断函数 $f(x) = x|x|$ 的奇偶性，并画出其大致图像。

## 13.9 练习题答案与解析

### 第 1 题 解析

**(1) $f(x) = x^4 + 2x^2$**

第 1 步：定义域为 $\mathbb{R}$，关于原点对称。✅

第 2 步：$f(-x) = (-x)^4 + 2(-x)^2 = x^4 + 2x^2$

第 3 步：$f(-x) = x^4 + 2x^2 = f(x)$

第 4 步：$f(x) = x^4 + 2x^2$ 是**偶函数**。

> 💡 **规律**：只含 $x$ 的偶次幂的多项式是偶函数。

**(2) $f(x) = x^3 - 2x$**

第 1 步：定义域为 $\mathbb{R}$，关于原点对称。✅

第 2 步：$f(-x) = (-x)^3 - 2(-x) = -x^3 + 2x$

第 3 步：$-f(x) = -(x^3 - 2x) = -x^3 + 2x$，所以 $f(-x) = -f(x)$

第 4 步：$f(x) = x^3 - 2x$ 是**奇函数**。

> 💡 **规律**：只含 $x$ 的奇次幂的多项式是奇函数。

**(3) $f(x) = x^2 + x$**

第 1 步：定义域为 $\mathbb{R}$，关于原点对称。✅

第 2 步：$f(-x) = (-x)^2 + (-x) = x^2 - x$

第 3 步：$f(x) = x^2 + x$，$-f(x) = -x^2 - x$。$f(-x) = x^2 - x$ 既不等于 $f(x)$ 也不等于 $-f(x)$。

第 4 步：$f(x) = x^2 + x$ **既不是奇函数也不是偶函数**（非奇非偶）。

> 💡 **规律**：同时含有 $x$ 的奇次幂和偶次幂的多项式，一般是非奇非偶。

---

### 第 2 题 解析

**(1) $f(x) = \dfrac{1}{x}$（$x \neq 0$）**

第 1 步：定义域为 $(-\infty, 0) \cup (0, +\infty)$，关于原点对称。✅

第 2 步：$f(-x) = \dfrac{1}{-x} = -\dfrac{1}{x}$

第 3 步：$-f(x) = -\dfrac{1}{x}$，所以 $f(-x) = -f(x)$

第 4 步：$f(x) = \dfrac{1}{x}$ 是**奇函数**。

**(2) $f(x) = \sqrt{x}$**

第 1 步：定义域为 $[0, +\infty)$（偶次根号下必须 $\geq 0$）。

检查是否关于原点对称：$x = 1$ 在定义域内，但 $x = -1$ 不在定义域内。定义域**不关于原点对称**。

结论：$f(x) = \sqrt{x}$ **既不是奇函数也不是偶函数**（非奇非偶）。

> ⚠️ **关键提醒**：这道题必须停在第 1 步！$f(-x) = \sqrt{-x}$ 对于 $x > 0$ 根本没有意义（在实数范围内），所以不能去计算 $f(-x)$ 然后比较。定义域不对称 = 直接非奇非偶。

**(3) $f(x) = \dfrac{x}{x^2 + 1}$**

第 1 步：分母 $x^2 + 1 \geq 1 > 0$，永远不为 0。定义域为 $\mathbb{R}$，关于原点对称。✅

第 2 步：$f(-x) = \dfrac{-x}{(-x)^2 + 1} = \dfrac{-x}{x^2 + 1} = -\dfrac{x}{x^2 + 1}$

第 3 步：$-f(x) = -\dfrac{x}{x^2 + 1}$，所以 $f(-x) = -f(x)$

第 4 步：$f(x) = \dfrac{x}{x^2 + 1}$ 是**奇函数**。

> 💡 **规律**：分子是奇函数（$x$），分母是偶函数（$x^2 + 1$），奇÷偶 = 奇。

---

### 第 3 题 解析

**已知**：$f(x)$ 是 $\mathbb{R}$ 上的奇函数，且当 $x > 0$ 时 $f(x) = x^2 + 1$。

**(1) 求 $f(0)$**：

因为 $f(x)$ 是奇函数，所以 $f(-x) = -f(x)$ 对任意 $x \in \mathbb{R}$ 成立。

令 $x = 0$：$f(-0) = -f(0)$，即 $f(0) = -f(0)$。

移项：$2f(0) = 0$，所以 $f(0) = 0$。

> 💡 **重要结论**：如果奇函数在 $x = 0$ 处有定义，则 $f(0) = 0$。这是奇函数的一个重要性质。

**(2) 求 $f(-2)$**：

$f(-2) = -f(2)$（奇函数性质）。

因为 $2 > 0$，所以 $f(2) = 2^2 + 1 = 4 + 1 = 5$。

因此 $f(-2) = -5$。

**(3) 求 $f(-3)$**：

$f(-3) = -f(3)$（奇函数性质）。

因为 $3 > 0$，所以 $f(3) = 3^2 + 1 = 9 + 1 = 10$。

因此 $f(-3) = -10$。

> 🧠 **反思**：这道题的关键是理解奇函数的含义：$f(-x) = -f(x)$。当给定 $x > 0$ 的解析式时，求 $x < 0$ 的函数值，只需要先求对应的正数函数值，再取相反数即可。

---

### 第 4 题 解析

**已知**：$f(x)$ 是奇函数，$g(x)$ 是偶函数，且 $f(x) + g(x) = x^2 + x + 1$。

**解法**：利用奇偶性构造方程组。

因为 $f(x) + g(x) = x^2 + x + 1$ 对所有 $x$ 成立，将 $x$ 替换为 $-x$：

$$f(-x) + g(-x) = (-x)^2 + (-x) + 1$$

利用奇偶性：$f(-x) = -f(x)$，$g(-x) = g(x)$：

$$-f(x) + g(x) = x^2 - x + 1$$

现在有两个方程：

$$\begin{cases}
f(x) + g(x) = x^2 + x + 1 & \quad \text{①}\\[4pt]
-f(x) + g(x) = x^2 - x + 1 & \quad \text{②}
\end{cases}$$

① + ②：$2g(x) = 2x^2 + 2$，所以 $g(x) = x^2 + 1$。

① - ②：$2f(x) = 2x$，所以 $f(x) = x$。

**验证**：
- $f(x) = x$ 是奇函数：$f(-x) = -x = -f(x)$ ✅
- $g(x) = x^2 + 1$ 是偶函数：$g(-x) = (-x)^2 + 1 = x^2 + 1 = g(x)$ ✅
- $f(x) + g(x) = x + x^2 + 1 = x^2 + x + 1$ ✅

**答案**：$f(x) = x$，$g(x) = x^2 + 1$。

> 🧠 **反思**：这道题展示了一种重要的解题技巧——**利用奇偶性构造方程组**。遇到"奇函数 + 偶函数 = 某个式子"的题目，把 $x$ 换成 $-x$ 再写一个方程，利用 $f(-x) = -f(x)$ 和 $g(-x) = g(x)$，就能解出 $f(x)$ 和 $g(x)$。这是高考中常见的题型。

---

### 第 5 题 解析

**题目**：判断函数 $f(x) = x|x|$ 的奇偶性，并画出大致图像。

**第一步：判断奇偶性**。

第 1 步：定义域为 $\mathbb{R}$，关于原点对称。✅

第 2 步：计算 $f(-x)$。

$$f(-x) = (-x) \cdot |-x| = (-x) \cdot |x| = -x|x|$$

第 3 步：比较。

$-f(x) = -x|x|$，所以 $f(-x) = -f(x)$。

第 4 步：$f(x) = x|x|$ 是**奇函数**。

**第二步：画出大致图像**。

将 $f(x) = x|x|$ 写成分段函数的形式：

$$f(x) = \begin{cases}
x \cdot x = x^2, & x \geq 0 \\[4pt]
x \cdot (-x) = -x^2, & x < 0
\end{cases}$$

```
        y↑
         |        ↗
         |       /
         |      /
         |     /
         |    /
         |   /
    ─────┼──●──────────→ x
         |  / 0
         | /
         |/
         /
        /
       /
      ↙
```

图像特征：
- 当 $x \geq 0$ 时，$f(x) = x^2$（右半支是抛物线）。
- 当 $x < 0$ 时，$f(x) = -x^2$（左半支是倒抛物线）。
- 图像关于原点对称（奇函数的特征）。
- 在 $x = 0$ 处连续，$f(0) = 0$。

> 🧠 **反思**：$f(x) = x|x|$ 虽然含有绝对值，但它是奇函数。很多同学看到绝对值就以为是偶函数——这是错误的。判断奇偶性只能用定义，不能凭感觉。$|x|$ 本身是偶函数，但乘上 $x$（奇函数）后，奇×偶 = 奇，所以 $x|x|$ 是奇函数。

|--------|---------|-----------|
| 偶函数 | $f(-x) = f(x)$，图像关于 $y$ 轴对称 | 只有偶次幂的多项式是偶函数 |

| 判断步骤 | 求定义域 → 算 $f(-x)$ → 比较 → 下结论 | 四步走，一步不能少 |
| 运算性质 | 奇×奇=偶，偶×偶=偶，奇×偶=奇 | 把"奇"当负号、"偶"当正号来记 |
| 既是奇又是偶 | 只有 $f(x) = 0$（且定义域关于原点对称） | 考试中答"不存在"通常是错的 |
