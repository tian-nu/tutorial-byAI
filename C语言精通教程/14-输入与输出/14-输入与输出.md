## 14.1 输出：`printf` 详解

### 14.1.1 基本语法

```c
printf("格式字符串", 参数1, 参数2, ...);
```

`printf` 是"格式化打印"函数。格式字符串中的 `%` 开头的占位符会被后面的参数按顺序替换。

**比喻**：`printf` 像一个填空游戏。格式字符串是一张"模板卡片"，上面写着"亲爱的____，您的余额是____元"。`%` 占位符就是那些空格，后面的参数就是填进去的内容。

### 14.1.2 常用格式说明符

| 格式符 | 含义 | 示例 |
|--------|------|------|
| `%d` | 有符号十进制整数 | `printf("%d", 42);` → `42` |
| `%u` | 无符号十进制整数 | `printf("%u", 42U);` → `42` |
| `%f` | 浮点数（小数形式） | `printf("%f", 3.14);` → `3.140000` |
| `%e` | 浮点数（科学记数法） | `printf("%e", 3.14);` → `3.140000e+00` |
| `%c` | 单个字符 | `printf("%c", 'A');` → `A` |
| `%s` | 字符串 | `printf("%s", "Hello");` → `Hello` |
| `%p` | 指针地址 | `printf("%p", &x);` → `0x7fff...` |
| `%%` | 百分号本身 | `printf("%%");` → `%` |
| `%x` | 十六进制整数（小写） | `printf("%x", 255);` → `ff` |
| `%X` | 十六进制整数（大写） | `printf("%X", 255);` → `FF` |

### 14.1.3 格式控制：宽度与精度

你可以在 `%` 和格式符之间加入数字来控制输出格式：

```c
#include <stdio.h>

int main(void)
{
    int num = 42;

    /* 宽度：最少占多少字符 */
    printf("[%d]\n", num);      /* [42] */
    printf("[%5d]\n", num);     /* [   42]  （右对齐，5个字符宽） */
    printf("[%-5d]\n", num);    /* [42   ]  （左对齐，5个字符宽） */
    printf("[%05d]\n", num);    /* [00042]  （用0填充） */

    /* 浮点数精度 */
    double pi = 3.1415926535;
    printf("%f\n", pi);         /* 3.141593   （默认6位小数） */
    printf("%.2f\n", pi);       /* 3.14       （保留2位小数） */
    printf("%10.2f\n", pi);     /*       3.14 （宽度10，精度2） */
    printf("%.10f\n", pi);      /* 3.1415926535 （精度10位） */

    return 0;
}
```

**比喻**：格式控制就像编辑文档的"段落格式"。`%5d` 是"右对齐，这个数最少占5个字符的位置"。`%.2f` 是"小数部分保留2位，多了四舍五入，少了补零"。

## 14.2 输入：`scanf` 详解

### 14.2.1 基本语法

```c
scanf("格式字符串", &变量1, &变量2, ...);
```

`scanf` 从 `*标准输入*（此术语见附录F）`（通常是键盘）读取数据，按照格式字符串解析，然后存入指定的变量。

### 14.2.2 那个著名的 `&` 符号

```c
int age;
scanf("%d", &age);   /* 注意：是 &age，不是 age！ */
```

`&` 是取地址运算符。`scanf` 需要知道"把数据放到哪里"，`&age` 就是变量 `age` 的内存地址。如果不写 `&`，`scanf` 不知道往哪放，程序会崩溃。

**比喻**：`scanf` 是一个快递员。你告诉他要送什么（"%d"表示一个整数包裹），还得告诉他送到哪里（`&age` 是地址）。如果你只说"送个快递"而不给地址，快递员会把包裹扔在街上——这就是不写 `&` 导致的崩溃。

### 14.2.3 常用格式说明符

与 `printf` 基本一致，但有细微差别：

| 格式符 | 含义 | 对应的变量类型 |
|--------|------|---------------|
| `%d` | 读取整数 | `int` |
| `%f` | 读取浮点数 | `float` |
| `%lf` | 读取双精度浮点数 | `double`（注意！是 `%lf` 不是 `%f`） |
| `%c` | 读取单个字符 | `char` |
| `%s` | 读取字符串 | `char[]`（不加 `&`） |

**特别注意**：`scanf` 中 `double` 用 `%lf`，但 `printf` 中 `double` 用 `%f`。这是C语言一个历史遗留的不一致之处。

```c
double d;
scanf("%lf", &d);   /* scanf: double 用 %lf */
printf("%f", d);    /* printf: double 用 %f */
```

### 14.2.4 一次性读取多个值

```c
int day, month, year;
printf("请输入日期（格式：日 月 年）：");
scanf("%d %d %d", &day, &month, &year);
printf("你输入的是 %d年%d月%d日\n", year, month, day);
```

用户输入时，可以用空格、Tab或回车分隔多个值。

## 14.3 `*缓冲区*（此术语见附录F）`

### 14.3.1 什么是缓冲区？

你按键盘上的键，字符不会立刻送到程序里。它们先被存在一个叫"缓冲区"的临时仓库里。当你按下**回车键**时，整个缓冲区的内容才被一起送给程序。

```
键盘输入流程：
   按键 → 终端驱动程序 → 输入缓冲区 → 按下回车 → scanf收到数据
   ──────────────────────────────────────────────────────────────
   你打字          字符暂存            触发发送       程序读取
```

**比喻**：缓冲区就像超市的购物车。你把商品（字符）一个一个放进购物车，但收银员（scanf）不会一个一个处理——你推着购物车到收银台（按回车），收银员才一次性处理整车的商品。

### 14.3.2 缓冲区带来的问题

```c
#include <stdio.h>

int main(void)
{
    int age;
    char name[50];

    printf("请输入年龄：");
    scanf("%d", &age);

    printf("请输入姓名：");
    scanf("%s", name);   /* 这里可能会出问题！ */

    printf("年龄：%d，姓名：%s\n", age, name);

    return 0;
}
```

如果你输入 `25` 然后回车，`scanf("%d")` 读取了 `25`，但**回车符（`\n`）还留在缓冲区里**。下一个 `scanf("%s")` 会读到这个残留的回车符，导致 `name` 读到的内容为空或异常。

**解决方法**：在 `scanf` 后加一个 `getchar()` 吃掉残留的回车符，或者使用 `scanf(" %c", &ch)`（注意 `%c` 前的空格，它会跳过空白字符）。

## 14.4 字符级I/O：`getchar` 和 `putchar`

### 14.4.1 `putchar`：输出一个字符

```c
#include <stdio.h>

int main(void)
{
    putchar('H');
    putchar('e');
    putchar('l');
    putchar('l');
    putchar('o');
    putchar('\n');

    return 0;
}
```

输出：`Hello`

### 14.4.2 `getchar`：读取一个字符

```c
#include <stdio.h>

int main(void)
{
    int ch;   /* 注意：用 int，不是 char！ */

    printf("请输入一个字符：");
    ch = getchar();

    printf("你输入了 '%c'，ASCII 值是 %d\n", ch, ch);

    return 0;
}
```

**为什么用 `int` 而不是 `char`？** 因为 `getchar` 除了返回0-255的字符值外，还可能返回 `EOF`（End Of File，值为 -1）。`char` 类型存不下 `-1`（或者说存下了但是和255无法区分）。所以 `getchar` 的返回值必须用 `int` 接收。

## 14.5 完整示例：交互式计算器

```c
#include <stdio.h>

int main(void)
{
    int num1, num2;
    char operator;

    printf("=== 简单计算器 ===\n");
    printf("请输入第一个数字：");
    scanf("%d", &num1);

    /* 吃掉残留的换行符 */
    getchar();

    printf("请输入运算符（+ - * /）：");
    operator = getchar();

    printf("请输入第二个数字：");
    scanf("%d", &num2);

    printf("\n计算结果：\n");
    printf("%d %c %d = ", num1, operator, num2);

    switch (operator) {
        case '+':
            printf("%d\n", num1 + num2);
            break;
        case '-':
            printf("%d\n", num1 - num2);
            break;
        case '*':
            printf("%d\n", num1 * num2);
            break;
        case '/':
            if (num2 != 0) {
                printf("%.2f\n", (double)num1 / num2);
            } else {
                printf("错误：不能除以零！\n");
            }
            break;
        default:
            printf("错误：不支持的运算符！\n");
    }

    return 0;
}
```

## 14.6 常见错误

### 错误1：`scanf` 忘记 `&`

```c
/* ❌ 错误示例 */
int age;
scanf("%d", age);   /* 缺少 &，程序崩溃或行为异常 */

/* ✅ 正确示例 */
int age;
scanf("%d", &age);
```

### 错误2：`scanf` 中 `double` 用 `%f` 而不是 `%lf`

```c
/* ❌ 错误示例 */
double height;
scanf("%f", &height);   /* 应该用 %lf */

/* ✅ 正确示例 */
double height;
scanf("%lf", &height);
```

### 错误3：`printf` 格式符与参数类型不匹配

```c
/* ❌ 错误示例 */
int x = 42;
printf("%f\n", x);   /* 格式符 %f 期望 double，但传了 int */

/* ✅ 正确示例 */
int x = 42;
printf("%d\n", x);
```

### 错误4：缓冲区残留问题

```c
/* ❌ 错误示例 */
int age;
char name[50];
scanf("%d", &age);   /* 读取年龄后，回车符留在缓冲区 */
scanf("%s", name);   /* name 读到的是空字符串（直接读到了回车） */

/* ✅ 正确示例 */
int age;
char name[50];
scanf("%d", &age);
getchar();           /* 吃掉残留的换行符 */
scanf("%s", name);
```

### 错误5：`scanf` 读取字符串溢出

```c
/* ❌ 危险 */
char name[10];
scanf("%s", name);   /* 用户输入超过9个字符就会溢出！ */

/* ✅ 安全 */
char name[10];
scanf("%9s", name);  /* 限制最多读取9个字符（留1个给'\0'） */
```

## 14.7 我在做什么？做得对不对？不对怎么办？

### 我在做什么？

我在学习C语言的输入输出：用 `printf` 格式化输出各种类型的数据，用 `scanf` 从键盘读取用户输入，理解缓冲区的概念，以及用 `getchar`/`putchar` 做字符级I/O。

### 我做得对不对？

- 你能写出一个交互式程序，从用户读取数字并计算 → 输入输出过关。
- 你知道 `scanf` 中 `double` 用 `%lf`，`printf` 中 `double` 用 `%f` → 格式符过关。
- 你知道 `scanf` 需要 `&`，理解了缓冲区残留问题 → 安全意识过关。
- 你知道 `getchar` 返回值用 `int` 而不是 `char` → 字符I/O过关。

### 不对怎么办？

| 症状 | 第一步检查 |
|------|-----------|
| `scanf` 程序崩溃 | 是否忘了 `&`？ |
| 输入读取异常 | 是否缓冲区残留了换行符？用 `getchar()` 吃掉 |
| `double` 输入得到奇怪值 | `scanf` 是否用了 `%lf`？ |
| `printf` 输出乱码 | 格式符与参数类型是否匹配？ |
| `getchar` 跳过某些输入 | 是否缓冲区有残留字符？ |

## 本章小结

| 知识点 | 说明 |
|--------|------|
| `printf` 格式符 | `%d`(int) `%f`(float/double) `%c`(char) `%s`(字符串) `%p`(指针) |
| `printf` 宽度控制 | `%5d` 右对齐，`%-5d` 左对齐，`%05d` 零填充 |
| `printf` 精度控制 | `%.2f` 保留2位小数 |
| `scanf` 基本用法 | `scanf("%d", &var);` 注意 `&` |
| `scanf` double | 用 `%lf`（不是 `%f`！） |
| 缓冲区 | 用户输入暂存区，回车后一起发送给程序 |
| 缓冲区残留 | `scanf` 后回车符可能残留，用 `getchar()` 吃掉 |
| `getchar`/`putchar` | 字符级输入输出，`getchar` 返回值用 `int` |
| `EOF` | 值为 -1，表示输入结束 |

## 练习题

1. **个人名片**：写程序，让用户输入姓名、年龄、身高（浮点数），然后用 `printf` 格式化输出一张"名片"。

2. **温度转换**：写程序，让用户输入摄氏温度（浮点数），转换为华氏温度（`F = C * 9/5 + 32`），输出结果保留2位小数。

3. **缓冲区实验**：写程序，先用 `scanf("%d")` 读取一个整数，再用 `getchar()` 读取下一个字符。输入 `42x` 然后回车。观察 `getchar()` 读到了什么。解释原因。

4. **格式对齐**：输出以下表格（用 `printf` 格式控制）：
   ```
   姓名      年龄      分数
   Alice     20       95.5
   Bob       22       88.0
   Charlie   19       91.3
   ```

5. **字符统计**：用 `getchar()` 循环读取字符，直到用户输入 `\n`（回车）。统计用户输入了多少个字符（不包括最后的回车）。
