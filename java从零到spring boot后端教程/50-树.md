## 50.1 什么是树？

树是由节点（Node）和边（Edge）组成的**层级结构**，有且只有一个根节点（Root）。

```
          ① ← 根节点(root)
        /   \
       ②     ③
      / \   / \
     ④  ⑤ ⑥  ⑦
    / \
   ⑧  ⑨ ← 叶子节点（没有子节点）

术语：
- 父节点：②的父节点是①
- 子节点：①的子节点是②和③
- 深度：根节点深度0，②的深度1，⑧的深度3
- 高度：从该节点到最远叶子节点的边数
- 二叉树：每个节点最多有两个子节点
```

## 50.2 二叉树节点定义与构建

```java
public class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode() {}

    TreeNode(int val) {
        this.val = val;
    }

    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

// 构建以下树：
//       1
//      / \
//     2   3
//    / \
//   4   5
TreeNode root = new TreeNode(1,
    new TreeNode(2,
        new TreeNode(4),
        new TreeNode(5)
    ),
    new TreeNode(3)
);
```

## 50.3 二叉树的四种遍历（必须会！）

二叉树的遍历分为两大类：**深度优先（DFS）** 和 **广度优先（BFS，即层序遍历）**。

DFS根据访问根节点的顺序又分三种：

```
       1
      / \
     2   3
    / \
   4   5

前序(根左右): 1 → 2 → 4 → 5 → 3
中序(左根右): 4 → 2 → 5 → 1 → 3
后序(左右根): 4 → 5 → 2 → 3 → 1
层序(逐层)  : 1 → 2 → 3 → 4 → 5
```

### 前序遍历（Preorder：根→左→右）

```java
// 递归版
public void preorder(TreeNode root, List<Integer> result) {
    if (root == null) return;
    result.add(root.val);              // 先访问根
    preorder(root.left, result);       // 再左子树
    preorder(root.right, result);      // 再右子树
}

// 迭代版（用栈模拟递归）
public List<Integer> preorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;

    Deque<TreeNode> stack = new ArrayDeque<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.add(node.val);
        if (node.right != null) stack.push(node.right);  // 先右后左，因为栈是LIFO
        if (node.left != null) stack.push(node.left);
    }
    return result;
}
```

### 中序遍历（Inorder：左→根→右）

```java
// 递归版
public void inorder(TreeNode root, List<Integer> result) {
    if (root == null) return;
    inorder(root.left, result);
    result.add(root.val);
    inorder(root.right, result);
}

// 迭代版
public List<Integer> inorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode cur = root;

    while (cur != null || !stack.isEmpty()) {
        while (cur != null) {           // 一路向左走到尽头
            stack.push(cur);
            cur = cur.left;
        }
        cur = stack.pop();              // 弹出栈顶（最左节点）
        result.add(cur.val);            // 访问
        cur = cur.right;                // 转向右子树
    }
    return result;
}
```

**中序遍历BST的特性**：输出的结果是有序的（升序）！这是BST的一个重要性质。

### 后序遍历（Postorder：左→右→根）

```java
// 递归版
public void postorder(TreeNode root, List<Integer> result) {
    if (root == null) return;
    postorder(root.left, result);
    postorder(root.right, result);
    result.add(root.val);
}

// 迭代版（前序"根左右"反转 → "根右左"反转 → "左右根"）
public List<Integer> postorderIterative(TreeNode root) {
    LinkedList<Integer> result = new LinkedList<>();
    if (root == null) return result;

    Deque<TreeNode> stack = new ArrayDeque<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.addFirst(node.val);         // 头插法，前序反转
        if (node.left != null) stack.push(node.left);
        if (node.right != null) stack.push(node.right);
    }
    return result;
}
```

**后序遍历的用途**：删除文件夹时，必须先删子文件再删父文件夹（左右根）。

### 层序遍历（Level Order，BFS）

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int levelSize = queue.size();
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
```

## 50.4 二叉搜索树（BST — Binary Search Tree）

BST的定义：对于任意节点——
- **左子树所有节点 < 当前节点**
- **右子树所有节点 > 当前节点**

```
        8
      /   \
     3     10
    / \      \
   1   6      14
      / \     /
     4   7   13

查找14：8→10→14，3步找到（O(log n)）
查找7：8→3→6→7，4步找到
```

### BST的查找

```java
public TreeNode searchBST(TreeNode root, int target) {
    TreeNode cur = root;
    while (cur != null) {
        if (cur.val == target) {
            return cur;
        } else if (target < cur.val) {
            cur = cur.left;     // 目标更小，去左边找
        } else {
            cur = cur.right;    // 目标更大，去右边找
        }
    }
    return null;  // 没找到
}
// 时间复杂度 O(log n) — 平衡的情况
// 最坏 O(n) — 如果树退化成一侧链表（如插入1→2→3→4顺序）
```

### BST的插入

```java
public TreeNode insertBST(TreeNode root, int val) {
    if (root == null) return new TreeNode(val);

    TreeNode cur = root;
    while (true) {
        if (val < cur.val) {
            if (cur.left == null) {
                cur.left = new TreeNode(val);
                break;
            }
            cur = cur.left;
        } else {
            if (cur.right == null) {
                cur.right = new TreeNode(val);
                break;
            }
            cur = cur.right;
        }
    }
    return root;
}
```

## 50.5 从BST到B+树：数据库索引原理

MySQL的InnoDB引擎，索引底层是**B+树**——一种改进的多叉平衡树。

普通BST的问题：
- 数据量大时树很深（100万条→约20层），每层一次磁盘IO
- 每个节点只存一个key，空间利用率低

B+树的改进：
- **每个节点存几百个key**（通常一个节点=一页=16KB）
- **树高极低**（100万条数据，B+树只有3-4层）
- **所有数据存在叶子节点**，叶子节点之间用链表连接（支持范围查询）

```
B+树结构（简化，每个节点3个key）：
              [30, 60]
             /   |    \
    [10,20]   [40,50]   [70,80,90]
    /  |  \    /  |  \    /  |  |  \
  叶子→叶子→叶子→叶子→叶子→叶子（链表串联）

查找47：根[30,60]→47在30-60之间→中间节点[40,50]→47在40-50之间→叶子节点→找到
范围查询30-70：定位30→顺着叶子链表读到70
```

这就是为什么 `SELECT * FROM users WHERE age BETWEEN 20 AND 30` 能这么快——B+树的叶子节点链表就是为了范围查询设计的。

## 50.6 树在后端的其他应用

| 数据结构 | 后端应用 |
|---|---|
| 二叉树 | 表达式解析（抽象语法树AST）、文件系统目录 |
| BST/B+树 | 数据库索引、文件系统（NTFS的MFT用B+树）|

| 红黑树 | Java TreeMap/TreeSet、Linux内核CFS调度器 |
| 哈夫曼树 | 数据压缩（gzip） |

## 50.7 ❌ 常见错误

| 错误 | 后果 | 正确做法 |
|---|---|---|
| 递归遍历不写base case | StackOverflow | 先写`if (root == null) return;` |
| 混淆前中后序的输出顺序 | 怎么都调试不对 | 牢记：前=根左右，中=左根右，后=左右根 |
| 认为BST查找永远是O(log n) | 最坏O(n)（全插在一侧） | 用AVL树/红黑树保证平衡 |
| 迭代遍历时栈的push顺序搞反 | 前序变后序 | 栈是LIFO，前序要"先push右再push左" |
| 层序遍历用栈代替队列 | 变成DFS了 | 层序（BFS）必须用Queue |

## 小结

| 概念 | 解释 |
|---|---|
| 二叉树 | 每个节点最多两个子节点 |
| 前序遍历 | 根→左→右（根在最前面） |
| 中序遍历 | 左→根→右（BST中序遍历=升序输出） |
| 后序遍历 | 左→右→根（用于先删子节点再删父节点） |
| 层序遍历 | 逐层从左到右（BFS，用队列） |
| BST | 左<根<右，查找/插入O(log n) |
| B+树 | 多叉平衡树，所有数据在叶子层，叶子链表串联 |
