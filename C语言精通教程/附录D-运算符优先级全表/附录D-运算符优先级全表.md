## 我在做什么？

C 语言的运算符多达 40 多种，它们的优先级决定了表达式求值的顺序。比如 `a + b * c` 中，乘法先于加法执行。本附录列出完整的优先级表（从高到低），附带记忆口诀和常见陷阱。

## D.2 记忆口诀

**单目 > 算术 > 移位 > 关系 > 位 > 逻辑 > 条件 > 赋值 > 逗号**

用一句话记住：
> **单算移关位逻条赋逗**（优先级从高到低）

展开：
1. **单**：单目运算符（`!`, `~`, `++`, `--`, `*`, `&`, `sizeof`, `(type)`）
2. **算**：算术运算符（`*`, `/`, `%`, `+`, `-`）
3. **移**：移位运算符（`<<`, `>>`）
4. **关**：关系运算符（`<`, `<=`, `>`, `>=`）
5. **位**：位运算符（`&`, `^`, `|`）
6. **逻**：逻辑运算符（`&&`, `||`）
7. **条**：条件运算符（`?:`）
8. **赋**：赋值运算符（`=`, `+=`, `-=`, 等）
9. **逗**：逗号运算符（`,`）

## D.3 常见陷阱

### 陷阱 1：`*p++` 是 `*(p++)` 还是 `(*p)++`？

```c
int arr[] = {10, 20, 30};
int *p = arr;

int x = *p++;   // 等价于 *(p++)
// 先取 *p（得到 10），然后 p 指向下一个元素

printf("%d\n", x);   // 10
printf("%d\n", *p);  // 20
```

后置 `++` 优先级高于 `*`，所以 `*p++` 等价于 `*(p++)`。要得到 `(*p)++`，必须加括号。

### 陷阱 2：`&` 取地址 vs `&` 按位与

```c
int a = 5, b = 3;
int result = a & b;   // 按位与：0101 & 0011 = 0001 = 1

int *ptr = &a;         // 取地址：& 在赋值右边，是取地址
```

取地址 `&` 出现在赋值运算符右边时，通常作为一元运算符，优先级高于二元 `&`。

### 陷阱 3：`==` 和 `=` 混淆

```c
// ❌ 错误：把赋值当比较
if (x = 5) {   // 永远为真！因为赋值表达式的结果是 5（非零）
    printf("x is 5\n");
}

// ✅ 正确
if (x == 5) {  // 真正的比较
    printf("x is 5\n");
}

// ✅ 防御性写法（Yoda 条件）
if (5 == x) {  // 如果写错成 5 = x，编译器会报错
    printf("x is 5\n");
}
```

### 陷阱 4：`<<` 和 `+` 的优先级

```c
// ❌ 可能误解
int result = 1 << 2 + 3;  // 不是 (1 << 2) + 3 = 7
                          // 而是 1 << (2 + 3) = 1 << 5 = 32

// ✅ 明确意图，加括号
int result = (1 << 2) + 3;  // 7
```

`+` 优先级高于 `<<`，所以 `1 << 2 + 3` 等价于 `1 << (2 + 3)`。

### 陷阱 5：`&&` 和 `||` 优先级

```c
// ❌ 误解
if (a && b || c && d) { ... }
// 等价于: if ((a && b) || (c && d)) { ... }
// 因为 && 优先级高于 ||

// ✅ 明确意图
if ((a && b) || (c && d)) { ... }
```

### 陷阱 6：逗号运算符

```c
int x = 1, y = 2, z;
z = (x++, y++, x + y);  // x=2, y=3, z=5
// 逗号运算符从左到右求值，返回最后一个表达式的结果

// 注意：函数参数列表中的逗号不是运算符
func(a, b, c);  // 这里的逗号是分隔符，不是运算符
```

## D.4 结合性说明

**左结合**：从左到右计算。大多数二元运算符（`+`, `-`, `*`, `/`, `&&`, `||` 等）都是左结合。

```c
int result = 10 - 5 - 2;  // (10 - 5) - 2 = 3，不是 10 - (5 - 2) = 7
```

**右结合**：从右到左计算。单目运算符和赋值运算符是右结合。

```c
int a, b, c;
a = b = c = 0;  // a = (b = (c = 0))，所有变量都赋为 0

int *p = &*ptr;  // &(*(ptr))，先解引用再取地址
```

## D.5 三问

1. **`*p++` 和 `(*p)++` 有什么区别？** `*p++` 先取 `*p` 的值，然后 `p` 指针自增（指向下一个元素）；`(*p)++` 先取 `*p` 的值，然后该值自增（`p` 指向的值变大）。

2. **为什么赋值运算符的优先级最低（除了逗号）？** 因为赋值通常是最后一步操作。比如 `a = b + c`，必须先计算 `b + c` 再赋值。如果赋值优先级高，`a = b` 会先执行，表达式就变成 `(a = b) + c`。

3. **什么时候必须加括号？** 当你不确定优先级时，加括号！即使已经确定，加括号也能让代码更可读。`if ((a && b) || c)` 比 `if (a && b || c)` 更清晰。
