## 如何使用

1. **按顺序读**：每篇都以前一篇为基础，跳着读会卡住
2. **每章做练习**：读完后立刻做对应练习题，不做题 = 没学
3. **篇末做项目**：每篇结束做附录N的串讲项目，串联知识
4. **篇末自测**：用附录L检验是否真的掌握了
5. **卡住了看附录M**：排错指南覆盖30个最常见错误

## 学习节奏建议

- 每天学 1~2 章
- 地基篇（0~14章）约 2~3 周
- Go语言篇（15~39章）约 4~6 周
- 后续各篇各 2~4 周
- 总计约 **6~10 个月**完成全部内容
