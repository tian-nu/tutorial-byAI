## 22.3 用单位圆定义三角函数

### 22.3.1 什么是单位圆？

**单位圆**：以坐标原点为圆心、半径为 $1$ 的圆。

方程：$x^2 + y^2 = 1$（因为圆上任意一点 $(x, y)$ 到原点的距离 $\sqrt{x^2 + y^2} = 1$）

---

### 22.3.2 三角函数的单位圆定义

> 这是高中数学三角函数体系的核心。请放慢速度，确保理解每一个字。

设 $\alpha$ 是一个任意角，它的终边与单位圆交于点 $P(x, y)$。则：

$$\boxed{\sin\alpha = y \quad\text{（点 P 的纵坐标）}}$$

$$\boxed{\cos\alpha = x \quad\text{（点 P 的横坐标）}}$$

$$\boxed{\tan\alpha = \frac{y}{x} = \frac{\sin\alpha}{\cos\alpha} \quad (x \neq 0)}$$

**逐字解读**：

- $\sin\alpha$ 就是交点 $P$ 的纵坐标 $y$。角 $\alpha$ 越大，$P$ 点在单位圆上逆时针移动，$y$ 坐标在 $-1$ 到 $1$ 之间变化。
- $\cos\alpha$ 就是交点 $P$ 的横坐标 $x$。
- $\tan\alpha$ 是纵坐标与横坐标的比值。当 $x = 0$（即 $\alpha = 90°$ 或 $270°$ 等）时，$\tan\alpha$ 无定义。

> 📊 交互演示见 [visual/math1_unit_circle_visual.html](visual/math1_unit_circle_visual.html) — 拖动角度滑块，观察点 $P$ 在单位圆上移动时 $x$（余弦）和 $y$（正弦）的实时变化。

### 22.3.3 直角三角形定义与单位圆定义如何统一？

> 这是初学者最困惑的问题之一。初中你学的是直角三角形中的锐角三角函数，高中突然变成了"单位圆上点的坐标"——这两者是什么关系？

**初中定义**（仅适用于锐角）：

在直角三角形中，设锐角 $\alpha$ 的对边为 $a$、邻边为 $b$、斜边为 $c$，则：

$$\sin\alpha = \frac{a}{c} = \frac{\text{对边}}{\text{斜边}}, \quad \cos\alpha = \frac{b}{c} = \frac{\text{邻边}}{\text{斜边}}, \quad \tan\alpha = \frac{a}{b} = \frac{\text{对边}}{\text{邻边}}$$

**现在**：对于锐角 $\alpha$（$0 < \alpha < 90°$），它的终边与单位圆交于点 $P(x, y)$。从 $P$ 向 $x$ 轴作垂线，垂足为 $M$，则 $\triangle OMP$ 是直角三角形。

```
         y
         ↑
         |    P(x, y)
         |   /|
         |  / |
         | /  | y (= sin α)
         |/α  |
  ———————O——M——→ x
         | x
         | (= cos α)
```

- 斜边 $OP = 1$（单位圆的半径）
- 对边 $PM = y$
- 邻边 $OM = x$

按初中定义：$\sin\alpha = \dfrac{PM}{OP} = \dfrac{y}{1} = y$，$\cos\alpha = \dfrac{OM}{OP} = \dfrac{x}{1} = x$

**结论**：单位圆定义包含了初中直角三角形定义，而且适用范围更大——它还能处理 $\alpha \geq 90°$ 和 $\alpha < 0°$ 的情况，那些情况初中定义无法处理。

---

### 22.3.4 单位圆定义的三个关键推论

**推论 1**：因为点 $P(x, y)$ 在单位圆上，满足 $x^2 + y^2 = 1$，所以：

$$\sin^2\alpha + \cos^2\alpha = 1$$

（$\sin^2\alpha$ 是 $(\sin\alpha)^2$ 的简写，不是 $\sin(\alpha^2)$。）

**推论 2**：$\sin\alpha$ 和 $\cos\alpha$ 的取值范围都是 $[-1, 1]$。因为 $P$ 的横纵坐标都在 $-1$ 到 $1$ 之间。

**推论 3**：$\tan\alpha$ 可以取任意实数值（当 $\cos\alpha$ 趋近于 $0$ 时，$\tan\alpha$ 趋近于无穷）。

---

## 22.4 三角函数值在各象限的符号

### 22.4.1 从单位圆理解符号

根据单位圆定义，$\sin\alpha = y$（纵坐标），$\cos\alpha = x$（横坐标）。

| 象限 | $x$（横坐标） | $y$（纵坐标） | $\sin\alpha = y$ | $\cos\alpha = x$ | $\tan\alpha = \dfrac{y}{x}$ |
|------|:---:|:---:|:---:|:---:|:---:|
| 第一象限 | $+$ | $+$ | $+$ | $+$ | $+$ |
| 第二象限 | $-$ | $+$ | $+$ | $-$ | $-$ |
| 第三象限 | $-$ | $-$ | $-$ | $-$ | $+$ |
| 第四象限 | $+$ | $-$ | $-$ | $+$ | $-$ |

> 💡 **理解方法**：不要死记。在脑子里画单位圆——第一象限右上角，$x$ 和 $y$ 都是正的，所以 $\sin$、$\cos$、$\tan$ 全正。第二象限左上角，$x$ 负、$y$ 正，所以 $\sin$ 正、$\cos$ 负、$\tan$ 负。以此类推。

---

### 22.4.2 口诀：一全正，二正弦，三正切，四余弦

这个口诀的意思是按象限依次看：

- **第一象限**：**全**部为正（sin、cos、tan 都 $> 0$）
- **第二象限**：只有**正弦**为正（sin $> 0$，cos $< 0$，tan $< 0$）
- **第三象限**：只有**正切**为正（tan $> 0$，sin $< 0$，cos $< 0$）
- **第四象限**：只有**余弦**为正（cos $> 0$，sin $< 0$，tan $< 0$）

> ⚠️ **注意**："二正弦"的意思是"第二象限只有正弦为正"，不是说"第二象限正弦就一定是正数"——它确实为正，但"二正弦"的关键信息是"余弦和正切在第二象限是负的"。

## 22.5 特殊角的三角函数值

### 22.5.1 必须背熟的值

| $\alpha$（角度） | $0°$ | $30°$ | $45°$ | $60°$ | $90°$ |
|:---:|:---:|:---:|:---:|:---:|:---:|
| $\alpha$（弧度） | $0$ | $\dfrac{\pi}{6}$ | $\dfrac{\pi}{4}$ | $\dfrac{\pi}{3}$ | $\dfrac{\pi}{2}$ |
| $\sin\alpha$ | $0$ | $\dfrac{1}{2}$ | $\dfrac{\sqrt{2}}{2}$ | $\dfrac{\sqrt{3}}{2}$ | $1$ |
| $\cos\alpha$ | $1$ | $\dfrac{\sqrt{3}}{2}$ | $\dfrac{\sqrt{2}}{2}$ | $\dfrac{1}{2}$ | $0$ |
| $\tan\alpha$ | $0$ | $\dfrac{\sqrt{3}}{3}$ | $1$ | $\sqrt{3}$ | 无定义 |

---

### 22.5.2 记忆方法

**观察规律**：$\sin$ 值从 $0°$ 到 $90°$ 依次为：

$$\frac{\sqrt{0}}{2},\; \frac{\sqrt{1}}{2},\; \frac{\sqrt{2}}{2},\; \frac{\sqrt{3}}{2},\; \frac{\sqrt{4}}{2}$$

即：分母全是 $2$，分子是 $\sqrt{0}, \sqrt{1}, \sqrt{2}, \sqrt{3}, \sqrt{4}$ 依次递增。

$\cos$ 值则反过来：$\dfrac{\sqrt{4}}{2}, \dfrac{\sqrt{3}}{2}, \dfrac{\sqrt{2}}{2}, \dfrac{\sqrt{1}}{2}, \dfrac{\sqrt{0}}{2}$

$\tan$ 值 = $\dfrac{\sin}{\cos}$，直接除即可：
- $\tan 30° = \dfrac{1/2}{\sqrt{3}/2} = \dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}$
- $\tan 45° = \dfrac{\sqrt{2}/2}{\sqrt{2}/2} = 1$
- $\tan 60° = \dfrac{\sqrt{3}/2}{1/2} = \sqrt{3}$

---

### 22.5.3 用单位圆扩展到任意象限

有了第一象限特殊角的值，利用单位圆的对称性可以求出其他象限的值。例如：

- $\sin 150° = \sin(180° - 30°) = \sin 30° = \dfrac{1}{2}$（第二象限，正弦为正）
- $\cos 150° = \cos(180° - 30°) = -\cos 30° = -\dfrac{\sqrt{3}}{2}$（第二象限，余弦为负）
- $\sin 210° = \sin(180° + 30°) = -\sin 30° = -\dfrac{1}{2}$（第三象限，正弦为负）

> 💡 这个"对称"思路正是下一章（诱导公式）的核心。本章先有个印象即可。

## 22.6 同角三角函数的基本关系

### 22.6.1 两个基本关系式

**关系一（平方关系）**：

$$\boxed{\sin^2\alpha + \cos^2\alpha = 1}$$

推导：已在 22.3.4 节用单位圆方程 $x^2 + y^2 = 1$ 得出。

**关系二（商数关系）**：

$$\boxed{\tan\alpha = \frac{\sin\alpha}{\cos\alpha} \quad (\cos\alpha \neq 0)}$$

推导：直接来自单位圆定义 $\tan\alpha = \dfrac{y}{x} = \dfrac{\sin\alpha}{\cos\alpha}$。

---

### 22.6.2 知一求二：已知一个三角函数值，求另外两个

> 这是考试中最常见的题型。核心思路：用平方关系求出第二个值，再用商数关系求出第三个值。**关键难点在于符号的确定**——你需要根据角所在的象限来判断。

**通用步骤**：

1. 根据已知条件确定角所在的象限
2. 用 $\sin^2\alpha + \cos^2\alpha = 1$ 求第二个值（开方时根据象限定符号）
3. 用 $\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha}$ 求第三个值

---

### 22.6.3 推导出的常用变形

从两个基本关系式可以推导出以下常用变形（不必死记，但遇到时应该能认出来）：

| 变形 | 推导过程 |
|------|---------|
| $\sin^2\alpha = 1 - \cos^2\alpha$ | 平方关系移项 |
| $\cos^2\alpha = 1 - \sin^2\alpha$ | 平方关系移项 |
| $\sin\alpha = \cos\alpha \cdot \tan\alpha$ | 商数关系两边乘 $\cos\alpha$ |
| $\sin\alpha = \pm\sqrt{1 - \cos^2\alpha}$ | 开方，符号由象限决定 |
| $\cos\alpha = \pm\sqrt{1 - \sin^2\alpha}$ | 开方，符号由象限决定 |

## 22.7 例题

### 例题 1：用终边上任意一点求三角函数值

**题目**：已知角 $\alpha$ 终边上一点 $P(-3, 4)$，求 $\sin\alpha, \cos\alpha, \tan\alpha$。

**解**：

> 注意：$P(-3, 4)$ 不在单位圆上（因为 $(-3)^2 + 4^2 = 25 \neq 1$）。但我们可以利用相似三角形：先求出 $P$ 到原点的距离 $r$，然后单位圆上的对应点坐标就是 $(\frac{x}{r}, \frac{y}{r})$。

**步骤1**：求 $P$ 到原点的距离 $r$。

$$r = \sqrt{(-3)^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

**步骤2**：计算三角函数值。

$$\sin\alpha = \frac{y}{r} = \frac{4}{5}$$

$$\cos\alpha = \frac{x}{r} = \frac{-3}{5} = -\frac{3}{5}$$

$$\tan\alpha = \frac{y}{x} = \frac{4}{-3} = -\frac{4}{3}$$

**步骤3**：验证。

检查 $\sin^2\alpha + \cos^2\alpha = \left(\dfrac{4}{5}\right)^2 + \left(-\dfrac{3}{5}\right)^2 = \dfrac{16}{25} + \dfrac{9}{25} = \dfrac{25}{25} = 1$。✅

---

### 例题 2：知一求二（给定象限）

**题目**：已知 $\sin\alpha = \dfrac{3}{5}$，且 $\alpha$ 是第二象限角，求 $\cos\alpha$ 和 $\tan\alpha$。

**解**：

**步骤1**：由 $\sin^2\alpha + \cos^2\alpha = 1$ 得：

$$\cos^2\alpha = 1 - \sin^2\alpha = 1 - \left(\frac{3}{5}\right)^2 = 1 - \frac{9}{25} = \frac{16}{25}$$

**步骤2**：开方，$\cos\alpha = \pm \dfrac{4}{5}$。

**步骤3**：确定符号。$\alpha$ 是第二象限角，第二象限的余弦为负。

$$\cos\alpha = -\frac{4}{5}$$

**步骤4**：求 $\tan\alpha$。

$$\tan\alpha = \frac{\sin\alpha}{\cos\alpha} = \frac{3/5}{-4/5} = -\frac{3}{4}$$

---

### 例题 3：知一求二（未给定象限——需要分类讨论）

**题目**：已知 $\cos\alpha = -\dfrac{5}{13}$，求 $\sin\alpha$ 和 $\tan\alpha$。

**解**：

**步骤1**：由 $\sin^2\alpha + \cos^2\alpha = 1$ 得：

$$\sin^2\alpha = 1 - \cos^2\alpha = 1 - \left(-\frac{5}{13}\right)^2 = 1 - \frac{25}{169} = \frac{144}{169}$$

**步骤2**：开方，$\sin\alpha = \pm \dfrac{12}{13}$。

**步骤3**：题目没有给出 $\alpha$ 的象限，但 $\cos\alpha = -\dfrac{5}{13} < 0$，说明 $\alpha$ 在第二象限或第三象限。需要分情况讨论：

- **情况一**：$\alpha$ 在第二象限（$\sin\alpha > 0$）

  $\sin\alpha = \dfrac{12}{13}$，$\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha} = \dfrac{12/13}{-5/13} = -\dfrac{12}{5}$

- **情况二**：$\alpha$ 在第三象限（$\sin\alpha < 0$）

  $\sin\alpha = -\dfrac{12}{13}$，$\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha} = \dfrac{-12/13}{-5/13} = \dfrac{12}{5}$

> ⚠️ **易错点**：很多人看到 $\cos\alpha$ 是负数就直接开方取负号，忘了 $\sin\alpha$ 可能正也可能负。当题目没有明确象限时，**必须分类讨论**。

---

### 例题 4：利用基本关系式化简证明

**题目**：证明 $\dfrac{\sin\alpha}{1 - \cos\alpha} = \dfrac{1 + \cos\alpha}{\sin\alpha}$（$\sin\alpha \neq 0$）。

**解**：

**步骤1**：从左边出发，分子分母同乘 $(1 + \cos\alpha)$。

$$\frac{\sin\alpha}{1 - \cos\alpha} = \frac{\sin\alpha(1 + \cos\alpha)}{(1 - \cos\alpha)(1 + \cos\alpha)}$$

**步骤2**：分母用平方差公式 $(a-b)(a+b) = a^2 - b^2$。

$$(1 - \cos\alpha)(1 + \cos\alpha) = 1 - \cos^2\alpha$$

**步骤3**：由 $\sin^2\alpha + \cos^2\alpha = 1$ 知 $1 - \cos^2\alpha = \sin^2\alpha$。

$$= \frac{\sin\alpha(1 + \cos\alpha)}{\sin^2\alpha}$$

**步骤4**：约去一个 $\sin\alpha$（因为 $\sin\alpha \neq 0$）。

$$= \frac{1 + \cos\alpha}{\sin\alpha}$$

证毕。✅

## 22.8 练习题

### 基础练习

**1.** 已知角 $\alpha$ 终边上一点 $P(4, -3)$，求 $\sin\alpha, \cos\alpha, \tan\alpha$。

**2.** 判断下列各式的符号：
- (1) $\sin 120° \cdot \cos 210°$
- (2) $\tan 300°$

**3.** 已知 $\cos\alpha = \dfrac{4}{5}$，且 $\alpha$ 是第四象限角，求 $\sin\alpha$ 和 $\tan\alpha$。

**4.** 求值：$\sin^2 30° + \cos^2 60° + \tan 45°$。

**5.** 已知 $\sin\alpha = \dfrac{1}{3}$，求 $\cos\alpha$ 和 $\tan\alpha$（提示：需要分类讨论）。

---

### 提高练习

**6.** 已知 $\tan\alpha = 2$，求 $\dfrac{\sin\alpha + \cos\alpha}{\sin\alpha - \cos\alpha}$ 的值。（提示：分子分母同除以 $\cos\alpha$）

**7.** 证明：$(\sin\alpha + \cos\alpha)^2 + (\sin\alpha - \cos\alpha)^2 = 2$。

**8.** 已知 $\sin\alpha + \cos\alpha = \dfrac{1}{2}$，求 $\sin\alpha \cdot \cos\alpha$ 的值。（提示：两边平方，利用平方关系）

---

### 练习答案

**1.** 
- $r = \sqrt{4^2 + (-3)^2} = \sqrt{16 + 9} = \sqrt{25} = 5$
- $\sin\alpha = \dfrac{y}{r} = \dfrac{-3}{5} = -\dfrac{3}{5}$
- $\cos\alpha = \dfrac{x}{r} = \dfrac{4}{5}$
- $\tan\alpha = \dfrac{y}{x} = \dfrac{-3}{4} = -\dfrac{3}{4}$

**2.**
- (1) $\sin 120° > 0$（第二象限），$\cos 210° < 0$（第三象限），乘积为负。
- (2) $\tan 300°$：$300°$ 是第四象限角，第四象限 $\tan$ 为负，所以 $\tan 300° < 0$。具体地，$\tan 300° = \tan(360° - 60°) = -\tan 60° = -\sqrt{3}$。

**3.**
- $\sin^2\alpha = 1 - \cos^2\alpha = 1 - \left(\dfrac{4}{5}\right)^2 = 1 - \dfrac{16}{25} = \dfrac{9}{25}$
- 第四象限正弦为负，$\sin\alpha = -\dfrac{3}{5}$
- $\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha} = \dfrac{-3/5}{4/5} = -\dfrac{3}{4}$

**4.**
- $\sin 30° = \dfrac{1}{2}$，$\sin^2 30° = \dfrac{1}{4}$
- $\cos 60° = \dfrac{1}{2}$，$\cos^2 60° = \dfrac{1}{4}$
- $\tan 45° = 1$
- 原式 $= \dfrac{1}{4} + \dfrac{1}{4} + 1 = \dfrac{1}{2} + 1 = \dfrac{3}{2}$

**5.**
- $\cos^2\alpha = 1 - \sin^2\alpha = 1 - \dfrac{1}{9} = \dfrac{8}{9}$
- $\cos\alpha = \pm \dfrac{2\sqrt{2}}{3}$
- $\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha} = \dfrac{1/3}{\pm 2\sqrt{2}/3} = \pm \dfrac{1}{2\sqrt{2}} = \pm \dfrac{\sqrt{2}}{4}$
- 当 $\alpha$ 在第一象限时：$\cos\alpha = \dfrac{2\sqrt{2}}{3}$，$\tan\alpha = \dfrac{\sqrt{2}}{4}$
- 当 $\alpha$ 在第二象限时：$\cos\alpha = -\dfrac{2\sqrt{2}}{3}$，$\tan\alpha = -\dfrac{\sqrt{2}}{4}$

**6.**
- 分子分母同除以 $\cos\alpha$（$\cos\alpha \neq 0$，因为 $\tan\alpha = 2$ 有意义）：
- $\dfrac{\sin\alpha + \cos\alpha}{\sin\alpha - \cos\alpha} = \dfrac{\tan\alpha + 1}{\tan\alpha - 1} = \dfrac{2 + 1}{2 - 1} = \dfrac{3}{1} = 3$

**7.**
- 左边 $= (\sin^2\alpha + 2\sin\alpha\cos\alpha + \cos^2\alpha) + (\sin^2\alpha - 2\sin\alpha\cos\alpha + \cos^2\alpha)$
- $= (\sin^2\alpha + \cos^2\alpha) + 2\sin\alpha\cos\alpha + (\sin^2\alpha + \cos^2\alpha) - 2\sin\alpha\cos\alpha$
- $= 1 + 1 = 2$。证毕。

**8.**
- 两边平方：$(\sin\alpha + \cos\alpha)^2 = \left(\dfrac{1}{2}\right)^2$
- $\sin^2\alpha + 2\sin\alpha\cos\alpha + \cos^2\alpha = \dfrac{1}{4}$
- $1 + 2\sin\alpha\cos\alpha = \dfrac{1}{4}$
- $2\sin\alpha\cos\alpha = \dfrac{1}{4} - 1 = -\dfrac{3}{4}$
- $\sin\alpha\cos\alpha = -\dfrac{3}{8}$

| 1 | 单位圆定义 | $\sin\alpha = y$，$\cos\alpha = x$，$\tan\alpha = \dfrac{y}{x}$（$P(x,y)$ 是角 $\alpha$ 终边与单位圆的交点） |
| 2 | 终边上任意点 | 若终边上一点 $P(x_0, y_0)$，$r = \sqrt{x_0^2 + y_0^2}$，则 $\sin\alpha = \dfrac{y_0}{r}$，$\cos\alpha = \dfrac{x_0}{r}$，$\tan\alpha = \dfrac{y_0}{x_0}$ |
| 3 | 平方关系 | $\sin^2\alpha + \cos^2\alpha = 1$ |
| 4 | 商数关系 | $\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha}$（$\cos\alpha \neq 0$） |
| 5 | 符号口诀 | 一全正，二正弦，三正切，四余弦 |
| 6 | 特殊角值 | $0°, 30°, 45°, 60°, 90°$ 的 sin、cos、tan 值 |

### 三个最容易出错的地方

1. ⚠️ **开方不判断符号**。用 $\sin^2\alpha + \cos^2\alpha = 1$ 求值时要开方，必须根据 $\alpha$ 所在象限确定正负号。如果题目没给象限，必须分类讨论。
2. ⚠️ **把 $\sin^2\alpha$ 理解为 $\sin(\alpha^2)$**。$\sin^2\alpha$ 表示 $(\sin\alpha)^2$，即先算 $\sin\alpha$ 再平方。$\sin^2 30° = (\frac{1}{2})^2 = \frac{1}{4}$，不是 $\sin(900°) = \sin 180° = 0$。
3. ⚠️ **用 $\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha}$ 时忘了分母不为零**。$\alpha = 90° + k \cdot 180°$ 时 $\cos\alpha = 0$，$\tan\alpha$ 无定义。
