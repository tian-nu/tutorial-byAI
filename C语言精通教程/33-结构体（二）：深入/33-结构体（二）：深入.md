## 一、结构体数组

### 1.1 什么是结构体数组

*结构体数组* 就是把多个相同类型的结构体变量排列在一起，用下标访问。它的本质和 `int` 数组、`char` 数组完全一样，只是每个元素不是简单类型，而是一个结构体。

```text
struct Student class[3]:

  class[0]           class[1]           class[2]
+---------------+  +---------------+  +---------------+
| name: "张三"  |  | name: "李四"  |  | name: "王五"  |
| age:  20      |  | age:  21      |  | age:  19      |
| score: 85.5   |  | score: 90.0   |  | score: 78.0   |
+---------------+  +---------------+  +---------------+
```

### 1.2 定义与初始化

```c
#include <stdio.h>

typedef struct {
    char name[20];
    int age;
    float score;
} Student;

int main() {
    // 定义时同时初始化所有元素
    Student class[3] = {
        {"张三", 20, 85.5},
        {"李四", 21, 90.0},
        {"王五", 19, 78.0}
    };

    // 遍历打印
    for (int i = 0; i < 3; i++) {
        printf("学生%d: %s, %d岁, %.1f分\n",
               i + 1, class[i].name, class[i].age, class[i].score);
    }

    return 0;
}
```

注意 `class[i].name` 的写法：先通过 `class[i]` 拿到第 i 个结构体变量，再用 `.name` 访问它的成员。两个运算符的结合顺序从左到右。

### 1.3 向结构体数组输入数据

```c
Student class[40];
int n;

printf("请输入学生人数: ");
scanf("%d", &n);

for (int i = 0; i < n; i++) {
    printf("第%d个学生:\n", i + 1);
    printf("  姓名: ");
    scanf("%s", class[i].name);          // 字符串数组名本身就是地址
    printf("  年龄: ");
    scanf("%d", &class[i].age);          // 基本类型需要取地址
    printf("  成绩: ");
    scanf("%f", &class[i].score);        // 基本类型需要取地址
}
```

**关键注意**：`class[i].name` 是字符数组，数组名本身已经是地址，不需要 `&`。但 `class[i].age` 是整数变量，需要 `&` 取地址。这和不使用结构体时的规则完全一致。

## 二、结构体嵌套

### 2.1 什么是结构体嵌套

一个结构体的成员可以是另一个结构体。比如：

```text
struct Date {
    int year;
    int month;
    int day;
};

struct Student {
    char name[20];
    struct Date birthday;    // 嵌套了一个 Date 结构体
    float score;
};
```

每个学生的生日信息本身包含年月日三个字段，把它们打包成一个 `Date` 结构体，再嵌入到 `Student` 中。

```text
+-----------------------------------+
| Student                           |
|  name: "张三"                     |
|  birthday:  +-------------------+ |
|             | year:  2000       | |
|             | month: 5          | |
|             | day:   15         | |
|             +-------------------+ |
|  score: 85.5                      |
+-----------------------------------+
```

### 2.2 嵌套结构体的初始化

```c
typedef struct {
    int year;
    int month;
    int day;
} Date;

typedef struct {
    char name[20];
    Date birthday;
    float score;
} Student;

int main() {
    Student s1 = {"张三", {2000, 5, 15}, 85.5};

    // 访问嵌套成员：逐层用 . 钻进去
    printf("%s 出生于 %d年%d月%d日\n",
           s1.name,
           s1.birthday.year,   // 先进入 birthday，再进入 year
           s1.birthday.month,
           s1.birthday.day);

    return 0;
}
```

访问路径：`s1.birthday.year`。读作"`s1` 的 `birthday` 的 `year`"。

### 2.3 嵌套结构体的逐层赋值

```c
Student s1;
strcpy(s1.name, "李四");
s1.birthday.year = 2001;
s1.birthday.month = 8;
s1.birthday.day = 22;
s1.score = 92.0;
```

每一项都通过多层 `.` 来定位。

## 三、结构体作为函数参数

### 3.1 传值（传递结构体本身）

```c
#include <stdio.h>

typedef struct {
    int x;
    int y;
} Point;

// 传值：函数收到的是原结构体的完整副本
void print_point(Point p) {
    printf("(%d, %d)\n", p.x, p.y);
    p.x = 999;  // 修改的是副本，不影响原变量
}

int main() {
    Point pt = {10, 20};
    print_point(pt);
    printf("原始值: (%d, %d)\n", pt.x, pt.y);  // 仍然是 (10, 20)
    return 0;
}
```

**传值的特点**：
- 函数收到一份完整副本，修改副本不影响原值。
- 缺点是当结构体很大时（比如几百字节），拷贝开销大。

### 3.2 传指针（推荐做法）

```c
// 传指针：函数收到的是原结构体的地址
void move_point(Point *p, int dx, int dy) {
    p->x += dx;  // -> 运算符，等价于 (*p).x += dx
    p->y += dy;
}

int main() {
    Point pt = {10, 20};
    move_point(&pt, 5, -3);
    printf("移动后: (%d, %d)\n", pt.x, pt.y);  // (15, 17)
    return 0;
}
```

**传指针的特点**：
- 只传递 4 或 8 字节的地址，无论结构体多大。
- 可以通过指针直接修改原始数据。
- 需要配合 `->` 运算符使用。

### 3.3 传值 vs 传指针对比

| 方式 | 语法 | 修改原值？ | 拷贝开销 | 推荐场景 |
|------|------|-----------|---------|---------|
| 传值 | `void f(Point p)` | 否 | 大 | 结构体很小（< 16 字节）且不需要修改 |
| 传指针 | `void f(Point *p)` | 是 | 小 | 结构体较大，或需要修改原值 |

**经验法则**：除非你有特殊原因，否则**结构体参数永远传指针**。

## 四、结构体指针与 `->` 运算符

### 4.1 `->` 是什么

当你有一个指向结构体的指针时，不能直接用 `.` 访问成员——`.` 要求左边是一个结构体变量，而指针是一个地址值。

```c
Point pt = {10, 20};
Point *p = &pt;

// ❌ 错误：p 是指针，不是结构体变量
printf("%d\n", p.x);        // 编译失败

// ✅ 方法一：先解引用，再用 .
printf("%d\n", (*p).x);     // 可行但括号太多了

// ✅ 方法二：用 -> 运算符（推荐）
printf("%d\n", p->x);        // 简洁优雅
```

`p->x` 等价于 `(*p).x`。*箭头运算符* *此术语见附录F* `->` 就是"解引用然后访问成员"的快捷写法。

```text
p -> x
 |    |
 |    +-- 成员名
 +------- 指向结构体的指针
```

### 4.2 `->` 的实际应用

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    char name[20];
    int age;
} Student;

// 用指针改写学生信息
void set_student(Student *s, const char *name, int age) {
    strcpy(s->name, name);
    s->age = age;
}

// 打印学生信息（只读，传指针但加 const 防止误改）
void print_student(const Student *s) {
    printf("%s, %d岁\n", s->name, s->age);
}

int main() {
    Student s1;
    set_student(&s1, "张三", 20);
    print_student(&s1);
    return 0;
}
```

### 4.3 结构体数组 + 指针

```c
Student class[3] = {
    {"张三", 20},
    {"李四", 21},
    {"王五", 19}
};

Student *p = class;   // 指向数组首元素

for (int i = 0; i < 3; i++) {
    printf("%s, %d岁\n", p->name, p->age);
    p++;              // 指针移动到下一个 Student 元素
}
```

`p++` 会让指针跳过整整一个 `Student` 结构体的大小（不是 1 字节），自动指向下一个结构体的起始位置。

## 五、综合示例：学生成绩管理系统

将本章所有知识点串联起来：

```c
#include <stdio.h>
#include <string.h>

#define MAX_STUDENTS 100
#define NAME_LEN 20

typedef struct {
    char name[NAME_LEN];
    int age;
    float math;
    float english;
    float c_lang;
    float average;
} Student;

// 计算平均分（传指针）
void calc_average(Student *s) {
    s->average = (s->math + s->english + s->c_lang) / 3.0f;
}

// 打印单个学生信息（const 指针，只读）
void print_student(const Student *s) {
    printf("%-10s  %2d岁  数学:%-6.1f 英语:%-6.1f C语言:%-6.1f 平均:%-6.1f\n",
           s->name, s->age, s->math, s->english, s->c_lang, s->average);
}

// 打印全班信息
void print_all(const Student *class, int count) {
    printf("\n===== 全班成绩单 =====\n");
    for (int i = 0; i < count; i++) {
        printf("[%2d] ", i + 1);
        print_student(&class[i]);
    }
}

// 查找最高平均分的学生（返回指针）
const Student* find_best(const Student *class, int count) {
    if (count == 0) return NULL;
    const Student *best = class;
    for (int i = 1; i < count; i++) {
        if (class[i].average > best->average) {
            best = &class[i];
        }
    }
    return best;
}

int main() {
    Student class[MAX_STUDENTS];
    int count = 0;

    printf("请输入学生人数: ");
    scanf("%d", &count);

    for (int i = 0; i < count; i++) {
        printf("\n第%d个学生:\n", i + 1);
        printf("  姓名: ");
        scanf("%s", class[i].name);
        printf("  年龄: ");
        scanf("%d", &class[i].age);
        printf("  数学成绩: ");
        scanf("%f", &class[i].math);
        printf("  英语成绩: ");
        scanf("%f", &class[i].english);
        printf("  C语言成绩: ");
        scanf("%f", &class[i].c_lang);

        calc_average(&class[i]);
    }

    print_all(class, count);

    const Student *best = find_best(class, count);
    if (best != NULL) {
        printf("\n最高分学生: %s, 平均分: %.1f\n",
               best->name, best->average);
    }

    return 0;
}
```

## 六、常见错误

### 6.1 对指针用 `.` 而非 `->`

```c
Student *p = &s1;
// ❌ 错误
printf("%s\n", p.name);

// ✅ 正确
printf("%s\n", p->name);
```

编译器会报：`request for member 'name' in something not a structure or union`——意思是你对一个指针直接用了 `.`。

### 6.2 传值后奇怪"为什么没改"

```c
// ❌ 错误版本——函数改了，但 main 里没变
void grow(Student s) {
    s.age += 1;
}

// ✅ 正确版本——传指针
void grow(Student *s) {
    s->age += 1;
}
```

### 6.3 结构体数组下标越界后指针访问

```c
Student class[3];
Student *p = class;
for (int i = 0; i < 100; i++) {  // 越界！
    p->age = 0;   // 访问到不属于你的内存
    p++;
}
```

数组越界的后果在结构体场景下同样危险，因为结构体可能擦除其他变量的数据。

## 七、我做得对不对？

创建下面这个程序并运行：

```c
#include <stdio.h>

typedef struct {
    int width;
    int height;
} Rectangle;

// 计算面积（传指针）
int area(const Rectangle *r) {
    return r->width * r->height;
}

// 缩放矩形（传指针，会修改原值）
void scale(Rectangle *r, double factor) {
    r->width = (int)(r->width * factor);
    r->height = (int)(r->height * factor);
}

int main() {
    Rectangle rects[3] = {
        {10, 5},
        {8, 8},
        {12, 3}
    };

    for (int i = 0; i < 3; i++) {
        printf("矩形%d: %d x %d, 面积 = %d\n",
               i + 1,
               rects[i].width,
               rects[i].height,
               area(&rects[i]));
    }

    scale(&rects[0], 2.0);
    printf("放大后矩形1: %d x %d\n", rects[0].width, rects[0].height);

    return 0;
}
```

预期输出：
```
矩形1: 10 x 5, 面积 = 50
矩形2: 8 x 8, 面积 = 64
矩形3: 12 x 3, 面积 = 36
放大后矩形1: 20 x 10
```

## 八、不对怎么办？

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| `'->' applied to non-pointer` | 对结构体变量用了 `->` | 结构体变量用 `.`，指针用 `->` |
| `request for member in something not a structure` | 对指针用了 `.` | 把 `.` 换成 `->`，或者先解引用 |
| 函数内修改了但 main 里没变 | 传的是值而非指针 | 函数参数加上 `*`，调用时传 `&变量` |
| `incompatible types when assigning to type 'Student'` | typedef 没生效或类型名写错 | 检查 typedef 语法，确保别名在定义之后使用 |
| 打印嵌套成员时编译失败 | 某层中间类型不是结构体 | 逐层检查：`a.b.c` 要求 `b` 是一个结构体成员 |
