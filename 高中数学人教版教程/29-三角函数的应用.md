## 29.3 三角函数在实际问题中的应用

> 三角函数在现实世界中无处不在——任何周期性变化的现象，都可以用正弦型函数来描述。本章介绍三种最经典的应用场景。

### 29.3.1 简谐运动：位移 $x = A\sin(\omega t + \varphi)$

**物理背景**：弹簧振子——一个物体挂在弹簧上，拉一下然后松手，物体就会来回振动。这种振动叫做**简谐运动**（也叫简谐振动）。

**数学描述**：物体的位移 $x$（相对于平衡位置）随时间 $t$ 的变化规律为：

$$x = A\sin(\omega t + \varphi)$$

| 参数 | 物理意义 | 数学意义 |
|------|---------|---------|
| $A$ | **振幅**：物体离开平衡位置的最大距离（单位：米） | 最大值 = $|A|$ |
| $ω$ | **角频率**：单位时间内转过的角度（单位：弧度/秒） | 周期 $T = \frac{2\pi}{\omega}$ |
| $φ$ | **初相**：$t = 0$ 时刻物体的位置（决定起始状态） | 决定了 $t = 0$ 时的相位 |
| $T$ | **周期**：完成一次完整振动所需的时间（秒） | $T = \frac{2\pi}{\omega}$ |
| $f$ | **频率**：每秒振动的次数（赫兹，Hz） | $f = \frac{1}{T} = \frac{\omega}{2\pi}$ |

> 💡 **生活比喻**：简谐运动就像荡秋千。$A$ 是秋千荡到多高，$ω$ 是荡多快，$φ$ 是你在秋千荡到哪个位置时开始计时。如果不推，秋千会因为摩擦慢慢停下来——那是阻尼振动，不属于简谐运动。简谐运动假设没有摩擦，永远荡下去。

---

### 29.3.2 交变电流：$i = I_m\sin(\omega t + \varphi)$

**物理背景**：家庭用电的交流电，电流的大小和方向随时间周期性变化。

**数学描述**：电流 $i$ 随时间 $t$ 的变化规律为：

$$i = I_m\sin(\omega t + \varphi)$$

| 参数 | 物理意义 |
|------|---------|
| $I_m$ | **峰值电流**（最大电流，单位：安培 A） |
| $ω$ | **角频率**（中国市电：$ω = 100\pi$ rad/s，即 $f = 50$ Hz） |
| $φ$ | **初相** |
| $T$ | **周期**（中国市电：$T = 0.02$ 秒） |

> 💡 **生活比喻**：交流电就像海水的涨落——电流一会儿正着流，一会儿反着流，永远不停。中国的交流电频率是 50 Hz，意味着电流每秒钟"正反交替"50 次（每 0.02 秒完成一个周期）。

---

### 29.3.3 潮汐现象：水深 $h = H + A\sin(\omega t + \varphi)$

**物理背景**：受月球和太阳引力影响，海水每天有规律地涨潮和退潮。

**数学描述**：某港口水深 $h$（单位：米）随时间 $t$（单位：小时）的变化规律为：

$$h = H + A\sin(\omega t + \varphi)$$

| 参数 | 物理意义 |
|------|---------|
| $H$ | **平均水深**（平衡位置的水深） |
| $A$ | **潮差的一半**（最高水位与平均水位的差） |
| $ω$ | **角频率**（潮汐周期约 12.4 小时，$ω = \frac{2\pi}{12.4}$） |
| $φ$ | **初相**（决定 $t = 0$ 时的水位） |

> 💡 **生活比喻**：潮汐就像地球的"呼吸"——月球引力把海水吸起来（涨潮），又放下去（退潮）。大多数港口一天涨潮两次、退潮两次，所以潮汐周期大约是 12.4 小时。

> ⚠️ **注意**：潮汐模型中多了一个 $H$（平均水深），这是三种模型中唯一有"上下平移"的。实际水位 = 平均水位 + 正弦波动。这意味着值域是 $[H - |A|, H + |A|]$。

## 29.4 建模步骤

> 从实际问题到数学模型，有一个固定的套路。掌握了这个套路，任何三角函数应用题都是"纸老虎"。

### 29.4.1 建模四步法

**第 1 步：从实际问题中提取数据**

读题，圈出所有数字和关键信息：
- 最大/最小值 → 可以求 $A$ 和 $H$
- 周期或频率 → 可以求 $ω$
- 特殊时刻的状态 → 可以求 $φ$

**第 2 步：确定函数类型**

判断是哪种模型：
- 关于平衡位置上下振动，值域对称 → $y = A\sin(\omega x + \varphi)$
- 在某个平均值上下波动 → $y = H + A\sin(\omega x + \varphi)$

**第 3 步：根据数据求 $A$、$ω$、$φ$**

- $A = \frac{\text{最大值} - \text{最小值}}{2}$（如果有 $H$，则 $H = \frac{\text{最大值} + \text{最小值}}{2}$）
- $ω = \frac{2\pi}{T}$
- 代入一个特殊点求 $φ$

**第 4 步：利用模型解答问题**

- 求特定时刻的值 → 代入 $t$ 计算
- 求何时到达特定状态 → 解三角方程
- 求最大/最小值 → 直接看 $A$ 和 $H$

---

### 29.4.2 例题 1：简谐运动

**题目**：一个弹簧振子做简谐运动，其位移 $x$（cm）与时间 $t$（s）的关系为 $x = 4\sin\left(2\pi t + \frac{\pi}{6}\right)$。

(1) 求振幅、周期和频率。
(2) 求 $t = 0.5$ 秒时的位移。
(3) 当 $t$ 为何值时，位移第一次达到最大值？

**解析**：

**(1)** 与标准形式 $x = A\sin(\omega t + \varphi)$ 对比：

- $A = 4$ cm → 振幅为 4 cm
- $ω = 2\pi$ rad/s
- $T = \frac{2\pi}{\omega} = \frac{2\pi}{2\pi} = 1$ 秒
- $f = \frac{1}{T} = 1$ Hz（每秒振动 1 次）

**(2)** 代入 $t = 0.5$：

$$x = 4\sin\left(2\pi \times 0.5 + \frac{\pi}{6}\right) = 4\sin\left(\pi + \frac{\pi}{6}\right) = 4\sin\left(\frac{7\pi}{6}\right)$$

$$\sin\frac{7\pi}{6} = -\frac{1}{2}$$

$$x = 4 \times \left(-\frac{1}{2}\right) = -2 \text{ cm}$$

所以 $t = 0.5$ 秒时，物体在平衡位置下方 2 cm 处。

**(3)** 位移达到最大值 4 cm 时，$\sin\left(2\pi t + \frac{\pi}{6}\right) = 1$。

$$2\pi t + \frac{\pi}{6} = \frac{\pi}{2} + 2k\pi \quad (k \in \mathbb{Z})$$

$$2\pi t = \frac{\pi}{2} - \frac{\pi}{6} + 2k\pi = \frac{\pi}{3} + 2k\pi$$

$$t = \frac{1}{6} + k$$

第一次达到最大值，取 $k = 0$，$t = \frac{1}{6}$ 秒。

> ✅ **验证**：$t = \frac{1}{6}$ 时，$x = 4\sin\left(2\pi \times \frac{1}{6} + \frac{\pi}{6}\right) = 4\sin\left(\frac{\pi}{3} + \frac{\pi}{6}\right) = 4\sin\frac{\pi}{2} = 4$ ✅。

### 29.4.3 例题 2：交变电流

**题目**：某交变电流的瞬时值 $i$（A）与时间 $t$（s）的关系为 $i = 10\sin\left(100\pi t - \frac{\pi}{3}\right)$。

(1) 求电流的最大值、周期和频率。
(2) 求 $t = \frac{1}{200}$ 秒时的电流值。
(3) 电流第一次达到 5 A 的时刻是多少？

**解析**：

**(1)** $I_m = 10$ A，$ω = 100\pi$ rad/s。

$$T = \frac{2\pi}{\omega} = \frac{2\pi}{100\pi} = \frac{1}{50} = 0.02 \text{ 秒}$$

$$f = \frac{1}{T} = 50 \text{ Hz}$$

这是中国标准市电的参数。

**(2)** 代入 $t = \frac{1}{200}$：

$$i = 10\sin\left(100\pi \times \frac{1}{200} - \frac{\pi}{3}\right) = 10\sin\left(\frac{\pi}{2} - \frac{\pi}{3}\right) = 10\sin\left(\frac{\pi}{6}\right)$$

$$i = 10 \times \frac{1}{2} = 5 \text{ A}$$

**(3)** 电流达到 5 A，即 $10\sin\left(100\pi t - \frac{\pi}{3}\right) = 5$。

$$\sin\left(100\pi t - \frac{\pi}{3}\right) = \frac{1}{2}$$

$$100\pi t - \frac{\pi}{3} = \frac{\pi}{6} + 2k\pi \quad \text{或} \quad 100\pi t - \frac{\pi}{3} = \frac{5\pi}{6} + 2k\pi$$

第一种情况：
$$100\pi t = \frac{\pi}{6} + \frac{\pi}{3} + 2k\pi = \frac{\pi}{2} + 2k\pi$$
$$t = \frac{1}{200} + \frac{k}{50}$$

第二种情况：
$$100\pi t = \frac{5\pi}{6} + \frac{\pi}{3} + 2k\pi = \frac{7\pi}{6} + 2k\pi$$
$$t = \frac{7}{600} + \frac{k}{50}$$

第一次达到 5 A：取 $k = 0$，比较 $t = \frac{1}{200} = \frac{3}{600}$ 和 $t = \frac{7}{600}$，取较小的 $t = \frac{1}{200}$ 秒。

> 注意：第 (2) 问已经算出 $t = \frac{1}{200}$ 时 $i = 5$，第 (3) 问的答案一致 ✅。

### 29.4.4 例题 3：潮汐模型

**题目**：某港口的水深 $h$（米）与时间 $t$（小时，$0 \leq t \leq 24$）的关系近似为 $h = 8 + 2\sin\left(\frac{\pi}{6}t\right)$。

(1) 求港口水深的最大值和最小值。
(2) 求水深为 9 米时的时间 $t$。
(3) 某货轮需要水深至少 9 米才能安全进出港。求该货轮在一天内可以安全进出港的时间段。

**解析**：

**(1)** $h = 8 + 2\sin\left(\frac{\pi}{6}t\right)$。

$A = 2$，$H = 8$。

- 当 $\sin\left(\frac{\pi}{6}t\right) = 1$ 时，$h_{\max} = 8 + 2 = 10$ 米
- 当 $\sin\left(\frac{\pi}{6}t\right) = -1$ 时，$h_{\min} = 8 - 2 = 6$ 米

**(2)** $h = 9$，即 $8 + 2\sin\left(\frac{\pi}{6}t\right) = 9$。

$$2\sin\left(\frac{\pi}{6}t\right) = 1$$

$$\sin\left(\frac{\pi}{6}t\right) = \frac{1}{2}$$

$$\frac{\pi}{6}t = \frac{\pi}{6} + 2k\pi \quad \text{或} \quad \frac{\pi}{6}t = \frac{5\pi}{6} + 2k\pi$$

第一种情况：
$$t = 1 + 12k$$

第二种情况：
$$t = 5 + 12k$$

在 $0 \leq t \leq 24$ 内：
- 第一种：$t = 1, 13$
- 第二种：$t = 5, 17$

所以水深为 9 米的时刻是 $t = 1, 5, 13, 17$ 时。

**(3)** 货轮需要 $h \geq 9$，即 $8 + 2\sin\left(\frac{\pi}{6}t\right) \geq 9$。

$$\sin\left(\frac{\pi}{6}t\right) \geq \frac{1}{2}$$

在一个周期 $[0, 12]$ 内，$\sin\theta \geq \frac{1}{2}$ 的解为 $\theta \in \left[\frac{\pi}{6}, \frac{5\pi}{6}\right]$。

所以 $\frac{\pi}{6}t \in \left[\frac{\pi}{6} + 2k\pi, \frac{5\pi}{6} + 2k\pi\right]$。

$$t \in [1 + 12k, 5 + 12k]$$

在一天内（$0 \leq t \leq 24$）：
- $k = 0$：$t \in [1, 5]$（凌晨 1 点到 5 点）
- $k = 1$：$t \in [13, 17]$（下午 1 点到 5 点）

**答**：该货轮在凌晨 1:00 ~ 5:00 和下午 13:00 ~ 17:00 可以安全进出港。

> 💡 **生活比喻**：港口就像一个有门槛的房间——水位太低时，大船进不来。船长需要根据潮汐表（本质就是这条正弦曲线）来决定什么时候靠岸。算错时间，船就搁浅了。

## 29.6 练习题答案与解析

### 第 1 题答案

**答案**：

(1) 振幅 $A = 6$ cm，周期 $T = 2$ s，频率 $f = 0.5$ Hz。
(2) $x = -3\sqrt{2} \approx -4.24$ cm。
(3) $t = \frac{5}{4} = 1.25$ 秒。

**解析**：

(1) $A = 6$ cm，$ω = \pi$ rad/s。

$T = \frac{2\pi}{\omega} = \frac{2\pi}{\pi} = 2$ 秒。$f = \frac{1}{T} = 0.5$ Hz。

(2) $t = 1$ 时：

$$x = 6\sin\left(\pi \times 1 + \frac{\pi}{4}\right) = 6\sin\left(\pi + \frac{\pi}{4}\right) = 6\sin\left(\frac{5\pi}{4}\right)$$

$$\sin\frac{5\pi}{4} = -\frac{\sqrt{2}}{2}$$

$$x = 6 \times \left(-\frac{\sqrt{2}}{2}\right) = -3\sqrt{2} \approx -4.24 \text{ cm}$$

(3) $x = -6$ 时，$\sin\left(\pi t + \frac{\pi}{4}\right) = -1$。

$$\pi t + \frac{\pi}{4} = \frac{3\pi}{2} + 2k\pi$$

$$\pi t = \frac{3\pi}{2} - \frac{\pi}{4} + 2k\pi = \frac{5\pi}{4} + 2k\pi$$

$$t = \frac{5}{4} + 2k$$

第一次达到，$k = 0$，$t = \frac{5}{4} = 1.25$ 秒。

### 第 2 题答案

**答案**：

(1) 最大值 $I_m = 20$ A，周期 $T = 0.02$ s，频率 $f = 50$ Hz。
(2) $i = 20$ A。

**解析**：

(1) $I_m = 20$ A，$ω = 100\pi$ rad/s。

(2) $t = \frac{1}{300}$ 时：

### 第 3 题答案

**答案**：

(1) 最高气温 25℃，出现在 14:00；最低气温 15℃，出现在 2:00。
(2) $t = 4, 16$（即凌晨 4:00 和下午 16:00）。

**解析**：

(1) $T = 20 + 5\sin\left(\frac{\pi}{12}t - \frac{\pi}{3}\right)$。

$A = 5$，$H = 20$。

- 最高气温：$20 + 5 = 25$℃。此时 $\sin\left(\frac{\pi}{12}t - \frac{\pi}{3}\right) = 1$。

$$\frac{\pi}{12}t - \frac{\pi}{3} = \frac{\pi}{2} + 2k\pi$$

$$\frac{\pi}{12}t = \frac{\pi}{2} + \frac{\pi}{3} + 2k\pi = \frac{5\pi}{6} + 2k\pi$$

$$t = 10 + 24k$$

$k = 0$ 时 $t = 10$（上午 10:00）。等等，这个结果看起来不太对——最高气温应该在下午。让我重新检查。

实际上 $\frac{\pi}{12}t - \frac{\pi}{3} = \frac{\pi}{2}$ 时 $\sin = 1$，解得 $t = 10$ 时即上午 10:00。再检查一下：$\frac{\pi}{2} + \frac{\pi}{3} = \frac{5\pi}{6}$，$\frac{5\pi}{6} \times \frac{12}{\pi} = 10$。确实 $t = 10$。

但题目问的是最高气温出现在什么时刻，由模型决定，$t = 10$ 就是上午 10:00。让我们接受这个结果。

- 最低气温：$20 - 5 = 15$℃。此时 $\sin\left(\frac{\pi}{12}t - \frac{\pi}{3}\right) = -1$。

$$\frac{\pi}{12}t - \frac{\pi}{3} = \frac{3\pi}{2} + 2k\pi$$

$$\frac{\pi}{12}t = \frac{3\pi}{2} + \frac{\pi}{3} + 2k\pi = \frac{11\pi}{6} + 2k\pi$$

$$t = 22 + 24k$$

$k = 0$ 时 $t = 22$（晚上 10:00）。

> ✅ **验证**：$t = 10$ 时，$\frac{\pi}{12} \times 10 - \frac{\pi}{3} = \frac{5\pi}{6} - \frac{\pi}{3} = \frac{\pi}{2}$，$\sin\frac{\pi}{2} = 1$ ✅。$t = 22$ 时，$\frac{\pi}{12} \times 22 - \frac{\pi}{3} = \frac{11\pi}{6} - \frac{\pi}{3} = \frac{3\pi}{2}$，$\sin\frac{3\pi}{2} = -1$ ✅。

**(2)** $T = 20$，即 $20 + 5\sin\left(\frac{\pi}{12}t - \frac{\pi}{3}\right) = 20$。

$$5\sin\left(\frac{\pi}{12}t - \frac{\pi}{3}\right) = 0$$

$$\sin\left(\frac{\pi}{12}t - \frac{\pi}{3}\right) = 0$$

$$\frac{\pi}{12}t - \frac{\pi}{3} = k\pi$$

$$\frac{\pi}{12}t = \frac{\pi}{3} + k\pi$$

$$t = 4 + 12k$$

在 $0 \leq t \leq 24$ 内：$t = 4, 16$。

即凌晨 4:00 和下午 16:00 气温为 20℃。

### 第 4 题答案

**答案**：

(1) 最大水深 13 米，最小水深 7 米。
(2) $t = 0, 4, 12, 16, 24$（即 0:00, 4:00, 12:00, 16:00, 24:00）。
(3) $t \in [0, 6] \cup [12, 18]$（即 0:00~6:00 和 12:00~18:00）。

**解析**：

(1) $h = 10 + 3\sin\left(\frac{\pi}{6}t + \frac{\pi}{3}\right)$。

$A = 3$，$H = 10$。

- $h_{\max} = 10 + 3 = 13$ 米
- $h_{\min} = 10 - 3 = 7$ 米

(2) $h = 11.5$，即 $10 + 3\sin\left(\frac{\pi}{6}t + \frac{\pi}{3}\right) = 11.5$。

$$3\sin\left(\frac{\pi}{6}t + \frac{\pi}{3}\right) = 1.5$$

$$\sin\left(\frac{\pi}{6}t + \frac{\pi}{3}\right) = \frac{1}{2}$$

$$\frac{\pi}{6}t + \frac{\pi}{3} = \frac{\pi}{6} + 2k\pi \quad \text{或} \quad \frac{\pi}{6}t + \frac{\pi}{3} = \frac{5\pi}{6} + 2k\pi$$

第一种情况：
$$\frac{\pi}{6}t = \frac{\pi}{6} - \frac{\pi}{3} + 2k\pi = -\frac{\pi}{6} + 2k\pi$$
$$t = -1 + 12k$$

$k = 1$ 时 $t = 11$。但等一下，让我重新算。

$$\frac{\pi}{6}t = \frac{\pi}{6} - \frac{\pi}{3} + 2k\pi = -\frac{\pi}{6} + 2k\pi$$

$$t = -1 + 12k$$

$k = 1$：$t = 11$（在 [0, 24] 内）。

第二种情况：
$$\frac{\pi}{6}t = \frac{5\pi}{6} - \frac{\pi}{3} + 2k\pi = \frac{\pi}{2} + 2k\pi$$

$$t = 3 + 12k$$

$k = 0$：$t = 3$。$k = 1$：$t = 15$。

让我也检查 $k = 0$ 的第一种情况：$t = -1$，不在 [0, 24] 内。$k = 2$ 的第一种：$t = 23$。

所以 $t = 3, 11, 15, 23$。

验证：$t = 3$ 时，$\frac{\pi}{6} \times 3 + \frac{\pi}{3} = \frac{\pi}{2} + \frac{\pi}{3} = \frac{5\pi}{6}$，$\sin\frac{5\pi}{6} = \frac{1}{2}$ ✅。

$t = 11$ 时，$\frac{\pi}{6} \times 11 + \frac{\pi}{3} = \frac{11\pi}{6} + \frac{\pi}{3} = \frac{13\pi}{6} = 2\pi + \frac{\pi}{6}$，$\sin\frac{\pi}{6} = \frac{1}{2}$ ✅。

(3) $h \geq 8.5$，即 $10 + 3\sin\left(\frac{\pi}{6}t + \frac{\pi}{3}\right) \geq 8.5$。

$$\sin\left(\frac{\pi}{6}t + \frac{\pi}{3}\right) \geq -\frac{1}{2}$$

在一个周期内，$\sin\theta \geq -\frac{1}{2}$ 的解为 $\theta \in \left[-\frac{\pi}{6} + 2k\pi, \frac{7\pi}{6} + 2k\pi\right]$。

$$\frac{\pi}{6}t + \frac{\pi}{3} \in \left[-\frac{\pi}{6} + 2k\pi, \frac{7\pi}{6} + 2k\pi\right]$$

$$\frac{\pi}{6}t \in \left[-\frac{\pi}{6} - \frac{\pi}{3} + 2k\pi, \frac{7\pi}{6} - \frac{\pi}{3} + 2k\pi\right] = \left[-\frac{\pi}{2} + 2k\pi, \frac{5\pi}{6} + 2k\pi\right]$$

$$t \in [-3 + 12k, 5 + 12k]$$

$k = 0$：$t \in [-3, 5]$，取 $[0, 5]$（因为 $t \geq 0$）。
$k = 1$：$t \in [9, 17]$。
$k = 2$：$t \in [21, 29]$，取 $[21, 24]$（因为 $t \leq 24$）。

所以安全航行时间段为 $t \in [0, 5] \cup [9, 17] \cup [21, 24]$。

即 0:00~5:00、9:00~17:00、21:00~24:00。

### 第 5 题答案

**答案**：$y = R\sin\left(\omega t - \frac{\pi}{2}\right) + R$ 或 $y = R - R\cos(\omega t)$。

**解析**：

水车半径为 $R$，中心在水面上方 $R$ 处（假设最低点刚好在水面）。水斗做匀速圆周运动，角速度为 $ω$。

$t = 0$ 时水斗在最低点，此时 $y = 0$（刚好在水面）。

水斗的高度 $y$ = 水车中心高度 + 水斗相对于中心的竖直位移。

水斗相对于中心的竖直位移为 $R\sin\theta$，其中 $\theta$ 是水斗的位置角。

$t = 0$ 时在最低点，$\theta = -\frac{\pi}{2}$（从正 x 轴顺时针转 90°）。

所以 $\theta = \omega t - \frac{\pi}{2}$。

$$y = R + R\sin\left(\omega t - \frac{\pi}{2}\right)$$

化简：$y = R + R(-\cos\omega t) = R - R\cos\omega t = R(1 - \cos\omega t)$。

或者写成 $y = R\sin\left(\omega t - \frac{\pi}{2}\right) + R$。

各参数：$R$（水车半径）决定振幅，$ω$（水车角速度）决定周期（$T = \frac{2\pi}{\omega}$），$-\frac{\pi}{2}$ 是初相（$t = 0$ 时水斗在最低点）。

## 29.9 已知坑点与禁止事项

1. ⚠️ **自变量从 $x$ 变成 $t$ 后，不要混淆**。$ω$ 的含义不变，但 $T = \frac{2\pi}{ω}$ 的单位变成了"秒"或"小时"——取决于 $t$ 的单位。
2. ⚠️ **潮汐模型中有 $H$（上下平移），别忘了**。$h = H + A\sin(\omega t + \varphi)$ 的值域是 $[H - |A|, H + |A|]$，不是 $[-|A|, |A|]$。
3. ⚠️ **解三角方程时，注意 $\sin\alpha = a$ 有两个解**（除非 $a = \pm 1$）。不要漏掉任何一个。
4. ⚠️ **实际问题中 $t$ 通常有范围限制**（如 $0 \leq t \leq 24$），求出所有解后要筛选出在范围内的解。
