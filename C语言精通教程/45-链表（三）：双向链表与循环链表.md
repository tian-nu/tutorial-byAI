## 我在做什么？

单向链表有一个致命的缺点：**你只能往前走，永远回不了头**。就像一条单行道，错过了出口就必须绕一大圈才能回来。如果你在链表中间，想知道前一个节点是什么，唯一的办法是从头再走一遍。

本章解决两个问题：
1. **双向链表**：每个节点既有"往前"的指针也有"往后"的指针，像一条双向六车道。
2. **循环链表**：最后一个节点不指向 NULL，而是指回头节点，形成闭环。

## 45.1 双向链表——两条腿走路

`*此术语见附录F*`

### 节点结构体：两个指针

```c
typedef struct DNode {
    int            data;
    struct DNode  *prev;   /* 指向前一个节点 */
    struct DNode  *next;   /* 指向后一个节点 */
} DNode;
```

```
        ┌──────────┐     ┌──────────┐     ┌──────────┐
NULL <--│prev      │<────│prev      │<────│prev      │
        │ data: A  │     │ data: B  │     │ data: C  │
        │ next ────┼────>│ next ────┼────>│ next ────┼──> NULL
        └──────────┘     └──────────┘     └──────────┘
```

双向链表的代价：每个节点多存一个指针（8字节在64位系统上），但换来的是前后都能走的灵活性。

### 插入操作：需要调整4个指针

在节点 `prev` 和 `prev->next` 之间插入新节点 `n`：

```
插入前:  ... <-> [prev] <-> [prev->next] <-> ...

步骤1: n->prev = prev
步骤2: n->next = prev->next
步骤3: prev->next->prev = n    (如果 prev->next 不为 NULL)
步骤4: prev->next = n
```

```
插入后:  ... <-> [prev] <-> [n] <-> [prev->next(旧)] <-> ...
```

⚠️ **与单向链表的关键区别**：单向链表插入只需改 2 个指针（`new->next` 和 `prev->next`），双向链表要改 4 个。**顺序非常重要**——如果先改 `prev->next`，原来的 `prev->next` 就找不到了！

```c
/*
 * dlist_insert_after: 在指定节点之后插入新节点
 * @node: 在此节点之后插入
 * @value: 要插入的数据
 * @return: 成功返回新节点指针，失败返回 NULL
 */
DNode* dlist_insert_after(DNode *node, int value) {
    if (node == NULL) return NULL;

    DNode *n = (DNode*)malloc(sizeof(DNode));
    if (n == NULL) return NULL;
    n->data = value;

    n->prev = node;               /* 步骤1: n 的前驱 */
    n->next = node->next;         /* 步骤2: n 的后继 */
    if (node->next != NULL) {
        node->next->prev = n;     /* 步骤3: 原后继的前驱指向 n */
    }
    node->next = n;               /* 步骤4: node 的后继指向 n */
    return n;
}
```

### 删除操作：改 2 个指针 + 释放

```
删除节点 [B]：

删除前:  ... <-> [A] <-> [B] <-> [C] <-> ...

步骤1: A->next = B->next   (即指向 C)
步骤2: C->prev = B->prev   (即指向 A)
步骤3: free(B)

删除后:  ... <-> [A] <-> [C] <-> ...
```

```c
/*
 * dlist_delete: 删除指定节点
 * @head: 指向头指针的指针（可能删除头节点）
 * @node: 要删除的节点
 * @return: 成功返回 0，失败返回 -1
 */
int dlist_delete(DNode **head, DNode *node) {
    if (node == NULL) return -1;

    /* 调整前驱的后继 */
    if (node->prev != NULL) {
        node->prev->next = node->next;
    } else {
        /* 删除的是头节点，更新头指针 */
        *head = node->next;
    }

    /* 调整后继的前驱 */
    if (node->next != NULL) {
        node->next->prev = node->prev;
    }

    free(node);
    return 0;
}
```

注意：双向链表删除**不需要遍历找到前驱**，因为 `node->prev` 直接就是。这是双向链表比单向链表最大的优势——删除操作从 O(n) 降到了 O(1)。

## 45.2 双向链表——完整代码

```c
/*
 * doubly_linked_list.c —— 双向链表完整实现
 */

#include <stdio.h>
#include <stdlib.h>

typedef struct DNode {
    int            data;
    struct DNode  *prev;
    struct DNode  *next;
} DNode;

/* 创建节点 */
DNode* dlist_create_node(int value) {
    DNode *n = (DNode*)malloc(sizeof(DNode));
    if (n == NULL) return NULL;
    n->data = value;
    n->prev = NULL;
    n->next = NULL;
    return n;
}

/* 头插法 */
int dlist_insert_head(DNode **head, int value) {
    DNode *n = dlist_create_node(value);
    if (n == NULL) return -1;
    n->next = *head;
    if (*head != NULL) {
        (*head)->prev = n;
    }
    *head = n;
    return 0;
}

/* 尾插法 */
int dlist_insert_tail(DNode **head, int value) {
    DNode *n = dlist_create_node(value);
    if (n == NULL) return -1;
    if (*head == NULL) {
        *head = n;
        return 0;
    }
    DNode *cur = *head;
    while (cur->next != NULL) cur = cur->next;
    cur->next = n;
    n->prev = cur;
    return 0;
}

/* 在指定节点后插入 */
DNode* dlist_insert_after(DNode *node, int value) {
    if (node == NULL) return NULL;
    DNode *n = dlist_create_node(value);
    if (n == NULL) return NULL;
    n->prev = node;
    n->next = node->next;
    if (node->next != NULL) node->next->prev = n;
    node->next = n;
    return n;
}

/* 删除节点 */
int dlist_delete(DNode **head, DNode *node) {
    if (node == NULL) return -1;
    if (node->prev != NULL) node->prev->next = node->next;
    else *head = node->next;
    if (node->next != NULL) node->next->prev = node->prev;
    free(node);
    return 0;
}

/* 正向打印 */
void dlist_print_forward(DNode *head) {
    if (head == NULL) { printf("(空链表)\n"); return; }
    printf("正向: ");
    DNode *cur = head;
    while (cur != NULL) {
        printf("%d", cur->data);
        if (cur->next) printf(" <-> ");
        cur = cur->next;
    }
    printf(" -> NULL\n");
}

/* 反向打印（利用 prev 指针，体现双向链表的优势） */
void dlist_print_backward(DNode *head) {
    if (head == NULL) { printf("(空链表)\n"); return; }
    /* 先走到尾部 */
    DNode *tail = head;
    while (tail->next != NULL) tail = tail->next;
    printf("反向: NULL <- ");
    while (tail != NULL) {
        printf("%d", tail->data);
        if (tail->prev) printf(" <-> ");
        tail = tail->prev;
    }
    printf("\n");
}

/* 释放链表 */
void dlist_free(DNode **head) {
    DNode *cur = *head;
    while (cur != NULL) {
        DNode *next = cur->next;
        free(cur);
        cur = next;
    }
    *head = NULL;
}

int main(void) {
    DNode *list = NULL;

    printf("===== 双向链表演示 =====\n");
    dlist_insert_tail(&list, 10);
    dlist_insert_tail(&list, 20);
    dlist_insert_tail(&list, 30);
    dlist_print_forward(list);
    dlist_print_backward(list);

    printf("\n在节点 20 后插入 25:\n");
    DNode *node20 = list->next;  /* 第二个节点 */
    dlist_insert_after(node20, 25);
    dlist_print_forward(list);

    printf("\n删除节点 20:\n");
    dlist_delete(&list, list->next);  /* 删除第二个节点 */
    dlist_print_forward(list);

    dlist_free(&list);
    return 0;
}
```

## 45.3 循环链表——首尾相连

`*此术语见附录F*`

循环链表有两种：**单向循环链表**和**双向循环链表**。它们的共同点：尾节点的 `next` 不指向 NULL，而是指向头节点。

```
单向循环链表:
    ┌──────────────────────────────────────┐
    │                                      │
    ▼                                      │
  [A] -> [B] -> [C] -┐                    │
                      └────────────────────┘
  没有 NULL！尾节点指向头节点。

双向循环链表:
    ┌──────────────────────────────────────┐
    │                                      │
    ▼                                      │
  [A] <-> [B] <-> [C] ─┐                  │
    ▲                    └──────────────────┘
    └──────────────────────────────────────┘
  头节点的 prev 指向尾节点，尾节点的 next 指向头节点。
```

### 遍历循环链表的关键变化

```c
/* ❌ 单向链表的遍历条件（在循环链表上会死循环！） */
while (current != NULL) {
    /* ... */
    current = current->next;
}
/* 循环链表没有 NULL，会无限循环 */

/* ✅ 循环链表的遍历：记住起点，转一圈回到起点就停 */
Node *start = head;
if (head != NULL) {
    do {
        printf("%d ", current->data);
        current = current->next;
    } while (current != start);
}
```

### 插入和删除的区别

循环链表与普通链表的插入/删除逻辑**几乎相同**，唯一区别：
- 尾节点的 `next` 指向头节点（而不是 NULL）
- 双向循环链表中，头节点的 `prev` 指向尾节点

```c
/* 单向循环链表：尾插法 */
void clist_insert_tail(Node **head, int value) {
    Node *n = create_node(value);
    if (n == NULL) return;

    if (*head == NULL) {
        *head = n;
        n->next = n;  /* 自己指向自己，形成一个节点的环 */
        return;
    }

    /* 找到尾节点（next 指向 head 的那个） */
    Node *tail = *head;
    while (tail->next != *head) {
        tail = tail->next;
    }
    tail->next = n;
    n->next = *head;  /* 新尾节点指向头节点 */
}
```

## 45.4 循环链表——完整代码

```c
/*
 * circular_linked_list.c —— 单向循环链表
 */

#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int          data;
    struct Node *next;
} Node;

Node* clist_create_node(int value) {
    Node *n = (Node*)malloc(sizeof(Node));
    if (n == NULL) return NULL;
    n->data = value;
    n->next = n;  /* 默认自己指向自己 */
    return n;
}

void clist_insert_head(Node **head, int value) {
    Node *n = clist_create_node(value);
    if (n == NULL) return;
    if (*head == NULL) {
        *head = n;
        return;
    }
    /* 找到尾节点 */
    Node *tail = *head;
    while (tail->next != *head) tail = tail->next;
    n->next = *head;
    tail->next = n;
    *head = n;
}

void clist_insert_tail(Node **head, int value) {
    Node *n = clist_create_node(value);
    if (n == NULL) return;
    if (*head == NULL) {
        *head = n;
        return;
    }
    Node *tail = *head;
    while (tail->next != *head) tail = tail->next;
    tail->next = n;
    n->next = *head;
}

void clist_print(Node *head) {
    if (head == NULL) { printf("(空链表)\n"); return; }
    Node *cur = head;
    int count = 0;
    printf("循环链表: ");
    do {
        printf("%d", cur->data);
        cur = cur->next;
        if (cur != head) printf(" -> ");
        count++;
    } while (cur != head && count < 100);  /* 安全上限，防止死循环 */
    printf(" -> (回到起点)\n");
}

void clist_free(Node **head) {
    if (*head == NULL) return;
    Node *cur = (*head)->next;
    while (cur != *head) {
        Node *next = cur->next;
        free(cur);
        cur = next;
    }
    free(*head);
    *head = NULL;
}

int main(void) {
    Node *clist = NULL;

    printf("===== 循环链表演示 =====\n");
    clist_insert_tail(&clist, 1);
    clist_insert_tail(&clist, 2);
    clist_insert_tail(&clist, 3);
    clist_print(clist);

    printf("从任意位置开始遍历一圈:\n");
    Node *start = clist->next;  /* 从第二个节点开始 */
    Node *cur = start;
    printf("从 %d 开始: ", start->data);
    do {
        printf("%d ", cur->data);
        cur = cur->next;
    } while (cur != start);
    printf("(回到 %d)\n", start->data);

    clist_free(&clist);
    return 0;
}
```

## 45.5 三种链表对比

| 特性 | 单向链表 | 双向链表 | 循环链表 |
|------|----------|----------|----------|
| 节点结构 | data + next | data + prev + next | 同单向或双向 |
| 遍历方向 | 只能向前 | 向前+向后 | 可一直循环 |
| 插入(已知位置) | O(1) | O(1) | O(1) |
| 删除(已知节点) | 需找前驱 O(n) | O(1) | 需找前驱 O(n) |
| 内存开销 | 1个指针 | 2个指针 | 1-2个指针 |
| 结束条件 | `next == NULL` | `next == NULL` | `next == head` |
| 典型应用 | 简单序列 | 需要双向遍历 | 轮询调度、缓冲区 |

## 我做得对不对？

1. **双向链表**：正向打印和反向打印，两者顺序应相反。
2. **双向链表插入**：插入后 `prev` 和 `next` 都正确，没有节点"丢失"。
3. **循环链表**：遍历一圈后回到起点，不会死循环。
4. **内存释放**：所有节点都被正确释放，无泄漏。

---

## 不对怎么办？

### 常见Bug 1：双向链表插入时指针顺序错误

```c
/* ❌ 先改了 node->next，原来的 node->next 找不到了 */
node->next = n;           /* 先改这个 */
n->next = node->next;     /* 此时 node->next 已经是 n 了！n 指向自己 */
```

✅ 正确顺序：先设置 `n` 的两个指针（它不依赖任何即将被修改的东西），再修改外部指针。

### 常见Bug 2：循环链表无限循环

```c
/* ❌ 用 != NULL 判断循环链表 */
while (current != NULL) {  /* 永远不会为 NULL，死循环！ */
    /* ... */
    current = current->next;
}
```

✅ 用 `do { ... } while (current != head);` 或加计数器设置上限。

### 常见Bug 3：双向链表删除时忘记处理头节点

```c
/* ❌ 删除头节点时没有更新 head */
dlist_delete(DNode **head, DNode *node) {
    node->prev->next = node->next;  /* 如果 node 是头节点，prev 是 NULL！ */
    /* ... */
}
```

✅ 必须判断 `node->prev == NULL`，此时更新 `*head`。

## 本章小结

| 要点 | 说明 |
|------|------|
| 双向链表节点 | `prev` + `next` 两个指针，前后都可走 |
| 双向链表插入 | 调整 4 个指针，注意顺序：先设新节点的指针，再改外部 |
| 双向链表删除 | 只需改 2 个指针 + `free`，不需要遍历找前驱 |
| 循环链表 | 尾节点 `next` 指向头节点，无 NULL |
| 循环链表遍历 | 用 `do-while` 或记录起点，避免死循环 |
| 选择建议 | 简单场景用单向，需要双向遍历用双向，需要轮询用循环 |
