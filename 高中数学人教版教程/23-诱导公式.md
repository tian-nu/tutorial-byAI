## 23.3 诱导公式的本质 —— 角的对称

> 在学任何公式之前，先建立几何直觉。诱导公式的本质是：**单位圆上，终边有对称关系的两个角，它们的三角函数值之间也有确定的关系。**

---

### 23.3.1 终边关于 x 轴对称（公式三）

**几何关系**：角 $-\alpha$ 的终边与角 $\alpha$ 的终边关于 $x$ 轴对称。

```
         y
         ↑
         |   P(x, y)  ← 角 α 的终边与单位圆的交点
         |  /
         | / α
  ———————O————————→ x
         | \ -α
         |  \
         |   Q(x, -y)  ← 角 -α 的终边与单位圆的交点
```

如果 $P(x, y)$ 是角 $\alpha$ 终边与单位圆的交点，那么关于 $x$ 轴对称的点 $Q$ 就是 $(x, -y)$，这正是角 $-\alpha$ 终边与单位圆的交点。

根据单位圆定义：
- $\sin(-\alpha) = -y = -\sin\alpha$
- $\cos(-\alpha) = x = \cos\alpha$
- $\tan(-\alpha) = \dfrac{-y}{x} = -\dfrac{y}{x} = -\tan\alpha$

**公式三**：

$$\boxed{\sin(-\alpha) = -\sin\alpha}$$

$$\boxed{\cos(-\alpha) = \cos\alpha}$$

$$\boxed{\tan(-\alpha) = -\tan\alpha}$$

> 💡 **通俗理解**：正弦是"奇函数"——取负角时它变号；余弦是"偶函数"——取负角时它不变。这跟它们的图像特征一致（正弦图像关于原点对称，余弦图像关于 y 轴对称，第24章会学到）。

---

### 23.3.2 终边关于 y 轴对称（公式二）

**几何关系**：角 $\pi - \alpha$ 的终边与角 $\alpha$ 的终边关于 $y$ 轴对称。

```
         y
         ↑
  Q(-x, y)│   P(x, y)
        \ │  /
   π-α  \│/ α
  ———————O————————→ x
```

如果 $P(x, y)$ 是角 $\alpha$ 终边与单位圆的交点，那么关于 $y$ 轴对称的点 $Q$ 就是 $(-x, y)$，这正是角 $\pi - \alpha$ 终边与单位圆的交点。

根据单位圆定义：
- $\sin(\pi - \alpha) = y = \sin\alpha$
- $\cos(\pi - \alpha) = -x = -\cos\alpha$
- $\tan(\pi - \alpha) = \dfrac{y}{-x} = -\dfrac{y}{x} = -\tan\alpha$

**公式二**：

$$\boxed{\sin(\pi - \alpha) = \sin\alpha}$$

$$\boxed{\cos(\pi - \alpha) = -\cos\alpha}$$

$$\boxed{\tan(\pi - \alpha) = -\tan\alpha}$$

> 💡 **通俗理解**：补角的正弦相等，补角的余弦互为相反数。这是初中就有的直觉——$\sin 150° = \sin 30° = \frac{1}{2}$，但 $\cos 150° = -\cos 30° = -\frac{\sqrt{3}}{2}$。

---

### 23.3.3 终边关于原点对称（公式四）

**几何关系**：角 $\pi + \alpha$ 的终边与角 $\alpha$ 的终边关于原点对称。

```
         y
         ↑
         |   P(x, y)
         |  /
         | / α
  ———————O————————→ x
    π+α /|
       / |
  Q(-x,-y)
```

如果 $P(x, y)$ 是角 $\alpha$ 终边与单位圆的交点，那么关于原点对称的点 $Q$ 就是 $(-x, -y)$，这正是角 $\pi + \alpha$ 终边与单位圆的交点。

根据单位圆定义：
- $\sin(\pi + \alpha) = -y = -\sin\alpha$
- $\cos(\pi + \alpha) = -x = -\cos\alpha$
- $\tan(\pi + \alpha) = \dfrac{-y}{-x} = \dfrac{y}{x} = \tan\alpha$

**公式四**：

$$\boxed{\sin(\pi + \alpha) = -\sin\alpha}$$

$$\boxed{\cos(\pi + \alpha) = -\cos\alpha}$$

$$\boxed{\tan(\pi + \alpha) = \tan\alpha}$$

> 💡 **通俗理解**：加 $180°$（半圈），正弦和余弦都变号，正切不变（因为负负得正）。

---

### 23.3.4 先看一个完整的推导例子

现在你已经有了三种对称关系，来做一个完整的推导练习：

**问题**：求 $\sin 210°$。

**思路**：$210° = 180° + 30°$，即 $\pi + \alpha$ 的形式（$\alpha = 30°$）。

用公式四（关于原点对称）：

$$\sin 210° = \sin(180° + 30°) = -\sin 30° = -\frac{1}{2}$$

**几何验证**：在单位圆上，$210°$ 的终边在第三象限。$210° = 180° + 30°$，从 $180°$ 再逆时针转 $30°$。终边与单位圆的交点的纵坐标 $y$ 是负的，且绝对值等于 $30°$ 时纵坐标的绝对值。$30°$ 时 $y = \frac{1}{2}$，所以 $210°$ 时 $y = -\frac{1}{2}$。$\sin 210° = -\frac{1}{2}$。✅

## 23.4 口诀："奇变偶不变，符号看象限"

> 这是高中数学中最著名的口诀之一。但很多学生只背口诀不理解含义，导致用错。本节把每个字都拆开来讲。

---

### 23.4.1 口诀适用的场景

这个口诀用于处理形如 $\dfrac{k\pi}{2} \pm \alpha$（$k \in \mathbb{Z}$）的角的三角函数。

即：目标是 $\dfrac{\pi}{2}$ 的整数倍加减一个角 $\alpha$。

---

### 23.4.2 "奇变偶不变"——函数名是否改变

**规则**：看 $\dfrac{k\pi}{2}$ 中的 $k$ 是奇数还是偶数。

- 若 $k$ 为**偶数**（即 $\dfrac{k\pi}{2}$ 是 $\pi$ 的整数倍，如 $0, \pi, 2\pi, 3\pi, \ldots$）：**函数名不变**。
- 若 $k$ 为**奇数**（即 $\dfrac{k\pi}{2}$ 是 $\dfrac{\pi}{2}$ 的奇数倍，如 $\dfrac{\pi}{2}, \dfrac{3\pi}{2}, \dfrac{5\pi}{2}, \ldots$）：**函数名改变**（$\sin \leftrightarrow \cos$，$\tan$ 变成什么后面说）。

**逐字解释**：
- "奇"：$k$ 是奇数
- "变"：函数名改变，$\sin$ 变成 $\cos$，$\cos$ 变成 $\sin$
- "偶"：$k$ 是偶数
- "不变"：函数名不变，$\sin$ 还是 $\sin$，$\cos$ 还是 $\cos$

**例子**：

| 原式 | $k$ | 奇偶 | 函数名变化 | 中间结果 |
|------|-----|------|-----------|---------|
| $\sin(\pi + \alpha)$ | $\pi = \dfrac{2\pi}{2}$，$k=2$ | 偶 | 不变 | $\sin(\pi+\alpha) = \pm \sin\alpha$ |
| $\sin\left(\dfrac{\pi}{2} - \alpha\right)$ | $\dfrac{\pi}{2}$，$k=1$ | 奇 | 变 | $\sin(\frac{\pi}{2}-\alpha) = \pm \cos\alpha$ |
| $\cos(\pi - \alpha)$ | $\pi = \dfrac{2\pi}{2}$，$k=2$ | 偶 | 不变 | $\cos(\pi-\alpha) = \pm \cos\alpha$ |
| $\cos\left(\dfrac{3\pi}{2} + \alpha\right)$ | $\dfrac{3\pi}{2}$，$k=3$ | 奇 | 变 | $\cos(\frac{3\pi}{2}+\alpha) = \pm \sin\alpha$ |

---

### 23.4.3 "符号看象限"——确定正负号

**规则**：把 $\alpha$ **视为锐角**（$0 < \alpha < \dfrac{\pi}{2}$），看 $\dfrac{k\pi}{2} \pm \alpha$ 所对应的**原函数**在目标角所在象限的符号。

**逐字解释**：
- "符号"：正号还是负号
- "看象限"：看 $\dfrac{k\pi}{2} \pm \alpha$ 这个角落在第几象限，然后根据**原函数**在该象限的符号确定

**操作步骤**（以 $\sin(\pi + \alpha)$ 为例）：

1. 把 $\alpha$ 视为锐角（想象 $\alpha = 30°$）
2. 确定 $\pi + \alpha$ 在第几象限：$\pi + \alpha$ 略大于 $180°$，在第三象限
3. 查看**原函数**（这里是 $\sin$）在第三象限的符号：$\sin$ 在第三象限为**负**
4. 所以结果前加**负号**

结合"奇变偶不变"：$k=2$（偶），函数名不变，结果为 $-\sin\alpha$。

$$\sin(\pi + \alpha) = -\sin\alpha$$

这跟我们在 23.3.3 节从对称性推导的结果完全一致。

---

### 23.4.4 完整示范

**例 1**：用口诀推导 $\cos\left(\dfrac{\pi}{2} - \alpha\right)$。

1. $k = 1$（$\dfrac{\pi}{2}$ 中 $\pi$ 的系数为 $\frac{1}{2}$，$k$ 是 1），**奇数** → 函数名**变**，$\cos$ 变成 $\sin$
2. 把 $\alpha$ 视为锐角，$\dfrac{\pi}{2} - \alpha$ 在第一象限
3. 原函数 $\cos$ 在第一象限为**正**
4. 结果：$\cos\left(\dfrac{\pi}{2} - \alpha\right) = +\sin\alpha$

$$\boxed{\cos\left(\frac{\pi}{2} - \alpha\right) = \sin\alpha}$$

**例 2**：用口诀推导 $\tan(\pi + \alpha)$。

1. $k = 2$（$\pi = \dfrac{2\pi}{2}$），**偶数** → 函数名**不变**
2. 把 $\alpha$ 视为锐角，$\pi + \alpha$ 在第三象限
3. 原函数 $\tan$ 在第三象限为**正**（口诀"三正切"）
4. 结果：$\tan(\pi + \alpha) = +\tan\alpha$

$$\boxed{\tan(\pi + \alpha) = \tan\alpha}$$

---

### 23.4.5 $\tan$ 的"变"是什么？

当 $k$ 为奇数时，$\tan$ 变成 $\dfrac{1}{\tan}$（即 $\cot$，余切）。但高中阶段通常不要求掌握 $\cot$，所以遇到 $\dfrac{\pi}{2}$ 的奇数倍时，更推荐用 $\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha}$ 分别处理。

例如：$\tan\left(\dfrac{\pi}{2} + \alpha\right) = \dfrac{\sin(\frac{\pi}{2} + \alpha)}{\cos(\frac{\pi}{2} + \alpha)} = \dfrac{\cos\alpha}{-\sin\alpha} = -\dfrac{1}{\tan\alpha}$

---

## 23.5 诱导公式汇总

### 23.5.1 六组公式表

| 公式编号 | 角的形式 | $\sin$ | $\cos$ | $\tan$ | 记忆线索 |
|:---:|------|------|------|------|------|
| 一 | $2k\pi + \alpha$ | $\sin\alpha$ | $\cos\alpha$ | $\tan\alpha$ | 终边相同的角（转整圈，值不变） |
| 二 | $\pi + \alpha$ | $-\sin\alpha$ | $-\cos\alpha$ | $\tan\alpha$ | 关于原点对称 |
| 三 | $-\alpha$ | $-\sin\alpha$ | $\cos\alpha$ | $-\tan\alpha$ | 关于 $x$ 轴对称 |
| 四 | $\pi - \alpha$ | $\sin\alpha$ | $-\cos\alpha$ | $-\tan\alpha$ | 关于 $y$ 轴对称 |
| 五 | $\dfrac{\pi}{2} - \alpha$ | $\cos\alpha$ | $\sin\alpha$ | — | 余角关系（$\sin$ 和 $\cos$ 互换） |
| 六 | $\dfrac{\pi}{2} + \alpha$ | $\cos\alpha$ | $-\sin\alpha$ | — | 余角加 $\frac{\pi}{2}$ |

---

### 23.5.2 统一记忆方法：利用单位圆对称性

如果你觉得口诀还是太抽象，那就回归几何：

- **公式一**（$2k\pi + \alpha$）：转整圈，终边不变，三角函数值不变。
- **公式二**（$\pi + \alpha$）：终边关于原点对称，$x$ 和 $y$ 都变号。$\sin$（$y$）变号，$\cos$（$x$）变号，$\tan$（$y/x$）不变。
- **公式三**（$-\alpha$）：终边关于 $x$ 轴对称，$y$ 变号，$x$ 不变。$\sin$ 变号，$\cos$ 不变，$\tan$ 变号。
- **公式四**（$\pi - \alpha$）：终边关于 $y$ 轴对称，$x$ 变号，$y$ 不变。$\sin$ 不变，$\cos$ 变号，$\tan$ 变号。
- **公式五**（$\frac{\pi}{2} - \alpha$）：在单位圆上，$\frac{\pi}{2} - \alpha$ 是 $\alpha$ 的余角。终边上的点 $(x, y)$ 变成了 $(y, x)$。所以 $\sin$ 和 $\cos$ 互换。
- **公式六**（$\frac{\pi}{2} + \alpha$）：从 $\frac{\pi}{2} - \alpha$ 再推一步——$\frac{\pi}{2} + \alpha = \pi - (\frac{\pi}{2} - \alpha)$，用公式四和公式五组合。

## 23.6 例题

### 例题 1：利用诱导公式求值

**题目**：求下列各式的值：
- (1) $\sin 210°$
- (2) $\cos(-60°)$
- (3) $\tan 150°$

**解**：

**(1) $\sin 210°$**：

$210° = 180° + 30° = \pi + 30°$

用公式二（或口诀：$k=2$ 偶，不变；$\pi+\alpha$ 在第三象限，$\sin$ 为负）：

$$\sin 210° = -\sin 30° = -\frac{1}{2}$$

**(2) $\cos(-60°)$**：

用公式三（关于 $x$ 轴对称，$\cos$ 不变）：

$$\cos(-60°) = \cos 60° = \frac{1}{2}$$

**(3) $\tan 150°$**：

$150° = 180° - 30° = \pi - 30°$

用公式四（或口诀：$k=2$ 偶，不变；$\pi-\alpha$ 在第二象限，$\tan$ 为负）：

$$\tan 150° = -\tan 30° = -\frac{\sqrt{3}}{3}$$

---

### 例题 2：化简含多个诱导公式的表达式

**题目**：化简 $\sin(180° + \alpha) \cdot \cos(90° + \alpha) + \sin(90° - \alpha) \cdot \cos(180° - \alpha)$。

**解**：

**逐项化简**：

- $\sin(180° + \alpha) = \sin(\pi + \alpha) = -\sin\alpha$（公式二）
- $\cos(90° + \alpha) = \cos\left(\dfrac{\pi}{2} + \alpha\right)$

  用口诀：$k=1$（奇），变 → $\cos$ 变 $\sin$；$\dfrac{\pi}{2}+\alpha$ 在第二象限，$\cos$ 为负 → $-\sin\alpha$

- $\sin(90° - \alpha) = \sin\left(\dfrac{\pi}{2} - \alpha\right)$

  用口诀：$k=1$（奇），变 → $\sin$ 变 $\cos$；$\dfrac{\pi}{2}-\alpha$ 在第一象限，$\sin$ 为正 → $\cos\alpha$

- $\cos(180° - \alpha) = \cos(\pi - \alpha) = -\cos\alpha$（公式四）

**代入**：

原式 $= (-\sin\alpha) \cdot (-\sin\alpha) + \cos\alpha \cdot (-\cos\alpha)$

$= \sin^2\alpha - \cos^2\alpha$

**进一步化简**：$\sin^2\alpha - \cos^2\alpha = -(\cos^2\alpha - \sin^2\alpha) = -\cos 2\alpha$

（如果你还没学二倍角公式，保留 $\sin^2\alpha - \cos^2\alpha$ 即可。）

---

### 例题 3：已知值求值（利用诱导公式转化）

**题目**：已知 $\sin 53° \approx 0.8$，求 $\cos 143°$ 的值。

**解**：

**步骤1**：把 $143°$ 与已知角 $53°$ 建立联系。

$143° = 90° + 53° = \dfrac{\pi}{2} + 53°$

**步骤2**：用诱导公式。

$\cos 143° = \cos\left(\dfrac{\pi}{2} + 53°\right)$

用口诀：$k=1$（奇），$\cos$ 变 $\sin$；$\dfrac{\pi}{2}+53°$ 在第二象限，$\cos$ 为负。

$$\cos 143° = -\sin 53° \approx -0.8$$

---

### 例题 4：证明三角恒等式

**题目**：证明 $\dfrac{\sin(540° - \alpha)}{\cos(-\alpha)} \cdot \dfrac{1}{\tan(\alpha - 180°)} = 1$。

**解**：

**步骤1**：逐项化简。

- $540° - \alpha = 360° + 180° - \alpha = 360° + (180° - \alpha)$

  $\sin(540° - \alpha) = \sin(360° + (180° - \alpha)) = \sin(180° - \alpha)$（公式一：转整圈，值不变）

  $\sin(180° - \alpha) = \sin\alpha$（公式四）

- $\cos(-\alpha) = \cos\alpha$（公式三）

- $\alpha - 180° = -(180° - \alpha)$

  $\tan(\alpha - 180°) = \tan(-(180° - \alpha)) = -\tan(180° - \alpha)$（公式三）

  $\tan(180° - \alpha) = -\tan\alpha$（公式四）

  所以 $\tan(\alpha - 180°) = -(-\tan\alpha) = \tan\alpha$

**步骤2**：代入原式。

原式 $= \dfrac{\sin\alpha}{\cos\alpha} \cdot \dfrac{1}{\tan\alpha} = \tan\alpha \cdot \dfrac{1}{\tan\alpha} = 1$

证毕。✅

- (1) $\sin 240°$
- (2) $\cos(-45°)$
- (3) $\tan 315°$

**2.** 化简：$\sin(180° + \alpha) + \cos(90° + \alpha)$。

**3.** 已知 $\cos 20° \approx 0.94$，求 $\sin 110°$ 的值。

**4.** 化简：$\dfrac{\sin(180° - \alpha) \cdot \cos(360° + \alpha)}{\tan(-\alpha) \cdot \cos(180° + \alpha)}$。

**5.** 用口诀推导 $\sin\left(\dfrac{3\pi}{2} + \alpha\right)$ 的化简结果。

---

### 提高练习

**6.** 已知 $\sin(\pi + \alpha) = \dfrac{3}{5}$，$\alpha$ 是第三象限角，求 $\cos(\alpha - 2\pi)$ 的值。

**7.** 证明：$\sin\left(\dfrac{5\pi}{2} + \alpha\right) = \cos\alpha$。

**8.** 已知 $\sin\alpha = \dfrac{1}{3}$，$\alpha$ 是第二象限角，求 $\dfrac{\sin(180° + \alpha) - \sin(90° + \alpha)}{\cos(360° - \alpha)}$ 的值。

---

### 练习答案

**1.**
- (1) $\sin 240° = \sin(180° + 60°) = -\sin 60° = -\dfrac{\sqrt{3}}{2}$
- (2) $\cos(-45°) = \cos 45° = \dfrac{\sqrt{2}}{2}$
- (3) $\tan 315° = \tan(360° - 45°) = \tan(-45°) = -\tan 45° = -1$

**2.**
- $\sin(180° + \alpha) = -\sin\alpha$
- $\cos(90° + \alpha) = -\sin\alpha$
- 原式 $= -\sin\alpha + (-\sin\alpha) = -2\sin\alpha$

**3.**
- $\sin 110° = \sin(90° + 20°)$
- $k=1$（奇），$\sin$ 变 $\cos$；$90°+20°$ 在第二象限，$\sin$ 为正
- $\sin 110° = \cos 20° \approx 0.94$

**4.**
- $\sin(180° - \alpha) = \sin\alpha$
- $\cos(360° + \alpha) = \cos\alpha$
- $\tan(-\alpha) = -\tan\alpha$
- $\cos(180° + \alpha) = -\cos\alpha$
- 原式 $= \dfrac{\sin\alpha \cdot \cos\alpha}{(-\tan\alpha) \cdot (-\cos\alpha)} = \dfrac{\sin\alpha \cdot \cos\alpha}{\tan\alpha \cdot \cos\alpha} = \dfrac{\sin\alpha}{\tan\alpha} = \dfrac{\sin\alpha}{\sin\alpha/\cos\alpha} = \cos\alpha$

**5.** $\sin\left(\dfrac{3\pi}{2} + \alpha\right)$：
- $k = 3$（奇），$\sin$ 变 $\cos$
- $\dfrac{3\pi}{2} + \alpha$ 在第四象限，$\sin$ 为负
- $\sin\left(\dfrac{3\pi}{2} + \alpha\right) = -\cos\alpha$

**6.**
- $\sin(\pi + \alpha) = -\sin\alpha = \dfrac{3}{5}$，所以 $\sin\alpha = -\dfrac{3}{5}$
- $\alpha$ 是第三象限角，$\sin\alpha < 0$（符合），$\cos\alpha < 0$
- $\cos^2\alpha = 1 - \sin^2\alpha = 1 - \dfrac{9}{25} = \dfrac{16}{25}$
- $\cos\alpha = -\dfrac{4}{5}$（第三象限，余弦为负）
- $\cos(\alpha - 2\pi) = \cos( -(2\pi - \alpha) ) = \cos(2\pi - \alpha)$（公式三）
- $= \cos(-\alpha)$（公式一：$2\pi$ 是整圈）
- $= \cos\alpha$（公式三）
- $= -\dfrac{4}{5}$

**7.**
- $\dfrac{5\pi}{2} = 2\pi + \dfrac{\pi}{2}$
- $\sin\left(\dfrac{5\pi}{2} + \alpha\right) = \sin\left(2\pi + \dfrac{\pi}{2} + \alpha\right)$
- $= \sin\left(\dfrac{\pi}{2} + \alpha\right)$（公式一：转整圈，值不变）
- $k=1$（奇），$\sin$ 变 $\cos$；$\dfrac{\pi}{2}+\alpha$ 在第二象限，$\sin$ 为正
- $= \cos\alpha$。证毕。

**8.**
- $\sin\alpha = \dfrac{1}{3}$，$\alpha$ 在第二象限
- $\cos^2\alpha = 1 - \sin^2\alpha = 1 - \dfrac{1}{9} = \dfrac{8}{9}$
- $\cos\alpha = -\dfrac{2\sqrt{2}}{3}$（第二象限，余弦为负）
- $\sin(180° + \alpha) = -\sin\alpha = -\dfrac{1}{3}$
- $\sin(90° + \alpha) = \cos\alpha = -\dfrac{2\sqrt{2}}{3}$
- $\cos(360° - \alpha) = \cos(-\alpha) = \cos\alpha = -\dfrac{2\sqrt{2}}{3}$
- 原式 $= \dfrac{-\frac{1}{3} - (-\frac{2\sqrt{2}}{3})}{-\frac{2\sqrt{2}}{3}} = \dfrac{-\frac{1}{3} + \frac{2\sqrt{2}}{3}}{-\frac{2\sqrt{2}}{3}}$
- 分子分母同乘 $3$：$= \dfrac{-1 + 2\sqrt{2}}{-2\sqrt{2}} = \dfrac{1 - 2\sqrt{2}}{2\sqrt{2}}$
- 有理化：$= \dfrac{(1 - 2\sqrt{2}) \cdot \sqrt{2}}{2\sqrt{2} \cdot \sqrt{2}} = \dfrac{\sqrt{2} - 4}{4}$

## 23.8 本章小结

### 核心公式速查

| 公式编号 | 角 | $\sin$ | $\cos$ | $\tan$ |
|:---:|------|------|------|------|
| 一 | $2k\pi + \alpha$ | $\sin\alpha$ | $\cos\alpha$ | $\tan\alpha$ |
| 二 | $\pi + \alpha$ | $-\sin\alpha$ | $-\cos\alpha$ | $\tan\alpha$ |
| 三 | $-\alpha$ | $-\sin\alpha$ | $\cos\alpha$ | $-\tan\alpha$ |
| 四 | $\pi - \alpha$ | $\sin\alpha$ | $-\cos\alpha$ | $-\tan\alpha$ |
| 五 | $\dfrac{\pi}{2} - \alpha$ | $\cos\alpha$ | $\sin\alpha$ | — |
| 六 | $\dfrac{\pi}{2} + \alpha$ | $\cos\alpha$ | $-\sin\alpha$ | — |

### 口诀完整解读

| 口诀字 | 含义 | 操作 |
|--------|------|------|
| 奇 | $\dfrac{k\pi}{2}$ 中 $k$ 为奇数 | 触发函数名变化 |
| 变 | 函数名改变 | $\sin \leftrightarrow \cos$ |
| 偶 | $\dfrac{k\pi}{2}$ 中 $k$ 为偶数 | 函数名不变 |
| 不变 | 函数名保持 | $\sin$ 还是 $\sin$，$\cos$ 还是 $\cos$ |
| 符号 | 正负号 | 根据原函数在目标象限的符号确定 |
| 看象限 | 把 $\alpha$ 视为锐角 | 确定目标角 $\dfrac{k\pi}{2} \pm \alpha$ 落在第几象限 |

### 三个最容易出错的地方

1. ⚠️ **"奇变偶不变"只适用于 $\dfrac{k\pi}{2} \pm \alpha$ 的形式**。如果是 $\pi \pm \alpha$（$k=2$），函数名不变；如果是 $\dfrac{\pi}{2} \pm \alpha$（$k=1$），函数名改变。不要把 $\pi \pm \alpha$ 误判为"奇"。
2. ⚠️ **"符号看象限"看的是原函数**。例如 $\cos(\pi + \alpha)$，看的是 $\cos$ 在第三象限的符号（负），而不是 $\sin$ 在第三象限的符号。
3. ⚠️ **把 $\alpha$ 视为锐角只是为了确定符号，实际 $\alpha$ 可以是任意角**。口诀只帮助你判断符号，公式本身对任意角 $\alpha$ 都成立。
