## 我在做什么？

到目前为止，我们学过的数据结构查找数据都需要"比较"：数组按下标直接访问 O(1) 但需要知道索引，链表遍历 O(n)，BST 二分比较 O(log n)。

有没有一种方法，能像数组按下标一样快速，但用任意值（字符串、ID等）作为"下标"？答案是**哈希表**。

想象一个巨大的图书馆，每本书都有一个唯一的 ISBN 号。你想找一本书，不是按分类号慢慢找，而是用一个**神奇的公式**把 ISBN 号直接转换成一个书架编号，然后径直走到那个书架前——书就在那里。

这个"神奇的公式"就是**哈希函数**。哈希表就是靠哈希函数把任意类型的键（key）映射到数组索引，从而实现 O(1) 的查找、插入和删除。

`*此术语见附录F*`：哈希表、哈希函数、键值对。

---

## 49.1 哈希表的核心思想

```
哈希表 = 数组 + 哈希函数

传统数组:  索引只能是整数 0, 1, 2, ...
           你必须有确切的索引才能访问

哈希表:    输入任意键（字符串、整数、结构体...）
           哈希函数把键映射成数组索引
           你就可以用 O(1) 时间存/取

示例:
  键 "alice"  →  hash("alice")  →  索引 3  →  table[3]
  键 "bob"    →  hash("bob")    →  索引 5  →  table[5]
  键 "charlie"→  hash("charlie")→  索引 1  →  table[1]
```

**键值对**（Key-Value Pair）：哈希表存储的是键值对，每个键对应一个值。就像字典中每个单词（键）对应一个释义（值）。

## 49.2 哈希函数

哈希函数把任意类型的键转换为一个整数（数组索引）。好的哈希函数要求：
1. **确定性**：同样的输入永远产生同样的输出。
2. **均匀分布**：输出尽可能分散，避免都挤到同一个索引。
3. **快速计算**：O(1) 时间复杂度。

### 最简单的哈希函数：除留余数法

```c
int hash_int(int key, int table_size) {
    return key % table_size;
}
```

### 字符串哈希函数：djb2

DJB2 是 Dan Bernstein 发明的一个简单但效果很好的字符串哈希函数：

```c
unsigned long hash_djb2(const char *str) {
    unsigned long hash = 5381;
    int c;
    while ((c = *str++)) {
        hash = ((hash << 5) + hash) + c;  /* hash * 33 + c */
    }
    return hash;
}
```

为什么是 5381 和 33？这是经验值，经过大量测试效果很好。你可以理解为一种"魔法数字"。

### 常见的字符串哈希函数

```c
/* 1. DJB2（推荐） */
unsigned long hash_djb2(const char *str) {
    unsigned long hash = 5381;
    int c;
    while ((c = *str++)) {
        hash = hash * 33 + c;
    }
    return hash;
}

/* 2. SDBM */
unsigned long hash_sdbm(const char *str) {
    unsigned long hash = 0;
    int c;
    while ((c = *str++)) {
        hash = c + (hash << 6) + (hash << 16) - hash;
    }
    return hash;
}

/* 3. 简单加法（不推荐，容易冲突） */
unsigned long hash_simple(const char *str) {
    unsigned long hash = 0;
    while (*str) {
        hash += *str;
        str++;
    }
    return hash;
}
```

> **为什么需要好的哈希函数**？如果哈希函数把所有键都映射到同一个索引，哈希表就退化成了链表，查找变成 O(n)。

## 49.3 哈希冲突

两个不同的键，经过哈希函数得到相同的索引，这就是**哈希冲突**。

```
hash("alice")  →  3
hash("bob")    →  3  ← 冲突！两个键映射到同一个位置
```

冲突不可避免（因为键的集合远大于数组大小），但我们可以解决冲突。两种主流方法：

### 链地址法（Chaining）

每个桶（数组的每个位置）是一条链表。冲突时把新节点追加到链表中。

```
table[0] → NULL
table[1] → [key:"charlie", val:100] → NULL
table[2] → NULL
table[3] → [key:"alice", val:200] → [key:"bob", val:300] → NULL
table[4] → NULL
```

查找时：先计算哈希值找到桶，然后遍历桶中的链表。

### 开放地址法（Open Addressing）

冲突时，用某种探测方式在数组中找下一个空位。常见方法：
- 线性探测：`index = (hash + i) % size`，i = 0, 1, 2, ...
- 平方探测：`index = (hash + i*i) % size`
- 双重哈希：用第二个哈希函数确定步长

本章重点实现**链地址法**，因为它实现简单且删除操作自然。

## 49.4 链地址法实现

### 数据结构定义

```c
/* 哈希表节点（链表节点） */
typedef struct HashNode {
    char             *key;      /* 键（字符串） */
    int               value;    /* 值 */
    struct HashNode  *next;     /* 链地址法：下一个节点 */
} HashNode;

/* 哈希表 */
typedef struct {
    HashNode **buckets;  /* 桶数组，每个元素是指向链表的指针 */
    int        size;     /* 桶的数量 */
    int        count;    /* 当前键值对数量 */
} HashTable;
```

### 创建和销毁

```c
HashTable* ht_create(int size) {
    HashTable *ht = (HashTable*)malloc(sizeof(HashTable));
    if (ht == NULL) return NULL;

    /* 桶的数量最好是质数，减少冲突 */
    ht->size = size;
    ht->count = 0;

    /* 分配桶数组，全部初始化为 NULL */
    ht->buckets = (HashNode**)calloc(size, sizeof(HashNode*));
    if (ht->buckets == NULL) {
        free(ht);
        return NULL;
    }
    return ht;
}

void ht_destroy(HashTable *ht) {
    if (ht == NULL) return;
    /* 释放每个桶的链表 */
    for (int i = 0; i < ht->size; i++) {
        HashNode *node = ht->buckets[i];
        while (node != NULL) {
            HashNode *next = node->next;
            free(node->key);   /* 注意：key 是 strdup 分配的 */
            free(node);
            node = next;
        }
    }
    free(ht->buckets);
    free(ht);
}
```

### 哈希函数（取 djb2 结果对表大小取模）

```c
static unsigned long hash_djb2(const char *str) {
    unsigned long hash = 5381;
    int c;
    while ((c = *str++)) {
        hash = ((hash << 5) + hash) + c;
    }
    return hash;
}

static int hash_index(const HashTable *ht, const char *key) {
    return (int)(hash_djb2(key) % ht->size);
}
```

### 插入（若键已存在则更新值）

```c
int ht_put(HashTable *ht, const char *key, int value) {
    if (ht == NULL || key == NULL) return -1;

    int index = hash_index(ht, key);

    /* 先检查桶中是否已有相同 key，有则更新 */
    HashNode *node = ht->buckets[index];
    while (node != NULL) {
        if (strcmp(node->key, key) == 0) {
            node->value = value;  /* 更新值 */
            return 0;
        }
        node = node->next;
    }

    /* 不存在，创建新节点插入头部（头插法，O(1)） */
    HashNode *new_node = (HashNode*)malloc(sizeof(HashNode));
    if (new_node == NULL) return -1;

    new_node->key = strdup(key);  /* 复制一份 key 字符串 */
    if (new_node->key == NULL) {
        free(new_node);
        return -1;
    }
    new_node->value = value;
    new_node->next = ht->buckets[index];  /* 头插法 */
    ht->buckets[index] = new_node;
    ht->count++;
    return 0;
}
```

### 查找

```c
int ht_get(const HashTable *ht, const char *key, int *out) {
    if (ht == NULL || key == NULL || out == NULL) return -1;

    int index = hash_index(ht, key);
    HashNode *node = ht->buckets[index];

    while (node != NULL) {
        if (strcmp(node->key, key) == 0) {
            *out = node->value;
            return 0;  /* 找到了 */
        }
        node = node->next;
    }
    return -1;  /* 没找到 */
}
```

### 删除

```c
int ht_delete(HashTable *ht, const char *key) {
    if (ht == NULL || key == NULL) return -1;

    int index = hash_index(ht, key);
    HashNode *node = ht->buckets[index];
    HashNode *prev = NULL;

    while (node != NULL) {
        if (strcmp(node->key, key) == 0) {
            /* 找到了，删除 */
            if (prev == NULL) {
                /* 删除的是链表头 */
                ht->buckets[index] = node->next;
            } else {
                prev->next = node->next;
            }
            free(node->key);
            free(node);
            ht->count--;
            return 0;
        }
        prev = node;
        node = node->next;
    }
    return -1;  /* 没找到 */
}
```

### 检查是否包含

```c
int ht_contains(const HashTable *ht, const char *key) {
    int dummy;
    return ht_get(ht, key, &dummy) == 0;
}
```

## 49.5 负载因子与扩容

**负载因子（Load Factor）** = 键值对数量 / 桶数量。

当负载因子过高时（比如 > 0.75），链表变长，查找效率下降。此时需要**扩容**：创建一个更大的哈希表，把所有元素重新哈希插入。

```c
/* 扩容：创建一个新的更大的哈希表，重新插入所有元素 */
int ht_resize(HashTable *ht, int new_size) {
    if (ht == NULL || new_size <= ht->size) return -1;

    /* 创建新桶数组 */
    HashNode **new_buckets = (HashNode**)calloc(new_size, sizeof(HashNode*));
    if (new_buckets == NULL) return -1;

    /* 遍历旧表，把每个节点重新哈希到新表 */
    for (int i = 0; i < ht->size; i++) {
        HashNode *node = ht->buckets[i];
        while (node != NULL) {
            HashNode *next = node->next;

            /* 重新计算哈希 */
            int new_index = (int)(hash_djb2(node->key) % new_size);

            /* 头插法插入新桶 */
            node->next = new_buckets[new_index];
            new_buckets[new_index] = node;

            node = next;
        }
    }

    /* 释放旧桶数组，替换为新桶 */
    free(ht->buckets);
    ht->buckets = new_buckets;
    ht->size = new_size;
    /* count 不变 */

    return 0;
}
```

## 49.6 完整代码清单

```c
/*
 * hash_table.c —— 哈希表完整实现（链地址法）
 * 编译: gcc -std=c99 -Wall -o hash hash_table.c
 */

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

/* ==================== 哈希表结构 ==================== */
typedef struct HashNode {
    char             *key;
    int               value;
    struct HashNode  *next;
} HashNode;

typedef struct {
    HashNode **buckets;
    int        size;
    int        count;
} HashTable;

/* ==================== 哈希函数 ==================== */
static unsigned long hash_djb2(const char *str) {
    unsigned long hash = 5381;
    int c;
    while ((c = *str++)) {
        hash = ((hash << 5) + hash) + c;
    }
    return hash;
}

static int hash_index(const HashTable *ht, const char *key) {
    return (int)(hash_djb2(key) % ht->size);
}

/* ==================== 创建/销毁 ==================== */
HashTable* ht_create(int size) {
    HashTable *ht = (HashTable*)malloc(sizeof(HashTable));
    if (ht == NULL) return NULL;
    ht->size = size;
    ht->count = 0;
    ht->buckets = (HashNode**)calloc(size, sizeof(HashNode*));
    if (ht->buckets == NULL) {
        free(ht);
        return NULL;
    }
    return ht;
}

void ht_destroy(HashTable *ht) {
    if (ht == NULL) return;
    for (int i = 0; i < ht->size; i++) {
        HashNode *node = ht->buckets[i];
        while (node != NULL) {
            HashNode *next = node->next;
            free(node->key);
            free(node);
            node = next;
        }
    }
    free(ht->buckets);
    free(ht);
}

/* ==================== 插入 ==================== */
int ht_put(HashTable *ht, const char *key, int value) {
    if (ht == NULL || key == NULL) return -1;
    int index = hash_index(ht, key);

    /* 检查是否已存在 */
    HashNode *node = ht->buckets[index];
    while (node != NULL) {
        if (strcmp(node->key, key) == 0) {
            node->value = value;
            return 0;
        }
        node = node->next;
    }

    /* 头插法 */
    HashNode *new_node = (HashNode*)malloc(sizeof(HashNode));
    if (new_node == NULL) return -1;
    new_node->key = strdup(key);
    if (new_node->key == NULL) {
        free(new_node);
        return -1;
    }
    new_node->value = value;
    new_node->next = ht->buckets[index];
    ht->buckets[index] = new_node;
    ht->count++;
    return 0;
}

/* ==================== 查找 ==================== */
int ht_get(const HashTable *ht, const char *key, int *out) {
    if (ht == NULL || key == NULL || out == NULL) return -1;
    int index = hash_index(ht, key);
    HashNode *node = ht->buckets[index];
    while (node != NULL) {
        if (strcmp(node->key, key) == 0) {
            *out = node->value;
            return 0;
        }
        node = node->next;
    }
    return -1;
}

/* ==================== 删除 ==================== */
int ht_delete(HashTable *ht, const char *key) {
    if (ht == NULL || key == NULL) return -1;
    int index = hash_index(ht, key);
    HashNode *node = ht->buckets[index];
    HashNode *prev = NULL;

    while (node != NULL) {
        if (strcmp(node->key, key) == 0) {
            if (prev == NULL) {
                ht->buckets[index] = node->next;
            } else {
                prev->next = node->next;
            }
            free(node->key);
            free(node);
            ht->count--;
            return 0;
        }
        prev = node;
        node = node->next;
    }
    return -1;
}

int ht_contains(const HashTable *ht, const char *key) {
    int dummy;
    return ht_get(ht, key, &dummy) == 0;
}

/* ==================== 扩容 ==================== */
int ht_resize(HashTable *ht, int new_size) {
    if (ht == NULL || new_size <= ht->size) return -1;
    HashNode **new_buckets = (HashNode**)calloc(new_size, sizeof(HashNode*));
    if (new_buckets == NULL) return -1;

    for (int i = 0; i < ht->size; i++) {
        HashNode *node = ht->buckets[i];
        while (node != NULL) {
            HashNode *next = node->next;
            int new_index = (int)(hash_djb2(node->key) % new_size);
            node->next = new_buckets[new_index];
            new_buckets[new_index] = node;
            node = next;
        }
    }

    free(ht->buckets);
    ht->buckets = new_buckets;
    ht->size = new_size;
    return 0;
}

/* ==================== 打印（调试用） ==================== */
void ht_print(const HashTable *ht) {
    if (ht == NULL) return;
    printf("哈希表 (size=%d, count=%d, load=%.2f)\n",
           ht->size, ht->count, (double)ht->count / ht->size);
    for (int i = 0; i < ht->size; i++) {
        if (ht->buckets[i] != NULL) {
            printf("  [%d]: ", i);
            HashNode *node = ht->buckets[i];
            while (node != NULL) {
                printf("(%s: %d)", node->key, node->value);
                if (node->next) printf(" -> ");
                node = node->next;
            }
            printf("\n");
        }
    }
}

/* ==================== 演示 ==================== */
int main(void) {
    HashTable *ht = ht_create(7);  /* 7个桶，小一点方便看冲突 */

    printf("===== 插入 =====\n");
    ht_put(ht, "alice", 100);
    ht_put(ht, "bob", 200);
    ht_put(ht, "charlie", 300);
    ht_put(ht, "dave", 400);
    ht_put(ht, "eve", 500);
    ht_put(ht, "frank", 600);
    ht_put(ht, "grace", 700);
    ht_put(ht, "heidi", 800);
    ht_print(ht);

    printf("\n===== 查找 =====\n");
    int val;
    if (ht_get(ht, "alice", &val) == 0) {
        printf("alice: %d\n", val);
    } else {
        printf("alice: 未找到\n");
    }
    if (ht_get(ht, "zoe", &val) == 0) {
        printf("zoe: %d\n", val);
    } else {
        printf("zoe: 未找到\n");
    }
    printf("包含 'bob'? %s\n", ht_contains(ht, "bob") ? "是" : "否");

    printf("\n===== 更新 =====\n");
    ht_put(ht, "alice", 999);
    ht_get(ht, "alice", &val);
    printf("alice 更新后: %d\n", val);

    printf("\n===== 删除 =====\n");
    ht_delete(ht, "bob");
    ht_delete(ht, "charlie");
    ht_print(ht);
    printf("包含 'bob'? %s\n", ht_contains(ht, "bob") ? "是" : "否");

    printf("\n===== 扩容 =====\n");
    ht_resize(ht, 17);
    ht_print(ht);

    ht_destroy(ht);
    return 0;
}
```

## 我做得对不对？

1. **插入**：插入8个键值对，负载因子 > 1，观察哪些桶有冲突（链表长度 > 1）。
2. **查找**：`alice` 能查到，`zoe` 查不到。
3. **更新**：`alice` 的值从 100 变为 999。
4. **删除**：删除 `bob` 和 `charlie` 后，`count` 正确减少。
5. **扩容**：从 size=7 扩容到 size=17，所有元素都还在，`count` 不变。

---

## 不对怎么办？

### 常见Bug 1：`strcmp` 比较的是指针而不是内容

```c
/* ❌ 用 == 比较字符串，实际上比较的是地址 */
if (node->key == key) {  /* 永远不要这样比较字符串！ */
    node->value = value;
}
```

✅ 必须用 `strcmp(node->key, key) == 0` 来比较字符串内容。

### 常见Bug 2：忘记 `strdup`，所有节点共享同一个 key

```c
/* ❌ 直接赋值指针，多个节点指向同一块内存 */
new_node->key = key;  /* key 是调用者传入的，可能被修改或释放 */
```

✅ 用 `strdup(key)` 复制一份，或者 `malloc + strcpy`。这样每个节点拥有自己的 key 副本。

### 常见Bug 3：删除时 `prev` 为 NULL 但忘记更新桶头

```c
/* ❌ prev 是 NULL 时，只执行了 free，没有更新桶头 */
if (prev == NULL) {
    free(node->key);
    free(node);
    /* 忘记写 ht->buckets[index] = node->next; */
}
```

✅ 删除链表头节点时，必须更新 `ht->buckets[index]`。

### 常见Bug 4：扩容后忘记释放旧桶数组

```c
/* ❌ 直接替换指针，旧桶数组泄漏 */
ht->buckets = new_buckets;
/* 忘记 free 旧的 ht->buckets */
```

✅ 先 `free(ht->buckets)`，再赋值。

### 常见Bug 5：哈希函数中 `%` 的使用错误

```c
/* ❌ 负数取模在C语言中行为不确定，可能得到负数索引 */
int index = hash_str(key) % size;  /* 如果 hash 返回负数？ */
```

✅ 用 `unsigned long` 类型的哈希值，或者 `(hash % size + size) % size`。

## 本章小结

| 要点 | 说明 |
|------|------|
| 哈希表原理 | 哈希函数将键映射为数组索引，O(1) 查找 |
| 哈希函数 | 确定性+均匀分布+快速，djb2 是常用字符串哈希 |
| 哈希冲突 | 不同键映射到同一索引，不可避免 |
| 链地址法 | 每个桶是一条链表，冲突时追加到链表 |
| 开放地址法 | 冲突时在数组中找下一个空位（线性探测等） |
| 负载因子 | count/size，过高时链表变长，需要扩容 |
| 扩容 | 创建更大的桶数组，所有元素重新哈希插入 |
| 时间复杂度 | 平均 O(1)，最坏（全冲突）O(n) |
| 与数组的区别 | 哈希表可以用任意类型作为"下标"（键） |

## 练习题

**题1（编程）**：实现一个函数 `ht_keys(HashTable *ht)`，返回所有键的字符串数组。提示：遍历所有桶的所有链表，收集键。

**题2（编程）**：实现开放地址法（线性探测）的哈希表。要求：每个桶不使用链表，冲突时线性探测下一个位置。需要标记"已删除"状态（使用 tombstone 标记）。

**题3（编程）**：用哈希表实现一个简单的"词频统计器"。输入一段文本（字符串），统计每个单词出现的次数，输出词频最高的前 5 个单词。

**题4（选做）**：比较链地址法和开放地址法在不同负载因子下的性能。生成随机键，分别测试插入和查找的时间，负载因子从 0.2 到 0.95，画出性能曲线。
