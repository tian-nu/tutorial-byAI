# 第17章 · 变量与常量

> "变量就是程序的记忆——给数据贴个标签，以后用名字就能找到它。Java对变量有一套严格的'户籍制度'：什么类型、占多少内存、默认值是什么，全都有规定。这一章帮你搞清楚Java的8种基本类型和它们的行为规则。"

---

## 17.1 变量的本质：内存里的一间房

变量有三个要素：
1. **名字**：你给它起的标签
2. **类型**：决定了这间房能存什么（整数？小数？字符？）
3. **值**：房间里实际装的东西

```java
int age = 25;
```

`int` 是类型，`age` 是名字，`25` 是值。这条语句做了两件事：在内存里划出一块32位的空间（int占32位），在里面写上25，然后给这块空间贴上"age"的标签。

### Java是强类型语言

每个变量必须先声明类型，才能使用。你不能把一个字符串赋值给一个int变量，也不能把一个数字直接当作布尔值。

```java
int age = "25岁";   // ❌ 编译错误！类型不匹配
int score = 95;
boolean isPass = score;  // ❌ 编译错误！int不能当boolean用
```

## 17.2 8种基本类型：Java的"原子类型"

Java有8种基本类型（primitive types），它们不是对象，直接存值，效率最高。其他所有类型（String、数组、类等）都是引用类型（存的是内存地址）。

### 整数四兄弟

| 类型 | 大小 | 范围 | 默认值 | 示例 |
|------|------|------|--------|------|
| `byte` | 8位（1字节） | -128 ~ 127 | 0 | `byte b = 100;` |
| `short` | 16位（2字节） | -32,768 ~ 32,767 | 0 | `short s = 30000;` |
| `int` | 32位（4字节） | -21亿 ~ 21亿 | 0 | `int i = 2000000000;` |
| `long` | 64位（8字节） | -9×10¹⁸ ~ 9×10¹⁸ | 0L | `long l = 9000000000000L;` |

**注意 `long`**：给long赋值时，如果数字超过int范围（21亿），必须在数字后面加 `L` 或 `l`（推荐大写L，小写l容易看成1）：

```java
long population = 8000000000L;  // ✅ 世界人口80亿，超过int范围
long population2 = 8000000000;  // ❌ 编译错误！整数默认是int类型，80亿超过int范围
```

### 小数两兄弟

| 类型 | 大小 | 精度 | 默认值 | 示例 |
|------|------|------|--------|------|
| `float` | 32位（4字节） | 约7位有效数字 | 0.0f | `float f = 3.14f;` |
| `double` | 64位（8字节） | 约16位有效数字 | 0.0d | `double d = 3.1415926535;` |

**Java里小数默认是double类型**。给float赋值必须在数字后加 `f` 或 `F`：

```java
float f = 3.14;    // ❌ 编译错误！3.14默认是double
float f = 3.14f;   // ✅
double d = 3.14;   // ✅ 默认就是double
```

> ❌ **常见错误**：`float price = 19.99;` 编译报错。记住：Java中小数默认为double，给float赋值必须加 `f`。

### 字符类型：char

| 类型 | 大小 | 范围 | 默认值 | 示例 |
|------|------|------|--------|------|
| `char` | 16位（2字节） | 0 ~ 65,535（Unicode） | '\u0000' | `char c = 'A';` |

`char` 用**单引号**（不是双引号！），存一个Unicode字符：

```java
char letter = 'A';
char chinese = '中';
char digit = '9';
char unicode = '\u4e2d';  // '中'的Unicode编码

// char运算时会变成int
char c = 'A';
System.out.println(c + 1);   // 66
System.out.println((char)(c + 1));  // 'B'
```

### 布尔类型：boolean

| 类型 | 大小 | 值 | 默认值 | 示例 |
|------|------|------|--------|------|
| `boolean` | JVM实现决定 | `true` 或 `false` | `false` | `boolean flag = true;` |

Java的 `boolean` **只能是 `true` 或 `false`**，不能用0和1代替（和C语言不同）：

```java
boolean isAdult = true;
boolean isPass = (score >= 60);  // 表达式的结果是boolean

// boolean isAdult = 1;   // ❌ 编译错误！Java不能用数字表示布尔值
```

## 17.3 声明与初始化的三种写法

```java
// 方式一：先声明，后赋值
int age;
age = 25;

// 方式二：声明同时赋值（最常用）
int age = 25;

// 方式三：一行声明多个同类型变量
int x = 1, y = 2, z = 3;
int a, b, c;  // 三个都未赋值
```

**局部变量必须先赋值才能使用**（类成员变量有默认值，后面讲）：

```java
int score;
System.out.println(score);  // ❌ 编译错误！局部变量使用前必须初始化
```

## 17.4 var关键字：让编译器猜类型（Java 10+）

Java 10引入了 `var`，可以省略局部变量的类型声明，让编译器自动推断：

```java
var name = "张三";      // 编译器推断为 String
var age = 25;           // 编译器推断为 int
var price = 19.99;      // 编译器推断为 double
var list = new ArrayList<String>();  // 编译器推断为 ArrayList<String>
```

**限制**：
- `var` 只能用于**局部变量**（方法里的变量），不能用于类成员变量、方法参数或返回类型
- 声明时必须初始化（编译器需要右边的值来推断类型）
- 一个var变量一旦确定类型，就不能改变

```java
var x = 10;    // x是int
x = "hello";   // ❌ 编译错误！x已经是int类型

var y;         // ❌ 编译错误！编译器无法推断类型
```

> 🤔 **想多一点**：`var` 是不是让Java变成了动态类型语言？不是。`var name = "张三"` 编译后等于 `String name = "张三"`，类型在**编译时**就确定了。`var` 只是帮你少打几个字，不改变Java的静态类型本质。生产代码里建议：简单类型（int、String）直接用类型名更清晰，复杂泛型类型（后面章节会讲）用 `var` 更简洁。

## 17.5 final常量：一旦赋值，终身不变

`final` 关键字修饰的变量称为常量——赋值后不能修改：

```java
final int MAX_SCORE = 100;
MAX_SCORE = 120;  // ❌ 编译错误！不能修改final变量

final double PI = 3.1415926535;
```

常量通常用**全大写+下划线**命名（业界惯例）：

```java
final int MAX_RETRY_COUNT = 3;
final String API_BASE_URL = "https://api.example.com";
final long ONE_DAY_MILLIS = 24 * 60 * 60 * 1000L;
```

`final` 可以修饰类成员变量和局部变量：

```java
public class Config {
    public static final String APP_NAME = "MyApp";  // 类常量（静态+final）
    public static final int DEFAULT_PORT = 8080;
}
```

## 17.6 命名规范：驼峰命名法

Java社区有很严格的命名规范，面试和Code Review时会被挑出来说：

| 类型 | 规范 | 示例 |
|------|------|------|
| 类名 | 大驼峰（PascalCase），每个单词首字母大写 | `HelloWorld`, `UserController`, `StudentManager` |
| 方法名 | 小驼峰（camelCase），第一个单词首字母小写 | `helloWorld()`, `getUserName()`, `calculateTotal()` |
| 变量名 | 小驼峰 | `userName`, `totalScore`, `maxRetryCount` |
| 常量名 | 全大写+下划线 | `MAX_VALUE`, `DEFAULT_TIMEOUT` |
| 包名 | 全小写，用 `.` 分隔 | `com.example.demo` |

## 17.7 类型转换：自动 vs 手动

### 自动类型转换（隐式转换）

小范围类型可以自动转换为大范围类型（不会丢数据）：

```java
byte → short → int → long → float → double
          ↑
         char
```

```java
int i = 100;
long l = i;      // ✅ int自动转long
float f = l;     // ✅ long自动转float
double d = f;    // ✅ float自动转double

char c = 'A';
int ascii = c;   // ✅ char自动转int，结果是65
```

### 强制类型转换（显式转换）

大范围转小范围必须手动写，可能**丢失数据**：

```java
double d = 3.99;
int i = (int) d;   // i = 3，小数部分被截断！

long big = 300;
byte small = (byte) big;  // small = 44（数据溢出！）
```

> ❌ **常见错误**：`int i = (int) 3.99;` 结果是 `3` 不是 `4`。Java的强制类型转换是**截断**不是四舍五入。要四舍五入，用 `Math.round()`。

## 本章小结

| 知识点 | 核心要点 |
|--------|----------|
| 8种基本类型 | byte/short/int/long（整数）、float/double（小数）、char（字符）、boolean（布尔） |
| 默认值 | 整数=0、小数=0.0、char='\u0000'、boolean=false。仅限成员变量，局部变量不适用 |
| long后缀 | 超过int范围的数字加 `L`：`8000000000L` |
| float后缀 | 小数默认double，给float赋值加 `f`：`3.14f` |
| var | Java 10+，编译器自动推断类型，只能用于局部变量 |
| final | 常量，赋值后不可修改。命名全大写+下划线 |
| 命名规范 | 大驼峰（类名）、小驼峰（变量/方法）、全大写下划线（常量） |
| 自动转换 | 小→大自动（byte→int→long→float→double） |
| 强制转换 | 大→小手动 `(type)`，截断处理 |
