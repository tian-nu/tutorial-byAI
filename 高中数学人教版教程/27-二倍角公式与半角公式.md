## 27.3 二倍角公式——令 β = α

> 二倍角公式的推导没有任何新思想——就是把第26章的和角公式里的 $\beta$ 换成 $\alpha$。

### 27.3.1 正弦的二倍角公式

由 $S_{(\alpha+\beta)}$：$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$。

令 $\beta = \alpha$：

$$\begin{aligned}
\sin 2\alpha &= \sin(\alpha + \alpha) \\
&= \sin\alpha\cos\alpha + \cos\alpha\sin\alpha \\
&= 2\sin\alpha\cos\alpha
\end{aligned}$$

$$\boxed{\sin 2\alpha = 2\sin\alpha\cos\alpha}$$

💡 **生活比喻**：$\sin 2\alpha$ 不是"$\sin\alpha$ 翻倍"，而是"$\sin\alpha$ 和 $\cos\alpha$ 的乘积再翻倍"。初学者最容易犯的错误就是 $\sin 2\alpha = 2\sin\alpha$——这是把 $\sin$ 当成线性函数了。

### 27.3.2 余弦的二倍角公式（三个等价形式）

由 $C_{(\alpha+\beta)}$：$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$。

令 $\beta = \alpha$：

$$\begin{aligned}
\cos 2\alpha &= \cos(\alpha + \alpha) \\
&= \cos\alpha\cos\alpha - \sin\alpha\sin\alpha \\
&= \cos^2\alpha - \sin^2\alpha
\end{aligned}$$

$$\boxed{\cos 2\alpha = \cos^2\alpha - \sin^2\alpha}$$

这是最原始的形式。利用 $\sin^2\alpha + \cos^2\alpha = 1$，可以推出另外两个等价形式：

- 用 $\cos^2\alpha = 1 - \sin^2\alpha$ 代入：

$$\cos 2\alpha = (1 - \sin^2\alpha) - \sin^2\alpha = 1 - 2\sin^2\alpha$$

$$\boxed{\cos 2\alpha = 1 - 2\sin^2\alpha}$$

- 用 $\sin^2\alpha = 1 - \cos^2\alpha$ 代入：

$$\cos 2\alpha = \cos^2\alpha - (1 - \cos^2\alpha) = 2\cos^2\alpha - 1$$

$$\boxed{\cos 2\alpha = 2\cos^2\alpha - 1}$$

> 🤔 **三个形式，何时用哪个？** 取决于题目中已知的是 $\sin\alpha$ 还是 $\cos\alpha$，以及你想保留 $\sin$ 还是 $\cos$：
> - 已知 $\sin\alpha$，用 $\cos 2\alpha = 1 - 2\sin^2\alpha$（含 $\sin$ 不含 $\cos$）。
> - 已知 $\cos\alpha$，用 $\cos 2\alpha = 2\cos^2\alpha - 1$（含 $\cos$ 不含 $\sin$）。
> - 已知 $\sin\alpha$ 和 $\cos\alpha$ 都有，用 $\cos 2\alpha = \cos^2\alpha - \sin^2\alpha$（两者都有）。

### 27.3.3 正切的二倍角公式

由 $T_{(\alpha+\beta)}$：$\tan(\alpha + \beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta}$。

令 $\beta = \alpha$：

$$\tan 2\alpha = \frac{\tan\alpha + \tan\alpha}{1 - \tan\alpha\tan\alpha} = \frac{2\tan\alpha}{1 - \tan^2\alpha}$$

$$\boxed{\tan 2\alpha = \frac{2\tan\alpha}{1 - \tan^2\alpha}}$$

> ⚠️ **前提**：$1 - \tan^2\alpha \neq 0$（即 $\tan\alpha \neq \pm 1$，$\alpha \neq \frac{\pi}{4} + \frac{k\pi}{2}$），且 $\tan\alpha$ 本身要有定义。

## 27.4 降幂公式——二倍角公式的逆用

> 🤔 二倍角公式 $\cos 2\alpha$ 的后两种形式中，$\sin^2\alpha$ 和 $\cos^2\alpha$ 出现在等式右边。如果我们把 $\sin^2\alpha$ 和 $\cos^2\alpha$ 解出来，就能把二次的三角函数降为一次。

### 27.4.1 从 $\cos 2\alpha = 1 - 2\sin^2\alpha$ 解出 $\sin^2\alpha$

$$\begin{aligned}
\cos 2\alpha &= 1 - 2\sin^2\alpha \\
2\sin^2\alpha &= 1 - \cos 2\alpha \\
\sin^2\alpha &= \frac{1 - \cos 2\alpha}{2}
\end{aligned}$$

$$\boxed{\sin^2\alpha = \frac{1 - \cos 2\alpha}{2}}$$

### 27.4.2 从 $\cos 2\alpha = 2\cos^2\alpha - 1$ 解出 $\cos^2\alpha$

$$\begin{aligned}
\cos 2\alpha &= 2\cos^2\alpha - 1 \\
2\cos^2\alpha &= 1 + \cos 2\alpha \\
\cos^2\alpha &= \frac{1 + \cos 2\alpha}{2}
\end{aligned}$$

$$\boxed{\cos^2\alpha = \frac{1 + \cos 2\alpha}{2}}$$

> 💡 **为什么叫"降幂"？** 左边是 $\sin^2\alpha$（二次），右边只有 $\cos 2\alpha$（一次三角函数）。从"二次"降到"一次"——这就是降幂的含义。反之，如果从右边的 $\cos 2\alpha$ 出发得到左边的 $\sin^2\alpha$，就叫"升幂"。

### 27.4.3 降幂公式的用途

降幂公式最常用于**积分计算**和**三角恒等式化简**中。当表达式里出现 $\sin^2\alpha$ 或 $\cos^2\alpha$ 时，用降幂公式可以将其转化为一次三角函数，使后续计算简化。

> 📝 **记忆口诀**：降幂公式 = 二倍角公式倒过来写。"$\sin^2$ 是一减，$\cos^2$ 是一加，都除以二。"

## 27.5 半角公式——把降幂公式里的 α 换成 α/2

> 降幂公式中左边是 $\sin^2\alpha$，右边是 $\cos 2\alpha$。如果把 $\alpha$ 换成 $\frac{\alpha}{2}$，左边就变成 $\sin^2\frac{\alpha}{2}$，右边变成 $\cos\alpha$。开方后得到半角公式。

### 27.5.1 推导 $\sin\frac{\alpha}{2}$

在 $\sin^2\alpha = \frac{1 - \cos 2\alpha}{2}$ 中，把 $\alpha$ 换成 $\frac{\alpha}{2}$：

$$\sin^2\frac{\alpha}{2} = \frac{1 - \cos\alpha}{2}$$

开方（注意 $\pm$ 号由 $\frac{\alpha}{2}$ 所在象限决定）：

$$\boxed{\sin\frac{\alpha}{2} = \pm\sqrt{\frac{1 - \cos\alpha}{2}}}$$

### 27.5.2 推导 $\cos\frac{\alpha}{2}$

在 $\cos^2\alpha = \frac{1 + \cos 2\alpha}{2}$ 中，把 $\alpha$ 换成 $\frac{\alpha}{2}$：

$$\cos^2\frac{\alpha}{2} = \frac{1 + \cos\alpha}{2}$$

开方：

$$\boxed{\cos\frac{\alpha}{2} = \pm\sqrt{\frac{1 + \cos\alpha}{2}}}$$

### 27.5.3 推导 $\tan\frac{\alpha}{2}$

由 $\tan\theta = \frac{\sin\theta}{\cos\theta}$：

$$\tan\frac{\alpha}{2} = \frac{\sin\frac{\alpha}{2}}{\cos\frac{\alpha}{2}} = \pm\sqrt{\frac{1 - \cos\alpha}{1 + \cos\alpha}}$$

但更常用的形式是**不带根号的有理式**：

$$\tan\frac{\alpha}{2} = \frac{\sin\alpha}{1 + \cos\alpha} = \frac{1 - \cos\alpha}{\sin\alpha}$$

$$\boxed{\tan\frac{\alpha}{2} = \frac{\sin\alpha}{1 + \cos\alpha} = \frac{1 - \cos\alpha}{\sin\alpha}}$$

> 🤔 **这两个有理式怎么来的？** 分子分母同乘共轭：
> - $\frac{\sin\frac{\alpha}{2}}{\cos\frac{\alpha}{2}} = \frac{2\sin\frac{\alpha}{2}\cos\frac{\alpha}{2}}{2\cos^2\frac{\alpha}{2}} = \frac{\sin\alpha}{1 + \cos\alpha}$
> - $\frac{\sin\frac{\alpha}{2}}{\cos\frac{\alpha}{2}} = \frac{2\sin^2\frac{\alpha}{2}}{2\sin\frac{\alpha}{2}\cos\frac{\alpha}{2}} = \frac{1 - \cos\alpha}{\sin\alpha}$

> ⚠️ **± 号怎么判断？** 半角公式中的 $\pm$ 取决于 $\frac{\alpha}{2}$ 所在的象限（不是 $\alpha$ 所在的象限！）。例如 $\alpha = 240^\circ$（第三象限），则 $\frac{\alpha}{2} = 120^\circ$（第二象限），此时 $\sin\frac{\alpha}{2} > 0$，$\cos\frac{\alpha}{2} < 0$。

## 27.6 万能公式——全部用 $\tan\frac{\alpha}{2}$ 表示

> 万能公式的名字来自它的"万能"——不论 $\sin\alpha$、$\cos\alpha$、$\tan\alpha$，都可以用 $\tan\frac{\alpha}{2}$ 的有理式表达。这有什么用？当你不确定如何处理 $\sin\alpha$ 和 $\cos\alpha$ 时，统一换成 $\tan\frac{\alpha}{2}$，问题就变成了一个关于 $t = \tan\frac{\alpha}{2}$ 的代数方程。

### 27.6.1 推导 $\sin\alpha$

由二倍角公式 $\sin 2\theta = 2\sin\theta\cos\theta$，令 $\theta = \frac{\alpha}{2}$：

$$\sin\alpha = 2\sin\frac{\alpha}{2}\cos\frac{\alpha}{2}$$

分子分母同除以 $\cos^2\frac{\alpha}{2}$：

$$\sin\alpha = \frac{2\sin\frac{\alpha}{2}\cos\frac{\alpha}{2}}{\sin^2\frac{\alpha}{2} + \cos^2\frac{\alpha}{2}} = \frac{2\tan\frac{\alpha}{2}}{1 + \tan^2\frac{\alpha}{2}}$$

$$\boxed{\sin\alpha = \frac{2\tan\frac{\alpha}{2}}{1 + \tan^2\frac{\alpha}{2}}}$$

### 27.6.2 推导 $\cos\alpha$

由二倍角公式 $\cos 2\theta = \cos^2\theta - \sin^2\theta$，令 $\theta = \frac{\alpha}{2}$：

$$\cos\alpha = \cos^2\frac{\alpha}{2} - \sin^2\frac{\alpha}{2}$$

分子分母同除以 $\cos^2\frac{\alpha}{2}$：

$$\cos\alpha = \frac{\cos^2\frac{\alpha}{2} - \sin^2\frac{\alpha}{2}}{\sin^2\frac{\alpha}{2} + \cos^2\frac{\alpha}{2}} = \frac{1 - \tan^2\frac{\alpha}{2}}{1 + \tan^2\frac{\alpha}{2}}$$

$$\boxed{\cos\alpha = \frac{1 - \tan^2\frac{\alpha}{2}}{1 + \tan^2\frac{\alpha}{2}}}$$

### 27.6.3 推导 $\tan\alpha$

$$\tan\alpha = \frac{\sin\alpha}{\cos\alpha} = \frac{\frac{2\tan\frac{\alpha}{2}}{1 + \tan^2\frac{\alpha}{2}}}{\frac{1 - \tan^2\frac{\alpha}{2}}{1 + \tan^2\frac{\alpha}{2}}} = \frac{2\tan\frac{\alpha}{2}}{1 - \tan^2\frac{\alpha}{2}}$$

$$\boxed{\tan\alpha = \frac{2\tan\frac{\alpha}{2}}{1 - \tan^2\frac{\alpha}{2}}}$$

> 💡 **注意**：$\tan\alpha$ 的万能公式和 $\tan 2\alpha$ 的二倍角公式**形式完全一样**——只是把 $2\alpha$ 换成了 $\alpha$，$\alpha$ 换成了 $\frac{\alpha}{2}$。本质上，万能公式就是二倍角公式的另一种写法。

## 27.8 例题精讲

### 例1 · 已知 $\sin\alpha$，求二倍角

已知 $\sin\alpha = \frac{3}{5}$，$\alpha \in \left(0, \frac{\pi}{2}\right)$（第一象限）。求 $\sin 2\alpha$、$\cos 2\alpha$、$\tan 2\alpha$。

> **思路**：用 $\sin 2\alpha = 2\sin\alpha\cos\alpha$ 需要 $\cos\alpha$。用 $\cos 2\alpha = 1 - 2\sin^2\alpha$ 可以避开求 $\cos\alpha$（但本题 $\cos\alpha$ 很好求）。$\tan 2\alpha$ 用 $\frac{\sin 2\alpha}{\cos 2\alpha}$ 或直接用公式都可以。

**解**：

第一步：由 $\sin^2\alpha + \cos^2\alpha = 1$，$\alpha$ 在第一象限（$\cos\alpha > 0$）：

$$\cos\alpha = \sqrt{1 - \sin^2\alpha} = \sqrt{1 - \frac{9}{25}} = \sqrt{\frac{16}{25}} = \frac{4}{5}$$

第二步：求 $\sin 2\alpha$：

$$\sin 2\alpha = 2\sin\alpha\cos\alpha = 2 \cdot \frac{3}{5} \cdot \frac{4}{5} = \frac{24}{25}$$

第三步：求 $\cos 2\alpha$（用两种方法互相验证）：

- 方法一：$\cos 2\alpha = 1 - 2\sin^2\alpha = 1 - 2 \cdot \frac{9}{25} = 1 - \frac{18}{25} = \frac{7}{25}$
- 方法二：$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = \frac{16}{25} - \frac{9}{25} = \frac{7}{25}$ √

第四步：求 $\tan 2\alpha$：

$$\tan 2\alpha = \frac{\sin 2\alpha}{\cos 2\alpha} = \frac{\frac{24}{25}}{\frac{7}{25}} = \frac{24}{7}$$

**答**：$\sin 2\alpha = \frac{24}{25}$，$\cos 2\alpha = \frac{7}{25}$，$\tan 2\alpha = \frac{24}{7}$。

> 💡 **自检技巧**：$\sin 2\alpha$ 和 $\cos 2\alpha$ 应该满足 $\sin^2 2\alpha + \cos^2 2\alpha = 1$。验算：$\left(\frac{24}{25}\right)^2 + \left(\frac{7}{25}\right)^2 = \frac{576 + 49}{625} = \frac{625}{625} = 1$ √。

---

### 例2 · 利用 $\cos 2\alpha$ 的三种形式选择最优路径

已知 $\cos\alpha = -\frac{1}{3}$，$\alpha$ 在第二象限。求 $\cos 2\alpha$。

> **思路**：只给了 $\cos\alpha$，用 $\cos 2\alpha = 2\cos^2\alpha - 1$ 最直接——不需要求 $\sin\alpha$。

**解**：

$$\cos 2\alpha = 2\cos^2\alpha - 1 = 2 \cdot \left(-\frac{1}{3}\right)^2 - 1 = 2 \cdot \frac{1}{9} - 1 = \frac{2}{9} - 1 = -\frac{7}{9}$$

**答**：$\cos 2\alpha = -\frac{7}{9}$。

> 💡 **对比**：如果不用 $2\cos^2\alpha - 1$ 而用 $\cos^2\alpha - \sin^2\alpha$，你必须先求 $\sin\alpha = \sqrt{1 - \frac{1}{9}} = \frac{2\sqrt{2}}{3}$（第二象限正弦为正），再算 $\cos 2\alpha = \frac{1}{9} - \frac{8}{9} = -\frac{7}{9}$。多了两步，但结果一样。**选对公式可以省步骤。**

---

### 例3 · 化简（降幂 + 二倍角综合）

化简：$\frac{1 + \cos 2\alpha}{\sin 2\alpha}$。

> **思路**：分子 $1 + \cos 2\alpha$ 和降幂公式有关（$1 + \cos 2\alpha = 2\cos^2\alpha$），分母 $\sin 2\alpha = 2\sin\alpha\cos\alpha$。

**解**：

$$\begin{aligned}
\frac{1 + \cos 2\alpha}{\sin 2\alpha}
&= \frac{2\cos^2\alpha}{2\sin\alpha\cos\alpha} \quad \text{（降幂公式 + 二倍角公式）} \\
&= \frac{\cos\alpha}{\sin\alpha} \quad \text{（约去 } 2\cos\alpha \text{，注意 } \cos\alpha \neq 0 \text{）} \\
&= \cot\alpha
\end{aligned}$$

**答**：化简结果为 $\cot\alpha$（即 $\frac{1}{\tan\alpha}$）。

> 🤔 **想一想**：这个结果和半角公式 $\tan\frac{\alpha}{2} = \frac{\sin\alpha}{1 + \cos\alpha}$ 的倒数形式一致吗？取倒数：$\frac{1 + \cos\alpha}{\sin\alpha} = \cot\frac{\alpha}{2}$。本题中 $\frac{1 + \cos 2\alpha}{\sin 2\alpha} = \cot\alpha$，确实形式一致（把 $\alpha$ 换成 $2\alpha$）。

### 例4 · 半角公式（带 ± 号判断）

已知 $\cos\alpha = -\frac{3}{5}$，$\alpha \in \left(\pi, \frac{3\pi}{2}\right)$（第三象限）。求 $\sin\frac{\alpha}{2}$ 和 $\cos\frac{\alpha}{2}$。

> **思路**：先判断 $\frac{\alpha}{2}$ 在哪个象限。$\alpha$ 在第三象限（$\pi < \alpha < \frac{3\pi}{2}$），所以 $\frac{\pi}{2} < \frac{\alpha}{2} < \frac{3\pi}{4}$，即 $\frac{\alpha}{2}$ 在第二象限。第二象限中 $\sin > 0$，$\cos < 0$。

**解**：

$$\sin\frac{\alpha}{2} = +\sqrt{\frac{1 - \cos\alpha}{2}} = \sqrt{\frac{1 - \left(-\frac{3}{5}\right)}{2}} = \sqrt{\frac{1 + \frac{3}{5}}{2}} = \sqrt{\frac{\frac{8}{5}}{2}} = \sqrt{\frac{4}{5}} = \frac{2}{\sqrt{5}} = \frac{2\sqrt{5}}{5}$$

$$\cos\frac{\alpha}{2} = -\sqrt{\frac{1 + \cos\alpha}{2}} = -\sqrt{\frac{1 + \left(-\frac{3}{5}\right)}{2}} = -\sqrt{\frac{1 - \frac{3}{5}}{2}} = -\sqrt{\frac{\frac{2}{5}}{2}} = -\sqrt{\frac{1}{5}} = -\frac{1}{\sqrt{5}} = -\frac{\sqrt{5}}{5}$$

**答**：$\sin\frac{\alpha}{2} = \frac{2\sqrt{5}}{5}$，$\cos\frac{\alpha}{2} = -\frac{\sqrt{5}}{5}$。

> ⚠️ **易错点**：有人在 $\sin\frac{\alpha}{2}$ 的公式里写 $\sin\frac{\alpha}{2} = \sqrt{\frac{1 - \cos\alpha}{2}}$，忘了 $\pm$ 号。也有人判断 $\frac{\alpha}{2}$ 象限时出错——**记住先算 $\frac{\alpha}{2}$ 的范围，再判断正负，不要凭直觉。**

## 27.9 常见错误：绝对不要犯

### ❌ 错误1：$\sin 2\alpha = 2\sin\alpha$

这是和角公式错误中"$\sin(\alpha+\beta) = \sin\alpha + \sin\beta$"的二倍角版本。取 $\alpha = 30^\circ$ 验证：

- ❌ 错误：$\sin 60^\circ \stackrel{?}{=} 2\sin 30^\circ = 2 \cdot \frac{1}{2} = 1$
- ✅ 正确：$\sin 60^\circ = \frac{\sqrt{3}}{2} \approx 0.866$
- 显然 $1 \neq 0.866$。

### ❌ 错误2：$\cos 2\alpha = 2\cos\alpha$

同理，取 $\alpha = 60^\circ$：

- ❌ 错误：$\cos 120^\circ \stackrel{?}{=} 2\cos 60^\circ = 2 \cdot \frac{1}{2} = 1$
- ✅ 正确：$\cos 120^\circ = -\frac{1}{2}$
- 显然 $1 \neq -\frac{1}{2}$。

### ❌ 错误3：半角公式忘了 ± 号

写 $\sin\frac{\alpha}{2} = \sqrt{\frac{1 - \cos\alpha}{2}}$ 而不加 $\pm$，默认取正。这是错误的——根号前必须有 $\pm$，需要根据 $\frac{\alpha}{2}$ 的象限来决定取正还是取负。

### ❌ 错误4：$\tan 2\alpha$ 分母为零时不处理

$\tan 2\alpha = \frac{2\tan\alpha}{1 - \tan^2\alpha}$。当 $\tan\alpha = \pm 1$（即 $\alpha = \frac{\pi}{4} + \frac{k\pi}{2}$）时分母为零，公式无意义。此时 $2\alpha = \frac{\pi}{2} + k\pi$，$\tan 2\alpha$ 不存在（趋于无穷大）。答题时必须说明"$\tan 2\alpha$ 不存在"。

## 27.10 练习题

### 基础巩固

**1.** 已知 $\cos\alpha = \frac{3}{5}$，$\alpha$ 在第一象限，求 $\sin 2\alpha$ 和 $\cos 2\alpha$。

**2.** 已知 $\tan\alpha = 2$，求 $\tan 2\alpha$。

**3.** 化简：$2\cos^2 15^\circ - 1$。

**4.** 已知 $\cos\alpha = \frac{1}{3}$，求 $\cos\frac{\alpha}{2}$（$\alpha$ 在第一象限）。

**5.** 用降幂公式将 $\sin^2 x$ 表示为 $\cos 2x$ 的一次式。

### 能力提升

**6.** 化简：$\frac{\sin 2\alpha}{1 + \cos 2\alpha} \cdot \frac{\cos\alpha}{1 + \cos\alpha}$。（提示：先化简第一个分式，识别半角公式结构。）

**7.** 已知 $\sin\alpha = \frac{4}{5}$，$\alpha \in \left(\frac{\pi}{2}, \pi\right)$，求 $\sin\frac{\alpha}{2}$、$\cos\frac{\alpha}{2}$、$\tan\frac{\alpha}{2}$。

**8.** 证明：$\frac{1 + \sin 2\alpha}{\cos 2\alpha} = \frac{1 + \tan\alpha}{1 - \tan\alpha}$。（提示：左边用二倍角公式展开。）

**9.** 已知 $\tan\frac{\alpha}{2} = 2$，用万能公式求 $\sin\alpha$ 和 $\cos\alpha$。

## 27.11 练习答案

**1.** $\sin\alpha = \frac{4}{5}$，$\sin 2\alpha = 2 \cdot \frac{3}{5} \cdot \frac{4}{5} = \frac{24}{25}$，$\cos 2\alpha = 2\cos^2\alpha - 1 = 2 \cdot \frac{9}{25} - 1 = -\frac{7}{25}$

**2.** $\tan 2\alpha = \frac{2 \cdot 2}{1 - 2^2} = \frac{4}{-3} = -\frac{4}{3}$

**3.** $2\cos^2 15^\circ - 1 = \cos(2 \times 15^\circ) = \cos 30^\circ = \frac{\sqrt{3}}{2}$（逆用二倍角公式）

**4.** $\alpha$ 在第一象限，$\frac{\alpha}{2}$ 在第一象限（$\cos\frac{\alpha}{2} > 0$）。$\cos\frac{\alpha}{2} = +\sqrt{\frac{1 + \cos\alpha}{2}} = \sqrt{\frac{1 + \frac{1}{3}}{2}} = \sqrt{\frac{\frac{4}{3}}{2}} = \sqrt{\frac{2}{3}} = \frac{\sqrt{6}}{3}$

**5.** $\sin^2 x = \frac{1 - \cos 2x}{2}$

**6.** $\frac{\sin 2\alpha}{1 + \cos 2\alpha} = \frac{2\sin\alpha\cos\alpha}{2\cos^2\alpha} = \frac{\sin\alpha}{\cos\alpha} = \tan\alpha$。原式 $= \tan\alpha \cdot \frac{\cos\alpha}{1 + \cos\alpha} = \frac{\sin\alpha}{1 + \cos\alpha} = \tan\frac{\alpha}{2}$。

**7.** $\alpha$ 在第二象限，$\cos\alpha = -\sqrt{1 - \frac{16}{25}} = -\frac{3}{5}$。$\frac{\pi}{4} < \frac{\alpha}{2} < \frac{\pi}{2}$（第一象限），$\sin\frac{\alpha}{2} > 0$，$\cos\frac{\alpha}{2} > 0$。
$\sin\frac{\alpha}{2} = \sqrt{\frac{1 - (-\frac{3}{5})}{2}} = \sqrt{\frac{\frac{8}{5}}{2}} = \sqrt{\frac{4}{5}} = \frac{2\sqrt{5}}{5}$；
$\cos\frac{\alpha}{2} = \sqrt{\frac{1 + (-\frac{3}{5})}{2}} = \sqrt{\frac{\frac{2}{5}}{2}} = \sqrt{\frac{1}{5}} = \frac{\sqrt{5}}{5}$；
$\tan\frac{\alpha}{2} = \frac{2\sqrt{5}/5}{\sqrt{5}/5} = 2$。或用有理式：$\tan\frac{\alpha}{2} = \frac{1 - \cos\alpha}{\sin\alpha} = \frac{1 - (-\frac{3}{5})}{\frac{4}{5}} = \frac{\frac{8}{5}}{\frac{4}{5}} = 2$ √。

**8.** 左边 $= \frac{1 + 2\sin\alpha\cos\alpha}{\cos^2\alpha - \sin^2\alpha}$。分子分母同除以 $\cos^2\alpha$：
$= \frac{\frac{1}{\cos^2\alpha} + \frac{2\sin\alpha}{\cos\alpha}}{1 - \tan^2\alpha} = \frac{\sec^2\alpha + 2\tan\alpha}{1 - \tan^2\alpha}$。
或用更直接的方法：$1 + \sin 2\alpha = (\sin\alpha + \cos\alpha)^2$，$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = (\cos\alpha - \sin\alpha)(\cos\alpha + \sin\alpha)$。
所以 $\frac{1 + \sin 2\alpha}{\cos 2\alpha} = \frac{(\sin\alpha + \cos\alpha)^2}{(\cos\alpha - \sin\alpha)(\cos\alpha + \sin\alpha)} = \frac{\sin\alpha + \cos\alpha}{\cos\alpha - \sin\alpha}$。
分子分母同除以 $\cos\alpha$：$= \frac{\tan\alpha + 1}{1 - \tan\alpha} = \frac{1 + \tan\alpha}{1 - \tan\alpha}$。证毕。

**9.** 设 $t = \tan\frac{\alpha}{2} = 2$。
$\sin\alpha = \frac{2t}{1 + t^2} = \frac{4}{1 + 4} = \frac{4}{5}$；
$\cos\alpha = \frac{1 - t^2}{1 + t^2} = \frac{1 - 4}{1 + 4} = -\frac{3}{5}$。

## 27.12 公式速查卡

| 类别 | 公式 | 用途 |
|------|------|------|
| 二倍角（正弦） | $\sin 2\alpha = 2\sin\alpha\cos\alpha$ | 已知 $\sin\alpha$、$\cos\alpha$ 求 $\sin 2\alpha$ |
| 二倍角（余弦1） | $\cos 2\alpha = \cos^2\alpha - \sin^2\alpha$ | 已知 $\sin\alpha$ 和 $\cos\alpha$ 都有 |
| 二倍角（余弦2） | $\cos 2\alpha = 2\cos^2\alpha - 1$ | 已知 $\cos\alpha$，避开 $\sin\alpha$ |
| 二倍角（余弦3） | $\cos 2\alpha = 1 - 2\sin^2\alpha$ | 已知 $\sin\alpha$，避开 $\cos\alpha$ |
| 二倍角（正切） | $\tan 2\alpha = \frac{2\tan\alpha}{1 - \tan^2\alpha}$ | 已知 $\tan\alpha$ 求 $\tan 2\alpha$ |
| 降幂（正弦） | $\sin^2\alpha = \frac{1 - \cos 2\alpha}{2}$ | 二次降为一次 |
| 降幂（余弦） | $\cos^2\alpha = \frac{1 + \cos 2\alpha}{2}$ | 二次降为一次 |
| 半角（正弦） | $\sin\frac{\alpha}{2} = \pm\sqrt{\frac{1 - \cos\alpha}{2}}$ | 已知 $\cos\alpha$ 求 $\sin\frac{\alpha}{2}$ |
| 半角（余弦） | $\cos\frac{\alpha}{2} = \pm\sqrt{\frac{1 + \cos\alpha}{2}}$ | 已知 $\cos\alpha$ 求 $\cos\frac{\alpha}{2}$ |
| 半角（正切） | $\tan\frac{\alpha}{2} = \frac{\sin\alpha}{1 + \cos\alpha} = \frac{1 - \cos\alpha}{\sin\alpha}$ | 不用根号，直接有理式 |
| 万能（正弦） | $\sin\alpha = \frac{2t}{1 + t^2}$（$t = \tan\frac{\alpha}{2}$） | 统一为 $t$ 的有理式 |
| 万能（余弦） | $\cos\alpha = \frac{1 - t^2}{1 + t^2}$ | 同上 |
| 万能（正切） | $\tan\alpha = \frac{2t}{1 - t^2}$ | 同上 |
