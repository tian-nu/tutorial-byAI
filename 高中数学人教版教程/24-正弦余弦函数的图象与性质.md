## 24.3 正弦函数 $y = \sin x$ 的图像

### 24.3.1 从单位圆到正弦曲线

在单位圆中，$\sin\alpha$ 是角 $\alpha$ 的终边与单位圆交点的**纵坐标**。当 $\alpha$ 从 $0$ 连续变化到 $2\pi$ 时，这个纵坐标也在连续变化。

把 $\alpha$ 当作横坐标（自变量），$\sin\alpha$ 当作纵坐标（函数值），在坐标系中描点连线，就得到了**正弦曲线**。

> 📊 **交互演示见 [visual/math1_sin_cos_visual.html](visual/math1_sin_cos_visual.html)**——在演示中，你可以看到一个点在单位圆上运动，同时右侧坐标系中同步画出对应的正弦曲线。这是理解"单位圆→正弦曲线"的最佳方式。**强烈建议现在就打开。**

---

### 24.3.2 五点法作图

要在 $[0, 2\pi]$ 上画出 $y = \sin x$ 的图像，不需要描几十个点——只需要**五个关键点**就够了。

**哪五个点？**

| $x$ | $0$ | $\dfrac{\pi}{2}$ | $\pi$ | $\dfrac{3\pi}{2}$ | $2\pi$ |
|-----|-----|-----|-----|-----|-----|
| $\sin x$ | $0$ | $1$ | $0$ | $-1$ | $0$ |

**为什么是这五个点？**

这五个点分别对应正弦函数在一个周期内的**起点、最高点、中点、最低点、终点**：

- $x = 0$：周期起点，$\sin 0 = 0$。——曲线从这里出发。
- $x = \frac{\pi}{2}$：最高点，$\sin \frac{\pi}{2} = 1$。——曲线到达波峰。
- $x = \pi$：与 $x$ 轴的交点，$\sin \pi = 0$。——曲线穿过 $x$ 轴。
- $x = \frac{3\pi}{2}$：最低点，$\sin \frac{3\pi}{2} = -1$。——曲线到达波谷。
- $x = 2\pi$：周期终点，$\sin 2\pi = 0$。——曲线回到起点高度，准备进入下一个周期。

这五个点把正弦曲线的一个完整周期分成了四个等长的段落，每个段落的长度都是 $\frac{\pi}{2}$。在这五个点之间，曲线是平滑的圆弧——不需要额外描点，直接用平滑曲线连接即可。

> 💡 **记忆口诀**：正弦五点——"0, 1, 0, -1, 0"（按 $x$ 从 $0$ 到 $2\pi$ 的顺序）。也可以记成"零、一、零、负一、零"。

---

### 24.3.3 五点法作图的步骤

**第 1 步**：在坐标系中标出五个关键点：
$$(0, 0),\; (\frac{\pi}{2}, 1),\; (\pi, 0),\; (\frac{3\pi}{2}, -1),\; (2\pi, 0)$$

**第 2 步**：用平滑的曲线依次连接这五个点。注意：
- 在 $x = 0$ 到 $x = \frac{\pi}{2}$ 之间，曲线从 $0$ 上升到 $1$，形状是**上凸**的。
- 在 $x = \frac{\pi}{2}$ 到 $x = \pi$ 之间，曲线从 $1$ 下降到 $0$，形状是**下凸**的。
- 在 $x = \pi$ 到 $x = \frac{3\pi}{2}$ 之间，曲线从 $0$ 下降到 $-1$。
- 在 $x = \frac{3\pi}{2}$ 到 $x = 2\pi$ 之间，曲线从 $-1$ 上升到 $0$。

**第 3 步**：利用周期性，将 $[0, 2\pi]$ 上的图像向左、向右重复延伸，得到完整的正弦曲线。

```
   y↑
   1 ┤     ╱‾‾‾╲         ╱‾‾‾╲
     │    ╱     ╲       ╱     ╲
     │   ╱       ╲     ╱       ╲
   0 ┼──●─────────●───●─────────●──→ x
     │ ╱   0       ╲ π╱   2π     ╲
     │╱             ╲ ╱           ╲
  -1 ┤               ●
     │              3π/2
     │  一个周期 →
```

> ⚠️ **注意**：这不是一条折线！五个点之间用**平滑曲线**连接，不要画成折线段。最高点和最低点处的曲线是圆滑的"弧顶"，不是尖角。

---

### 24.3.4 正弦曲线的图像特征

| 特征 | 描述 |
|------|------|
| 形状 | 波浪形（无限延伸的"～"形曲线） |
| 周期 | $2\pi$——每隔 $2\pi$，曲线完全重复自身 |
| 值域 | $[-1, 1]$——函数值在 $-1$ 和 $1$ 之间波动 |
| 与 $x$ 轴的交点 | $x = k\pi$（$k \in \mathbb{Z}$），即 $x = 0, \pm\pi, \pm2\pi, \ldots$ |
| 最高点 | $x = \frac{\pi}{2} + 2k\pi$，$y = 1$ |
| 最低点 | $x = -\frac{\pi}{2} + 2k\pi$（或 $x = \frac{3\pi}{2} + 2k\pi$），$y = -1$ |

> 🤔 **想多一点**：为什么正弦函数的值域是 $[-1, 1]$？因为正弦的几何定义是"单位圆上点的纵坐标"，而单位圆上任何点的纵坐标都在 $-1$ 到 $1$ 之间。这个几何直觉比死记硬背可靠得多。

## 24.4 余弦函数 $y = \cos x$ 的图像

### 24.4.1 五点法作图

用同样的方法，取 $[0, 2\pi]$ 上的五个关键点：

| $x$ | $0$ | $\dfrac{\pi}{2}$ | $\pi$ | $\dfrac{3\pi}{2}$ | $2\pi$ |
|-----|-----|-----|-----|-----|-----|
| $\cos x$ | $1$ | $0$ | $-1$ | $0$ | $1$ |

**为什么是这五个点？**

- $x = 0$：周期起点，$\cos 0 = 1$。——曲线从最高点出发。
- $x = \frac{\pi}{2}$：与 $x$ 轴的交点，$\cos \frac{\pi}{2} = 0$。——曲线穿过 $x$ 轴。
- $x = \pi$：最低点，$\cos \pi = -1$。——曲线到达波谷。
- $x = \frac{3\pi}{2}$：与 $x$ 轴的交点，$\cos \frac{3\pi}{2} = 0$。——曲线再次穿过 $x$ 轴。
- $x = 2\pi$：周期终点，$\cos 2\pi = 1$。——曲线回到最高点。

> 💡 **记忆口诀**：余弦五点——"1, 0, -1, 0, 1"（按 $x$ 从 $0$ 到 $2\pi$ 的顺序）。也可以记成"一、零、负一、零、一"。

```
   y↑
   1 ┤●───╲         ╱───●
     │     ╲       ╱
     │      ╲     ╱
   0 ┼───────●───●───────→ x
     │       ╲ π╱ 3π/2
     │        ╲ ╱
  -1 ┤         ●
     │         π
     │  一个周期 →
```

---

### 24.4.2 余弦曲线与正弦曲线的关系

**核心关系**：
$$\boxed{\cos x = \sin(x + \frac{\pi}{2})}$$

这个公式的几何含义是：**余弦曲线就是正弦曲线向左平移 $\dfrac{\pi}{2}$ 个单位。**

**验证**：

取 $x = 0$：$\cos 0 = 1$，而 $\sin(0 + \frac{\pi}{2}) = \sin\frac{\pi}{2} = 1$。✅

取 $x = \frac{\pi}{2}$：$\cos \frac{\pi}{2} = 0$，而 $\sin(\frac{\pi}{2} + \frac{\pi}{2}) = \sin\pi = 0$。✅

取 $x = \pi$：$\cos\pi = -1$，而 $\sin(\pi + \frac{\pi}{2}) = \sin\frac{3\pi}{2} = -1$。✅

**这意味着什么？**

如果你已经画好了正弦曲线，只需把它整体向左移动 $\frac{\pi}{2}$，就得到了余弦曲线。反过来，正弦曲线是余弦曲线向右平移 $\frac{\pi}{2}$：

$$\sin x = \cos(x - \frac{\pi}{2})$$

> 💡 **记忆技巧**：正弦和余弦是"同一根曲线，只是起点不同"。正弦从 $0$ 开始（$x=0$ 时 $y=0$），余弦从 $1$ 开始（$x=0$ 时 $y=1$）。余弦比正弦"领先" $\frac{\pi}{2}$。

---

### 24.4.3 正弦曲线与余弦曲线的对比

| 对比项 | $y = \sin x$ | $y = \cos x$ |
|--------|-------------|-------------|
| 五点法坐标 | $(0,0), (\frac{\pi}{2},1), (\pi,0), (\frac{3\pi}{2},-1), (2\pi,0)$ | $(0,1), (\frac{\pi}{2},0), (\pi,-1), (\frac{3\pi}{2},0), (2\pi,1)$ |
| 形状 | 波浪形 | 波浪形 |
| 两者关系 | $\sin x = \cos(x - \frac{\pi}{2})$ | $\cos x = \sin(x + \frac{\pi}{2})$ |
| 周期 | $2\pi$ | $2\pi$ |
| 值域 | $[-1, 1]$ | $[-1, 1]$ |

## 24.5 正弦、余弦函数的性质

### 24.5.1 定义域与值域

| 函数 | 定义域 | 值域 |
|------|--------|------|
| $y = \sin x$ | $\mathbb{R}$（全体实数） | $[-1, 1]$ |
| $y = \cos x$ | $\mathbb{R}$（全体实数） | $[-1, 1]$ |

正弦和余弦的定义域都是全体实数——因为任何角都有正弦值和余弦值。值域都是 $[-1, 1]$——因为单位圆上点的纵坐标（正弦）和横坐标（余弦）都在 $-1$ 到 $1$ 之间。

> ⚠️ **重要**：$y = \sin x$ 的最大值是 $1$，最小值是 $-1$。当题目中出现 $\sin x$ 的系数 $A$ 时，$y = A\sin x$ 的值域就变成了 $[-|A|, |A|]$。这个 $|A|$ 称为**振幅**。

---

### 24.5.2 周期性

**定义**：对于函数 $f(x)$，如果存在一个非零常数 $T$，使得对于定义域内的任意 $x$，都有
$$f(x + T) = f(x)$$
那么称 $f(x)$ 为**周期函数**，$T$ 称为这个函数的**周期**。

**最小正周期**：在所有正周期中，如果存在一个最小的正数，就称它为**最小正周期**。

> 对于正弦和余弦函数，$T = 2\pi$ 是它们的最小正周期。也就是说：
> $$\sin(x + 2\pi) = \sin x, \quad \cos(x + 2\pi) = \cos x$$
> 且 $2\pi$ 是使得这个等式成立的最小正数。

**为什么最小正周期是 $2\pi$？**

从单位圆的角度看：角 $\alpha$ 每增加 $2\pi$（即旋转一圈），终边回到原来的位置，正弦值和余弦值也就回到原来的值。而 $2\pi$ 是使终边首次回到原位的最小正角——所以最小正周期就是 $2\pi$。

**更一般地**：$y = A\sin(\omega x + \varphi)$ 的最小正周期是 $T = \dfrac{2\pi}{|\omega|}$。这个公式非常重要，后面会专门学到。

---

### 24.5.3 奇偶性

**正弦函数 $y = \sin x$ 是奇函数。**

因为 $\sin(-x) = -\sin x$ 对任意 $x \in \mathbb{R}$ 成立。从图像上看，正弦曲线关于**原点**对称。

**验证**：取 $x = \frac{\pi}{2}$，$\sin(-\frac{\pi}{2}) = -1$，$-\sin\frac{\pi}{2} = -1$。✅

**余弦函数 $y = \cos x$ 是偶函数。**

因为 $\cos(-x) = \cos x$ 对任意 $x \in \mathbb{R}$ 成立。从图像上看，余弦曲线关于 **$y$ 轴**对称。

**验证**：取 $x = \frac{\pi}{3}$，$\cos(-\frac{\pi}{3}) = \frac{1}{2}$，$\cos\frac{\pi}{3} = \frac{1}{2}$。✅

> 💡 **记忆技巧**：正弦是"奇"函数——"奇"和"正弦"没有直接关联，但你可以这样记：正弦曲线在原点处是"穿过"的（$x=0$ 时 $y=0$），这符合奇函数的特征（$f(0)=0$）；而余弦曲线在原点处是"最高点"（$x=0$ 时 $y=1$），这符合偶函数的特征（关于 $y$ 轴对称）。

---

### 24.5.4 单调性

**正弦函数 $y = \sin x$ 在一个周期 $[0, 2\pi]$ 上的单调性**：

| 区间 | 单调性 | 解释 |
|------|--------|------|
| $[0, \frac{\pi}{2}]$ | **单调递增** | 从 0 升到 1 |
| $[\frac{\pi}{2}, \frac{3\pi}{2}]$ | **单调递减** | 从 1 降到 -1 |
| $[\frac{3\pi}{2}, 2\pi]$ | **单调递增** | 从 -1 升到 0 |

推广到整个定义域（利用周期性，加上 $2k\pi$）：

- 单调递增区间：$[-\frac{\pi}{2} + 2k\pi,\; \frac{\pi}{2} + 2k\pi]$（$k \in \mathbb{Z}$）
- 单调递减区间：$[\frac{\pi}{2} + 2k\pi,\; \frac{3\pi}{2} + 2k\pi]$（$k \in \mathbb{Z}$）

**余弦函数 $y = \cos x$ 在一个周期 $[0, 2\pi]$ 上的单调性**：

| 区间 | 单调性 | 解释 |
|------|--------|------|
| $[0, \pi]$ | **单调递减** | 从 1 降到 -1 |
| $[\pi, 2\pi]$ | **单调递增** | 从 -1 升到 1 |

推广到整个定义域：

- 单调递减区间：$[2k\pi,\; \pi + 2k\pi]$（$k \in \mathbb{Z}$）
- 单调递增区间：$[\pi + 2k\pi,\; 2\pi + 2k\pi]$（$k \in \mathbb{Z}$）

> 💡 **简化记忆**：正弦在 $[-\frac{\pi}{2}, \frac{\pi}{2}]$ 上递增，余弦在 $[0, \pi]$ 上递减。这两个区间是考试中最常出现的。

---

### 24.5.5 对称性

**正弦函数 $y = \sin x$ 的对称性**：

- **对称轴**：$x = \frac{\pi}{2} + k\pi$（$k \in \mathbb{Z}$）——即 $x = \frac{\pi}{2}, \frac{3\pi}{2}, \frac{5\pi}{2}, \ldots$ 和 $x = -\frac{\pi}{2}, -\frac{3\pi}{2}, \ldots$
  - 对称轴穿过正弦曲线的最高点或最低点（波峰或波谷）。
- **对称中心**：$(k\pi, 0)$（$k \in \mathbb{Z}$）——即 $(0, 0), (\pi, 0), (2\pi, 0), \ldots$ 和 $(-\pi, 0), (-2\pi, 0), \ldots$
  - 对称中心是正弦曲线与 $x$ 轴的交点。

**余弦函数 $y = \cos x$ 的对称性**：

- **对称轴**：$x = k\pi$（$k \in \mathbb{Z}$）——即 $x = 0, \pi, 2\pi, \ldots$ 和 $x = -\pi, -2\pi, \ldots$
  - 对称轴穿过余弦曲线的最高点或最低点。
- **对称中心**：$(\frac{\pi}{2} + k\pi, 0)$（$k \in \mathbb{Z}$）——即 $(\frac{\pi}{2}, 0), (\frac{3\pi}{2}, 0), \ldots$
  - 对称中心是余弦曲线与 $x$ 轴的交点。

> 💡 **对比记忆**：正弦的对称轴在"最高/最低点"（$x=\frac{\pi}{2}+k\pi$），对称中心在"零点"（$x=k\pi$）；余弦恰好反过来——对称轴在"零点"（$x=k\pi$），对称中心在"最高/最低点"（$x=\frac{\pi}{2}+k\pi$）。

## 24.6 求三角函数的值域与最值——辅助角公式

### 24.6.1 问题引入

单独的正弦或余弦函数，值域一目了然：$y = \sin x$ 的值域是 $[-1, 1]$。但如果遇到 $y = \sin x + \cos x$ 或 $y = \sin x + \sqrt{3}\cos x$，值域是什么？

直接看是看不出来的——因为 $\sin x$ 和 $\cos x$ 同时变化，它们的和在什么时候最大、什么时候最小？需要一种方法把 $\sin x$ 和 $\cos x$ 的线性组合化为**单一的三角函数**。

---

### 24.6.2 辅助角公式

**公式**：
$$\boxed{A\sin x + B\cos x = \sqrt{A^2 + B^2}\,\sin(x + \varphi)}$$

其中 $\varphi$ 由以下条件确定：
$$\sin\varphi = \frac{B}{\sqrt{A^2 + B^2}}, \quad \cos\varphi = \frac{A}{\sqrt{A^2 + B^2}}$$

即 $\tan\varphi = \dfrac{B}{A}$（需结合 $A$ 和 $B$ 的符号确定 $\varphi$ 所在的象限）。

**推导**（理解这条公式的来源比死记硬背更重要）：

设 $A\sin x + B\cos x$，提取 $\sqrt{A^2 + B^2}$：

$$\begin{aligned} A\sin x + B\cos x &= \sqrt{A^2 + B^2}\left(\frac{A}{\sqrt{A^2 + B^2}}\sin x + \frac{B}{\sqrt{A^2 + B^2}}\cos x\right) \end{aligned}$$

注意到 $\left(\dfrac{A}{\sqrt{A^2 + B^2}}\right)^2 + \left(\dfrac{B}{\sqrt{A^2 + B^2}}\right)^2 = \dfrac{A^2 + B^2}{A^2 + B^2} = 1$。

所以存在一个角 $\varphi$，使得 $\cos\varphi = \dfrac{A}{\sqrt{A^2 + B^2}}$，$\sin\varphi = \dfrac{B}{\sqrt{A^2 + B^2}}$。

代入：
$$\begin{aligned} A\sin x + B\cos x &= \sqrt{A^2 + B^2}\,(\cos\varphi\sin x + \sin\varphi\cos x) \\ &= \sqrt{A^2 + B^2}\,\sin(x + \varphi) \end{aligned}$$

最后一步用了正弦的和角公式：$\sin(x + \varphi) = \sin x\cos\varphi + \cos x\sin\varphi$。

> 🤔 **想多一点**：辅助角公式的本质是**正弦的和角公式的反向使用**。正向是 $\sin(x+\varphi) = \sin x\cos\varphi + \cos x\sin\varphi$，反向是把 $\sin x$ 和 $\cos x$ 的线性组合"塞回"一个 $\sin(x+\varphi)$ 当中。这个思路在物理中也非常常见——比如两个同频率的简谐振动合成，用的就是辅助角公式。

---

### 24.6.3 利用辅助角公式求值域

化为 $y = \sqrt{A^2 + B^2}\,\sin(x + \varphi)$ 后，值域就一目了然了：

因为 $\sin(x + \varphi) \in [-1, 1]$，所以：
$$y \in \left[-\sqrt{A^2 + B^2},\; \sqrt{A^2 + B^2}\right]$$

- 最大值：$\sqrt{A^2 + B^2}$（当 $\sin(x + \varphi) = 1$ 时取得）
- 最小值：$-\sqrt{A^2 + B^2}$（当 $\sin(x + \varphi) = -1$ 时取得）

---

### 24.6.4 常见特例

| 函数 | 辅助角公式化归 | 值域 | 最大值 |
|------|---------------|------|--------|
| $y = \sin x + \cos x$ | $y = \sqrt{2}\sin(x + \frac{\pi}{4})$ | $[-\sqrt{2}, \sqrt{2}]$ | $\sqrt{2}$ |
| $y = \sin x - \cos x$ | $y = \sqrt{2}\sin(x - \frac{\pi}{4})$ | $[-\sqrt{2}, \sqrt{2}]$ | $\sqrt{2}$ |
| $y = \sin x + \sqrt{3}\cos x$ | $y = 2\sin(x + \frac{\pi}{3})$ | $[-2, 2]$ | $2$ |
| $y = \sqrt{3}\sin x + \cos x$ | $y = 2\sin(x + \frac{\pi}{6})$ | $[-2, 2]$ | $2$ |

> 💡 **记忆技巧**：$\sin x + \cos x$ 的辅助角是 $\frac{\pi}{4}$，$\sin x + \sqrt{3}\cos x$ 的辅助角是 $\frac{\pi}{3}$，$\sqrt{3}\sin x + \cos x$ 的辅助角是 $\frac{\pi}{6}$。这三个是最高频出现的，建议记熟。

## 24.7 例题精讲

### 例题 1 —— 五点法画正弦曲线

**题目**：用五点法画出 $y = \sin x$ 在 $[0, 2\pi]$ 上的图像，并标出五个关键点的坐标。

**解**：

**第 1 步**：确定五个关键点的 $x$ 坐标：$0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}, 2\pi$。

**第 2 步**：计算对应的 $y$ 值：

| $x$ | $0$ | $\frac{\pi}{2}$ | $\pi$ | $\frac{3\pi}{2}$ | $2\pi$ |
|-----|-----|-----|-----|-----|-----|
| $y = \sin x$ | $0$ | $1$ | $0$ | $-1$ | $0$ |

**第 3 步**：在坐标系中描出五个点，用平滑曲线依次连接。

**第 4 步**：标注关键点坐标：$(0, 0), (\frac{\pi}{2}, 1), (\pi, 0), (\frac{3\pi}{2}, -1), (2\pi, 0)$。

> ✅ **验证**：检查曲线是否经过原点 $(0, 0)$，最高点是否为 $(\frac{\pi}{2}, 1)$，最低点是否为 $(\frac{3\pi}{2}, -1)$。曲线在 $x = \pi$ 处是否穿过 $x$ 轴。

---

### 例题 2 —— 正弦与余弦的平移关系

**题目**：已知 $y = \sin x$ 的图像，描述如何通过平移得到下列函数的图像：

(1) $y = \cos x$
(2) $y = \sin(x + \frac{\pi}{3})$
(3) $y = \cos(x - \frac{\pi}{4})$

**解**：

**(1) $y = \cos x$**

因为 $\cos x = \sin(x + \frac{\pi}{2})$，所以将 $y = \sin x$ 的图像向**左**平移 $\frac{\pi}{2}$ 个单位，就得到 $y = \cos x$ 的图像。

**(2) $y = \sin(x + \frac{\pi}{3})$**

将 $y = \sin x$ 的图像向**左**平移 $\frac{\pi}{3}$ 个单位。

> 💡 口诀："左加右减"——$x$ 后面 $+ \frac{\pi}{3}$ 就是左移 $\frac{\pi}{3}$。

**(3) $y = \cos(x - \frac{\pi}{4})$**

方法一：先将 $y = \cos x$ 的图像向右平移 $\frac{\pi}{4}$ 个单位。

方法二：$\cos(x - \frac{\pi}{4}) = \sin(x - \frac{\pi}{4} + \frac{\pi}{2}) = \sin(x + \frac{\pi}{4})$，所以将 $y = \sin x$ 的图像向左平移 $\frac{\pi}{4}$ 个单位。

两种方法得出的图像完全相同。

---

### 例题 3 —— 判断正弦函数的单调区间

**题目**：求函数 $y = \sin x$ 在 $[0, 2\pi]$ 上的单调递增区间和单调递减区间。

**解**：

**第 1 步**：回顾正弦函数在一个周期内的单调性。

在 $[0, 2\pi]$ 上：
- $[0, \frac{\pi}{2}]$：单调递增（从 0 到 1）。
- $[\frac{\pi}{2}, \frac{3\pi}{2}]$：单调递减（从 1 到 -1）。
- $[\frac{3\pi}{2}, 2\pi]$：单调递增（从 -1 到 0）。

**第 2 步**：写出答案。

- 单调递增区间：$[0, \frac{\pi}{2}]$ 和 $[\frac{3\pi}{2}, 2\pi]$
- 单调递减区间：$[\frac{\pi}{2}, \frac{3\pi}{2}]$

> ⚠️ **注意**：这里有两个递增区间和一个递减区间。不能把两个递增区间合并写成 $[0, \frac{\pi}{2}] \cup [\frac{3\pi}{2}, 2\pi]$——虽然数学上不禁止这样写，但通常分别列出更清晰。

---

### 例题 4 —— 辅助角公式求值域

**题目**：求函数 $y = \sin x + \sqrt{3}\cos x$ 的值域和最大值。

**解**：

**第 1 步**：确定 $A$ 和 $B$。

$A = 1$（$\sin x$ 的系数），$B = \sqrt{3}$（$\cos x$ 的系数）。

**第 2 步**：计算 $\sqrt{A^2 + B^2}$。

$$\sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1 + 3} = \sqrt{4} = 2$$

**第 3 步**：确定 $\varphi$。

$$\cos\varphi = \frac{A}{\sqrt{A^2 + B^2}} = \frac{1}{2}, \quad \sin\varphi = \frac{B}{\sqrt{A^2 + B^2}} = \frac{\sqrt{3}}{2}$$

满足 $\cos\varphi = \frac{1}{2}$ 且 $\sin\varphi = \frac{\sqrt{3}}{2}$ 的角 $\varphi = \frac{\pi}{3}$。

**第 4 步**：写出辅助角公式化归结果。

$$y = \sin x + \sqrt{3}\cos x = 2\sin(x + \frac{\pi}{3})$$

**第 5 步**：求值域。

因为 $\sin(x + \frac{\pi}{3}) \in [-1, 1]$，所以 $y = 2\sin(x + \frac{\pi}{3}) \in [-2, 2]$。

- 值域：$[-2, 2]$
- 最大值：$2$（当 $\sin(x + \frac{\pi}{3}) = 1$ 时取得，即 $x + \frac{\pi}{3} = \frac{\pi}{2} + 2k\pi$，$x = \frac{\pi}{6} + 2k\pi$）

> ✅ **验证**：取 $x = \frac{\pi}{6}$，$y = \sin\frac{\pi}{6} + \sqrt{3}\cos\frac{\pi}{6} = \frac{1}{2} + \sqrt{3} \cdot \frac{\sqrt{3}}{2} = \frac{1}{2} + \frac{3}{2} = 2$。✅ 确实取到最大值 2。

---

### 例题 5 —— 求对称轴和对称中心

**题目**：求函数 $y = \sin x$ 的对称轴方程和对称中心坐标。

**解**：

**对称轴**：正弦函数的对称轴在最高点和最低点处，即 $x = \frac{\pi}{2} + k\pi$（$k \in \mathbb{Z}$）。

具体写出来：$x = \frac{\pi}{2}, \frac{3\pi}{2}, \frac{5\pi}{2}, \ldots$ 和 $x = -\frac{\pi}{2}, -\frac{3\pi}{2}, \ldots$

**对称中心**：正弦函数的对称中心在曲线与 $x$ 轴的交点处，即 $(k\pi, 0)$（$k \in \mathbb{Z}$）。

具体写出来：$(0, 0), (\pi, 0), (2\pi, 0), \ldots$ 和 $(-\pi, 0), (-2\pi, 0), \ldots$

> ✅ **验证**：以 $x = \frac{\pi}{2}$ 为对称轴，取 $x = \frac{\pi}{2} + \Delta$ 和 $x = \frac{\pi}{2} - \Delta$，验证 $\sin(\frac{\pi}{2} + \Delta) = \sin(\frac{\pi}{2} - \Delta)$。例如 $\Delta = \frac{\pi}{6}$：$\sin(\frac{\pi}{2} + \frac{\pi}{6}) = \sin\frac{2\pi}{3} = \frac{\sqrt{3}}{2}$，$\sin(\frac{\pi}{2} - \frac{\pi}{6}) = \sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}$。✅

## 24.8 常见错误辨析

| 错误 | 错在哪里？ | 正确做法 |
|------|-----------|---------|
| ❌ 五点法画正弦曲线时，把五个点用折线连接 | 正弦曲线是平滑的波浪形，不是折线 | 用平滑曲线依次连接五个关键点，在最高点和最低点处注意曲线是"弧顶"而非尖角 |
| ❌ 认为 $\sin x$ 是偶函数 | 正弦函数 $\sin(-x) = -\sin x$，是奇函数 | 记住：正弦关于原点对称，余弦关于 $y$ 轴对称 |
| ❌ 认为 $\cos x$ 是奇函数 | 余弦函数 $\cos(-x) = \cos x$，是偶函数 | 区分记忆：$\sin$（奇/odd）→ 奇函数，$\cos$（偶/even）→ 偶函数 |
| ❌ 把 $\sin x + \cos x$ 的最大值算成 $1+1=2$ | $\sin x$ 和 $\cos x$ 不能同时取到最大值 1。当 $\sin x = 1$ 时，$x = \frac{\pi}{2}$，此时 $\cos x = 0$ | 用辅助角公式：$\sin x + \cos x = \sqrt{2}\sin(x + \frac{\pi}{4})$，最大值是 $\sqrt{2}$，不是 $2$ |
| ❌ 辅助角公式中 $\varphi$ 算错 | 容易把 $\sin\varphi$ 和 $\cos\varphi$ 对应的分子搞反 | 记住：$\cos\varphi$ 对应 $\sin x$ 的系数 $A$，$\sin\varphi$ 对应 $\cos x$ 的系数 $B$ |
| ❌ 认为正弦和余弦的周期是 $\pi$ | 正弦和余弦的最小正周期是 $2\pi$，不是 $\pi$。正切函数的周期才是 $\pi$ | 正弦和余弦：$T = 2\pi$；正切：$T = \pi$ |
| ❌ 说"正弦函数在 $[0, 2\pi]$ 上单调递增" | 正弦在一个周期内并不是单调的——它先增后减再增 | 必须分区间说明：$[0, \frac{\pi}{2}]$ 递增，$[\frac{\pi}{2}, \frac{3\pi}{2}]$ 递减，$[\frac{3\pi}{2}, 2\pi]$ 递增 |

## 24.10 练习题答案与解析

### 第 1 题 解析

在 $[0, 2\pi]$ 上：

**$y = \sin x$ 的五点**：$(0, 0), (\frac{\pi}{2}, 1), (\pi, 0), (\frac{3\pi}{2}, -1), (2\pi, 0)$

**$y = \cos x$ 的五点**：$(0, 1), (\frac{\pi}{2}, 0), (\pi, -1), (\frac{3\pi}{2}, 0), (2\pi, 1)$

在同一坐标系中，正弦曲线从原点出发，余弦曲线从 $(0, 1)$ 出发。余弦曲线是正弦曲线向左平移 $\frac{\pi}{2}$ 的结果。

---

### 第 2 题 解析

(1) ❌ —— $y = \sin x$ 是**奇函数**，因为 $\sin(-x) = -\sin x$。图像关于原点对称。

(2) ❌ —— $y = \cos x$ 的最小正周期是 $2\pi$，不是 $\pi$。$\cos(x + \pi) = -\cos x \neq \cos x$，所以 $\pi$ 不是周期。$\cos(x + 2\pi) = \cos x$，$2\pi$ 才是最小正周期。

(3) ❌ —— $y = \sin x$ 的值域是 $[-1, 1]$（闭区间），不是 $(-1, 1)$（开区间）。因为 $\sin x$ 确实能取到 $1$ 和 $-1$（例如 $\sin\frac{\pi}{2} = 1$，$\sin\frac{3\pi}{2} = -1$）。

(4) ✅ —— $\sin(x - \frac{\pi}{2}) = \sin x\cos\frac{\pi}{2} - \cos x\sin\frac{\pi}{2} = \sin x \cdot 0 - \cos x \cdot 1 = -\cos x$。等等，这个是错的！

重新计算：$\sin(x - \frac{\pi}{2}) = \sin x\cos\frac{\pi}{2} - \cos x\sin\frac{\pi}{2} = \sin x \cdot 0 - \cos x \cdot 1 = -\cos x$。

所以 $\sin(x - \frac{\pi}{2}) = -\cos x \neq \cos x$。这句话是 ❌。

正确的公式是 $\cos x = \sin(x + \frac{\pi}{2})$ 或 $\sin x = \cos(x - \frac{\pi}{2})$。

> ⚠️ **关键提醒**：遇到这种公式判断，一定要动手算一遍，不要凭印象。很多同学记反"正弦和余弦谁领先谁 $\frac{\pi}{2}$"。

---

### 第 3 题 解析

**对称轴**：$x = \frac{\pi}{2} + k\pi$（$k \in \mathbb{Z}$）

**对称中心**：$(k\pi, 0)$（$k \in \mathbb{Z}$）

---

### 第 4 题 解析

$y = \sin x + \cos x$，$A = 1$，$B = 1$。

$\sqrt{A^2 + B^2} = \sqrt{1^2 + 1^2} = \sqrt{2}$。

$\cos\varphi = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$，$\sin\varphi = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$，所以 $\varphi = \frac{\pi}{4}$。

$$y = \sin x + \cos x = \sqrt{2}\sin(x + \frac{\pi}{4})$$

值域：$[-\sqrt{2}, \sqrt{2}]$。

最大值：$\sqrt{2}$，当 $\sin(x + \frac{\pi}{4}) = 1$ 时取得。

即 $x + \frac{\pi}{4} = \frac{\pi}{2} + 2k\pi$（$k \in \mathbb{Z}$），解得 $x = \frac{\pi}{4} + 2k\pi$（$k \in \mathbb{Z}$）。

---

### 第 5 题 解析

$y = \sqrt{3}\sin x - \cos x$，$A = \sqrt{3}$，$B = -1$。

$\sqrt{A^2 + B^2} = \sqrt{(\sqrt{3})^2 + (-1)^2} = \sqrt{3 + 1} = 2$。

$\cos\varphi = \frac{A}{\sqrt{A^2 + B^2}} = \frac{\sqrt{3}}{2}$，$\sin\varphi = \frac{B}{\sqrt{A^2 + B^2}} = \frac{-1}{2}$。

$\cos\varphi = \frac{\sqrt{3}}{2} > 0$ 且 $\sin\varphi = -\frac{1}{2} < 0$，所以 $\varphi$ 在第四象限，$\varphi = -\frac{\pi}{6}$。

$$y = \sqrt{3}\sin x - \cos x = 2\sin(x - \frac{\pi}{6})$$

值域：$[-2, 2]$。

> 💡 **注意**：$B = -1$，所以 $\sin\varphi = -\frac{1}{2}$，$\varphi = -\frac{\pi}{6}$（或 $\varphi = \frac{11\pi}{6}$，但通常取主值 $-\frac{\pi}{6}$）。

---

### 第 6 题 解析

$y = \sin x + \cos x = \sqrt{2}\sin(x + \frac{\pi}{4})$。

已知 $x \in [0, \frac{\pi}{2}]$，则 $x + \frac{\pi}{4} \in [\frac{\pi}{4}, \frac{3\pi}{4}]$。

在区间 $[\frac{\pi}{4}, \frac{3\pi}{4}]$ 上，$\sin$ 函数在 $[\frac{\pi}{4}, \frac{\pi}{2}]$ 上单调递增，在 $[\frac{\pi}{2}, \frac{3\pi}{4}]$ 上单调递减。

最大值在 $x + \frac{\pi}{4} = \frac{\pi}{2}$（即 $x = \frac{\pi}{4}$）处取得：$y_{\max} = \sqrt{2} \cdot 1 = \sqrt{2}$。

最小值在区间端点处，比较两个端点：
- $x + \frac{\pi}{4} = \frac{\pi}{4}$（即 $x = 0$）：$y = \sqrt{2} \cdot \frac{\sqrt{2}}{2} = 1$
- $x + \frac{\pi}{4} = \frac{3\pi}{4}$（即 $x = \frac{\pi}{2}$）：$y = \sqrt{2} \cdot \frac{\sqrt{2}}{2} = 1$

两个端点值相等，都是 $1$。所以 $y_{\min} = 1$。

**答案**：最大值 $\sqrt{2}$（$x = \frac{\pi}{4}$），最小值 $1$（$x = 0$ 或 $x = \frac{\pi}{2}$）。

> 🧠 **反思**：这道题的关键技巧是：用了辅助角公式之后，还需要考虑**给定的 $x$ 范围对新角 $x + \varphi$ 的范围的影响**。不是所有 $x$ 都能让 $\sin(x + \varphi)$ 取到 $-1$ 或 $1$。一定要先求出 $x + \varphi$ 的范围，再在这个范围内求 $\sin$ 的最值。

---

### 第 7 题 解析

在 $[0, 2\pi]$ 上：
- $[0, \pi]$：单调递减（从 1 到 -1）。
- $[\pi, 2\pi]$：单调递增（从 -1 到 1）。

**答案**：单调递减区间 $[0, \pi]$，单调递增区间 $[\pi, 2\pi]$。

| 术语 | 简要解释 |
|------|---------|
| 正弦曲线 | 函数 $y = \sin x$ 的图像，形状为无限延伸的波浪线 |
| 余弦曲线 | 函数 $y = \cos x$ 的图像，形状与正弦曲线相同，只是向左平移了 $\frac{\pi}{2}$ |
| 周期 | 对于函数 $f(x)$，若存在非零常数 $T$ 使得 $f(x+T) = f(x)$ 对所有 $x$ 成立，则 $T$ 为一个周期 |
| 最小正周期 | 所有正周期中最小的那个。正弦和余弦的最小正周期是 $2\pi$ |
| 振幅 | 正弦型函数 $y = A\sin(\omega x + \varphi)$ 中 $\|A\|$ 称为振幅，表示波动偏离平衡位置的最大距离 |
| 五点法 | 用五个关键点（起点、波峰、中点、波谷、终点）快速画出正弦或余弦一个周期草图的方法 |
| 辅助角公式 | 公式 $A\sin x + B\cos x = \sqrt{A^2+B^2}\sin(x+\varphi)$，用于将正弦和余弦的线性组合化为单一三角函数 |
