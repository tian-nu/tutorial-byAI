## 我在做什么？

前面你学会用数组把多个同类型的数据捆在一起。但如果想捆在一起的数据类型不同呢？比如一个学生：名字（字符串）、年龄（整数）、成绩（浮点数）。C 语言提供了结构体来做这件事。

读完本章你能：定义自己的结构体类型，创建结构体变量，读写结构体内部的成员，以及用 typedef 给结构体起别名。

---

## 一、为什么需要结构体？

### 1.1 一个真实的需求

假设你要在程序里存储一个学生的信息：姓名、年龄、三门课的成绩。用我们已经学过的知识，你可能会这样写：

```c
char name[20] = "张三";
int age = 20;
float score1 = 85.5;
float score2 = 90.0;
float score3 = 78.5;
```

问题很明显：这些变量各自独立，没有任何联系。当你把数据传给函数时，需要传递 5 个参数；当你要处理 30 个学生时，需要 150 个变量。

### 1.2 结构体的比喻

想象你去银行开户。柜员不会给你分别发"姓名卡"、"身份证卡"、"余额卡"三张独立的卡片，而是给你一张银行卡——这张卡上集成了你所有信息。

**结构体就是这张银行卡**。它把多个可能不同类型的数据打包成一个整体。

```text
+--------------------------------------------------+
|              结构体 Student                        |
|  +------------+ +--------+ +-------------------+ |
|  | name[20]   | | age    | | score[3]          | |
|  | "张三"     | | 20     | | {85.5, 90.0, 78.5}| |
|  +------------+ +--------+ +-------------------+ |
+--------------------------------------------------+
        一个变量，三个字段
```

## 二、定义结构体类型

*结构体* *此术语见附录F* 是一种自定义的数据类型。你需要先告诉编译器"我的结构体长什么样"，然后才能创建这种类型的变量。

### 2.1 基本定义语法

```c
struct 结构体名 {
    类型 成员名1;   // *成员* *此术语见附录F* 也叫 *字段* *此术语见附录F*
    类型 成员名2;
    // ...
};
```

注意末尾的分号——这是 C 语言里少数几种"大括号后面必须跟分号"的场合之一。忘记分号是一个非常常见的编译错误。

```c
struct Student {
    char name[20];
    int age;
    float score[3];
};  // <-- 这个分号不能省略！
```

这段代码告诉了编译器：有一个类型叫 `struct Student`，它内部包含一个 20 字符的数组、一个整数、一个浮点数数组。

### 2.2 声明结构体变量

定义了类型之后，就可以创建这种类型的变量了：

```c
struct Student s1;   // 创建一个 struct Student 类型的变量，名叫 s1
struct Student s2;   // 再创建一个
```

`s1` 和 `s2` 各自拥有一份完整的 name、age、score，互不影响。

**常见错误**——忘记写 `struct` 关键字：

```c
// ❌ 错误：Student 未被定义为一个类型名，编译器不认识
Student s1;

// ✅ 正确：必须带 struct 前缀
struct Student s1;
```

为什么必须写 `struct`？因为 C 语言里 `Student` 不是独立存在的类型名，只有 `struct Student` 合在一起才是。稍后我们会用 `typedef` 解决这个麻烦。

## 三、初始化结构体

### 3.1 定义时初始化

在声明结构体变量的同时，用大括号给各成员赋初值：

```c
struct Student s1 = {"张三", 20, {85.5, 90.0, 78.5}};
```

赋值的顺序必须与结构体定义的成员顺序一致：先 name，再 age，再 score。

### 3.2 指定成员初始化（C99）

从 C99 标准开始，你可以显式指定成员名来初始化，这样不需要记住顺序：

```c
struct Student s1 = {
    .name = "张三",
    .age = 20,
    .score = {85.5, 90.0, 78.5}
};
```

这种方式的好处是当结构体成员很多时，你不需要数位置，也不怕以后维护时成员顺序被调整。

### 3.3 部分初始化

如果你只初始化部分成员，其余成员会被自动初始化为 0（对于全局变量和静态变量）或未定义值（对于局部变量）：

```c
struct Student s1 = {"张三"};  // age 和 score 未明确赋值
```

**注意**：对于局部结构体变量，未初始化的成员值是未定义的（垃圾值）。养成全部初始化的习惯。

### 3.4 逐个成员赋值

你也可以先定义变量，再逐个给成员赋值（稍后讲到 `.` 运算符）。

## 四、访问结构体成员：`.` 运算符

定义好结构体变量之后，怎么读取或修改它内部的某个字段？用**点号运算符**。

```c
struct Student s1 = {"张三", 20, {85.5, 90.0, 78.5}};

printf("姓名: %s\n", s1.name);       // 输出 "张三"
printf("年龄: %d\n", s1.age);        // 输出 20
printf("成绩1: %.1f\n", s1.score[0]); // 输出 85.5

s1.age = 21;                         // 修改年龄
s1.score[0] = 88.0;                  // 修改第一门成绩
```

语法：`结构体变量.成员名`。把 `.` 理解为"的"——"s1 的 name"。

```text
表达式:   s1 . name
含义:     结构体变量 . 成员名
读法:     "s1 的 name"
```

### 4.1 完整的示例程序

```c
#include <stdio.h>
#include <string.h>

struct Student {
    char name[20];
    int age;
    float score[3];
};

int main() {
    struct Student s1;

    // 逐个赋值
    strcpy(s1.name, "李四");
    s1.age = 22;
    s1.score[0] = 92.0;
    s1.score[1] = 88.5;
    s1.score[2] = 95.0;

    // 打印
    printf("学生信息:\n");
    printf("  姓名: %s\n", s1.name);
    printf("  年龄: %d\n", s1.age);
    printf("  成绩: %.1f, %.1f, %.1f\n",
           s1.score[0], s1.score[1], s1.score[2]);

    return 0;
}
```

编译运行：
```bash
gcc student.c -o student
./student
```

预期输出：
```
学生信息:
  姓名: 李四
  年龄: 22
  成绩: 92.0, 88.5, 95.0
```

## 五、typedef：给结构体起别名

每次都要写 `struct Student` 很啰嗦。C 语言提供了 `typedef` *此术语见附录F* 关键字，让你给类型起一个简洁的别名。

### 5.1 语法

```c
typedef struct {
    char name[20];
    int age;
    float score[3];
} Student;
```

解读：先写一个完整的 `struct { ... }`，再用 `typedef` 给它起个名字 `Student`。这之后，`Student` 就是 `struct { ... }` 的别名了。

### 5.2 使用别名

```c
Student s1;          // 不再需要 struct 关键字！
Student s2 = {"王五", 19, {78.0, 82.5, 91.0}};

printf("%s\n", s1.name);  // 其他用法完全一样
```

### 5.3 两种写法的对比

```c
// 写法一：先定义，再取别名（分两步）
struct Student {
    char name[20];
    int age;
};
typedef struct Student Student;  // 给 struct Student 起别名 Student

// 写法二：一步到位（更常见）
typedef struct {
    char name[20];
    int age;
} Student;

// 写法三：保留标签名（两种方式都能用）
typedef struct Student {
    char name[20];
    int age;
} Student;
// 此时 struct Student 和 Student 等价
```

推荐使用**写法二**（一步到位，最简洁）。但当你需要在结构体内部引用自己时（比如链表节点），必须用写法三——下一章会讲到。

## 六、常见错误

### 6.1 忘记结构体末尾的分号

```c
// ❌ 错误：缺少分号
struct Point {
    int x;
    int y;
}   // 编译器报错：expected ';' after struct definition

// ✅ 正确
struct Point {
    int x;
    int y;
};
```

### 6.2 使用未定义的别名

```c
// ❌ 错误：typedef 还没执行完，别名还不能用
typedef struct {
    int id;
    Student friend;  // Student 此时还未定义！
} Student;

// ✅ 正确：如果要在结构体内引用自身，保留标签名
typedef struct Student {
    int id;
    struct Student *friend;  // 使用 struct Student 而非 Student
} Student;
```

### 6.3 把字符串直接赋值给字符数组

```c
typedef struct {
    char name[20];
} Student;

int main() {
    Student s1;
    // ❌ 错误：数组名不能被赋值
    s1.name = "张三";

    // ✅ 正确：用 strcpy 拷贝
    strcpy(s1.name, "张三");
    return 0;
}
```

回忆一下：数组名是一个常量指针，不能出现在赋值号左边。需要 `#include <string.h>` 后用 `strcpy`。

## 七、我做得对不对？

创建一个完整的程序并运行它：

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    char title[50];
    char author[30];
    int pages;
    float price;
} Book;

int main() {
    Book b1 = {"C语言程序设计", "谭浩强", 378, 39.80};
    Book b2;

    strcpy(b2.title, "深入理解计算机系统");
    strcpy(b2.author, "Randal E. Bryant");
    b2.pages = 1080;
    b2.price = 139.00;

    printf("书架:\n");
    printf("  《%s》 - %s, %d页, ￥%.2f\n",
           b1.title, b1.author, b1.pages, b1.price);
    printf("  《%s》 - %s, %d页, ￥%.2f\n",
           b2.title, b2.author, b2.pages, b2.price);

    return 0;
}
```

如果编译通过且输出两本书的信息，说明你已经掌握了结构体的定义、初始化、成员访问。

预期输出：
```
书架:
  《C语言程序设计》 - 谭浩强, 378页, ￥39.80
  《深入理解计算机系统》 - Randal E. Bryant, 1080页, ￥139.00
```

## 八、不对怎么办？

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| `expected ';' after struct definition` | 结构体定义末尾忘记分号 | 在大括号后加分号 `};` |
| `unknown type name 'Student'` | 用了 `Student` 但没加 `struct` 也没 typedef | 改为 `struct Student` 或加上 `typedef` |
| `assignment to expression with array type` | 用 `=` 给字符数组成员赋值 | 用 `strcpy(s1.name, "...")` |
| `expected identifier before '.' token` | 对类型而非变量用了 `.` | `.` 只能用于变量，不能用于类型名 |
| 打印出的浮点数全是 0 | 忘记给浮点成员赋值，局部变量初始值不确定 | 定义时用 `{0}` 全部初始化 |

## 本章小结

| 知识点 | 要点 |
|--------|------|
| 结构体定义 | `struct 结构体名 { 类型 成员; ... };` |
| 声明变量 | `struct 结构体名 变量名;` |
| 初始化 | `struct 结构体名 变量 = {值1, 值2, ...};` |
| 访问成员 | `变量.成员名` |
| typedef 别名 | `typedef struct { ... } 别名;` |
| 字符数组成员赋值 | 必须用 `strcpy`，不能用 `=` |
| 末尾分号 | 结构体定义的大括号后必须加分号 |
