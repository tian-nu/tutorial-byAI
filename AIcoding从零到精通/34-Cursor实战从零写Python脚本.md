上一章你写了一个网页，这一章我们写一个**Python脚本**——一个在电脑上真正能跑的实用工具。

**需求**：写一个文件批量重命名工具。能把你文件夹里的一堆文件，按照你指定的规则批量改名。

> ⚠️ **特别声明**：这一章刻意展示**完整的失败→修复→成功过程**。我们不会只给你看"完美的代码"——我们让你看到真实场景：AI写了代码，你运行报错了，你把错误给AI，AI修复，再试，成功。**这才是AI编程的日常。**

## 34.2 第一步：描述需求，让AI生成代码

### **1.** 按 `Ctrl + I` 打开Composer

### **2.** 输入以下Prompt：

```
帮我写一个Python文件批量重命名工具，要求：
1. 功能：扫描指定文件夹里的所有文件，按规则批量重命名
2. 规则1：添加前缀 —— 所有文件名前面加一个前缀，比如 "backup_"
3. 规则2：替换文字 —— 把文件名里的某个词替换成另一个词，比如把所有 "old" 替换成 "new"
4. 规则3：添加序号 —— 所有文件名前面加序号，比如 "001_文件名.jpg", "002_文件名.png"
5. 用户通过命令行参数选择规则和参数
6. 重命名前先预览，让用户确认后再执行

命令行使用示例：
python renamer.py --folder ./test --prefix "backup_"    （加前缀）
python renamer.py --folder ./test --replace "old" "new"  （替换文字）
python renamer.py --folder ./test --number               （加序号）
```

### **3.** 按回车，等待AI生成代码

**第1次运行**：报错，因为 `argparse` 参数名写错了

**第2次运行**：没报错，但重命名后文件找不到了

## 34.6 如果出错了怎么办

### 🔴 问题1：报错"python: can't open file"

**原因**：当前终端路径不对，不在 `renamer.py` 所在的文件夹。

**解决**：在终端里 `cd` 到 `file-renamer` 文件夹：

```bash
cd d:/你的路径/file-renamer
```

### 🔴 问题2：路径格式问题（Windows vs Mac）

**症状**：Windows用户用 `./test`，Mac用户用 `./test` 都能用，但如果代码里硬编码了Windows路径 `C:\xxx`，Mac上就跑不了。

**解决**：用 `os.path` 或 `pathlib` 模块处理路径，不要硬编码。如果AI写了 `C:\` 开头的路径，在Chat里说"请用跨平台路径写法，不要硬编码Windows路径"。

### 🔴 问题3：文件被重命名后找不到了

**原因**：可能是路径拼接错误，文件被移动到了其他目录。

**解决**：把错误现象描述给AI，让它检查路径拼接逻辑。或者先用 `--number` 规则（只加序号，不改变文件位置），确认功能正常后再用其他规则。

### 🔴 问题4：AI生成的代码太复杂，看不懂

**解决**：选中相关的代码，在Chat里问"这段代码是什么意思？逐行解释一下"。AI会帮你解释。
